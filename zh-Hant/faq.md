[[extra.faq_groups.items]]
slug = "what-is-the-working-hypothesis"
question = "工作假設是什麼？"
answer = """
該假設由五個相互關聯的部分組成：

1. **創世記中的 *埃洛希姆* 是字面上的複數。** 希伯來語原詞在語法上為複數名詞。語料庫按其字面來讀——創造人類的不是單一神祇，而是一群存在者。
2. **埃洛希姆是一支高度發達的人類文明，而非超自然存在。** 他們在生物學上與人類相親緣，技術上遠超人類，並在地球上從事一項有意識的科學計劃。
3. **我們作為「宗教」繼承下來的，是對他們工作的長久記憶。** 人類的宗教傳統以壓縮並常被神秘化的形式，儲存著關於埃洛希姆文明在約兩萬兩千年間介入的一份準確歷史證詞。
4. **這一工作沿歲差週期組織。** 地球自轉軸約 25,920 年一週的緩慢擺動，是記錄該計劃的日曆。十二個約 2,160 年的時代構成「大年」。
5. **當前時代是揭示的時代。** 始於 1950 年的水瓶時代，是這段長久記憶在科學上有能力評估它的人類面前變得可讀的時代。

該假設提供出來，是為了被試驗、評估、與其他選項相對照——並不作為既定事實而被斷言。
"""

[[extra.faq_groups.items]]
slug = "question-of-evidence"
question = "你們如何處理證據問題？"
answer = """
認真且公開。語料庫使用三類證據：

**文本證據**——宗教與神話來源實際上說了什麼，儘可能貼近原文閱讀。語料庫最有力的材料是解讀性的：當框架被應用於某一具體的希伯來語段落時，它是否產生比其他選項更連貫的解讀？

**比較證據**——獨立傳統之間反覆出現的圖式。當美索不達米亞、聖經、吠陀與中美洲來源保留著結構相似的敘述時，這種反覆本身就是值得審視的證據。

**科學證據**——當代遺傳學、天體生物學、行星科學與合成生物學就該框架主張的技術可行性提出什麼。語料庫在這裡的論證是條件性的：*如果* 框架成立，科學應當呈現某種特定面貌；語料庫追蹤科學實際情況與此的契合程度。

專案並不主張確定的證明。它主張的是一個連貫且具解釋力的框架，在可得證據之上誠實地加以應用——讀者被邀請來評估它。
"""

地球自轉軸傾斜約 23.5°，且在空間中並不指向固定方向。像陀螺一樣，地軸在天空中緩慢畫出一個圓，約每 25,920 年完成一圈。這一運動稱為 *分點歲差*。

實際後果是：春分日太陽背後所升起的星座，沿黃道帶緩慢向西移動。兩千年前是白羊；今天是雙魚；很快將是水瓶。這一週期約每 26,000 年重複一次。

[[extra.faq_groups.items]]
slug = "what-does-elohim-mean"
question = '「埃洛希姆」是什麼意思？專案為什麼把它讀作複數？'
answer = """
*埃洛希姆*（אֱלֹהִים）是常用聖經譯作「上帝」的那個希伯來詞。從語法上看，它是一個複數名詞——字尾 *-im* 是希伯來語標準的複數標記，與另一些陽性名詞的複數化（*基路伯*/*基路冰*，*拿非*/*拿非利*）一致。

通常的基督教與猶太教神學把這個複數讀作「莊嚴複數」（單數神被以複數加以呼喚，以示崇高，類似王室的「我們」），或讀作「三位一體」的佔位符。語料庫照字面來讀：複數的 *埃洛希姆* 指向多位存在。

希伯來文本的若干特徵支援這一讀法：

- 創世記 1:26 使用第一人稱複數動詞：*na'aseh adam be-tsalmenu kidmutenu*——「讓 *我們* 按照 *我們* 的形象、*我們* 的樣式造人。」
- 創世記 3:22 使用相似的複數：*hen ha-adam hayah ke-achad mimennu*——「看哪，人已成為 *我們* 中的一員。」
- 創世記 6:1–4 引入 *benei ha-Elohim*——「眾埃洛希姆的兒子們」——作為有別於人類的特殊範疇。
- 文本在不同段落中區分 *YHWH Elohim*（複數範疇中被命名的某位特定存在）與單獨的 *Elohim*（整個範疇）。

詳見 Wiki 詞條[埃洛希姆](/zh-Hant/wiki/elohim/)。
"""

語料庫並不否定作為「生物隨時間變化的描述」的進化。它否定的是這樣一種主張：單憑進化、不受引導，便能解釋生命與人類的起源。

按語料庫的讀法，化石記錄中的主要形態學過渡，是埃洛希姆文明所開展的設計-迭代階段；每個設計階段內部更小的進化變化，則按慣常機制（突變、選擇、漂變）發生。某些過渡的表觀突然性——寒武紀大爆發、*智人* 自先前的人科形態中迅速出現——正是設計-迭代計劃所會產生的樣子。

語料庫的框架更多暗示一種總體走向，而不是帶有具體日期的預測。水瓶時代被讀作漸進揭示的時代——長程歷史記錄變得日益可讀的過程，伴隨著人類向自身創造效能力的成熟。在接下來的 ~2,160 年裡，框架預期：合成生物學持續向埃洛希姆所具有的那種設計能力成熟；系外行星持續被發現，並可能偵測到生物特徵或技術特徵；宗教傳統與科學傳統持續整合；以及在第二個「大年」中，人類終將自身成為一個創造性文明。

語料庫不就「接觸」、「揭示事件」或末世場景給出具體日期的預測。框架是對一條長弧的描述，而非對具體事件的預言。

語料庫依賴多個來源家族，並在其中明確劃分層級：

**主要**：雷爾派源材料，主要是 *講述真相的書*（沃利隆，1974 年）及其後續卷。

**次要**：歲差神話學傳統，基礎文獻是喬治奧·德·桑蒂利亞納與赫塔·馮·德欽德的 *Hamlet's Mill*（1969 年）。

**第三層**：讓·桑迪 1968–1972 年對聖經材料作為外星活動記錄的重讀（*La Lune, clé de la Bible*；*Les Dieux nous sont nés*；*L'Ère du Verseau*）。

**比較層**：聖經、美索不達米亞、吠陀、中美洲及其他一手宗教與神話來源，儘可能閱讀其原文。

**接觸但不背書**：更廣義的古代宇航員文獻（馮·丹尼肯、西欽、漢考克等），就具體論點選擇性接觸。

**以其自身術語對待**：聖經學、遺傳學、天體生物學、考古天文學與合成生物學等領域的當代同行評議學術。

完整的來源版圖見[圖書館](/zh-Hant/library/)與[資源](/zh-Hant/resources/)兩節。
"""

[[extra.faq_groups.items]]
slug = "handling-uncertainty"
question = "專案如何處理不確定性與分歧？"
answer = """
直接處理。專案的編輯風格指南明確要求：不確定性必須被命名，而不是被遮蔽：

- 「該日期存在爭議。」
- 「這一年代是傳統的，而非學術上的。」
- 「這一身份認定是解讀性的，源頭中並未明示。」
- 「此一平行具有暗示性，但並未在譜系學意義上被證明。」

學界共識與語料庫框架不一致的地方，語料庫會直接處理這種不一致，命名替代性讀法，並陳述語料庫為何得出不同結論的具體推理。證據確實懸而未決的地方，語料庫會直說。語料庫在思辨的地方，也會直說。
"""

[[extra.faq_groups.items]]
slug = "claim-badges"
question = '頁面徽章上的「直接」「推斷」「思辨」分別是什麼意思？'
answer = """
語料庫的每一頁都帶有一個小徽章，表明該頁主要論斷的認知論地位：

**直接論斷**——源文本本身所斷言的內容。*「創世記 1:26 寫道『我們要按照我們的形象造人』。」*這是最可靠的論斷；核實就是核對來源。

**推斷論斷**——學術從源頭中合理推出的結論。*「動詞的複數形式表明存在多個施動者。」*這類論斷涉及解讀工作，但仍緊貼證據。

**思辨論斷**——專案作為解讀性綜合所提出的內容。*「創世記中複數的埃洛希姆，可以與雷爾派源材料所描述的那個小型文明加以同認。」*這是專案最具特色的貢獻，也最有可能在未來被修訂。

徽章讓讀者能在「單項論斷」的層級上調校信任。一個思辨性論斷並不因為它是思辨的就錯了；它被 *標註為* 思辨，是為了讓讀者知道按何種權重去掂量它。
"""
