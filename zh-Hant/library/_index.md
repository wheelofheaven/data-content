# 圖書館

圖書館收藏本計畫所依據的原典——並非對它們的評註,而是文本本身,供直接閱讀。

居於核心的是雷爾運動的源頭書籍,本計畫將其視為基礎正典。環繞其周圍的,是與此正典並列閱讀的聖經、美索不達米亞、吠陀及其他古代著作。每本書都在互動式閱讀器中呈現,在語料庫具備相關資料之處,將原文、轉寫與譯文並列對照。

如果你想了解詮釋層面——本計畫如何解讀某段經文——請從[時間軸](/timeline/)或[百科](/wiki/)開始。圖書館是先以文本自身的方式閱讀它們的地方。
