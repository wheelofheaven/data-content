Wheel of Heaven 是一個開放的項目。歡迎任何人閱讀語料庫、與其論斷互動、提議更正、提出異議、貢獻翻譯，或建議新的方向。本頁描述了項目對所有在其公共空間——[GitHub Discussions](https://github.com/orgs/wheelofheaven/discussions)、問題追蹤器、Pull Request，以及項目未來可能新增的任何渠道——中參與者的期望標準。

## 承諾

項目承諾成為這樣一個空間：參與不取決於背景、身份，或事先認同語料庫的框架。實質性的不同意見是受歡迎的。基於身份的敵意則不是。項目認真對待這兩條承諾，而後者會在出現時被強制執行。

## 鼓勵的做法

- 與語料庫的實質內容互動——包括對其提出挑戰
- 提出誠實的問題，即使作者在別處可能已經回答過
- 在回應前認真閱讀，並引用你所回應的對象
- 在存在不確定性的地方承認自己的不確定
- 直接指出錯誤，並附上更正的來源
- 翻譯、引用、在其上繼續創作或重新發布作品（[CC0](https://creativecommons.org/publicdomain/zero/1.0/) 許可證主動邀請這樣做）

## 不歡迎的做法

- 針對任何參與者的人身攻擊、騷擾，或有針對性的敵意
- 基於身份、背景或信仰的歧視
- 釣魚挑釁、反覆的惡意參與，或試圖讓實質性討論脫軌
- 未經他人同意公開其他參與者的私人信息
- 改宗勸說——無論是為或反對語料庫的框架。項目不尋求皈依者，也無意從任何方向參與對抗性運動。

## 不同意見是受歡迎的；騷擾則不是

二者並非一回事，項目認真對待這種區分。一位讀者若認為語料庫錯了、認為雷爾派的原始材料是編造的、認為歲差框架是牽強的，或對項目在某一具體問題上的立場感到不被說服，是受歡迎在公共空間中說出來的。作者寧願聽到一份誠實的反對，也不要一份虛偽的贊同。

不受歡迎的，是利用公共空間因參與者是誰、相信什麼、來自何處、認同何種傳統而攻擊他們本人。實質內容是開放的；人則不是。

## 執行

作者負責審核項目的公共空間，並保留刪除內容、鎖定話題、或封禁違反此準則之參與者的權利。執行旨在做到比例適當且透明：輕微違規通常會獲得一條說明哪裡不妥的備註；反覆或嚴重的違規可能導致移除。

如要舉報違規，請在[倉庫](https://github.com/wheelofheaven)上開啟一個 Issue；或對不應公開的事項，通過 GitHub 上權威行為準則中列出的渠道直接聯繫維護者。
