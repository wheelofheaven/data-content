# 時間線

本文集透過歲差週期解讀人類歷史——地球地軸的二萬六千年擺動,劃分為十二個約二千一百六十年的時代,每個時代以春分日出時背景星座命名。在此框架內,十五個篇章追溯耶洛因——那個改造了地球生命的小型先進人類文明——做了什麼、何時做、為何而做。

系列以《起初》開篇,場景設於第一個時代之前,描述母星上促使流亡者前往地球的政治裂變。其後貫穿介入與再介入的深時篇章——摩羯座、射手座、天蠍座、天秤座、處女座、獅子座、巨蟹座、雙子座——進入金牛座、白羊座與雙魚座的歷史記錄,並以我們目前身處其中的水瓶座過渡作結,另附一章綜論輪盤本身。

每一篇章皆透過正典之鏡解讀其所屬時代——比較宗教、科學脈絡與批判學術皆涵納其中——但正典主張直接陳述,不加遮掩。
