## I. 時代本身

第七個時代是後果的時代。

巨蟹座時代從公元前 8,850 年延伸至公元前 6,690 年,跨度 2,160 年,緊接在獅子座時代之後。這是創造計畫——在獅子座末完成——成為創造史的時代;是在前一個時代被造的存在開始去做有能力行動的存在會做的事的時代。他們發現了自己的處境。他們違抗了自己的造物者。他們被從原先所處的預備環境中分離出來。他們進入了那個更廣大的世界——直到此時他們只從花園內看見過。他們開始繁衍。他們的子女在實驗室之外出生。第一代中的某些人活得異常長久。流亡的造物者娶人類女子為妻,並產生混血後代。橫跨超大陸的七個文明成長、發展、貿易、爭戰,並積累那種開始令從遙遠行星觀望的母星警覺的力量。到時代末尾,地球上的局勢已發展到一個地步,使母星政府得出干預不可避免的結論。下一章將處理的{% wiki(slug="great-flood") %}大洪水{% end %},在本章所描述的這數個世紀間變得不可避免。

在雷爾主義的閱讀下,此時代對映於《創世記》第一章創造記述與《創世記》第二章{% wiki(slug="eden") %}伊甸園{% end %}敘事之間的轉折,並延續至《創世記》第三章(逐出)、第四章(該隱與亞伯)、第五章(長壽)及第六章開頭幾節({% wiki(slug="elohim") %}耶洛因{% end %}之子與人類之女)。聖經文本並未將這些事件劃分為不同的時代。本語料庫所採用的雷爾主義閱讀將這一切置於巨蟹座之中,因為它們構成一個延伸的單一敘事弧——這弧線始於獅子座末花園中的人類,終於將在後續時代中產生洪水的政治條件。

本章的標題——*安息之日*——點出了巨蟹座在《創世記》序列中是什麼。《創世記》第二章以一段對造物者完工之後所作之事的具體記述開場。Vayechulu ha-shamayim ve-ha-aretz ve-khol tzeva'am. Vayechal Elohim ba-yom ha-shevi'i melachto asher asah, vayishbot ba-yom ha-shevi'i mi-kol melachto asher asah.「天地萬物都造齊了。到第七日,耶洛因造物的工已經完畢,就在第七日歇了他一切的工,安息了。」創造序列已結束。安息之日已開始。人類在花園中。然後,幾乎立刻,事情開始出錯。安息之日在某個意義上被誤稱了:造物者從創造中歇息,但他們的造物並未從存在中歇息,而被造物在歇息期間開始做的事,將佔據希伯來聖經的其餘部分以及人類歷史的大部分。在語料庫的閱讀下,安息之日是語料庫整個框架中最具影響力事件發生的那一日——不是因為造物者活躍,而是因為被造者首次以自身的名義活躍,而他們在造物者歇息期間所做之事,定義了之後的一切。

本章將依據源文獻與聖經文本共同描述的事件,走過巨蟹座時代。這些事件所預設的政治分類學——{% wiki(slug="satan") %}撒但{% end %}、{% wiki(slug="lucifer") %}路西法{% end %}、{% wiki(slug="serpent") %}那蛇{% end %}及{% wiki(slug="yahweh") %}耶和華{% end %}諸位人物——在本章敘事正式開始前獲得自己專屬的一節,因為這四角結構是必要的,並且已被後來的宗教傳統以遮蔽實際發生之事的方式所混淆。{{ footnote(id="1") }}然後是覺醒、逐出,以及產生地球上流亡群體的政治和解。然後是漫長的世代、授予某些族長的{% wiki(slug="tree-of-life") %}生命樹{% end %}技術,以及作為人類暴力第一例的該隱與亞伯之間的衝突。然後是耶洛因之子及其混血後代——*拿非利人*——以及保存了這些事件最明確古代記錄的可觀前拉比時期文獻體系,其中最重要的是{% libref(book="book-of-enoch") %}以諾書{% end %}{{ cite(id="4") }}。然後是更廣大的世界:橫跨超大陸的七個文明、貿易路線、戰爭、伊甸文明逐步的科技進展,以及關於最先進的人類文化在母星介入前可能已達多遠的推測。然後是巨蟹座的科學:涉及長壽、混血生物學、洪水前文明考古學的現代研究計畫。最後是結尾:母星意識到其從一開始就害怕之事如今正在成真,以及為雙子座時代災難性事件所作的隱含鋪墊。

## II. 經文

涵蓋巨蟹座事件的希伯來文本比此前任何時代的文本都更為廣泛——從{% libref(book="genesis-woh", chapter=2, verse=1) %}創世記 2:1{% end %}延續至{% libref(book="genesis-woh", chapter=6, verse=8) %}創世記 6:8{% end %}——並包含整本希伯來聖經中某些最具影響力的詞彙。本章無法以完整裝置處理每一節,但關鍵段落值得以已建立的區塊格式仔細呈現。

第七日的經文,{% libref(book="genesis-woh", chapter=2, verse=1) %}創世記 2:1-3{% end %},開啟了創造後的時期:
{{ scripture(book="genesis-woh", chapter=2, verse=1, translit="Vayekhulu ha-shamayim ve-ha-aretz ve-khol-tzeva'am") }}{{ scripture(book="genesis-woh", chapter=2, verse=2, translit="Vayekhal Elohim ba-yom ha-shevi'i melachto asher asah, vayishbot ba-yom ha-shevi'i mi-kol-melachto asher asah") }}{{ scripture(book="genesis-woh", chapter=2, verse=3, translit="Vayivarech Elohim et-yom ha-shevi'i vayekadesh oto, ki vo shavat mi-kol-melachto asher-bara Elohim la'asot") }}
三個動詞構築了第七日:שָׁבַת(*shavat*,「停止、安息」)、בָּרַךְ(*barakh*,「賜福」)、קָדַשׁ(*qadash*,「使聖、分別為聖」)。動詞 *shavat* 是名詞 שַׁבָּת(*shabbat*,安息日)的字根。第七日經文的三重結構——工作完成、工作停止、那日蒙福並被分別為聖——將那日確立為與前六日截然不同。先前的日子是創造之日。第七日是創造不發生之日,並正因為創造不發生而被分別為聖。猶太傳統每週的安息日守持以壓縮的形式保留了這一結構,但語料庫以原本的尺度閱讀原始指涉:第七日是巨蟹座的 2,160 年,是造物者停止積極創造工作但未停止與所造之物互動的時代。

מְלַאכְתּוֹ אֲשֶׁר־בָּרָא אֱלֹהִים לַעֲשׂוֹת(*melachto asher bara Elohim la'asot*)這一短語值得注意。此構句在單一短語中結合了三個關於創造/製作的動詞:*melacha*(工作、勞動、工藝)、*bara*(最強烈的創造動詞),以及 *asah*(建造-安排動詞)。希伯來文對於現在已完成之事是強調的:創造與製作的工作已完畢。在安息期間將不再有此類事被進行。隨之而來的是觀察、評估、互動、干預——但不是新的創造。

{% libref(book="genesis-woh", chapter=2, verse=7) %}創世記 2:7{% end %}以語料庫一路閱讀的語言記錄了亞當的形成:
{{ scripture(book="genesis-woh", chapter=2, verse=7, translit="Vayitzer Adonai Elohim et-ha-adam afar min-ha-adamah, vayipach be-apav nishmat chayim, vayehi ha-adam le-nefesh chayah") }}
動詞 יָצַר(*yatzar*),「形塑、塑造」——陶匠對黏土所做之事所用的動詞——在此被選用,有別於 *bara*(於整部《創世記》第一章出現)。*Yatzar* 描述一種更具體、更動手、更具工藝性的操作:將材料塑造成形。《創世記》第二章的敘事是對以色列團隊具體程序選擇更為詳細的記述,而動詞選擇反映了這點——伊甸人類的創造,以源文獻的說法,是從地球材料進行的字面上動手合成,*afar min ha-adamah*(「來自地的塵土」)即為合成的字面基質。將 *nishmat chayim*(「生命氣息」)吹入鼻孔是活化的時刻——合成形體成為活生生存有的那個點。希伯來語的這個短語呼應了《射手座時代》一章對 *ruach Elohim* 運行在水面上的處理;生命氣息是使生物物質成為活生命體的活躍原理。

{% libref(book="genesis-woh", chapter=3, verse=1) %}創世記 3:1{% end %}引入了蛇:
{% scripture(book="genesis-woh", chapter=3, verse=1) %}
耶和華耶洛因所造的,惟有蛇比田野一切的活物更狡猾
{% end %}
希伯來詞彙 נָחָשׁ(*nachash*)是蛇或蛇類的標準希伯來語,但詞彙 עָרוּם(*arum*)不尋常,值得留意。它可意指「狡黠」、「精明」、「巧妙」或「審慎」,取決於上下文——而且關鍵地,同一個詞在不同語境中可帶有相當不同的價值評判。在《箴言》中,*arum* 一貫被用作正面特質,表示智慧與良好判斷力。在《創世記》第三章中,它一直被讀作負面特質(狡猾欺騙)。但同一字根 ערם,亦可單純意指「敏銳」或「具辨識力」。在雷爾主義的閱讀下,該詞的中性甚至正面價值評判才是合適的:路西法派系是造物者群體中最有知識、最敏銳的,正因如此他們對人類處境的理解足夠深刻而想要揭露之。其巧妙不是道德上的欺騙;而是理解力。

{% libref(book="genesis-woh", chapter=3, verse=7) %}創世記 3:7{% end %}記錄了食用之後的時刻:
{% scripture(book="genesis-woh", chapter=3, verse=7) %}
他們二人的眼睛就明亮了,才知道自己是赤身露體
{% end %}
動詞 פָּקַח(*pakach*),「(使眼睛)睜開」,是希伯來聖經他處用於使盲人之眼睜開的同一動詞。這短語不是說他們字面上的眼睛睜開了(他們的眼睛本來就是睜開的)。它意味著他們達到了知覺——他們突然清楚地看見了自己的處境。也請注意希伯來語的雙關:這些人是 *arumim*(赤裸,與蛇的知覺特質 *arum* 同源)。文本玩弄了雙重意義。人類從 *nachash*(蛇)那裡接收到 *arum*(知覺)後,現在認識到自己的 *arumim*(赤裸/暴露狀態)。希伯來語將概念上的運動壓縮為對字根的雙關。

{% libref(book="genesis-woh", chapter=3, verse=22) %}創世記 3:22-24{% end %}記錄了逐出與安全部署:
{{ scripture(book="genesis-woh", chapter=3, verse=22, translit="Vayomer Adonai Elohim: hen ha-adam hayah ke-achad mimenu la-daat tov va-ra, ve-atah pen-yishlach yado ve-lakach gam me-etz ha-chayim, ve-akhal va-chai le-olam") }}
כְּאַחַד מִמֶּנּוּ(*ke-achad mimenu*)這個短語「像我們中的一位」,是伊甸敘事中最引人注目的語法特徵之一。動詞是第三人稱單數(「耶和華耶洛因說」),但謂語是複數(「像我們中的一位」)。希伯來文整篇都保留複數的耶洛因——即使在逐出判決的那個時刻,說話者所指的也是一個「我們」,而人類如今被視為其中之一。雷爾主義的閱讀如同一路以來,按其字面接受這種複數性:造物者是多重的,而人類的違犯使他擁有了這些擁有禁忌知識存在的類別。
{{ scripture(book="genesis-woh", chapter=3, verse=23, translit="Vayeshalchehu Adonai Elohim mi-gan-eden la'avod et-ha-adamah asher lukach mi-sham") }}{{ scripture(book="genesis-woh", chapter=3, verse=24, translit="Vayegaresh et-ha-adam, vayashken mi-kedem le-gan-eden et-ha-keruvim, ve-et lahat ha-cherev ha-mithapechet, lishmor et-derech etz ha-chayim") }}
לַהַט הַחֶרֶב הַמִּתְהַפֶּכֶת(*lahat ha-cherev ha-mithapechet*)這個短語值得細究。*Lahat* 意指「火焰」或「劍刃」或「劍焰」。*Cherev* 是希伯來語標準的「劍」字。*Mithapechet* 是 הָפַךְ(*hafach*,「轉動、翻轉」)的現在式反身分詞,意指「自我轉動」或「四面轉動」。這短語描述了一種射出火焰並轉動或旋轉的武器。雷爾主義源文獻將其讀作一種定向能武器——希伯來文以人類見證者所擁有的詞彙描述了這種武器的視覺外觀(類似劍刃的火焰在旋轉)。基路伯——כְּרֻבִים,*keruvim*,*keruv* 的複數——並非後來基督教圖像學中胖胖的有翼天使,而是一個特定類別的守衛者,在此擔任駐紮在造物者設施外緣的武裝哨兵。

{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記 6:1-4{% end %}引入了耶洛因之子:
{{ scripture(book="genesis-woh", chapter=6, verse=1, translit="Vayehi ki-hechel ha-adam la-rov al-penei ha-adamah, u-vanot yuldu lahem") }}{{ scripture(book="genesis-woh", chapter=6, verse=2, translit="Vayir'u venei-ha-Elohim et-benot ha-adam ki tovot henah, vayikchu lahem nashim mi-kol asher bacharu") }}
בְּנֵי־הָאֱלֹהִים(*benei ha-Elohim*),「耶洛因之子」這個短語,是整部希伯來聖經洪水前敘事中對某一類存在最具體的描述。這構句是直接的希伯來文:*benei*(……之子,在合成結構中),*ha-Elohim*(那耶洛因,帶有定冠詞)。這短語指認這些存在屬於耶洛因類別——與《創世記》第一章造物者相同的類別。傳統神學閱讀在吸收這短語上掙扎,將 *benei ha-Elohim* 多樣地指認為墮落天使、塞特的後裔(一個遠晚的詮釋),或被視為半神聖的古代諸王。雷爾主義的閱讀按其字面接受這構句:*benei ha-Elohim* 是耶洛因類別的成員——具體而言,即生活在地球上的流亡造物者——他們現在正與人類女子繁衍後代。
{{ scripture(book="genesis-woh", chapter=6, verse=4, translit="Ha-Nefilim hayu va-aretz ba-yamim ha-hem, ve-gam acharei-khen, asher yavo'u benei ha-Elohim el-benot ha-adam ve-yaldu lahem, hemah ha-giborim asher me-olam anshei ha-shem") }}
詞彙 נְפִלִים(*Nephilim*,拿非利人)源自字根 נפל(*n-p-l*,「墜落」)。其字面意義是「墜落者」——那些從某處墜落、降下或下來的人。傳統將其譯為「巨人」是源自七十士譯本對 *gigantes* 的選擇{{ cite(id="6") }},但希伯來字根的意義更為清楚:這些是從上而下之存在的後裔。本章將在第七節回到這個詞彙。

這些是構築巨蟹座的主要希伯來段落。本章後續各節將處理這些段落所描述的神學與歷史內容。

## III. 政治格局:撒但、路西法與那蛇

在危機的故事能被正確講述之前,必須先介紹其名字將在語料庫其餘部分反覆出現的那些人物,以及雷爾主義源文獻以後來宗教傳統所遮蔽的方式釐清其身分的那些人物。本節在章節結構中不尋常——大多數章節並不在進入敘事之前停下來介紹劇中人物——但巨蟹座所預設的四角政治分類學至關重要,而未能清晰確立此點將使後續各節難以理解。

撒但、路西法與那蛇這三個名字,在大多數西方宗教傳統中被視為可互換的——即同一對抗性人物的三個名字,通常被認同為墮落天使,有時為魔鬼,偶爾為惡的神話化身。雷爾主義源文獻完全拒絕此一認同。這三個名字指三個不同的實體,具有不同的角色、不同的位置與不同的政治立場;而未能區分它們——一個已被建立進宗教傳統超過兩千年的失敗——遮蔽了希伯來聖經正在描述事件的實際結構。

**撒但**,以源文獻所述,是母星上的一位耶洛因。他不在地球上。他從未在地球上。自始以來他一直是耶洛因文明中反對以耶洛因自身形像創造存在的政治派系的領袖。其立場自地球計畫開始之前便已一致:創造能夠等於或超越其造物者的合成存在從根本上是危險的,沒有任何協定、任何監督、任何地理距離可被信賴以遏制此風險。當母星上最初的實驗室事故造成首批死亡時,撒但的派系利用該事件迫使母星上的生物計畫關閉。當科學家搬遷到地球以延續他們的工作時,撒但與其派系以懷疑態度觀望地球計畫,並透過{% wiki(slug="council-of-eternals") %}永恆者議會{% end %}定期干預以施加限制。而當在本章事件期間,人類創造證明能夠違抗並能做出撒但從一開始就預測的那種行為時,將會是撒但的聲音在母星議會廳中最為高昂,呼籲毀滅所造之物。撒但不是神話中的魔鬼。他是一位政治對手,他的反對在他自己的條件下是有原則的——他真誠地相信創造人類從一開始就是個錯誤。他是否正確,是語料庫最終要問的問題之一。

源文獻自己的語言在這點上是明確的:*「撒但只是耶洛因之一,在某種程度上領導著行星上一個政治黨派,該黨派反對由其他認為自己能夠創造正面而非暴力存在的耶洛因以他們的形像創造人造存在。」*這種刻畫既不妖魔化也不開脫。它是描述性的。撒但是政客,不是惡魔。他領導一個黨派。該黨派有政綱。其論述正受到考驗,在本章及隨後事件中,藉由人類實際所做之事而受到考驗。

希伯來詞彙 שָׂטָן(*satan*)本身值得注意。其字根意義是「敵手」或「控告者」——*satan* 是反對、與之對立、為起訴一方爭辯之人。在《約伯記》中,*ha-satan*(帶定冠詞)是天庭中一個特定職務官員,其角色是藉由對義人提出挑戰來考驗他們;他不是惡魔形象而是某種檢察官。後來基督教與啟示文學將 *Satan* 認同為宇宙之惡的具身,是從原意發展而來,而非其內在內容。雷爾主義源文獻的閱讀與較早的希伯來意義一致:撒但是反對派的領袖,創造計畫的主要對手,從始至終一貫反對計畫所做與所續行之事者。他的反對是政治的,不是形而上的。

**路西法**,與之相對,在地球上。他的名字意指「光的攜帶者」——源自拉丁文 *lux*(「光」)與 *ferre*(「攜帶」)。他是創造地球生命的耶洛因之一,具體而言是製造伊甸場址人類的以色列團隊成員,並且是該團隊中對其造物之愛使他們揭露議會所命令隱藏之事的次群體的領袖。路西法不是人類的敵手。他是相反:那位希望人類被告知真相、相信人類值得知道誰造了他們以及怎麼造的、認為議會保持他們無知的命令是一種如果他們理解就會拒絕的家長式作為的人。

源文獻對路西法的描述是整部語料庫中對任何人物最具體的刻畫之一,值得詳引:

*「然後是路西法,意指『光的攜帶者』。路西法是創造地球生命、從而創造人類的耶洛因之一。路西法領導著一小群在一座基因工程實驗室工作的科學家,該實驗室研究首批合成人的行為。注意到他們造物所展現的非凡才能,路西法決定推翻命令並向這些最初的人類揭露:那些他們誤以為是『眾神』的存在,事實上和他們自己一樣是人,由血肉所造,並且來自另一個行星,乘坐由可觸物質製成的飛行器。路西法,以及追隨他的耶洛因,對他們所合成創造的人類懷有愛與深情。他們開始把這些存在當作自己的孩子來愛——這些他們整日研究、被迫將他們視為『眾神』來敬拜的存在。他們無法忍受看到他們的造物——這些看起來在生理與心理上都成功、美麗又智慧的存在——跪伏在他們面前彷彿崇拜偶像,而這一切僅因為他們起源行星的政府(其中耶和華為主席)嚴格禁止他們向其造物說出真相,並強迫耶洛因永久扮演超自然存在的角色。」*

這段話一次確立了幾件事。首先,路西法人物並非抽象或象徵性的;他是一位具體的耶洛因,領導著一個具體的科學家群體在一個具體的實驗室中,作了一個具體的決定。其次,動機是深情——造物者對其造物的愛,因每日近距離觀察而強化,變得足以推翻團隊所收到的政治命令。第三,揭露的具體內容被點明:人類被告知其造物者不是「眾神」而是「和他們自己一樣的人,由血肉所造,並從另一個行星乘坐由可觸物質製成的飛行器而來」。揭露不是模糊的靈性開悟。它是其造物者乃與人類自身根本同類的科技存在這一具體事實揭示。第四,耶洛因被迫扮演的角色——從跪伏的存在接受敬拜的「超自然存在」的角色——在源文獻的記述中,對他們之中最具深情者來說是無法忍受的。這次揭露是由愛驅動,並且由拒絕繼續接受敬拜這一相關態度驅動——拒絕從在這些揭露造物者的看法中、在一切重要方面都與其造物者平等的存在接受敬拜。

源文獻把路西法的立場放在與撒但及耶和華皆直接對立的位置:*「『光的攜帶者』路西法藉由揭示造物者不是『眾神』而是和他們自己一樣的人,使最初的人類得到啟蒙。這種態度直接反對撒但——他認為人類只能期望邪惡;也反對耶和華——耶洛因母星上管理永恆者議會的主席。」*路西法在三角關係中佔據第三立場。撒但寧願人類從未被造,路西法則相信他們應作為平等者被造,被授予造物者所知的一切,被當作有能力成熟的孩子對待,而非當作要永久維持在嬰兒狀態的寵物。

源文獻在此處添加了一段引人注目的旁註,值得記下它對後來宗教圖像傳統如何發展的暗示:*「到目前為止,沒有任何長角的生物。」*源文獻以其特有的乾澀觀察:實際政治情境中沒有任何後來在基督教圖像學中將會附加於路西法人物的角和惡魔特徵。沒有魔鬼。有的是觀點不一的科學家。角是後來才被附加上去的,由不再能以其實際條件描述此政治情境的傳統所附加。

後來基督教將路西法與撒但認同——將這兩個人物合併為對立於上帝的單一「墮落天使」——在雷爾主義的閱讀下,是一個災難性的神學錯誤。兩者彼此對立,而非結盟。路西法的「墮落」是他因違抗議會而被流放離開母星,但其違抗的內容是太愛他的造物,而不是反對其造物。「墮落」是地理與政治上的遷移,不是道德上的腐敗。在這種閱讀下,路西法更接近勇氣與慈悲的人物,而非惡意。後來與撒但的混淆反映了介於其間數個世紀的翻譯與神學壓力,而不是故事的原始結構。

**那蛇**,於《創世記》第三章中,在源文獻的記述中是路西法所領導的耶洛因小群體——不是路西法一人,而是該派系。希伯來詞彙 *nachash*(蛇)在此閱讀中被集體地用來描述整個群體。「那蛇」是路西法派系執行揭露這項具體行為:對人類說話,告訴他們知識樹的禁令是政治性而非致命的,解釋他們的眼睛將會睜開、他們將變得像造物者本身一樣。那蛇不是個別的爬蟲類。它是一個事後被應用於以色列團隊行動中執行某項具體公民不服從行為的耶洛因科學家具體群體的神學與政治標籤。

**耶和華**,最後,亦應置於這個分類學中,因為他的立場已在語料庫中被描述,但尚未相對於其他三者完全刻畫。耶和華是永恆者議會的主席——耶洛因文明母星上的治理機構。他是地球計畫中的最高權威,是發布路西法群體所違抗命令者,並且是在本章政治危機末對該違抗作出判決者。耶和華自己的立場是複雜的。他不是撒但——他支持人類的創造並繼續支持其保存。他也不是路西法——他認為保持人類無知的命令是有理由的,因為需要保護母星免於一個若被完全啟蒙可能成為威脅的造物。他的立場是這個三角關係中的溫和派:保存人類,但將他們限制起來。後續聖經敘事的大部分將是這溫和立場長期實際產生何種結果、以及它如何不同於撒但毀滅性立場和路西法完全揭露立場的展開。

這個四角分類學——撒但從母星反對所有人類創造、耶和華在母星上調和議會的決定並監督計畫、路西法在地球上領導以色列團隊內的異議派系、那蛇(集體的路西法派系)作為《創世記》第三章敘事中的揭露行動者——是語料庫其餘部分將要運作的政治結構。讀者現在可以帶著已命名並區分的角色進入巨蟹座敘事。

## IV. 覺醒

危機始於以色列團隊內部——七個派系團隊中最有成就的一支,其工作產生了伊甸場址,其人類是七個人類群體中最具智慧的。

源文獻以直接的措辭描述所發生之事:*「該團隊中的某些科學家對他們小小的人類,他們的『造物』,感到深刻的愛,他們想要給予他們完整的教育,以使他們成為和自己一樣的科學家。所以他們告訴這些幾近成年的年輕人,他們可以追求自己的科學研究,如此他們將變得和他們的造物者一樣有知識。」*

《創世記》第三章的記述以蛇的試探之形保存了這點。在語料庫的閱讀下,那蛇——路西法派系——首先對女人說話。希伯來文本在{% libref(book="genesis-woh", chapter=3, verse=4) %}創世記 3:4–5{% end %}中記錄的對話簡短但精確:
{{ scripture(book="genesis-woh", chapter=3, verse=4, translit="Vayomer ha-nachash el-ha-isha: lo-mot temutun.") }}
{{ scripture(book="genesis-woh", chapter=3, verse=5, translit="Ki yode'a Elohim ki be-yom akhalkhem mimenu ve-nifkechu eineikhem, vihyitem ke-Elohim yodei tov va-ra.") }}

路西法派系告訴人類其處境的真相。他們不是在一個將永遠維持為樂園的樂園中。他們在一個實驗室裡,被決定使他們對自己可能成為何物保持無知的造物者所看管。針對知識樹的禁令不是道德禁令而是政治禁令。樹的果子不致命。食用它不會產生死亡;它產生知識,而知識是母星上協調議會專門命令以色列團隊扣留之物。而且——最重要——獲得這知識的後果是人類將變得 *ke-Elohim*,「像耶洛因」,具有 *yodei tov va-ra*「知道善與惡」的具體能力。短語 *tov va-ra* 是命名整個道德知識的合用法。人類將變成有自身權利的道德主體,有能力評估、有能力判斷,不再受制於其前知識狀態所特有的單純順服。

人類聽了。他們吃了。後果正如那蛇所預測。*Vatipakachna einei shneihem, vayed'u ki erumim hem.*「他們二人的眼睛就明亮了,才知道自己是赤身露體。」第二節已指出,動詞 *pakach* 在他處用於使盲人之眼睜開。人類突然看見。他們首先且最直接看見的,是自己的身體狀況——他們是 *erumim*,赤裸、暴露。但更深的睜開是概念上的。他們看見了自己的處境。他們首次理解到自己是被造的存在。他們理解到他們的造物者本身就是存在——不是神,不是超自然力量,而是和他們自己一樣根本同類的實體,擁有相同的基本智能裝置,因此受相同的基本評估所制約。並且,關鍵地,他們理解到自己是被刻意保持在無知狀態中,理由是服務造物者而非他們自己。

源文獻描述了後果:*「這些新人類於是理解到他們也能輪到自己成為造物者,並且因為他們的『父母』把他們阻隔在科學書籍之外、把他們當作危險的實驗動物對待而對他們生氣。」*這是人類-造物者關係取得其在語料庫餘下部分將保留的性格的時刻。在這時刻之前,人類是孩子。在此之後,他們是某種更為複雜的存在——有能力評判其造物者、有能力怨恨其造物者、有能力把自己與其造物者相比較並發現比較結果令人不滿的造物。關係如今受到爭議。造物者對其造物的權威——曾經被視為理所當然——如今受到協商。在聖經語言中,這就是眼睛被睜開的意思。

還有一個文本細節值得提及。《創世記》第三章中的對話跨越數節,女人對蛇說話,然後將果子遞給男人,男人吃了。文本並未以後來神學傳統常常閱讀的方式將女人定為主要違犯者;希伯來文呈現食用為共同行為,兩位人類同等牽連並同等受隨後之事影響。雷爾主義源文獻添加了進一步限定傳統閱讀的脈絡:揭露被提供給兩位人類,揭露造物者的深情對兩者皆有,覺醒的後果同等地落在兩者身上。將夏娃讀為唯一有罪者的厭女詮釋傳統,是後來的發展,而不是文本本身的框架。

## V. 逐出與和解

母星上協調議會的反應,以及地球上遵從其命令之耶洛因的反應,是立即的。

人類被從花園移走。源文獻按字面閱讀此逐出:實驗室-花園是造物者的居所,而人類如今知道自己的處境,不再能被允許居住其中。*Vayeshalchehu Adonai Elohim mi-gan eden*——希伯來文的措辭不是隱喻。人類被遷至預備場址的界限之外,並被告知要在更廣大的環境中存活。

文本接著描述了安全部署。*Vayashken mi-kedem le-gan eden et ha-keruvim ve-et lahat ha-cherev ha-mithapechet, lishmor et derech etz ha-chayim.*「他在伊甸園的東邊安設基路伯,和四面轉動發火焰的劍,要把守生命樹的道路。」雷爾主義源文獻將此譯為其技術意義:*「持原子分解武器的士兵被部署在造物者居所的入口,以防止人類偷取更多科學知識。」*第二節已指出,短語 *lahat ha-cherev ha-mithapechet* 描述了一種射出火焰並旋轉的武器——以人類見證者所擁有的詞彙描述的定向能武器。基路伯不是後來圖像學中胖胖的有翼天使;他們是哨兵,駐紮在外緣,若人類試圖返回設施並取得他們被禁止之物,即被指示使用致命武力。

這值得停頓,因為它為聖經敘事的其餘部分確立了一個將在後續章節反覆出現的語域。希伯來聖經中所描述的天使、基路伯、撒拉弗、各種帶有不尋常武器與不尋常能力的天上存在——在雷爾主義的閱讀下,所有這些都是造物者文明的人員,執行具體行動中的具體職能,使用人類見證者無法辨認但我們在二十一世紀正開始接近的科技。定向能武器存在於我們自身當代的軍事研究中;它們是國防部大量投資的明確主題,而多國軍隊在過去十年中已部署運作系統。「四面轉動的火焰之劍」,在雷爾主義的閱讀下,是與我們自身文明所開始發展之物大致相當的技術——以對見證者尚不存在的技術詞彙未描述,而以當時確實存在的視覺詞彙描述。

人類如今在花園外,被給予他們生存所需。*Vaya'as Adonai Elohim le-Adam u-le-ishto kotnot or vayalbishem.*「耶和華耶洛因為亞當和他妻子用皮子作衣服給他們穿。」源文獻將此讀為基本生存手段的提供——不僅是衣服,而是人類在沒有造物者持續支援下於預備環境之外生活所需的最低物質資源。人類被放開。他們並非在被判定要死亡的意義上被遺棄;他們被釋放,並具備生存所需的裝備,並且被期望能生存下去。

接下來發生之事是創造序列中最具影響力的政治事件。母星上的協調議會——同一個勉強允許地球創造計畫進行的議會,以及以色列團隊以揭露禁忌知識違反其命令的議會——如今作出判決。源文獻描述結果:*「『那蛇』是這小群想要對{% wiki(slug="adam-and-eve") %}亞當與夏娃{% end %}說出真相的造物者,結果他們被自身行星的政府判處流放於地球,而所有其他科學家必須停止實驗並離開地球。」*

這是一個具體且不尋常的解決方案。議會並未毀滅人類創造。議會並未毀滅路西法派系。議會分開了他們。路西法群體——以色列團隊內違抗者,以及其他團隊中以團結加入他們者——被判處永久留在地球,處於監視之下,在他們選擇啟蒙的人類中度過餘生。其他科學家——以色列團隊的多數,加上來自超大陸其他區域的其他六支派系團隊——被命令停止他們的實驗工作並從行星撤離。政治和解將造物者群體分為兩組:一個返回母星的多數,以及一個永久流亡於地球的少數。

這種劃分將構築聖經敘事的其餘部分。流亡的造物者——文本將多樣地稱為墮落天使、守望者、耶洛因之子,以及後來拿非利人之父的群體——將留在地球,散佈於超大陸但中心於原伊甸地區。他們將與人類互動、教導他們、最終與他們交合,並在隨後兩千年的事件中扮演核心角色。母星上的議會將從遠處監視局勢,偶爾透過剩餘基礎設施介入但不再維持持續的創造性存在。兩個群體之間的關係——地球上流亡的耶洛因與母星上的議會——將緊張、有時敵對,並將塑造語料庫每一個後續事件的政治結構。

另一個值得注意的細節:在源文獻的記述中,流亡的造物者並不完全是罪犯。他們是出於對其造物的深情而行動的異議者,違反了他們認為過於嚴厲的命令。議會承認這點。流放是懲罰,但並未伴隨處決或毀滅已接受禁忌知識的人類。事實上,它是一種妥協——一種在制裁所為者的同時保存已做之事的方式,以及將問題在地理上隔離以免汙染母星的方式。流亡的造物者要與其選擇的後果共存,生活在那些他們選擇改變其命運的存在之中。

主要造物者團隊從地球的撤離,在這種閱讀下,本身是一個重大事件。七個團隊中有六個離開了各自的區域,放棄他們所建立的實驗室與基礎設施,撤回作為他們所創人類群體直接教師的人員。第七個團隊——以色列團隊——僅部分撤離;路西法派系留下,而團隊的其餘部分返回母星。從人類群體的角度來看,直接後果是大多數在伊甸期間持續在場之造物者人物的消失。其他六個世系將以自身版本的逐出來體驗此事——不是從某個具體花園,而是從與教導他們的造物者的持續直接接觸中。唯有伊甸世系保留了路西法派系的持續存在,以及最終將從這保留中流出的所有後果。

## VI. 生命樹與漫長的世代

政治和解一旦到位,進一步的發展出現了,源文獻以其特有的壓縮方式處理之,但其涵義是相當的。

花園外的人類開始繁衍。流亡的造物者如今生活在他們中間,與第一代人類領袖——希伯來聖經將其命名為亞當、塞特、以挪士、該南、瑪勒列、雅列、以諾、瑪土撒拉、拉麥及其他洪水前族長的人物——建立了關係。結果發現,這些人類領袖活了不尋常的長壽期間。《創世記》第五章的譜系以具體性記錄這些人物:亞當 930 年、塞特 912 年、以挪士 905 年、該南 910 年、瑪勒列 895 年、雅列 962 年、以諾 365 年(雖然以諾並未以平常意義死去,如第十二節將討論)、瑪土撒拉 969 年、拉麥 777 年、{% wiki(slug="noah") %}挪亞{% end %} 950 年。這些數字是具體的,而聖經文本將它們視為字面而非象徵。一個費心為每一世代記錄具體年數的譜系,是一個意圖讓那些年數被讀為具體年數、而非整數象徵的譜系。

源文獻以技術詞彙解釋此長壽。「生命樹」——在逐出時專門對亞當與夏娃禁止——是一種技術——一種延長生命的技法,類似源文獻在他處所描述為耶洛因長壽基礎的細胞轉移技法。流亡的造物者一旦獲得議會允許維持其與人類領袖的關係,便安排這些領袖受益於該長壽技術。*「在軍事監視下被留下的流亡造物者,敦促人類給他們帶來食物,以向自己的上級顯明新創造之民是好的、永遠不會反叛其造物者。如此他們設法為這些最初人類的領袖獲得『生命樹』之益,而這解釋了他們為何活得如此長久。」*

授予是有限的。源文獻指出,長壽不是遺傳的——壽命延長之族長的子女並不自動繼承該處置——並且最終*「生命的祕密失去了,而人類的進步減緩了。」*議會的勉強是可以理解的。如果每個人類都能活幾世紀,他們將比短壽群體更迅速地積累足以威脅母星的知識與經驗。長壽作為有限讓步被允許,授予一小群族長人物作為人類值得其造物者眷顧的展現,但從未延伸為將會把人類群體轉化為議會無法控制之事物的可遺傳特徵。

「生命樹」的技術內容值得注意,因為它在雷爾主義源材料中反覆出現。耶洛因自身的長壽,在源文獻的記述中,是透過一種細胞轉移技法達成{{ footnote(id="2") }}——意識在一個新生長的身體中重新具現,在多個連續身體形態中產生經驗的連續性。耶和華本人,在 1970 年代與雷爾對話之時,已歷經跨越 25,000 年的二十五具這樣的身體。授予洪水前族長的長壽很可能是此技法的一個變體,經改造以適應人類生理,並以個案而非普遍處置的方式允許。聖經中的年齡——930、912、905 等等——在這種閱讀下反映了接受該處置者所達成的實際壽命,以地球年計量。

這種不對稱值得強調,因為它將對本章後續各節變得關鍵。*某些人類接受了長壽處置。大多數沒有。*《創世記》第五章的聖經譜系追溯一個具體的族長世系——塞特的世系,最終下傳至挪亞——透過這世系傳遞長壽。語料庫的閱讀為:此世系是伊甸文明的領導世系。擁有權威職位、被視為值得有限長壽授予、能在數世紀而非數十年中積累知識的人類——這些就是長壽者。一般人類群體,包括該隱的後裔以及橫跨超大陸其他六個文明的更廣大群體,過著平常的人類壽命,以數十年而非數世紀為單位。

這種不對稱的涵義是相當的,本章將在第八節回到此。一個領袖活幾世紀的文明根本不同於領袖活數十年的文明。長壽的領袖能跨越多個正常世代追求項目。他們能跨越超出任何單一短壽者持續注意力能力之跨度精煉具體知識體。他們能整合一些領域的專業,而在短壽文明中,這些將分隔於不同從業者之間,沒有任何單一個體能跨越它們。個人記憶以任何短壽文明都無法匹敵的方式變成制度記憶的基質。擁有長壽族長領導與來自擁有原始耶洛因科學的流亡造物者持續接觸的伊甸文明,因此處於以其他六個文明——沒有長壽優勢且沒有持續直接教導——無法匹敵的速率在科技與智識上前進的位置。

與此同時,第一代生於花園外的人類是亞當與夏娃的子女。《創世記》第四章的聖經記述聚焦於其中兩位:該隱,年長者,以及亞伯,年幼者。他們的衝突故事——該隱對亞伯的謀殺,首次由另一個人類刻意造成的人類死亡——以細節保存。

源文獻對此敘事的貢獻是間接的。《創世記》第四章的記述按字面接受,作為兩個具體個體之間實際衝突的描述。源文獻所添加的是更廣大的脈絡。流亡的造物者鼓勵人類給他們帶來祭品——食物、農產品、牲畜——作為向母星議會展現人類行為良好並感激其造物者的方式。該隱帶來農產品。亞伯帶來肉。造物者(具體而言是耶和華,在此期間他要麼親臨地球,要麼透過允許評估祭品的技術連結臨在)對亞伯的祭品比該隱的更為偏好,產生了導致謀殺的怨恨。

衝突是否僅關乎祭品,或反映了第一代人類正發展之農業與遊牧生活方式之間的更深張力,不是源文獻所解決之事。源文獻所確立的是:首次謀殺不是神話事件。它是一個具體的人際衝突,具有具體的後果,因其展現了造物者的造物在被留下自行生活後變成了什麼,而被保留在記錄中。人類從知識樹吃了之後,現在完全有能力評估其處境、對其情境作出情緒回應,並以暴力對這些回應採取行動。

《創世記》第四章記述的延續——該隱的流放、他的婚姻、他建立城市、他發展具體技能與行業之後裔的出生——被源文獻視為實際人類文明的開始。{% libref(book="genesis-woh", chapter=4, verse=17) %}創世記 4:17-22{% end %}中給出的該隱世系譜系,其具體性令人矚目:該隱生以諾(與《創世記》第五章族長以諾不同的另一以諾),以諾生以拿,以拿生米戶雅利,米戶雅利生瑪土撒利,瑪土撒利生拉麥(也是另一個拉麥)。拉麥的子女接著以他們對文明的具體貢獻被指認:雅八是住帳棚牧養牲畜之人的祖師;猶八是一切彈琴吹簫之人的祖師;土八-該隱是打造各種銅鐵利器之人。文本明確:遊牧畜牧、音樂與冶金都源自此譜系所保留的具體名人,在逐出後幾個世代之內。這些不是模糊「原始人類」的發明;它們是歸於譜系所保留具體名人的具體文化與科技發展。

涵義是人類文明在逐出後最初幾代中迅速發展,有多個平行世系貢獻於其具體要素。該隱的世系產生了畜牧、音樂與冶金的奠基者。塞特的世系,長壽透過此世系傳遞,產生了將攜帶伊甸文明直至挪亞的族長領導。橫跨超大陸的其他六個文明,各自源自其奠基團隊,將會產生自身的平行文化奠基者集合,於其自身倖存傳統中歸於其自身名人(在有傳統倖存的地方)。到巨蟹座第一個千年末,超大陸已含有重要且文化多樣的人類文明,畜牧、農業、冶金、音樂、城市化等基礎科技已建立並傳播。

## VII. 耶洛因之子與人類之女

巨蟹座時代最具影響力的發展——以及將在下一個時代觸發產生洪水決定的事件——記載於{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記 6:1-4{% end %}。源文獻對此段落的閱讀,以其自身的措辭而言,是整個雷爾主義宇宙論中最具體、最重要的主張之一。而涉及此事的補充文獻,或許是語料庫框架最重要的前拉比時期猶太著作體系。

聖經文本,如第二節所引,如下:

*Vayehi ki hechel ha-adam la-rov al penei ha-adamah, u-vanot yuldu lahem. Vayir'u venei ha-Elohim et benot ha-adam ki tovot henah, vayikchu lahem nashim mi-kol asher bacharu... Ha-Nefilim hayu va-aretz ba-yamim ha-hem, ve-gam acharei khen, asher yavo'u venei ha-Elohim el benot ha-adam ve-yaldu lahem, hemah ha-giborim asher me-olam anshei ha-shem.*

「當人在世上多起來、又生女兒的時候,耶洛因的兒子們看見人的女子美貌,就隨意挑選,娶來為妻……那時候有拿非利人在地上,後來耶洛因的兒子們和人的女子們交合生子,那就是上古英武有名的人。」

源文獻按其字面接受{{ cite(id="1") }}。*「流亡中的造物者取了人類中最美麗的女兒為妻。」**benei ha-Elohim*,字面意義為「耶洛因之子」,是流亡的造物者——路西法群體及其同夥,他們自被逐出花園以來一直生活在地球上,依政治和解的條款允許他們留下但要求他們避免進一步的創造工作。在逐出後的數個世紀中,隨著人類群體成長與人類女兒成熟,流亡的造物者與她們建立了性與生殖關係。這些關係不是輕率的。文本將之描述為婚姻。並且它們產生了後代。

這些後代是 *拿非利人*,「墜落者」或「降下者」,源自字根 *nafal*,「墜落」。譯為「巨人」源自七十士譯本的 *gigantes*,並已成為英文翻譯傳統中的標準,但希伯來字根的意義是清楚的:這些是與墜落、降下、希伯來聖經將通篇應用於那些離開其原始狀態或地位之「墜落」類別相關聯的存在。在雷爾主義的閱讀下,*拿非利人* 是流亡造物者與人類女子的混血子女——這些存在按聖經文本的標準既非凡(英武有名之人)又有問題(與將導致洪水的道德敗壞相關聯)。「巨人」的閱讀可能並非全然錯誤,因為比人類基線實質地更大、更有能力的存在的混血後代,合理可能比一般人類更大、更有能力。但詞彙的詞源核心是降下,不是大小——*拿非利人* 是降下者,是從上而下之存在的子女。

**以諾書。**《創世記》第六章段落簡短——僅幾節,壓縮且省略。但闡述這些事件的可觀補充文獻體系值得自己的處理,因為沒有其他古代來源含有與雷爾主義框架如此直接一致的材料,而沒有其他文獻體系比此更系統地被繼承第二聖殿時期的宗教傳統邊緣化。

《以諾書》——具體而言學者所稱的《以諾一書》、衣索比亞文以諾書——是一部複合作品,含有五個不同的書或部分,跨越大約四個世紀作成,自公元前三世紀至公元後一世紀。五個部分為:《守望者之書》(《以諾一書》1-36 章),它將《創世記》第六章的敘事擴展為相當的細節;《比喻書》(《以諾一書》37-71 章);《天文書》(《以諾一書》72-82 章),保存了看似技術性的天文細節;《夢境之書》(《以諾一書》83-90 章);以及《以諾書信》(《以諾一書》91-108 章)。《守望者之書》是最古老的部分,也是與巨蟹座最相關的。

《守望者之書》所含內容,在雷爾主義框架下,是耶洛因在此時期實際是何物、做了什麼的最明確古代記錄。書中描述 200 位 *benei ha-Elohim*——稱為 *守望者*(希臘文 *Egregoroi*,亞蘭文 *Ireen*)——在一位名叫 Semjaza{{ cite(id="4") }}(某些音譯為 *Shemyaza*)的領導下降至地球。他們娶人類為妻。他們產生混血後代(*拿非利人*,雖然《守望者之書》稱之為 *gibborim*,「強悍者」,並闡述其本質)。他們教導人類守望者所知:冶金、武器製造、化妝品、天文學、占星術、草藥醫學、巫術、書寫與閱讀的技藝。文本給出具體的守望者名字及各自所揭露的具體知識:阿撒瀉勒(Azazel)教導刀劍、匕首、盾牌、胸甲的製作及化妝品的使用;Semjaza 教導咒語與根藥;Armaros 教導咒語的解除;Baraqijal 教導占星術;Kokabel 教導星座;Ezeqeel 教導關於雲的知識;Araqiel 教導大地的徵兆;Shamsiel 教導太陽的徵兆;Sariel 教導月亮的軌跡。具體細節的程度令人矚目。《守望者之書》並未描述模糊的靈性指導;它描述了從降下的存在向其所遇人類群體進行的具體技術知識轉移。

與雷爾主義源文獻的對應是直接的。以諾的守望者在語料庫的閱讀下,是雷爾主義源文獻的流亡造物者——耶洛因文明的一個具體子群體,在地球上,娶人類為妻並產生了將在導致洪水事件中扮演重要角色的混血後代。守望者教導人類具體技術知識的以諾細節,是雷爾主義源文獻關於流亡造物者違抗議會禁令繼續教導人類的記述的以諾對應。這兩個記述透過完全不同的傳播管道給出——一個由衣索比亞東正教會保存兩千年,一個於 1973 年給予一位法國記者——匯聚於同一基本敘事。

**經典歷史。**特別值得注意的是這個問題:儘管《以諾書》明顯重要且具有古老淵源,它如何被排除於經典希伯來聖經與大多數基督教經典之外?

該書在第二聖殿時期晚期(約公元前 200 年至公元 100 年)被廣泛閱讀並被視為權威。從 1947 年開始在庫姆蘭發現的死海古卷含有多種以諾類書籍的手稿殘片——《守望者之書》、《天文書》與《夢境之書》的亞蘭文殘片,可追溯至公元前二世紀與一世紀。這些殘片在庫姆蘭的存在表明,負責這些古卷的群體將以諾類文獻視為權威經文,與其他聖經與次經文本並列。該書在新約中被直接引用:猶大書 1:14-15 明確引用《以諾一書》1:9,將該引文歸於「亞當之後第七代的以諾」。幾位早期基督教作者——包括特土良、俄利根、亞歷山卓的革利免與愛任紐——將《以諾書》視為經典或半經典,在其神學著作中自由引用。

該書於拉比編纂過程中被排除於希伯來經典之外,該過程可能在公元一世紀末至二世紀完成。它在公元四至五世紀更廣泛的經典形成過程中被排除於大多數基督教經典之外,奧古斯丁在其《上帝之城》(第十五卷第 23 章)中以其古老性無法確立為由反對將以諾納入。羅馬天主教與大多數新教傳統將其排除。東正教傳統也將其排除。值得注意的例外是衣索比亞東正教泰瓦赫多教會,該教會將該書保存於其完整經典中並繼續這樣做,《以諾一書》在那裡被視為完全的經典經文。

為何被排除?傳統學術解釋是,以諾類文獻與正在興起的拉比與教父主流發現神學上有問題的啟示與宗派傳統相關聯。《守望者之書》的具體內容含有與後聖經時期猶太教與基督教正在發展的一神論神學難以調和的材料。守望者並非純粹靈體;他們有實體身體,與人類繁殖,教導冶金、天文學與醫學。他們被呈現為具有相當主動性的存在,他們選擇違抗神聖命令、採取實體存在,並透過普通生物過程產生後代。這比《創世記》第一章的記述更難置入乾淨的一神論框架中,後者的天上群體較為抽象、較少具身。

雷爾主義的閱讀直接解釋了該排除。《以諾書》含有對於耶洛因實際是什麼的最明確前拉比記錄——具實體的存在,能與人類繁殖,擁有人類作者以他們所掌握詞彙描述的科技。該書保存了希伯來經典經文編纂者已開始軟化之物:天上來訪者明確的、具身的、科技性的特徵。隨著後來的傳統圍繞抽象、屬靈、一神論神學鞏固,以諾類材料變得越來越難以納入。排除並不必然是惡意行動者單一有意識決定的結果;更謹慎的閱讀是,該材料對發展中的宗教傳統變得在神學上不可持續,於是被悄悄拋棄,其在衣索比亞教會的留存是某種地理隔絕的偶然。衣索比亞教會,因政治與地理因素與更廣大的基督教神學發展隔絕,保留了其餘傳統所拋棄之物。語料庫將此保留讀為一個幸運的偶然——語料庫框架最重要前拉比記錄的留存,在逃離了更廣泛神學馴化過程的單一管道中。{{ footnote(id="3") }}

對語料庫而言,這意味著《以諾書》應被視為與希伯來聖經並列的主要補充來源,對其關於守望者、其對人類的教導、其混血後代與其最終審判的具體事實主張給予同等重量。本章無法以該文獻應得的深度開展完整的以諾類材料;那將是語料庫中專門補充章節的主題。但本章必須登記:以諾類文獻存在;它以實質方式擴展並闡述《創世記》第六章的敘事;它在第二聖殿時期晚期是權威經文,並由衣索比亞教會保存;以及其內容與雷爾主義框架對應,精準到要求注意。

**以諾本人。**除了更廣泛的以諾類文獻之外,《創世記》第五章中以諾的人物本身值得具體關注,因為他在族長譜系中的處理是不尋常的。{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記 5:24{% end %}記載:*Vayithallekh Hanokh et ha-Elohim, ve-einenu ki lakach oto Elohim.*「以諾與耶洛因同行,耶洛因將他取去,他就不在世了。」以諾並未以平常意義死去。他被取走——被耶洛因本身從地球上移除。動詞 לָקַח(*lakach*),「拿、取」,是希伯來語標準的拿取動詞,而構句是事實性的:以諾不再在場(*einenu*,「他不在了」),因為耶洛因取走了他(*lakach oto Elohim*)。

雷爾主義的閱讀按字面接受這點。以諾被流亡造物者或議會訪客帶上母星。他後續的經歷——在《以諾書》與相關傳統中描述——構成任何來源所保存的耶洛因直接接觸最具體的古代記錄之一。在這種閱讀下,《以諾書》的天文與宇宙論細節源自以諾在其造訪期間實際所見。某些以諾段落的技術具體性——對宇宙地理的描述、曆法系統、諸天的結構——是一位古代人類與一個比自己實質更先進文明接觸的倖存記錄。

**猶太人作為直接後裔。**源文獻對拿非利人作出另一項值得仔細處理的主張,因為它對語料庫將追溯之後續歷史是具體且重要的。在雷爾於 1975 年第二次與耶洛因相遇期間,耶和華作出一個源文獻以不尋常強調呈現的揭示{{ cite(id="2") }}:*「猶太人是我們在地球上的直接後裔。這就是為何為他們保留了具體的命運。他們是『耶洛因之子和人類之女』(如《創世記》中所述)的後裔。他們最初的錯誤是與自己的科學造物交合。」*

主張是具體的。在源文獻的記述中,猶太人不僅僅是以色列團隊在獅子座原始創造期所產出的人類。他們是混血世系的後裔——流亡造物者與人類女子的後代,他們是耶洛因自身的直接基因後裔,而不僅是其造物。「上帝選民」的指認,在這種閱讀下,有一個生物學基礎:猶太人在其基因遺傳中保留了耶洛因世系的一部分,其他人類群體在同等濃度上無一所載。這就是使他們成為「直接後裔」之事。這就是為何為他們保留了具體的命運。

源文獻的框架值得仔細處理,語料庫必須明確說明它主張與不主張之事。該主張不是分層意義上斷言某個人類群體在道德上優於另一群體。在這個記述中,猶太人是流亡造物者的後裔——即違抗議會命令、被判處留在地球、然後進一步加重違規以與其造物交合的異議派系的後裔。*「他們最初的錯誤是與自己的科學造物交合」*是源文獻自身承認:產生猶太世系的那個結合本身,以源文獻自身的條件,是個違犯。猶太人並非以美德區分,而是以遺傳——具體而言,以一個複雜且道德上含混的起源遺傳,反映了流亡造物者對其造物的深情以及產生他們之結合的政治不規則性。

歷史上與猶太人相關聯的智識與文化成就,在這個記述中,是此遺傳的可見後果。一個承載較高比例造物者自身基因材料的群體,平均而言,預期將以高於沒有此特質的群體之比率展現造物者本身的認知與創造能力。源文獻並未將此呈現為驕傲或支配的理由。它呈現為起源的事實,具有相應的責任而非特權。為猶太人保留的「具體命運」,以源文獻的條件而言,並非統治其他人類的命運,而是在人類最終認識自身起源並與其造物者建立正確關係的長過程中扮演特定角色的命運。

語料庫必須明確,如其在獅子座一章中對更廣泛種族問題的處理那樣,此主張不允許任何現代政治應用。造物者的形像——*tzelem Elohim*——平等地賦予每一個人類,無論其基因祖先以較大比例包含哪一世系。後續的混血是廣泛的;當代沒有任何群體是任何單一古代世系的乾淨保留;對人類尊嚴重要的認知與道德能力在整個物種中是普遍的。源文獻關於某個世系歷史起源的主張不轉化為任何關於當代群體相對價值的現代主張。語料庫明確拒絕任何將此材料解讀為允許種族階層、反猶太主義、以差異道德地位形式存在的親猶太主義,或任何其他將任何人類群體視為較其他群體更值得或較不值得完整人類尊嚴的閱讀。源文獻的主張是其所是。其現代應用受語料庫始終確立的平等尊嚴更廣泛道德框架治理,任何關於祖先的歷史主張都不能凌駕之。

這是敏感材料。本章已以其應得的關注處理之。此處提供的閱讀是雷爾主義源文獻自身提供的閱讀;它是否正確是語料庫無法僅從源材料解決的問題,但該主張具體到足夠,且對希伯來聖經將描述的後續歷史重要到足夠,值得清楚陳述而非以改寫軟化。

## VIII. 更廣大的世界

本章迄今聚焦於原伊甸地區與以色列團隊世系,因為該世系是希伯來聖經以細節保存其故事者。但在此時必須矯正聚焦可能產生的畫面。巨蟹座時代不僅是伊甸故事的時代。它是在獅子座期間被創造的所有人類群體——七個團隊每一個的輸出,遍佈整個超大陸——開始繁衍、擴展、組織自身並發展各自文明的時代。

巨蟹座的超大陸是單一陸塊,尚未被將在雙子座時代將其分隔為我們現在所知諸大陸的板塊事件所破。七個團隊在其各自的區域基地創造其人類後,於獅子座期間將自身分布於此陸塊。在獅子座末逐出危機時主要造物者團隊撤離——帶走實驗基礎設施、活躍的設計計畫,以及大部分人員——之後,每個大陸上所留下的是一個已被教導農業、定居、社會組織及其團隊判定可安全傳授之基礎科技基本要素的人類群體。這些群體在巨蟹座開始時邁入其存在的第二個千年,現在成長、組織自身為社群、發展區域文化、建造將成為城市的聚落,並精緻化將攜帶他們度過時代其餘部分的文明基質。

地理分布大致如下,基於源文獻對原始團隊位置的識別以及洪水後世界所保存的現代大陸格局。以色列團隊位於將成為東地中海/黎凡特地區之處,涵蓋日後將是以色列、希臘、土耳其,以及超大陸周圍沿海地區。第二個團隊在將成為地中海地區以南的非洲,可能延伸至現在的撒哈拉(該地區在此期間實質地較濕潤,含有廣大的湖泊與河流系統,維持著現代沙漠所無法維持的群體)。第三個團隊在將成為南亞之處,印度河流域與周圍區域。第四個團隊在將成為東亞之處,中國大陸與鄰近區域。第五個團隊在將成為美洲之處,在最終將成為中美洲或安第斯山脈的區域。第六個團隊在將成為大洋洲之處,可能在超大陸後來將成為澳洲與太平洋群島鏈的部分。第七個團隊在將成為北歐之處,在後來將是斯堪地那維亞與周圍北方土地的區域。確切識別是推測性的;源文獻並未給出完整的地理規格。源文獻確立的是七個團隊在超大陸的七個不同區域工作,且這些區域大致對應於洪水後世界將繼承之主要人類群體的地理起源。

超大陸的地理在分裂前允許跨越後來將變得無法通行之距離的貿易與文化接觸。在巨蟹座晚期,一位從伊甸地區到東亞地區的商人原則上可以走完整段距離——可能三、四千公里跨越連續陸地的旅程,艱苦但可行。沿超大陸海岸線的海上文化能用幾乎肯定比洪水後考古記錄所暗示更發達的船舶科技在相當距離上移動貨物——考慮到我們所知緊接洪水後時期存在的連續造船傳統,以及語料庫框架所暗示的洪水前科技實質地比倖存證據所表明更為先進。

這意味著,在語料庫的閱讀下,洪水前人類是*全球互聯*的。貨物在七個文明之間移動。觀念移動。科技移動。基因材料透過個體在更廣大超大陸的遷移與通婚而移動。七個文明的文化傳統雖具區別性、可被辨認,但將處於持續接觸與持續相互影響之中。畫面不是七個獨立發展的孤立群體;而是一個全球文明網絡,在其互聯性上可比擬於中世紀絲綢之路或近現代歐洲探索網絡,實質差異在於連接是大陸性而非海洋性,且網絡的持續時間在於兩千年量級而非幾世紀。

七個文明之間的政治關係並非總是和平。源文獻明確指出洪水前時期包括相當的人類間衝突,以各文明所發展的科技進行:*「事實上,人類選擇用此軍火庫進行可憎的彼此戰爭。」*所稱的「軍火庫」是流亡造物者所教導伊甸世系的科技集合,並透過貿易與文化接觸傳播到其他文明。源文獻的框架暗示此軍火庫是相當的——足夠破壞性使其所支持的戰鬥可被稱為「可憎的戰爭」,並且重要到母星政府對這些戰爭的察覺將促成他們最終的干預決定。我們不知道涉及哪些具體科技,但暗示是洪水前的人類戰爭達到了在洪水後世界中直到工業時代、可能直到二十世紀才再次接近的規模與強度。

流亡造物者與這些文明的關係是不平等的。路西法派系在逐出後留在伊甸地區,繼續教導伊甸世系實質地超過他們被授權之事。其他六個文明沒有對異議造物者-教師相同的直接持續接觸。他們在獅子座期間從其奠基團隊接受了開始文明所足夠的初始指導,但他們沒有將他們帶到伊甸世系正在達到之相同科技與科學進步水平的持續教導。結果是橫跨超大陸的不對稱進步分布:伊甸文明在前沿,其他六個以不同速度跟隨,取決於其具體情況以及哪些知識透過貿易與文化接觸傳播至他們。

正是在此脈絡下,失落先進文明的零散記憶——保存於全球各地傳統中、被現代學術大致歸檔於「神話」之下的記憶——變得值得認真看待。**柏拉圖的亞特蘭提斯**,保存於《蒂邁歐》與《克里底亞》對話中,描述了大約在柏拉圖自己時代之前九千年存在的一個重要文明。柏拉圖寫於公元前四世紀初;九千年之前將把亞特蘭提斯事件置於約公元前 9600 年——這以驚人的精確性落在語料庫時間軸上巨蟹座早期。柏拉圖的亞特蘭提斯科技先進、全球有影響力、圍繞中心城市的同心陸地與水環建造、由諸王邦聯統治、擁有相當的海上與農業能力,並最終在一場地震與洪水的災難事件中被摧毀。柏拉圖給出的細節——城市的大小、環的排列、工程作品、農業系統、政治結構——足夠具體,暗示柏拉圖(或他的來源,即據報告告訴梭倫原始故事的賽伊斯埃及祭司)接觸到一個關於真實洪水前文明的連貫傳統,而不僅是發明的寓言。

柏拉圖給出的地理位置——直布羅陀之柱以外,在大西洋中——是產生最多後續推測的細節。傳統位置在大西洋中部,並未獲得任何海洋學證據支持;在字面大西洋位置所要求的深度與位置上,未發現對應於柏拉圖描述的沉沒大陸或主要島鏈。在相當的相關文獻中提出的替代位置包括地中海(聖托里尼與米諾斯文明,學術上最受尊重的候選)、南極(在某些替代製圖學理論中曾經位於更接近赤道之處),以及世界各地的其他遺址。這些都未被主流接受為亞特蘭提斯位置,大多數比傳統地中海讀法實質地更弱。

一個較新的候選,自約 2018 年以來在替代考古學討論中出現,是茅利塔尼亞的**理查特結構**——也稱為非洲之眼或撒哈拉之眼。該結構是一個直徑約 40 公里的圓形地質構造,從太空可見,由差異侵蝕岩石的同心環組成。主流地質學解釋為深度侵蝕的地質穹丘——隆升後長期侵蝕暴露出相繼的岩層,並未暗示它曾經是建造的特徵。替代閱讀提出理查特結構以令人驚訝的精確性匹配柏拉圖對亞特蘭提斯的描述:柏拉圖所描述同心的陸地與水環、大小匹配柏拉圖以斯塔德為單位的測量在合理容忍範圍內、位置在西北非而非大西洋中部,以及周圍的沙漠平原對應於柏拉圖描述為環繞亞特蘭提斯的大平原。在語料庫年表所置於巨蟹座-雙子座邊界的災難性洪水事件,將會淹沒該結構,留下從上方可見的地質格局,但建築在隨後的千年中被侵蝕殆盡,平原隨氣候乾化而成為現今的撒哈拉。

語料庫不堅持理查特鑑定。主流將該結構地質性閱讀為自然構造是主導詮釋,替代考古學的閱讀未被專業考古學家或地質學家採納。語料庫所指出的是:理查特結構是柏拉圖所描述之事的候選位置,該候選與語料庫框架以更傳統候選所沒有的方式一致,並且該問題值得比主流考古學通常給予的輕慢處理更為認真的調查。

亞特蘭提斯與伊甸文明在語料庫的閱讀下有何關係?幾種可能性自我呈現。亞特蘭提斯傳統可能保存了伊甸文明本身的記憶,地理細節(大西洋位置、柱以外)反映了零散記憶的洪水後重新定位,而非文明的原始位置。亞特蘭提斯傳統可能保存了一個獨立文明的記憶——其他六個世系之一——它與伊甸文明有貿易與文化接觸但不同於它。亞特蘭提斯傳統可能保存了伊甸文明的主要殖民地或前哨的記憶,位於距中央伊甸地區相當距離,並作為自身的區域強權運作。或者,亞特蘭提斯傳統可能將多個不同文明的記憶崩塌為單一複合形象,傳承該傳統的希臘人無法區分其零散資訊所取自的各種來源。

語料庫無需承諾具體的閱讀。它所承諾的是更廣泛的框架:一個重要的洪水前文明曾存在,其記憶以零散形式倖存於後續傳統中,希臘傳統保存了一個這樣的片段為亞特蘭提斯,其他傳統以其他名字保存了其他片段。橫跨全球神話傳統的格局——赫西俄德的黃金種族、含有壽命極長的{% wiki(slug="antediluvian") %}洪水前{% end %}統治者的蘇美爾王表、含有先進文明興衰循環的印度教 yugas、含有人類先前嘗試的中美洲《波波爾·烏》、神與人同行的埃及 *zep tepi*(「最初之時」)、沉沒家園的玻里尼西亞傳統、海下失落城市的凱爾特傳統——並不是單一傳統。它是失落先進文明記憶的全球分布,以各文化特有的形式保存,但在結構層次上匯聚於同一故事。

一個文明曾存在。它被摧毀了。倖存的零散記憶經洪水後群體代代相傳,被長期傳承所扭曲但保留了曾經之事的結構輪廓。以任何合理的閱讀,這些傳統不是從無中發明的。它們是某事發生之物的不完美記錄。

**伊甸文明的科技軌跡。**本章現在已準備好提出其關於巨蟹座的核心詮釋主張,即透過前面各節一直在建立的主張。

伊甸文明對其他六個世系有多重複合的優勢。第一,源文獻對以色列團隊人類作為七個輸出中最具智慧者的具體刻畫。語料庫在獅子座一章及此處皆謹慎地限定其意義:不是類別優越性、不是允許階層,而是源文獻關於一個團隊特定輸出的具體事實主張。無論所涉認知區別性的確切性質為何,伊甸人類從其創造之刻起便擁有此優勢。

第二,路西法派系作為持續教師的持續直接存在。其他六個世系在獅子座期間從其奠基團隊接受了初始指導,然後在那些團隊撤離時失去了與造物者人物的持續直接接觸。伊甸世系永久保留了路西法派系。異議的耶洛因因為首先選擇揭露禁忌知識,跨越數個世紀持續教導,在兩千年中精煉伊甸世系對母星原本所禁科學與科技的理解。教導是持續的,跨越兩千年維持,並由其自身文明擁有語料庫框架中最先進科學的存在進行。

第三,授予伊甸世系領袖的長壽處置。《創世記》第五章的族長譜系記錄了一個具體的領袖世系——從亞當經塞特至挪亞——的具體年齡(930、912、905,直到瑪土撒拉的 969)。在源文獻的記述中,這些延長的壽命是路西法派系在有限與個案基礎上為伊甸文明的領導所取得之生命樹科技的後果。沒有此優勢的其他文明過著數十年量級的平常人類壽命。差異對文明後果是巨大的。一個活九百年的領袖能跨越多個正常世代追求研究項目,能跨越超出任何短壽者持續注意力能力之跨度精煉具體知識體,能整合在短壽文明中將分隔於不同從業者、沒有單一個體能跨越之領域的專業。個人記憶變為制度記憶的基質。數世紀累積的經驗可用於告知在短壽文明中將不得不由僅有數十年個人經驗可資的領袖作出之決定。

第四,*benei ha-Elohim* 與人類女子的混血後代,到巨蟹座中期已是伊甸文明中相當的群體。這些混血——拿非利人、「英武有名之人」——憑藉其雙重遺傳擁有接近耶洛因自身的能力。他們在一系列維度上比一般人類更強壯、更具智慧、更有能力。他們成為領袖、戰士、建造者、科學家,以及在產生他們的文明中自然而然會上升至相當權威與成就職位的那種人物。伊甸文明因此不僅由流亡的耶洛因教導,而且如今將耶洛因基因材料併入其自身的領導群體。

第五,流亡造物者所給予他們的科技——包括在源文獻框架中產生源文獻所描述「可憎戰爭」的武器。伊甸文明不僅擁有先進的和平科技,也擁有先進的軍事科技,足夠進行對超大陸其他文明的相當規模戰爭。其他文明透過自己的研究計畫以及伊甸科技經由貿易網絡的擴散發展了自身的軍事科技以作回應,但伊甸文明在整個時代中始終保持在前沿。

這些優勢的複合效應——有才能的奠基群體、造物者-教師的持續直接指導、領袖的長壽、領導庫存中的混血耶洛因基因材料、包括軍事軍火庫在內的先進科技——產生了一個文明,在巨蟹座時代末已進步至語料庫只能以實質謙遜推測之水平。洪水前文明的物理證據至多是零散的,其大部分存在已在下一章將處理的災難事件中被摧毀。倖存之物——獅身人面像若其替代年代正確、土耳其東南部的哥貝克力石陣與卡拉漢石陣、橫跨多大陸的巨石傳統、保存於柏拉圖、赫西俄德與蘇美爾來源中的文化記憶——僅給我們片段。這些片段暗示相當的精緻。但伊甸文明在其巔峰的實際能力是我們只能透過推斷重建之物。

語料庫的框架允許對伊甸文明可能進步至何種程度作認真推測。可能在某些方面可比擬於或超越我們自身當前的科技能力。可能擁有我們尚未發展的能源或材料科技。可能能夠在我們仍只是匹配或尚未匹配的尺度上進行工程壯舉。語料庫框架的戲劇性涵義是,該文明進步*得夠遠以至於母星政府將其視為真正的威脅*。不僅是煩擾。不僅是顧慮。一個真正的威脅,足以為下一章將描述的災難性干預作出辯護。無論何種能力水平在母星政府中觸發該判斷,巨蟹座末的伊甸文明已達到該水平。

這就是母星從一開始就害怕之事。撒但派系的論述——回溯至地球計畫甚至開始之前——一直是能夠等於或超越其造物者的合成造物從根本上是危險的,因為沒有任何協定能遏制它們最終將變成正是這樣威脅的風險。地球計畫越過撒但的反對而進行,母星希望地理距離與政治約束(不揭露禁忌知識、不給予人類長壽、限制科技)將足以防止威脅成真。但約束已被違反。路西法的派系揭露了知識。長壽已授予伊甸的領導。科技已傳遞。混血後代已被產生。伊甸文明已進步至接近或可能超越母星自身能力的水平。超大陸文明之間的戰爭已表明破壞能力是真實的並正在被使用。從母星的視角觀察這一切在兩千年中展開,結論是不可避免的:約束已失敗。地球實驗已產生原始反對者所警告的威脅。現在需要行動。

緊接其後的雙子座時代,將是行動的時代。母星政府在語料庫將於適當章節處理的大量辯論之後,將得出結論認為人類創造必須在它成為對母星文明本身實際威脅之前被摧毀。決定將透過源文獻描述為洪水的災難事件實施,該事件在聖經文本中於《創世記》6-9 章中記錄。透過方舟保存一群餘民——流亡造物者的工作,他們拒絕參與摧毀那些他們愛到願意啟蒙的存在——將設定定義洪水後時代的恢復。但這一切都是雙子座的領域。對於巨蟹座,相關的點是:到時代末,將會產生洪水的條件已就位,而母星政府的審議已開始匯聚於干預不可避免的結論。

## IX. 巨蟹座的科學

源文獻告訴我們巨蟹座期間發生之事的概略。事件的技術內容——長壽科技實際是什麼、拿非利人混血生物學將需要什麼、洪水前文明的考古學能告訴我們什麼——一如前面章節,在當前科學中可得,儘管它必須從多個專業文獻中匯集。

本節分為七個子節,遵循語料庫科學疊加層所建立的格局。第一,生命樹科技與現代長壽研究。第二,混血生物學以及耶洛因-人類繁殖所需條件的問題。第三,前陶器時代新石器安那托利亞的考古學,特別關注哥貝克力石陣與卡拉漢石陣。第四,新仙女木事件在涉及巨蟹座-雙子座轉折處的關係。第五,亞特蘭提斯傳統與洪水前文明記憶的更廣泛格局。第六,群體瓶頸問題與基因證據。第七,通往我們自身時刻的脈絡。

### IX.1. 生命樹與現代長壽研究

聖經中的族長壽命——亞當 930 年、瑪土撒拉 969 年,等等——描述了以任何當代生物標準衡量都非凡的長壽。現代有記錄已驗證的最長人類壽命是讓娜·卡爾芒(Jeanne Calment)的 122 年 164 天,她於 1997 年去世。洪水前的族長活得幾乎比這個最大值長一個數量級。要麼聖經數字是象徵性而非字面性的,要麼族長接觸到了將壽命延長至遠超出普通人類生物所提供之科技。語料庫依循雷爾主義源文獻,採取第二種閱讀。

關於人類老化生物學中對「如何達到這樣的長壽」這個問題具相關性之事,目前知道什麼?現代研究已確認了老化過程的若干主要貢獻因素,以及若干在實驗室與臨床情境中已展現能力減緩或部分逆轉老化具體面向的干預策略。

山中因子(Yamanaka factors)是一組四個轉錄因子——Oct4、Sox2、Klf4 與 c-Myc——當引入成體細胞時,能將那些細胞重新編程回類似胚胎幹細胞的多能狀態。此發現由京都大學的山中伸彌與其同事作出並於 2006 年發表,山中因此於 2012 年榮獲諾貝爾生理學或醫學獎。其對長壽研究的涵義是相當的。細胞老化與表觀遺傳標記的漸進變化相關聯——對 DNA 與組蛋白的化學修飾,它們調節基因表現而不改變底層 DNA 序列。這些表觀遺傳變化隨時間累積是細胞變老的主要機制之一。山中因子透過將細胞重新編程回多能狀態,也重置這些細胞表觀遺傳年齡的大部分。如果重新編程能被部分應用——足以重置表觀遺傳年齡但不足以使細胞去分化回多能性——結果將是生物年齡比其生理年齡更年輕、同時保留其專門身分的細胞。

關於部分細胞重新編程作為抗老化干預的相當研究正在進行中。哈佛醫學院大衛·辛克萊(David Sinclair)的實驗室在此領域特別活躍,在小鼠模型中證明部分重新編程能恢復老化小鼠的視力、逆轉各種組織中細胞老化的某些標記,並延長治療動物的健康壽期。其他實驗室,包括索爾克研究所的胡安·卡洛斯·伊斯皮蘇阿·貝爾蒙特(Juan Carlos Izpisua Belmonte)與史丹佛的維托里奧·塞巴斯蒂亞諾(Vittorio Sebastiano),已產生互補的發現。研究的軌跡是清楚的:細胞老化至少部分可逆,而逆轉它的技術正逐步精煉。

已確認並受積極干預研究關注的其他老化貢獻因素包括端粒縮短(細胞分裂期間染色體末端保護性 DNA 序列的漸進喪失)、衰老細胞的累積(停止分裂但抵抗正常細胞死亡途徑並產生促成組織功能障礙之發炎信號的細胞)、粒線體功能障礙、幹細胞功能的喪失,以及細胞間通訊的崩潰。每一項都是相當研究計畫的主題,而各種干預——端粒酶活化、選擇性殺死衰老細胞的衰老解除藥物(senolytic drugs)、針對粒線體的療法、幹細胞治療——已在實驗室模型以及在某些情況下早期臨床試驗中展現療效。

研究的當前狀態暗示在未來幾十年內,透過這些干預的某種組合,即使沒有超出當前軌跡的重大科學突破,人類壽命達到 150 或 200 年是可達成的。500 或 1,000 年的壽命將要求超出當前軌跡的實質突破,或者——這是與語料庫框架相關的可能性——當前生物學尚未發展的完全不同方法。雷爾主義源文獻歸於耶洛因的細胞轉移科技,其中意識在一個新生長的身體中重新具現,即為一種這樣的不同方法。現代科學目前無法做到這點。但各種組件——從幹細胞培養替代器官與組織、繪製記憶與人格的神經相關物、在基質之間轉移生物資訊——皆為積極研究的主題,在每一項上都有實質進展。朝向類似細胞轉移長壽的軌跡雖然緩慢但是可見的。

對語料庫框架而言,涵義是族長壽命在生物上並非不可能。它們可透過我們現在開始發展的科技,以及一個經歷數千年持續研究的更先進文明可能已使其成熟的科技,在生物上達成。雷爾主義源文獻的生命樹科技在語料庫的閱讀下是真實的科技,屬於我們現在處於發展早期階段的同類。族長壽命是有限地接受該科技之存在的傳記記錄。當前人類群體中此類長壽的缺席反映了原始授予的有限與最終撤回性質,而非底層過程的任何生物不可能性。

### IX.2. 混血生物學與耶洛因之子

《創世記》第六章的主張——*benei ha-Elohim* 取人類女子為妻並產生可存活的後代——承載了相當的生物涵義。在現代生物學中,跨物種繁殖受所涉物種之間的基因距離所限制。近緣物種有時能產生混血後代(馬與驢的騾、獅與虎的獅虎獸),但後代通常不育或生育力顯著降低,反映了當不同物種的染色體試圖在減數分裂期間配對時所產生的困難。較遠緣的物種完全無法產生可存活的後代;基因差異足以阻止成功的受精、胚胎發育或活產。

*benei ha-Elohim* 與人類女子要產生可存活、可生育的後代——拿非利人、*gibborim*、「英武有名之人」——耶洛因在生物上必須非常接近人類。具體而言,他們在基因上必須足夠接近,以至於人類與耶洛因的染色體能在減數分裂期間成功配對、胚胎發育所需的發育程序相容,且所得後代本身有能力進一步繁殖。這是對耶洛因必須是什麼的相當約束。

該約束與語料庫更廣泛框架一致。在源文獻的記述中,耶洛因透過修改靈長類模板——黑猩猩或南方古猿的起點——並添加使結果成為人類而非靈長類之具體修飾來設計人類。獅子座一章詳細討論了這點。如果耶洛因本身是更先進文明的生物學人類(而非根本不同生物學的外星人),則耶洛因與人類之間的基因距離將足夠小以支援混血繁殖。這基本上是源文獻全篇所主張之事:耶洛因不是外星;他們是和我們一樣的人類,從其母星上自身的生物祖系下傳,沿著類似於我們自身的演化或設計線進行。{% libref(book="genesis-woh", chapter=1, verse=26) %}創世記 1:26{% end %}的「按我們的形像」是字面意義:耶洛因看起來像我們,是因為我們像他們,以他們的模板為模型。

人類-耶洛因混血的生物後果是推測性的但可勾勒。混血後代將從兩個親代群體繼承特徵——來自人類母親的基本人類基質,來自 *benei ha-Elohim* 父親的具體耶洛因變異。這些變異可能包括增強的認知能力(如果耶洛因如獅子座一章討論的稍微高於人類設計的認知終點)、較長壽命(如果耶洛因生物學自然支持人類僅透過科技干預才能達到的長壽)、較強的身體能力,可能還有與耶洛因世系相關聯、可在混血群體後代中可被偵測的具體基因標記。聖經將拿非利人描述為 *gibborim*「強悍者」與 *anshei ha-shem*「英武有名之人」,與雙重遺傳產生相對於一般人類基線的增強能力之後代相符。

對現代讀者而言,涵義是語料庫框架預測基因記錄中可偵測的事物。如雷爾主義源文獻所主張,如果猶太群體(或其中的特定亞群體)從混血世系下傳,則耶洛因祖先的基因簽名原則上應可作為猶太祖先的具體成分被偵測,將其與沒有相同混血歷史之群體的祖先區分開來。現代群體遺傳學迄今未識別任何此類獨特成分,但分析需要知道要尋找什麼。耶洛因基因組將足夠接近人類以支援混血繁殖,但足夠不同以使具體標記可被足夠精緻的基因分析偵測。語料庫並未主張此分析已被進行;它指出框架預測了原則上可被測試的具體事物。

### IX.3. 前陶器時代新石器安那托利亞:哥貝克力石陣與石頭文明

對語料庫框架而言,過去三十年最具影響力的考古發展,是對土耳其東南部前陶器時代新石器遺址——在語料庫的閱讀下對應於原始伊甸領域的區域——的挖掘與研究。

**哥貝克力石陣**,即實質地推翻了早期人類文明理解的遺址,由德國考古學家克勞斯·施密特(Klaus Schmidt)識別並在其指導下自 1995 年開始挖掘。該遺址位於土耳其東南部尚勒烏爾法省的山頂,距現代尚勒烏爾法市東北約 15 公里。施密特與其團隊在二十年挖掘中所揭露之事,實質地推翻了先前佔主導地位的關於複雜人類文明何時開始的觀點。

哥貝克力石陣由多個圓形與橢圓形圍場組成,每個皆由按精心格局排列的巨大雕刻石灰岩柱所界定。最大的柱子是 T 形,高約 5.5 公尺,每根重量在 10-20 噸量級。許多柱子刻有動物的詳細浮雕——獅子、狐狸、鳥、蛇、蠍子、野豬、瞪羚——排列為具體的圖像構圖,其意義尚未被理解。某些柱子顯示抽象符號,被多樣解讀為原始文字、曆法標記或宗教圖像。

年代測定是關鍵細節。對該遺址有機材料的碳測年將其建造置於公元前第 10 與第 9 千年——具體而言,最早地層可追溯至約公元前 9600 年,建造延續約 1,500 年,直到遺址在公元前 8000 年左右被刻意掩埋(以瓦礫填入)。在語料庫年表上,這將哥貝克力石陣置於巨蟹座早至中期之內。該遺址在時間上是語料庫框架的直接後伊甸時期——逐出之後不久的幾個世紀,當時伊甸文明處於後花園發展的早期階段。

哥貝克力石陣的重要性在於,它展現了在此早期年代實質上比先前主導考古框架所允許更精緻的組織、技術能力與文化複雜性。在哥貝克力石陣之前,主流考古學的主導觀點是複雜的紀念性建築需要定居的農業社會,農業比此類建築早數個千年,而發展序列是:遊牧狩獵採集群體 → 定居農業村莊 → 帶有紀念性建築的早期城市。哥貝克力石陣顛倒了此序列。該遺址是由看起來像是狩獵採集或原始農業群體所建造,組織規模充分且技術能力充分,得以跨越超過千年的持續建造採石、運輸、雕刻並豎立多噸石柱。主導框架視為此類建築必要前提的農業,在哥貝克力石陣中,可能是*後果*而非前提——隨著遺址群體增長至需要比狩獵採集所能提供更可靠食物供應之規模,在周圍區域興起。

克勞斯·施密特本人在 2014 年去世前已開始闡述此涵義:傳統序列是錯的,紀念性宗教或儀式建築先於並可能驅動了農業的發展,以及此類建築所需的人類文化能力已被前一個框架實質低估。該涵義在過去十年中已實質地被主流考古學吸收。哥貝克力石陣不再是邊緣發現;它現在被視為理解從更新世到全新世過渡的奠基遺址之一,而相當的研究計畫正在調查其涵義。

對語料庫框架而言,哥貝克力石陣是直接與所預測畫面一致的物理證據。語料庫框架預測洪水前文明在巨蟹座早期、在伊甸地區以相當的精緻存在,並且如果有任何此類建築倖存於後續洪水與隨後長期侵蝕過程,將留下紀念性建築。哥貝克力石陣正是這樣的倖存證據:紀念性建築,具相當精緻,在所預測的區域,可追溯至所預測的時期。語料庫並未主張哥貝克力石陣可獨特地認同於任何具體伊甸相關事件。它所主張的是哥貝克力石陣是語料庫框架所預測之物,且此類遺址在此類地點、此類年代的發現——獨立於語料庫並透過普通考古方法作出——是語料庫年表框架最強的物理證據之一。

**卡拉漢石陣**,位於土耳其東南部同一區域的另一遺址,自 2019 年在伊斯坦堡大學的內米·卡魯爾(Necmi Karul)及其團隊指導下進行挖掘。卡拉漢石陣比哥貝克力石陣略小,但同樣古老——在某些初步年代測定中可能略古老——並顯示類似的紀念性 T 形柱、雕刻浮雕與複雜空間組織特徵。挖掘在撰寫之時正進行中,卡拉漢石陣所含的完整範圍尚未發表。已經清楚的是哥貝克力石陣並非獨特。土耳其東南部的前陶器時代新石器 A 文化在相當的地理區域產生了多個紀念性遺址,表明所涉的文化與技術能力不是單一孤立社群的工作,而是擁有多個中心之更廣大文明的工作。

更廣大的前陶器時代新石器複合體在這兩個具體遺址之外延伸至包括其他幾處——內瓦利·喬里(Nevali Çori)、漢贊台(Hamzan Tepe)、塞費爾台(Sefer Tepe)等——它們共同構成了伊甸地區巨蟹座早期文明的考古簽名。語料庫框架預測此文明存在;主流考古學現已實質地證實其存在。框架與主流之間的差異在於對此文明是什麼的詮釋——框架將其讀為從以色列團隊創造下傳的伊甸世系早期階段,主流考古學將其讀為早期人類文化複雜性令人矚目但未獲解釋的範例。框架的閱讀更為具體。它將哥貝克力石陣連接到更廣大的敘事;主流考古學擁有該遺址但尚未對其有解釋。

### IX.4. 新仙女木期

新仙女木期是發生於距今約 12,900 至 11,700 年前的一個氣候事件——也就是,在語料庫年表上獅子座時代晚期幾個世紀至巨蟹座時代早期幾個世紀。該事件涉及在北半球大部分地區突然回到接近冰川的寒冷狀況,持續約 1,200 年,然後氣候同樣突然地回到自上次冰川極盛期結束以來進行中的暖化趨勢。

新仙女木期的成因受到相當的辯論。傳統解釋是它由勞倫泰冰原退卻所致融水的突然排放擾亂北大西洋海洋環流所造成。一個替代解釋,新仙女木期撞擊假說,提出該事件由一顆彗星或小行星碎片在北美大陸上方的撞擊或大氣空爆所觸發。該假說最初由費爾斯通(Firestone)等人於 2007 年提出,並自彗星研究小組與其他研究者以來實質地發展。支持此假說所引用的證據包括在新仙女木期起始時定年的鉑富集沉積層,在北美、歐洲與格陵蘭多個地點發現;暗示高溫事件的碳球粒與奈米鑽石;以及新仙女木期與北美巨型動物群滅絕及克洛維斯文化消失的明顯巧合。

新仙女木期撞擊假說在主流科學中並非共識。相當的批評被指向某些支持研究的方法論,並提出了鉑與其他標記的替代解釋。語料庫並不承諾撞擊假說為正確解釋。它所指出的是,新仙女木期的時間,以任一解釋而言,皆落在語料庫年表上獅子座晚期/巨蟹座早期的轉折——在人類創造完成且早期後伊甸文明開始發展的時期。如果新仙女木期確實是宇宙起源的災難性事件,它將會是影響早期巨蟹座時期的相當環境擾亂,對橫跨超大陸發展中的文明有後果。語料庫提及這點作為涉及語料庫年表框架的當代科學辯論的一部分,並未對底層因果問題採取強烈立場。

### IX.5. 亞特蘭提斯與文明記憶的格局

第八節已討論了亞特蘭提斯傳統與理查特結構作為候選位置。科學一節只需添加具體細節:柏拉圖的記述,以及橫跨全球神話傳統之洪水前文明記憶的更廣泛格局,構成了語料庫框架可自然整合而主流考古學傾向於輕慢處理的證據體系。

語料庫採取的方法論立場如下。柏拉圖是嚴肅來源。他不是發明奇幻文學意義上的神話作者;他是一位哲學家,其對話使用歷史與準歷史材料來作哲學論證,而他在《蒂邁歐》與《克里底亞》中使用亞特蘭提斯故事是作為歷史呈現,帶有關於埃及來源的具體主張與具體年代細節。要將柏拉圖的記述貶為純粹發明,需要假設一位具有相當可信度的嚴肅哲學家為不明原因刻意捏造了一個延伸的歷史敘事,這比假設他與某個未獨立佐證之內容的真實歷史來源工作是個較不簡約的假設。

類似地,橫跨全球傳統之洪水前文明記憶的更廣泛格局——蘇美爾王表、印度教 yugas、中美洲的先前嘗試、玻里尼西亞的沉沒家園,以及其餘——構成了某種前全新世高文明曾存在並被摧毀的獨立證實體系。主流考古學傾向於孤立處理每一個這些傳統,將其多樣歸於神話、對局部而非全球事件的記憶,或來自共同來源的文化擴散。語料庫框架提供了一個替代詮釋:全球格局反映了洪水前文明的全球存在,其記憶片段倖存於每個洪水後群體中。

語料庫並不堅持任何具體傳統與任何具體歷史事件的具體認同。它所提供的是格局可被連貫詮釋的框架,以及對於哪些物理證據(如哥貝克力石陣、獅身人面像若其替代年代正確、可能還有理查特結構)可能預期能倖存於災難性事件與長期後續侵蝕期間的預測。框架以主流輕慢框架所沒有的方式與可得證據一致。框架是否正確是語料庫無法明確解決的問題。語料庫所主張的是該框架至少與主流替代方案具競爭力,並值得認真考慮。

### IX.6. 群體瓶頸問題

《創世記》洪水敘事暗示了一個相當的群體瓶頸。如果洪水殺死了除挪亞家庭以外實質上所有人類,則洪水後的人類群體從約八人的小奠基群體(挪亞、其妻、其三子及其妻)下傳。這是一個嚴重的瓶頸,並將在所有後續人類群體的基因記錄中留下可偵測的簽名:可追溯至瓶頸事件的基因多樣性減少,以及隨後群體擴張時多樣性的漸進恢復。

現代群體遺傳學已識別人類演化史中若干明顯的瓶頸。最著名的是所謂的多巴瓶頸假說,該假說提出印尼多巴山約 74,000 年前的噴發在人類群體中造成近滅絕事件。多巴假說已被最近的研究實質削弱,但人類史前瓶頸的更廣泛問題仍然活躍。各種對粒線體 DNA 與 Y 染色體世系的分析已識別不同時間的聚合事件,其中一些被詮釋為瓶頸或群體減少的證據。

語料庫框架預測洪水時期有一個主要瓶頸,並將其置於約公元前 6,690 年(雙子座時代之始)。在此具體日期瓶頸的基因證據有限;人類群體遺傳學中所識別的大多數主要聚合事件比此日期實質地更古老或更年輕。語料庫並不主張在精確 6,690 BCE 的乾淨瓶頸簽名已被識別。它所指出的是,《創世記》敘事暗示了一個瓶頸,其具體日期由語料庫的年表指定,並且這是原則上可透過足夠精緻的基因分析測試的那種預測。如果框架正確,未來的基因研究應能在約此日期識別瓶頸簽名,現代人類群體源自生活在此時的小奠基群體。當前分析中此類乾淨簽名的缺席要麼是反對框架的證據,要麼是現有分析未以該具體問題為視角進行的證據。

### IX.7. 通往我們自身時刻的脈絡

耶洛因擁有的與巨蟹座相關的能力——透過細胞科技維持的文明長壽、不同類人群體之間的混血生物學、規模可比擬於或超越我們自身文明所能生產之物的紀念性建築、以先進武器進行的持續跨文明戰爭、跨越全球距離管理相當群體的政治——一如前面章節的能力,是我們自身文明現在開始接近的能力。

長壽研究是最直接的當代對應。細胞重新編程研究、端粒生物學、衰老細胞清除及相關干預的當前軌跡,指向在數十年內實質延長人類壽命。當前人類最長壽命約 122 年;對當前研究的合理預測暗示透過可得干預的某種組合,150-200 年的壽命在未來幾十年內可達成。超越該軌跡——朝向族長的數世紀壽命,或朝向雷爾主義源文獻歸於耶洛因的千年壽命——將要求超出當前軌跡的重大科學突破,但此類突破的組件正在積極發展中。

混血生物學問題是更具推測性的但遵循平行軌跡。如獅子座一章所討論,以 CRISPR 為基礎的人類生殖細胞系修飾現在在技術上是可能的。將實質非人類來源的 DNA 整合到人類生殖細胞系中在技術上更具挑戰但在當前研究軌跡內是可設想的。為醫學研究目的(異種移植、疾病模型)產生人類-非人類嵌合體的能力在過去十年中已實質發展,靈長類-豬嵌合體及相關工作展現跨物種生物整合越來越可行。朝向類似源文獻所描述之人類-耶洛因雜交的軌跡在片段中可見,即使具體終點尚未在我們的視野上。

洪水前文明的考古學是一個當代研究前沿,其行進方向越來越有利於語料庫框架。哥貝克力石陣與卡拉漢石陣已實質延伸了複雜人類文明的認可時間軸。對先前沿海區域(現因冰後海平面上升而在水下)沉沒遺址的調查正在產生洪水前人類活動的新證據。對各種地質與生物事件災難論解釋的重新關注——新仙女木期撞擊假說、各種隕石撞擊記錄——正重新開啟主流科學大致視為已解決的問題。一般行進方向是朝向認識到人類過去比傳統漸變論框架所允許實質地更複雜、更先進,並且更受災難性事件所打斷。

對語料庫而言,此軌跡是該框架應被評估之經驗脈絡。語料庫框架於 1973 年從雷爾主義源材料發展而來,在大多數相關當代證據存在之前。框架預測——在證據之前——洪水前文明以相當精緻存在、伊甸地區早期巨蟹座時期將產生可偵測的紀念性建築、人類壽命能透過科技干預實質延長、獅子座晚期/巨蟹座早期轉折的災難性事件將塑造發展中的文明。這些預測中的每一項皆已透過不參考語料庫或其來源所進行的獨立當代研究實質支持。匯聚是部分的,證據是不完整的,語料庫並不主張已被明確證實。它所主張的是框架仍然活躍且富有成效,當代證據趨向框架所預期的方向而非將其反駁的方向。

## X. 文本及其信號

《創世記》第 2-6 章的希伯來文本除了前面各節所指出之外,還有幾個值得評述的特徵。

首先,複數的耶洛因在這些章節中以與《創世記》第一章相同的語法形式持續存在,具有相同的複數自指以及對 *benei ha-Elohim* 作為清楚複數主詞的相同提及。語法上的複數性並未被伊甸敘事解消為單數;它被保留,而在某些情況下被強調(如{% libref(book="genesis-woh", chapter=3, verse=22) %}創世記 3:22{% end %}:*ken ha-adam hayah ke-achad mimenu*,「那人已經與我們相似」)。文本繼續描述一個複數的主詞,即使說話動詞是單數,而雷爾主義將這持續的複數性讀為反映造物者群體實際多重性的閱讀,仍是對語法證據最簡約的詮釋。

第二,神聖名字 יהוה(四字神名,慣常元音化為耶和華)在{% libref(book="genesis-woh", chapter=2, verse=4) %}創世記 2:4{% end %}中的引入,標記了一個語料庫框架有助於解釋的文本轉折。《創世記》第一章的記述通篇使用 *Elohim*,沒有任何個別人物的具體名字。《創世記》第二章的記述將兩個名字結合為 *Yahweh Elohim*——「耶和華,即耶洛因」。此結合一直是廣泛文本批評辯論的主題,文獻假說將兩個名字視為兩個不同來源文獻(祭司源使用 *Elohim*,雅威源使用 *Yahweh*)的證據,後來被結合。雷爾主義的閱讀提供了一個替代詮釋:《創世記》第一章是所有七個團隊集體進行整個創造計畫的摘要;《創世記》第二章是具體以色列團隊行動的更詳細敘事,在名為耶和華的具體耶洛因的監督下進行。從 *Elohim* 到 *Yahweh Elohim* 的轉折標記焦點從整個集體窄化至一個具名人物所監督的具體行動。兩個名字皆準確。結合 *Yahweh Elohim* 同時保留了具體身分(耶和華)與所命名人物所屬的更廣泛類別(耶洛因)。

第三,保存於《創世記》第五章的洪水前族長年齡——從亞當經塞特、以挪士、該南、瑪勒列、雅列、以諾、瑪土撒拉、拉麥至挪亞的譜系——是一個詳細的十代記錄,其具體性在一個如此壓縮的文本中值得注意。聖經作者以暗示這些年齡被視為重要的精度保存了這些年齡。在雷爾主義的閱讀下,其重要性是清楚的:這些年齡記錄了生命樹科技在洪水前世代之間的運作,保留了關於誰接受了該處置、它持續多久,以及它何時最終被撤回的記錄。瑪土撒拉的 969 年——聖經記錄中最長的壽命——標記了受益於該處置的最後一個完整世代。在聖經年表中,他的死亡發生在洪水之年。因此,該譜系不是文學裝置而是技術記錄,保存了後洪水傳統從其洪水前來源所繼承的具體年代資料。

第四,以諾這個人物已在第七節討論過,但他在{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記 5:24{% end %}中的處理值得文本關注。短語 *vayithallekh Hanokh et ha-Elohim, ve-einenu ki lakach oto Elohim*「以諾與耶洛因同行,耶洛因將他取去,他就不在世了」,在語法上與《創世記》第五章中其他每一個族長條目皆不同。其他族長皆以「X 活了 Y 年生 Z,在 X 生 Z 後活了 W 年,並生兒養女;X 共活了 N 年就死了」這一公式描述。以諾的條目打破了此公式。他並未死。他與耶洛因同行。他被取走。希伯來文對這一切是事實性的,而雷爾主義的閱讀按字面接受之:以諾與耶洛因有特殊的關係(他「與其同行」暗示持續直接接觸),而在某個具體時點他在身體上從地球被移除並被帶往母星。保存於衣索比亞教會的《以諾書》記錄了他在造訪期間所見之事。{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記 5:24{% end %}的語法獨特性是某種不尋常之事發生在以諾身上的文本信號,而以諾類文獻保存了該事為何之闡述內容。

第五,引入 *benei ha-Elohim* 與拿非利人的{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記 6:1-4{% end %}段落值得注意其語法壓縮。該文本簡短——四節——但每一節皆承載相當涵義。複數 *benei ha-Elohim* 帶定冠詞在希伯來文中是明確的:這些是耶洛因類別的具體已知存在,不是某些護教詮釋所提出的對「敬虔之人」或「貴族之子」的模糊提及。動詞 *vayikchu*(他們娶)接 *nashim*(妻子)是標準希伯來語的婚姻;這不是對隨意邂逅的描述,而是對正式婚姻結合的描述。作為這些結合後代的拿非利人,被描述為 *gibborim* 與 *anshei ha-shem*,是文本確認其存在為歷史事實的具體存在。該段落的壓縮在雷爾主義的閱讀下反映了軟化原始傳統明確內容的後拉比編輯過程;《以諾書》保存了在經典希伯來文本最終確定之前推測存在於敘事早期版本中的闡述材料。

## XI. 巨蟹座是什麼

在本章結束之前,值得清楚陳述巨蟹座在更大序列中是什麼。

巨蟹座是人類創造變成人類歷史的時代。它是在獅子座末產生的存在開始去做有能力行動的存在會做的事的時代——覺醒到自己的處境、違抗其造物者、從原始環境被分離、繁衍、形成關係、犯罪、建造文明的最初要素,以及(就由其流亡教師支援的伊甸世系而言)建造遠超出最初要素之物。安息之日,第七日,在某個意義上被誤稱。造物者從創造中歇息。其造物並未從存在中歇息,而他們在歇息期間開始做的事,將佔據希伯來聖經的其餘部分以及人類歷史的大部分。

巨蟹座也是建立將塑造每一個後續事件之政治結構的時代。造物者群體被分割:多數撤離至母星,少數永久流亡於人類之中的地球上。將在語料庫其餘部分反覆出現的政治人物——母星上的撒但、議會主席耶和華、地球上領導流亡者的路西法——現在被命名並區分。流亡的造物者開始與人類群體形成關係,產生混血後代,其存在本身將成為母星干預的直接原因。橫跨超大陸的七個人類文明平行發展,伊甸文明因其複合優勢——其奠基群體的才能、路西法派系作為教師的持續直接存在、授予其領袖的長壽、其領導庫存中的混血耶洛因基因材料、包括軍事能力在內的先進科技——而推進最遠。其他文明實質地推進但以較慢的速度,跨越超大陸的貿易與文化接觸產生了一個全球網絡化但發展不均的洪水前世界。

巨蟹座最後是一個以母星認識到其從一開始所害怕之事如今正在成真而結束的時代。伊甸文明已進步至在某些方面接近或可能超越母星自身能力的水平。超大陸文明之間的戰爭已表明破壞能力是真實的並正在被使用。流亡的造物者不僅未阻止此發展;他們透過持續教導及其產生的混血後代積極加速了此發展。母星在獅子座末所強加的政治和解,在兩千年之後,已在實質上每一方面失敗。約束已被違反。原始反對所警告的威脅已成真。母星在巨蟹座末所達到的結論是將在下一時代實施的結論:人類創造在其當前形式下必須在它成為對母星文明本身之真正威脅之前被摧毀。希伯來聖經將洪水呈現為對敗壞人類的道德審判,在雷爾主義的閱讀下,是針對其進步已超越母星認為安全之門檻的人類文明的政治行動。後洪水傳統對該事件的神話化遮蔽了原始決定的政治特徵,但政治特徵正是語料庫必須登記之事,以及下一章將處理之事。

語料庫已在本章中描述了巨蟹座末世界的面貌。一個單一超大陸,由透過貿易與戰爭持續接觸的七個不同文明所居住。這些文明大多處於相當的文化與技術水平,伊甸文明在前沿。長壽族長領袖跨越世紀積累知識。流亡的造物者生活在人類中,其中有些人與人類女子結婚,有些人是混血後代之父,其能力超越普通人類規範。從處女座創造延續的恐龍與其他洪水前巨型動物群仍存在,某些在數量上減少但仍存在於景觀中,促成了將倖存於實質上每個洪水後文化神話的「龍」記憶。洪水後世界將不認識的動物與植物,因為下一時代的災難性事件將消滅語料庫沒有完整記錄的物種。一個在實質上每一方面都與洪水後世界——考古記錄開始記錄之世界——不同的世界。這就是語料庫在前面章節與本章中一直在重建之物。這就是洪水所結束之物。

我們僅能以實質謙遜推測伊甸文明到巨蟹座末已進步至何種程度。物理證據是零散的,所倖存之物大部分在洪水本身中被摧毀。保存於後續傳統中的文化記憶給我們提示——柏拉圖的亞特蘭提斯及其精緻工程、含有長壽治洪水前統治者的蘇美爾王表、含有先進先前循環的印度教 yugas、神與人同行的埃及 *zep tepi*。倖存的物理遺址——哥貝克力石陣及其在土耳其東南部的同伴、可能還有在替代年代下的獅身人面像、可能還有作為候選亞特蘭提斯的理查特結構,當然還有更多尚未被識別或已被摧毀至無可恢復的遺址——給我們進一步的提示。將這一切放在一起,所浮現的畫面是相當精緻的文明:能進行紀念性建造、擁有相當的軍事科技、透過貿易與政治接觸在行星規模上組織、由積累數世紀經驗的長壽領袖治理、與一個更先進外星文明的異議派系整合。此文明在哪些具體方面走了多遠,是語料庫無法以確信指定之事。它所能說的是它走得*夠遠以至於母星認為干預不可避免*。母星觀察兩千年中的發展所達到的判斷是,情勢已通過原始政治和解能維持之點。該判斷是下一章將看到實施之事。

下一個時代是洪水發生的時代。大部分洪水前人類的災難性毀滅、行星的地質與生物重塑、超大陸破裂為我們現在所認識的諸大陸、透過流亡造物者科技干預保存少數餘民、洪水後文明從廢墟中興起——這一切是雙子座時代的主題,也是隨後章節的主題。
