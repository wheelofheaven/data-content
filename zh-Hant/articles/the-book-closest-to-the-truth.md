在古代的最後幾個世紀某個時候——確切年代正是本文要探討的問題之一：一位希伯來作者寫下了神的身高。
他是用單位寫下的。這個數字由一位天使傳達，像宣誓證詞一樣立誓，並以*帕拉桑*{{ footnote(id="1") }}計價，那是旅人的雙腳當作一小時步程來丈量的波斯里格：從榮耀寶座向上，一百一十八無數；從榮耀寶座向下，一百一十八無數；其身高全長，二百三十六個「無數之千」。文本接著逐一量度腳掌、
腳踝、脖頸、頭頂，各自配上其秘名，然後，
彷彿預料到讀者會伸手去拿石板算數，它為自己的術語下了定義：一
帕拉桑是三哩，一哩是一萬肘，一肘是三拃——
而一拃即是整個世界的全幅。

大約十九個世紀之後，1973 年 12 月，一位二十七歲的
法國賽車記者克勞德·沃里隆——
{% wiki(slug="rael") %}雷爾{% end %}——聲稱有一位名為
{% wiki(slug="yahweh") %}耶和華{% end %}的存在，在奧弗涅一座火山口內，花了六天向他講解希伯來
聖經。在他次年出版的書中，這位存在在一個傳統之上停留，凌駕
於其餘一切之上：

{% library(book="the-book-which-tells-the-truth", chapter=5, verse=51) %}
真理的痕跡——不僅存在於聖經與福音書中；在幾乎所有的宗教裡都能找到見證。卡巴拉尤其
是見證最豐富的書之一，只是你要弄到一本並不容易。倘若有一天你能找到一本，你便能
在其中看到大量指向我們的暗示。尤其是《雅歌》（第五章）中有一段對創造者所居行星、
以及它與地球之間距離的描述。那裡說，「創造者的身高」是 236,000「帕拉桑」，而「他
腳跟的高度」是三千萬「帕拉桑」。
{% end %}

三段之後，便是那句給了本文標題的、平鋪直敘的最高級斷言：*「卡巴拉是最接近真理的書」*
（{% libref(book="the-book-which-tells-the-truth", chapter=5, verse=54) %}TBWTT 5:54{% end %}）。

雷爾正典中的大多數主張都是解讀——{% wiki(slug="genesis") %}創世記{% end %}背後的一座實驗室、
{% wiki(slug="great-flood") %}大洪水{% end %}背後的一次重置。而這一項卻是一則引文。它
點名了一部典籍、一個章次、兩個數字與一個單位，而每一項都可以
查證。誠實地做這番查證要花掉本文的全部篇幅，
因為這條線索穿過猶太神秘主義中最詭異的文本、一場
關於其年代的三方學術之爭、長達一千年的、關於其內容的神學
尷尬，以及——在最末——一本帶著露餡翻譯錯誤的廉價法文
平裝書。這一目的地值得
先行說明：**耶和華所引的那段文字確實存在。** 它名為
*希烏爾·科瑪*（希伯來文 שִׁעוּר קוֹמָה，*šiʿur qômāh*，「身量的量度」），
而 Wheel of Heaven 計畫已將它完整譯出：
[*希烏爾·科瑪*——Wheel of Heaven 譯本](/library/shiur-qomah-woh/)。
這些數字*意味著*什麼，正是主流與正典分道揚鑣之處，
本文將在行至該處時標明那條界線。

## 卡巴拉是什麼——又不是什麼

正典自身的措辭裡藏著一個謎題，而它始於那個詞。
{% wiki(slug="kabbalah") %}卡巴拉{% end %}並不是一本書。希伯來文
קַבָּלָה（*qabbalah*）意為「領受」、「所領受之物」，源自
詞根 ק־ב־ל（*q-b-l*），「去領受」——那是這門語言中一個尋常而實用的動詞
（在現代以色列，*kabbalah* 是一張商店收據；*kabbalat
Shabbat* 是「領受」安息日的週五晚間禮儀）。
一旦用於一套教導，這個詞就提出了一個明確而大膽的
主張：這套知識從未被撰寫、推演或發現。它是被
*親手交遞*的——口耳相傳，師徒相授，在約束之下，歷經
數個世紀。同一個動詞開啟了《米示拿》那句著名的傳承鏈：
*「摩西在西奈領受〔kibbel〕了妥拉，並把它傳給約書亞，
約書亞傳給眾長老，眾長老傳給眾先知，眾先知又傳給
大會堂的眾人」*（《先賢篇》1:1）。這一傳統的
自我描述——שַׁלְשֶׁלֶת הַקַּבָּלָה，*shalshelet ha-Kabbalah*，「傳承
之鏈」——一路上溯，穿過
{% wiki(slug="moses") %}摩西{% end %}並繼續往前。亞歷山大·薩夫蘭
（1910–2006），戰爭年間任羅馬尼亞首席拉比、其後半個世紀任
日內瓦首席拉比，用一句話道出了這一傳統主張：*「卡巴拉在
古老程度上超越西奈的啟示；它可追溯至史前時代。摩西只是把它
引入了以色列的歷史。」* 拉比猶太教把這份審慎與其
內容一同制度化了。早期拉比承認兩門秘傳課程——מַעֲשֵׂה
בְּרֵאשִׁית（*ma'aseh bereshit*），「創造之工」，即
創世記首章背後的奧秘；以及 מַעֲשֵׂה מֶרְכָּבָה（*ma'aseh
merkavah*），「戰車之工」，{{ footnote(id="3") }}即
以西結寶座異象背後的奧秘——而《米示拿》禁止哪怕只向一名學生講解
第二門，除非那人有智慧、能憑
己力領悟（哈基加篇 2:1）。

當這領受而來的教導終於凝結為書卷時，它是遲來且
分階段成形的。סֵפֶר יְצִירָה（*Sefer Yetzirah*，「形成之書」），
一部僅數千字的宇宙論小冊，成書於公元二至六世紀之間某個時候，
以「三十二條智慧之路」構築宇宙——即希伯來
字母的二十二個字母，加上它稱為 סְפִירוֹת（*sefirot*）的十個原初數字——
且其中並無對神的任何量度；現代學術甚至把它
堅稱神的「量度為十」讀作對那一
量度傳統的一記靜默斥責。《光輝之書》（*Sefer ha-Bahir*）
於十二世紀末在普羅旺斯浮現，把
*sefirot* 轉化為活生生的神聖力量。זֹהַר（*Zohar*，「輝煌
之書」），即「卡巴拉」一詞通常喚起的那部
浩瀚的亞蘭文傑作，成書於 1280 年代的卡斯提爾——被呈現為
一位二世紀賢者的教導，儘管現代學術主要把它
歸於萊昂的摩西——它闡述了那套成熟的體系：
不可知的無限神性，אֵין סוֹף（*Ein Sof*，「無盡」），
透過排列為{% wiki(slug="tree-of-life") %}生命之樹{% end %}的十個 *sefirot*
向世界展開。以撒·盧里亞在薩費德的圈子
於十六世紀圍繞宇宙災變與修復，再度重建了這套體系。這便是「卡巴拉」一詞
通常所指的那座藏書庫——而其中沒有任何一部是耶和華引文的來源。在
它的全部篇幅中，《光輝之書》只給了那古老的量度傳統
匆匆一瞥（Zohar II 175b–176a）。倘若「卡巴拉」指的是那些著名的
書卷，正典的引文便會落空。

它並未落空，因為還有一個更古老的地層——而描述它
需要一個多數讀者未曾遇過的詞。

## 卡巴拉之前：宮殿文獻

大約在公元三至八世紀之間——年代之爭將在
下文另闢一節——匿名的猶太神秘家寫下了一批
著作，學界稱之為 **Hekhalot 文獻**，源自
הֵיכָלוֹת（*hekhalot*），「宮殿」或「殿堂」。{{ footnote(id="2") }} 這些
文本描述入門者穿過七重同心的
天界宮殿——每一重都有持械的天使守門者——直抵
神的寶座之廳——即 מֶרְכָּבָה（*merkavah*），先知
以西結在迦巴魯河邊所見的那帶輪的戰車寶座。以一種
文本從不解釋的悖謬，行者把這一登天稱作
一次*下降*：他們是 יוֹרְדֵי מֶרְכָּבָה（*yordei merkavah*），「下降
至戰車者」。這位旅人需要裝備——在每道
門前吟誦的頌歌、守門者的名字、要出示的印記——而這些文獻
在很大程度上就是那套裝備：一頁又一頁無法讀出聲的
天使之名、咒誓與禮儀。與登天文本並列的，還有
把這趟旅程反向運行的儀式，把一位天使*召降*至
人間，以賜下這一文化珍視勝過一切的那份禮物：
對妥拉全備、永久、永不遺忘的知識。主要的著作
帶著這樣的名字：*Hekhalot Rabbati*（「大宮殿」）、*Hekhalot
Zutarti*（「小宮殿」）、*Merkavah Rabbah*（「大戰車」），
以及*《以諾三書》*——它們全都託名於那些偉大的
二世紀拉比{{ footnote(id="4") }}以實瑪利與阿基瓦，這二人在這一傳統中充當
其僅有的兩位獲准旅人：阿基瓦，在《塔木德》那則著名的
傳說裡，是進入*pardes*（果園，樂園）並全身而退的
四位賢者中唯一的一位。

關於這批文獻，有兩件事對我們的線索至關重要。第一是它的
地位。革舜·舒勒姆（1897–1982）——這位生於柏林的史學家於 1923 年
移居耶路撒冷，出任有史以來第一個專門獻給
猶太神秘主義的大學講座教席，並圍繞他自己的著述目錄
建立了整個現代學術領域——畢生堅稱這批材料絕非
異端的邊緣：「這些思辨的承載者，處於坦拿與塔木德時代
拉比猶太教的正中心……我們此處所討論的
靈知，是一種嚴格正統的猶太靈知。」第二則是
它的物理狀態。Hekhalot 文本存留於中世紀的抄本中，
這些抄本拒絕安定為固定的書卷；同樣的材料以
不同的編排、不同的標題，從一份抄本流傳到
下一份。嵌在這一流動的典籍之中、被它引用、藉它
流傳的，正是我們所追尋的那段量度文本。而正典那個
歸類上的口誤——把一個傳統稱作你或許可以「弄到一本」的「書」——結果
是一條線索而非一個錯誤，因為那段量度文本*確實*
成了一本你可以弄到的書：從 1701 年起，它就被收入
סֵפֶר רָזִיאֵל הַמַּלְאָךְ（*Sefer Raziel
ha-Malakh*，「天使拉結爾之書」）的印本之中，那是一部魔法書，其自身的
框架傳說稱天使拉結爾把它交給了始祖亞當。Sefaria
圖書館今日所印行的那個*希烏爾·科瑪*抄本系，正是
從那裡複製而來——即 1701 年阿姆斯特丹版的第 37b 頁。

## 身量的量度

*希烏爾·科瑪*把自己宣告為證詞——一份經天使公證的
供狀。拉比以實瑪利轉述天使
梅塔特隆{{ footnote(id="8") }}的誓言：

{% library(book="shiur-qomah-woh", chapter=1, verse=2, interlinear=true) %}
拉比以實瑪利說：見證大君梅塔特隆對我說：我
奉以色列的神耶洛因 YHWH——那活著並長存的神，至高至大、
崇高且被高舉、拯救我們、救贖我們、並救我脫離一切惡事——以此見證作證。
{% end %}

這文本的名字道出了它的營生。שִׁעוּר（*shiʿur*）是「量度」、
「數量」、「份量」——與現代以色列人用來指一堂
課的是同一個詞；קוֹמָה（*qomah*）是「身量」或「高度」——在現代希伯來語中，指樓房的
一「層」——而在那個時代的亞蘭語中，它可以僅僅指
「身體」。*Shiʿur qomah*：身量的量度；或者，如舒勒姆
所偏好聽到的，身體的量度。這文本所回答的問題就
問在其內，出自拉比以實瑪利自己的口：

{% library(book="shiur-qomah-woh", chapter=1, verse=9, interlinear=true) %}
拉比以實瑪利說：那向一切受造物隱藏的至聖者（願祂受頌讚）的身量之量度，
究竟有多大。
{% end %}

而答案在誓言之下到來：

{% library(book="shiur-qomah-woh", chapter=1, verse=3, interlinear=true) %}
從祂榮耀寶座向上，一百一十八無數；從祂榮耀
寶座向下，一百一十八無數。祂的身高是二百三十六個無數之千
帕拉桑。
{% end %}

這一對稱——向上一百一十八、向下一百一十八——是刻意的，那個成語也是；
要領會這兩者，須先明白希伯來語處理數字的兩個事實。第一：
這類文獻的希伯來語不寫任何數碼。數量是用
詞語拼寫出來的，由具名的單位塊堆疊而成——即 אֶלֶף（*elef*），一
千，以及 רִבּוֹא（*ribbo*），一萬的
無數。{{ footnote(id="5") }}「祂的身高是二百三十六個無數之千
帕拉桑」——רוּמֵיהּ מָאתַיִם וּשְׁלֹשִׁים וְשִׁשָּׁה רִבְבוֹת אֲלָפִים
פַּרְסָאוֹת——因此展開為 236 × 10,000 × 1,000：二十三億
六千萬帕拉桑。第二：每個希伯來字母都兼作一個
數字（א = 1，ב = 2，如此直到 ת = 400），這意味著每個希伯來
詞也都是一個總和——這門解經之術稱為
gematria。{{ footnote(id="7") }} 那個 236 便是由此而來。詩篇
147:5 讚美神為 גָּדוֹל אֲדוֹנֵינוּ וְרַב כֹּחַ（*gadol adoneinu
ve-rav koaḥ*），「我們的主為大，大有能力」——而
*ve-rav koaḥ* 諸字母加總為 ו 6 + ר 200 + ב 2 + כ 20 + ח 8 = 236。
神秘家把這節經文讀成彷彿它說的是：*我們的主為大——祂的量度
是 236*。從那顆種子起，文本逐一肢體、
由上而下、由內而外地展開，其次序猶如一名殿中侍者為一尊
塑像沐浴：腳掌、腳踝、
小腿、大腿、腰、脖頸、頭骨、鬍鬚、雙眼、雙肩、雙臂、手指、
腳趾——各有一個尺寸，也各有一個無法讀出的名字。腳掌
最先出場：

{% library(book="shiur-qomah-woh", chapter=1, verse=10, interlinear=true) %}
祂腳掌充滿整個世界，如經上所說：「耶和華如此說：天是我的
座位，地是我的腳凳。」
{% end %}

在*拉結爾之書*抄本系裡{{ footnote(id="6") }}——即那個印行的、
可弄到手的抄本系——這雙腳掌的高度是 שְׁלֹשׁ רִבְבוֹת אֲלָפִים
פַּרְסָאוֹת（*shalosh rivevot alafim parsa'ot*）：三個無數之
千，三千萬帕拉桑。而在 Sefer Haqqomah 抄本系裡——
由馬丁·薩繆爾·科恩從牛津 MS 1791 譯出，這位美國
拉比學者的猶太神學院博士論文成長為
該文本的標準現代研究，並在 1985 年成為其首部校勘本——
同一行卻寫作：「祂腳掌的高度是 30,000,000 帕拉桑；其
名為 PARMESEH。」耶和華的第二個數字就在
抄本裡。第一個數字則更奇特、也更耐人尋味。校勘
本說二百三十六個無數之千——二十三億六千萬。耶和華說 236,000。但
傳統本身早已帶著那個被截短的形式：十世紀的
卡拉派{{ footnote(id="11") }}論戰者 al-Qirqisānī，為了嘲笑而
引用該文本時，記述道：「祂的身高，從腳掌
直到其全身，共二十三萬六千帕拉桑。」抄手與滿懷敵意的
見證者早在奧弗涅那人之前一千年，就一直在
省去那些「無數」。這一截短，將在本文末尾的溯源
審查中帶著後果重新登場。

那個單位本身承載著一段歷史，而且在希伯來語中還藏著一個雙關。帕拉桑——
פַּרְסָה（*parsah*），複數 פַּרְסָאוֹת（*parsa'ot*）——是一個源自
波斯語 *farsang* 的外來詞，即阿契美尼德御道上的那個里格，
它以《塔木德》日常的長程單位之名進入了拉比希伯來語。
但同樣這三個詞根字母 פ־ר־ס，也拼出希伯來語裡的蹄
或腳掌——因此諸抄本系可以說（不論這是一次同音的
巧合，還是抄手一個不動聲色的玩笑）：祂雙腳的*parsot*
量得如許多的*parsa'ot*。而文本為自己做了總計，其
求和乾脆全然放棄了精確：

{% library(book="shiur-qomah-woh", chapter=1, verse=39, interlinear=true) %}
如此看來，全部的量度總計，其身高是眾多之眾多無數的無數之千
帕拉桑，其身寬則是千千個無數帕拉桑。
{% end %}

還有兩項文本特徵，對接下來的一切都很重要。第一
是它的換算表。有人——文本把這安排成梅塔特隆
搶在拉比以實瑪利的算術之前——為那個單位下了定義：

{% library(book="shiur-qomah-woh", chapter=1, verse=31, interlinear=true) %}
你要說明帕拉桑的計算——它們有多大。每一
帕拉桑是三哩；每一哩是一萬肘；每一肘是
⟨…⟩ 拃，按祂的拃計；而祂的一拃即是整個世界的
全幅。
{% end %}

這些是神聖的帕拉桑。每一個都由拃構成，而每一拃都
有一個宇宙那麼寬。第二項特徵，是知曉這一切所繫的那個
應許——它是這文本在整個後世傳統中被引用最多的幾行，
表面看來卻也最為古怪：

{% library(book="shiur-qomah-woh", chapter=1, verse=41, interlinear=true) %}
拉比以實瑪利說：關於此事，我與拉比阿基瓦作保——
凡知曉我們創造者這一量度、並知曉至聖者（願祂受頌讚）之
稱頌者，必得保證他是來世之子；惟一的
條件是他每一天都把它當作《米示拿》來誦讀。
{% end %}

兩位賢者以嚴格的法律意義上的保人身分，為凡是
*每日誦讀這些量度*之人的得救作保。無論這文本還可能是別的什麼，
它都是一種操練。

## 雅歌，第五章

正典為何說那描述位於「《雅歌》（第五章）」？乍
看之下，這又是一個歸類上的口誤——《雅歌》是一部聖經
書卷，八章情詩，其中任何地方都不出現帕拉桑。
它的第五章包含新婦對其良人身體的讚美：

{% library(book="song-of-solomon", chapter=5, verse=10, verse_end=15) %}
我的良人白而且紅，超乎萬人之上。他的頭
像至精的金……他的兩手好像金管，鑲嵌水蒼玉；他的
身體如同雕刻的象牙，周圍鑲嵌藍寶石。他的腿好像白玉石柱，
安在精金座上。
{% end %}

這一關聯比乍看之下更深。二世紀的傳統把
《雅歌》的良人讀作神自己，是全以色列在紅海與
西奈所見的那一位。而*希烏爾·科瑪*自身逐字引用了雅歌
5:10–16——而且在 Merkavah Rabbah 抄本系裡，這些經節甚至
被打亂聖經次序重新排列，以配合剛剛量度過的肢體
順序。這一關聯是文本內在的，而非外加於它的。基於
這一點，舒勒姆——連同掃羅·利伯曼（1898–1983），
這位猶太神學院教授被同儕視為
本世紀最偉大的塔木德學者——主張*希烏爾·科瑪*一開始是
關於雅歌第五章的秘傳米大示；利伯曼徑直稱它為對雅歌
5:10–16 的「一篇早期米大示」。舒勒姆又補上一個精彩的間接論證：奧利根（約 185–253），
其時代最博學的基督教學者，約於公元 240 年在該撒利亞
寫作時記述道：猶太人把
四段經文保留「到最末」才教給學生——即
創世記的開頭、以西結書的開頭與結尾，以及
《雅歌》。前三者是《米示拿》已知的秘傳課目。而
《雅歌》出現在這份受限清單上唯有在下述情況下才說得通，舒勒姆論證道：
即到奧利根的時代，它早已是那量度
教義的經文錨點。

製作了校勘本的科恩強力反駁：*希烏爾·科瑪*的神「既無金頭，
亦無象牙的腰、大理石的小腿」；《雅歌》是由頭往下
描述其良人，而*希烏爾·科瑪*是由腳往上量度；並且《雅歌》的經節在
文本中充當禮儀性的框架，像圍繞著它的那些詩篇一樣被引用，
從不作為任何單一量度的證明經文。依科恩的讀法，《雅歌》
賦予這文本正當性與氛圍；它並未生成這文本。這場爭論
仍在進行、懸而未決，而本計畫是保存它而非了結
它。對我們的目的而言，更重要也更確鑿的是這一點：從二
世紀起，傳統本身就把對神聖身體的異象繫於
《雅歌》第五章——以致當舒勒姆要為現代讀者總結這一
教義時，他用一句融合了兩者的話來做：這些殘篇描述創造者的身體
*「與《雅歌》第五章對良人身體的描述
極為相似……創造者的身高是 236,000 帕拉桑——
按另一個傳統，僅祂腳掌的高度就是三千萬
帕拉桑。」* 舒勒姆這一句話，結果正是本文以之作結的那個
溯源問題的樞紐。

## 這文本有多古老？

年代問題值得單闢一節，因為正典那句順帶之語
——*「你要弄到一本並不容易」*——恰好是這場學術之爭的
每一方都會連署背書的唯一主張。

舒勒姆把這一教義定於公元二世紀。他的證據是一
串早期的暗影：靈知派的馬爾庫斯（瓦倫提努斯的門徒，活躍
於約公元 170 年），舒勒姆把他那套帶著以字母命名之肢體的神聖
「真理之體」體系讀作對猶太源頭的借用；查士丁·馬爾提爾，
約於公元 160 年——作為規範性的猶太教導，而非異端——記述道：
神有形狀與器官；斯拉夫文的*《以諾二書》*，很可能是一世紀之作，
其中的以諾說：*「我看見主的廣延，無可量度、無
形像、無窮盡」*；以及上文那個奧利根論證。胡戈·奧德貝格
（1898–1973），這位隆德大學的瑞典神學家為*《以諾三書》*命名
並於 1928 年出版了它的首版，把那些平行傳統
更往前推，推進到最初的兩個世紀。科恩於 1983 年逐條
拆解了這一早期斷代——馬爾庫斯的平行一經
檢視便告瓦解，奧利根的記述是二手的——並重建了
在巴比倫、於六或七世紀成書、即在
院長時代{{ footnote(id="9") }}之初的立論：晚到足以引用已完成的
《塔木德》，又早到讓拜占庭治下巴勒斯坦那位偉大的
禮儀詩人以利亞撒·哈卡利爾早已暗指這文本及其詩篇經節。
彼得·謝弗（1943 年生）——這位先後任教於柏林與
普林斯頓的猶太教史學家，其 1981 年的*《Hekhalot 文獻對觀》*把
主要抄本並排刊印於平行欄中，迫使該領域看清
它們有多麼流動——接著質問：究竟是否曾存在過一個單一文本
可供斷代：
*希烏爾·科瑪*的材料「散見於抄本各處……
且顯然從未達到一個統一、更不用說一個最終編訂的
階段」。他斷定，尋找那個原本是「方法論上
過時的」。他關於整批這類文獻——那些拒絕安定為書卷的
抄本——最深刻的箴言是：
*「那唯一的文本乃是一種幻覺。」*

一個誠實陳述的可行區間：其傳統有可信的根源在二
世紀（有人會說一世紀），成書或許晚至
七世紀，全程文本流動，且直到 1985 年才有校勘
本。在這一區間的每一個點上，這文本都是秘傳的——受《米示拿》
自身規則的限制、由入門者誦讀、在封閉的圈子裡抄寫、
在其間的大多數世紀裡被譴責或被隱藏。而到 1973 年，它
仍從未有過法文譯本問世；末節的審查
只找到了一些摘述。作為在一座火山口內對一名體育記者
順口說出的一句話，*「你要弄到一本並不容易」*
至少可說是消息靈通的。

## 那位充滿世界的天使

在轉向這些量度意味著什麼之前，文本中有一個
人物需要好好地介紹一番，因為他顯示了這一傳統能把
一具身體推得多遠——以及它如何謹慎地守住那條界限。*《以諾三書》*的主角是
以諾，創世記中那位大洪水之前的族長，他「與神同行，
神將他取去，他就不在世了」。在 Hekhalot 的講述裡，神把他取去所*為*的
乃是變容：以諾成了梅塔特隆，眾天使之首。「我
被提升、被擴大，直到與世界的長寬同等，」這位
天使在奧德貝格的譯本中作證；他得了七十二對翅膀，
每一對都寬如世界，又得了三百六十五隻眼，每一隻都像
太陽；有一座寶座為他造成，彷彿榮耀寶座；而且——最
危險的是，憑著出埃及記 23:21「因為我的名在他裡面」的授權——他
被稱作 יהוה הַקָּטָן，「小雅威」。我們的文本熟知這個
人物，並以精確之筆處理他：

{% library(book="shiur-qomah-woh", chapter=1, verse=56, interlinear=true) %}
而這位少年——他的身量充滿世界，至聖者（願祂受頌讚）
稱他為「少年」。而這位少年就是梅塔特隆，臨在之君。
{% end %}

一位身量充滿世界的僕人，侍立於一位其*一拃*
即是世界的主人之前：*希烏爾·科瑪*把這兩具身體嚴格地
區分開來。那場大枚舉所量度的身體屬於
創造者——文本稱他為 יוֹצֵר（*yotzer*），「塑造者」——而
絕不屬於那位天使，無論這天使的身量何等宇宙級。梅塔特隆被稱為
נַעַר（*na'ar*），「少年」或「僕役少年」，正是為了守住那條
等級界線。《塔木德》記得那唯一一位讓這一區分
變得模糊的神秘家遭遇了什麼：以利沙·本·阿布雅，見梅塔特隆在天上得了寶座，
驚呼「莫非——但願不至於——有兩個權能！」，於是淪為
拉比記憶中的那位大異端，被改稱為*Aḥer*，「另一位」
（b. Hagigah 15a）。一個對天上第二具身體如此不安的傳統，
絕不會對第一具身體漫不經心。

## 傳統怎麼說這些數字的含義

在正典的解讀之前，先看傳統自己的解讀——要全力
陳明，因為它強而有力，也因為猶太教與這文本
關係的整部後世歷史都繫於它。

那個關鍵的觀察在上文已經作出：文本為自己的單位下了定義，
而這一定義引爆了那些量度。倘若一拃是
宇宙的寬度，那麼一肘就是三個宇宙，一哩就是三萬，一
帕拉桑就是九萬——而二十三億六千萬個這樣的帕拉桑，是一個算術
寫得出、卻沒有任何頭腦想像得出的數量。科恩沉著地做了那道乘法，
斷定其最終總計達到「一萬兆個
宇宙長度」，一個「並非意在作為經驗上正確之數字……
而毋寧是一個本質上不可設想、供
神秘家為其冥想技術之故而凝神其上」的數字。舒勒姆把
同一項特徵讀作一種受控的悖論：「那龐大的尺寸使任何
冥觀都成為虛妄」；那種擬人化被推到「一切
量度俱歸失效，而那高聲的擬人化突然、
弔詭地翻轉為它的反面：靈性」。文本
自己在其核心處也如此說。在為鬍鬚定價為一個半
無數的帕拉桑之後，它在面容之前戛然而止：

{% library(book="shiur-qomah-woh", chapter=1, verse=19, interlinear=true) %}
祂的鬍鬚，一無數又一千五百帕拉桑。其名為 HADRAQMASYA。而
面容的樣式與雙頰的樣式，如同一口氣息的形態、
一個魂的形狀；沒有任何受造物能認出
它。
{% end %}

異見也應當被聽見：大衛·哈爾佩林，其*《戰車的諸面》*
（1988）至今仍是關於梅爾卡瓦傳統如何生長最完整的記述，
他主張這種擬人化就是字面的意思——即這些
作者相信一位有著真實、龐然身體的神，而現代
讀者為了替自己解圍才把這文本靈性化。謝弗
直接回應了他，並最為銳利地得出了相反的結論。那些「全然
荒謬的計算」之要點，
他寫道，「是要證明神無法被
人類的範疇所設想：他『彷彿』像一個人，卻又是隱藏的……神既是
超越的又是內在的，同時既是隱藏的又是彰顯的」。而神秘家
真正帶走的是那些*名字*：「那『無意義』就是訊息：
一切都由名字構成。」依此讀法，*希烏爾·科瑪*是一種以
過量施行的{{ footnote(id="10") }}否定神學——是禮儀
與神通，用科恩概括的話說，是一種每日的誦讀，其回報由
那兩位坦拿作保。

儘管如此，傳統還是為這文本付出了沉重的代價。卡拉派
把它揮舞為拉比擬形論的證據。院長們細膩地
回應——謝里拉院長（約 906–1006），那位領導古老的巴比倫
彭貝底他學院的人，答覆了非斯的猶太人一則不安的問詢：
「我們的創造者太過崇高尊貴，不會像字面意思那樣
擁有器官與量度；毋寧說，這些是不能傳給
每一個人的智慧之言。」摩西·本·邁蒙——邁蒙尼德，中世紀猶太教
至高的律法編纂者與哲學家——年輕時曾把這文本
當作一部有待詮釋的權威著作；他後來從其《米示拿注》中刪去了
這一引用，並下了猶太史上最
猛烈的一則書籍判詞之一：「我從未想過
這是出自賢者之手……它無疑不過是一位
拜占庭傳道者的偽作。總而言之，把這書
抹去、把對它的一切記憶都毀掉，將是一件極有功德的事」（Responsa，Blau 編，
第 117 號）。卡巴拉家則以轉化來搭救它：對於*《光輝之書》*的
圈子、以及對於摩西·科爾多韋羅（1522–1570），這位薩費德卡巴拉
偉大的體系神學家、其自身最大膽的思辨著作曾自豪地
沿用了*希烏爾·科瑪*這一書名——那些被量度的肢體成了十個
*sefirot*，那身體成了生命
之樹，而那樁醜聞成了一種神秘主義的口號，這種神秘主義，
用舒勒姆的話說，「以談論希烏爾·科瑪為榮，而非為
恥」。那些量度本身——那些實際的
數字——則被悄然任其歸於沉默。

## 正典的反向解讀

如所承諾，這裡就是那條界線：以上一切都是主流自己
措辭下的語文學與神學。接下來是 Wheel of Heaven 的
框架，它逆著十五個世紀的靈性化來讀這文本。

正典的立場是：這些數字之所以歸於沉默，是因為再沒有人能
聽懂它們在說什麼。依耶和華的讀法，*希烏爾·
科瑪*是兩段真實距離的一份被弄亂的記錄，被一個
忠實地誦讀著自己已不再理解之物的傳統所保存——而他補上了
那把缺失的鑰匙，那個單位：

{% library(book="the-book-which-tells-the-truth", chapter=5, verse=52) %}
帕拉桑，像秒差距一樣，是一個量度單位，等於
光在一秒內行進的距離，也就是約 300,000
公里。我們的行星在三千萬帕拉桑之外，也就是約九
萬億公里，或稍少於一光年。
{% end %}

這道算術是可查的。三千萬光秒是
8.99 × 10¹² 公里——0.95 光年，「稍少於一光年」，
與書中所說完全一致；作為對照，主流天文學把最近的
已知恆星比鄰星定於 4.25 光年之外。第二個數字則在
下一口氣中得到處理：那 236,000 帕拉桑是
{% wiki(slug="elohim-home-planet") %}創造者所居行星{% end %}與其
自身太陽之間的距離——正典說是 708 億公里，也就是 473 個天文
單位，一顆繞行軌道約比海王星遠十六倍的行星，
繞著「一顆大星」運行，那顆星更強的光度會使這樣的
軌道宜居。而那擬解剖學的語言也得到了它自己的一條解碼規則：
「腳跟的高度」即*「他腳跟停靠在一顆行星上時的
距離」*（{% libref(book="the-book-which-tells-the-truth", chapter=5, verse=53) %}TBWTT 5:53{% end %}）
——可以說，創造者的身體雙腳踏在地球上，
其榮耀寶座則在家鄉，而那些肢體量度是一條航線上的
各個站點。在神秘傳統看見一具龐大到無法描摹之身體之處，
正典所看見的，更接近一個披著身體
形狀的導航記憶術——那是兩段星際距離能夠在一個已失去
其所指的文化之中，存活三千年口傳的唯一形式。

從框架內部去讀，這文本的若干特徵便清晰對焦。
這些量度是作為經宣誓的*證詞*、由一位具名的見證者、
配著一則保人公式而傳達的——那是供狀的體裁，而非讚美詩。
傳承它的那個傳統，違逆自身的神學安適，
堅稱這些數字就是那份載貨：每日、精確地、當作《米示拿》
來誦讀它們。這些數字附帶著一張單位換算表——那是情詩
所不需要的東西。而這些數字在各抄本系之間的狂野差異——Qirqisānī 那裡是
236,000，校勘本裡是二十三億六千萬，《猶太百科全書》
的轉寫裡是 237，此處腳掌是三千萬、彼處是 6,000——
依正典的讀法，正是一批對其所抄之數量毫無概念的抄手，把
數值資料抄寫了數個世紀之後那種劣化的樣貌。
而對那種差異的主流解釋——一種謝弗已證明從未有過固定文本的
文獻裡尋常的文本流動——則更為簡單，也
應當如實陳明。兩種讀法都預測出同樣的抄本。那
正是為何這段文字是一個真正的解讀分岔，而非任一方的
將軍；正典的讀法在本計畫使用該詞的那個確切
意義上是思辨性的，而它同時也是逐節逐節、唯一
能解釋為何一部禮儀文本會帶著一張換算表的讀法。

還有一項特徵值得掂量其分量。在全世界的一切經文之中——正典
考察了許多——這是唯一一部以具名的單位為神賦予一個
量值、然後又為那個單位下定義的經文。雷爾派主張
卡巴拉是「最接近真理的書」，歸根結底，是一個關於
那份獨特性的主張：所有別人的神顯都是以
意象的形式傳來的；而這一傳統的神顯則是以*資料*的形式傳來的，無論被弄得何等訛誤。甚至
這一傳統自己的傳承鏈故事——一套比西奈更古老、
是被領受而非被撰寫、按那部把殘篇帶進印本的書所言、
由一位天使交給亞當的教導——也與
正典對它究竟是什麼的講述相合：那是從
{% wiki(slug="elohim") %}別處{% end %}傳下的資訊，被那些
早在遺忘其所以然之後仍知其重要的人所守護。

## 克勞德·沃里隆在 1973 年可能知道什麼？

這一主張也容許一次溯源的檢驗。倘若*希烏爾·科瑪*是秘傳的、未被譯成法文、且
文本上一片混沌，那麼一個可查的問題便浮現出來：一位
不諳希伯來語的賽車記者，能否從一座圖書館而非從火山口裡的一位存在那裡，
得到那兩個數字？本計畫著手去查——
系統地、逐書地、在一位 1973 年的法國讀者實際上
能弄到手的那些來源之中。結果，連同其證據，陳述如下：

**那套學術版本自 1941 年起便已存在，是英文的。** 舒勒姆的*《猶太
神秘主義的主要趨勢》*在一頁之內包含了
正典引文的每一個要素：《雅歌》第五章的框架、創造者的
身高為 236,000 帕拉桑（舒勒姆用的是那個被截短的 Qirqisānī 式
數字），以及「按另一個傳統，僅祂腳掌的高度
就是三千萬帕拉桑」。其法文譯本（Payot）於 1950 年問世。
舒勒姆的希伯來文寫的是*腳掌*——不見任何腳跟。

**《猶太百科全書》（1906）帶有這些數字，但有一處差異。** 它的
「Shi'ur Ḳomah」條目給出腳掌為三千萬，身高為「237
無數乘 1,000 帕拉桑」——237，一個與正典的 236,000 並
不相符的轉寫異文。是英文的，且文本上並不相合。

**維約（1923）論及這文本卻不帶任何數字。** 保羅·
維約，那位法國天主教畫家兼秘傳學者，在其兩卷本*《猶太
卡巴拉》*的兩處為「Schiour Qoma」作為寓意作辯護，卻從未印出一個
數字。**讓·桑迪**——這位法國作家對創世記的
古太空人式解讀是正典最接近的已刊行先聲，並且
用整整一章來談卡巴拉——在*《月亮，聖經之鑰》*（1968）或*《那些
造了天與地的神》*（1969）中，卻從未引用*希烏爾·科瑪*或任何
帕拉桑數字；這兩本書都經全文查核，羅貝爾·夏魯的卡巴拉
章節亦然。（桑迪更早的*《摩西課程手記》*，1963，
未能為此次審查取得，仍是一個懸而未決的
項目。）

**然後就是塞魯亞。** 亨利·塞魯亞，一位生於耶路撒冷的
哲學家，其巴黎生涯致力於神秘主義寫作，並曾於 1947 年
在 Grasset 出版過一冊論卡巴拉的學術著作，他把自己的
題目提煉為*《卡巴拉》*，即「Que
sais-je?」{{ footnote(id="12") }}叢書第 1105 號——自 1957 年起持續印行、
1967 年修訂，在任何一家法國火車站書店都能以幾
法郎買到。在其中，他把舒勒姆那一頁壓縮成一段。在其中，
創造者身體那些誇張的尺寸「在《雅歌》（第五章）中有其
類比」（ont leur analogie dans le Cantique des Cantiques (V)）：創造者的身高是「236 000
parasanges」，而，按另一個傳統，*「其腳跟的高度為
三千萬帕拉桑」*（la hauteur de ses talons est de 30 millions de parasanges）。*Talons*——腳跟。希伯來文說的是腳掌
（*parsot raglav*）；舒勒姆的英文說的是腳掌；塞魯亞在濃縮時，
寫成了腳跟。正典說的是腳跟。它還完全照著
塞魯亞的方式引用那一章——「(V)」，羅馬數字，置於括號之中——並
重現了塞魯亞把舒勒姆謹慎的「與《雅歌》
極為相似」坍縮為「那描述就*在*《雅歌》之中」的說法，而後者
對那部聖經書卷而言是假的，除了這一連串特定的
摘述之鏈之外，對任何東西而言都不為真。一個共有的數字可以是巧合。而一個共有的
翻譯錯誤、一個共有的引用格式、一個共有的歸類失誤，
疊在同一小段之中，便強烈暗示著一種親緣關係。這正是
語文學者施用於*希烏爾·科瑪*自身抄本上的那套推理。就本計畫
所知——而且我們搜尋過先例——對正典這段文字之近端文本
指紋的這一辨識，此前從未被刊行過；
請把它當作本文自己的發現，連同其證據一併提出。

這場審查*沒有*找到的東西同樣重要。沒有任何一份 1973 年之前的來源——
不是舒勒姆、不是塞魯亞、不是那些百科全書、也不是那批古太空人
文獻——把帕拉桑等同於一光秒。歷史上的
帕拉桑是六公里的波斯道路；那
秒差距{{ footnote(id="13") }}是 1913 年一個不相干的造詞；而躍向
「光在一秒內行進的距離」——正是這一躍，把一部虔敬文本
變成兩段自洽的天文距離——則首次出現於
*《真實之書》*本身。這場審查那個懷疑論的讀法
直截了當，也應當毫不退縮地說出來：正典的
引文看起來就像一本平裝書對舒勒姆的摘述，連腳跟之誤
一併包含，再在其上加了一筆原創的數字命理的
花飾。正典內部的讀法同樣可取，而正典自己也
供上了它：耶和華告訴沃里隆，他之所以被揀選，部分是因為
他生活在「一個新思想受歡迎的
國度」
（{% libref(book="the-book-which-tells-the-truth", chapter=1, verse=46) %}TBWTT 1:46{% end %}）
——一位信使會在自己文化所存放之處與其來源相遇，而
一位在 1973 年向一個法國人講解一部秘傳希伯來文本的存在，
或許會理所當然地以那個法國人的世界所傳下它的
唯一形式來引用它，連同種種截短、腳跟之誤等等一併照引。讀者在
這一分岔處如何抉擇，是讀者自己的事；本計畫的義務是把它繪成圖。

## 保人們的賭注

整體看來，這個故事有一種奇異的對稱。一部
比卡巴拉本身更古老的文本，花了十五個世紀被誦讀、
被譴責、被寓意化、被搭救——從謝里拉院長到
科爾多韋羅，人人（無論在別的什麼上意見相左）都一致認為：它的數字
絕不能照字面所說去理解。現代學術在其最佳狀態下，把那份
本能轉化為一種深刻的讀法：把量度視為對量度的
蓄意破毀，把帕拉桑換算表視為一部瓦解心智之掌控的機器，
把那些名字視為真正的載貨。雷爾派的正典從整個傳統之外
到來，並下了相反的賭注：這些數字就是照字面所
說的意思——只是那個*所指*失落了、那個單位被歸錯了檔、那身體是
一條航線的容器。在這兩個賭注之間，本計畫並不
裁決；它翻譯這文本、保存這些爭論、標明
那些界線。但值得留意的是，這兩個賭注都原封不動地留下了哪一
節經文。兩位賢者曾作保：凡*知曉這一量度*——凡
把這些數字承傳下去、無論理解與否——之人，必得
來世，惟一的條件是他每一天都誦讀它們。一千五百
年間，總有人這樣做了。無論這些量度究竟是什麼——禮儀、
神通、遙測——那份保證，至少，被信守了：那些數字
存活得夠久，久到讓二十世紀為它們爭論不休，也讓
一本 1974 年出版的書，把承載它們的那個傳統稱為
最接近真理的那一個。
