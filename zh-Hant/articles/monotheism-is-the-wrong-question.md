一尊{% wiki(slug="idolatry") %}偶像{% end %}，在它進了博物館之後，是最容易被
擊敗的。

隔著玻璃，斷絕於香火、遊行、音樂、食物、衣袍、祭司，以及那曾使它成為一座
城市之靜止核心的建築，這尊雕像看上去正如先知所說的那樣。它有眼卻不能看。
它有口卻不能答。一位館員戴著棉手套便能把它舉起。以賽亞筆下的木匠可以燒掉
半根木頭取暖，把另一半雕成像，俯伏在它面前，求那木料拯救自己。案子彷彿在
說明牌都還沒讀之前，就已了結。

但古代的那場爭辯，從來不曾這樣輕易。那位雕出一尊美索不達米亞膜拜神像的
工匠，知道是他把它雕出來的。那位為它洗口的祭司，知道它不久之前還無法進食。
那些抬著它穿過城市的敬拜者，並不需要一位希伯來先知來說明香柏木出自一棵樹。
他們關於神像的神學要更為精微：種種儀式把這件器物從作坊移交進神聖的職事
之中；那尊像便成了一位存有——一位並不被眼前的物質所窮盡的存有——的一具
享有特權的身體、臨在，或會遇之所。聖經的論戰攻擊那一主張的辦法，是把每一
層都重新坍縮回木料。這是精彩的修辭。它不是中立的民族誌。

這事關緊要，因為同一套論戰，後來被派去承載一個更為龐大的論證。
{% wiki(slug="religion") %}西方宗教{% end %}的歷史，通常被描繪為
{% wiki(slug="monotheism") %}一神論{% end %}的勝利，是**一**對**多**的勝利。
以色列逃離了{% wiki(slug="plurality-of-gods") %}迦南的多神信仰{% end %}，發現了
唯一的神；基督教把他普世化；伊斯蘭教則把這一結果純化為
{% wiki(slug="hanafiyya") %}認主獨一{% end %}（*tawhid*），即阿拉那不容妥協的
獨一性。在這條界線的另一側，站著異教徒與不信者，他們繁衍出眾神，或因他們
尚未抵達那份抽象，或因他們已從中墮落。宗教的版圖於是變成了算術。

麻煩在於，古代的文本總是拒絕好好地計數。《申命記》一面要求以色列單單事奉
{% wiki(slug="yahweh") %}耶和華{% end %}，一面卻記得列國被分配給了別的神聖
之子。詩篇 82 篇讓耶洛因端坐於一個議會之中，並讓他*在眾神之間*施行審判。
保羅可以寫下只有一位神、一位主，卻又在同一口氣裡，承認「許多的神、許多的
主」。《古蘭經》可以譴責以物配主（*shirk*），同時又一再使它的對手承認阿拉
創造了天與地。在每一種情形裡，那個活生生的問題都不僅僅是*究竟存在多少
存有？*而是：*你將事奉誰？你可以透過誰去趨近？什麼可以代表那不在場者？誰
擁有權柄？誰配得創造之功的歸榮？*{{ footnote(id="1") }}

這篇解說推展兩項主張，而它們必須被擺在不同的證據距離上。第一項是歷史性的：
**對古代的反偶像文本而言，一神論與多神論往往是拙劣的主導範疇**。它們所遮蔽
的，可能多於它們所揭示的，因為它們把本體論、祭祀、效忠、表徵與創造，壓縮
成了單獨一個數字。這一主張如今背後已有一大批學術文獻。

第二項是 Wheel of Heaven 的假說：**先知向偶像宣的戰，或許保存著一種反覆
出現的記憶尋回，把人類從死寂的替代品那裡召喚回來，回到他們真正的製造者
——{% wiki(slug="elohim") %}耶洛因{% end %}——身邊，而這通常是藉由一位被
命名的 Eloha（耶洛因中的單獨一員）或信使的權威來完成的**。依這一解讀，
後來那位單一、非物質、無所不在的神，並不是那些最古老的論戰所要捍衛的東西。
他是數百年的神學從一段更為具體的歷史中造出來的產物。那個主張是
`speculative`（臆測性的）。它不是標準的聖經學術，也不是伊斯蘭神學。它的
價值取決於：它是否比它想要取代的那些範疇，更好地解釋了文本的殘餘。更宏大
的正典架構，在[*眾宗教之宗教*](/articles/the-religion-of-religions/)中有所展開。

## 一把十七世紀的尺，橫置於古代諸世界之上

*一神論*（monotheism）這個詞，比望遠鏡還要年輕。它是在十七世紀下半葉，由
基督教哲學家所鑄造並推廣的；一般把它在英語中的首次使用歸於亨利·摩爾
（Henry More）。這個詞指稱著一個真實而重要的信念家族，但它並不是摩西、
保羅與穆罕默德所共有的一種古代自我描述。它是一把在某一間思想作坊裡造出來、
又被回溯性地橫置於種種迥然不同之宗教世界之上的尺。

那把尺通常只問一個問題：**這個宗教說有多少位神存在？**答案把各個傳統歸入
不同的格子。一位，是一神論。多於一位，是多神論。學者們為那些不肯配合的
證據，造出了一些中間的術語。*單一神崇拜*（monolatry）指稱對一位神的崇拜，
而其他的神仍可能存在；*單一主神論*（henotheism）指稱在一個複數的神群之中對
一位至高神的專奉。{{ footnote(id="2") }}然而，即便是這些修補，也仍把數目擺在
中心。

一個更好的初步做法，是一個矩陣：

| 提問 | 它所問的 | 一個古代的回答可能是 |
|---|---|---|
| 本體論 | 存在著哪些種類的超人類存有？ | 一位至高神、諸神聖之子、信使、鬼魔、祖先 |
| 祭祀 | 誰領受獻祭、禱告、許願或朝聖？ | 一位守護神、若干地方性的權能，或藉由中介者而歸於一位 |
| 效忠 | 誰的律法與政治權威約束著這個民族？ | 我們的聖約之主，而非別族的守護神 |
| 表徵 | 一個存有能否居於一尊像、一塊石、一個名、一座寶座、一件聖物或一座殿宇之中，或藉之運作？ | 臨在而不等同其身份；中介而不等於其本身 |
| 創造 | 誰造了世界、人類，或這個民族？ | 一位至高的造物者、一個議會、一位工匠，或數位行事者 |

這些格子如今可以落在意想不到的地方。一個群體可以承認眾多的天上存有，卻只
向其中一位獻祭。另一個群體可以稱一位至高的存有為造物者，同時卻讓從屬的
諸神充滿它的殿宇。第三個群體可以否認一尊像本身就是神，同時卻把它當作那位
神所授權的身體來對待。第四個群體可以承認一位神，同時卻透過聖徒或天使來
導引禱告。這些都不是迴避。它們是不同的宗教系統，在回答不同的問題。

丹·麥克萊倫（Dan McClellan）晚近對這一範疇的綜述，稱*一神論*「含混而棘手」，
並指出它曾服務於種種關於文明進步與衰落的敘事。馬克·S·史密斯（Mark S.
Smith）對{% wiki(slug="hebrew-bible") %}希伯來聖經{% end %}得出了一個更為
具體的結論：在那裡，一神論並不單純是多神論之後的一個新階段，而是一套強化
以色列與其神之排他關係的修辭。葆拉·弗雷德里克森（Paula Fredriksen）對於
古代則走得更遠：關於獨一神性的種種陳述，往往宣告的是一位神的權能與敬拜者
的忠誠，而非那位神的孤然獨存。一旦這一可能性被承認，若干著名的經文便會
改變其溫度。

## 「在我以外不可有別的耶洛因」

十誡並不以一個哲學證明開場。它以一次拯救、以及一項管轄權的主張開場——正是
那個貫穿[白羊座時代](/timeline/age-of-aries/)所追溯的聖約世界：

{% library(book="exodus", chapter=20, verse=2, verse_end=5) %}
我是耶和華你的神，曾將你從埃及地、從為奴之家領出來。
除了我以外，你不可有別的神。
不可為自己雕刻偶像，也不可做甚麼形像，彷彿上天、下地，和地底下、水中的百物。
不可跪拜那些像，也不可事奉它們。
{% end %}

這一先後次序關係重大。「我把你領出來」確立了一種關係；「在我以外不可有別
的*耶洛因*」使這關係成為排他的；而禁像之令則巡守著它那可見的邊界。所給出
的理由是嫉妒，是一份因對手效忠而受冒犯的聖約所用的語言。這條誡命並沒有說：
「不存在別的屬於*耶洛因*這一範疇的存有。」的確，「在我以外」對於安放子虛
烏有之物來說會是個奇怪的位置。它所說的是：以色列不可在耶和華面前設立、
事奉或跪拜它們。

其他一些早期的表述是比較性的，而非消去性的。渡海之後，出埃及記問道：
「耶和華啊，眾神之中誰能像你？」詩篇 89 篇問，大能者的眾子之中有誰堪與
耶和華相比。在日常的言談裡，「眾王之中誰能像這位王？」抬高的是一位君主；
它並不廢除其他每一座王座。

摩西之歌保存著最為棘手的證據。在《申命記》32:8–9 那個更古老的、由一份
死海古卷與希臘文見證所佐證的讀法中，至高者「按神的兒子們的數目」劃分列國，
而耶和華則分得雅各作為他所得的份。{{ footnote(id="3") }}再過幾行，以色列向
*shedim* 獻祭——那些「非 Eloha」的權能，那些「新近興起的神」。這首詩所
譴責的，是忠誠的轉移。它並不棲居於一片空蕩的天空。
[*在元年甦醒的書卷*](/articles/the-scrolls-that-woke-in-year-one/)詳細追溯了
這批手稿的發現，以及那個更古老的「神的兒子們」（sons of Elohim）讀法。

詩篇 82 篇，是本典籍{% wiki(slug="plurality-of-gods") %}諸神複數性解讀{% end %}
中的一個核心見證，它更是難以被壓平：

{% library(book="psalms", chapter=82, verse=1) %}
神站在神的會中，在眾神之間施行審判。
{% end %}

希伯來原文重複了同一個名詞：耶洛因（*Elohim*）站在 El 的會中；在眾耶洛因
（*elohim*）之間，他施行審判。那些被控告的統治者被告知：「你們是神，你們
都是至高者的兒子」，隨後又被判處「像世人一樣死去」。詮釋各不相同。它們曾
被解讀為凡間的審判官、外邦的神、一個神聖議會的成員，或一首刻意把舊神降格
為必死之物的詩。這首詩無論如何都無法被弄得像那個乾淨俐落的現代命題——即
希伯來聖經只知道神聖範疇中有唯一一位居住者。

因此，班傑明·索默（Benjamin Sommer）為*一神論*一詞的有限度使用作了辯護：
希伯來聖經之所以能提及別的神，是因為它們作為獨立的行事者，從不能與耶和華
那終極的掌控相匹敵。這是一個嚴肅的回答。它把定義從數目挪移到了主權。然而
這一挪移恰恰讓出了爭論的要點：聖經中的「合一」，並不總是數目上的孤然獨存。
它可以意味著對一個住滿了存有的天界的無可比擬之統治。

## 以賽亞的斧，以及它究竟砍斷了什麼

支持本體論一神論的最強證據，出現在{% wiki(slug="prophet") %}先知{% end %}
以賽亞被擄時期的詩篇裡，即第 40 至 55 章。在這裡耶和華說：「在我以先沒有
造作的神，在我以後也必沒有」；「我是首先的，我是末後的；除我以外再沒有神」；
「我是耶和華，此外再無別神」。這些不僅僅是敬拜的命令。它們聽起來像是在
否認有任何堪與相比的神存在。

接著便是聖經對偶像作坊的那場偉大諷刺。鐵匠在他的爐前疲乏。木匠拉起準繩，
選定一棵樹，做出一個好看的人形。木料的一半用來煮他的飯食。在剩下的那一半
面前，他跪了下去：

{% library(book="isaiah", chapter=44, verse=16, verse_end=17) %}
他把一半燒在火中，用這一半烤肉吃；他烤肉吃飽，又自己取暖，說：「啊哈，我暖和了，我見了火。」
他用剩下的一半做了一位神，就是他的雕像；他向它俯伏敬拜，向它禱告，說：「求你拯救我，因你是我的神。」
{% end %}

假如有任何一段聖經文本的意思是「只有一位存有存在，而每一位所謂的神都是
無生命的物質」，那麼這就是那個候選者。一份負責任的論點絕不可把它解釋掉。
那些被擄時期的作者，確實朝著一位普世造物者移動，而這位造物者的獨一性，比起
《申命記》32 章那種神聖分配，要遠遠更接近後來猶太教、基督教與伊斯蘭教的
一神論。

然而這套修辭仍然運作在不止一個層面上。掃羅·奧利安（Saul Olyan）論證說，
以賽亞那些「無可比擬之神」的套語，與別處用於諸神的頌讚相似：「無一像你」
抬高的是位階與效能，並不必然充當一份對每一位超人類存有的清點。馬克·史密斯
則把這套語言安置在被擄那場政治災難之中。當列國傾覆，它們的守護神看似戰敗
了。以賽亞的回應，不是承認耶和華之死，而是擴張他的管轄範圍：征服者居魯士
（Cyrus）也是他的器具；以色列的民族之神，是歷史唯一的主宰。

換句話說，以賽亞的斧朝兩個方向砍。它把神像砍倒，斥之為一件無能的人造物。
它也砍去了那幅古老的民族版圖——在那幅圖裡，每個民族都有一位範圍相當的
守護神。這是一次真實的神學發展。但它仍然不能使每一段更早的反偶像經文，都
變成一堂關於某位非物質絕對者之屬性的講課。

## 拜偶像者知道那木頭從何而來

那個先知式的笑話之所以奏效，是因為它壓制了對手的回答。古代美索不達米亞的
宗教，通常並不主張那個木質的核心憑其自身創造了天。一尊新製成的雕像要經歷
*mis pi* 與 *pit pi*，即洗口與開口之禮。它被從作坊領往河岸與果園，受潔淨、
與它的工匠分離、被穿衣、餵食，然後被安置於一座殿宇之中。咒語把它的誕生歸於
工藝諸神，並宣告它「生於天，造於地」。{{ footnote(id="4") }}

最後那份張力，正是神學之所在。人人都能看見，是人的雙手在地上做了工。那個
儀式所否認的是：那雙手窮盡了這尊像的來源。它把一件人工製品轉化為一具膜拜
的身體，一位神藉之得以觀看、聆聽、吃享祭物、受衣、在遊行中行走，並升座
理事。這尊雕像既不是單純的肖像，也不是一位神的全部。它是一處經授權的臨在
之所。

即便在那些以禁像聞名的傳統裡，這一區分也並不陌生。約櫃、基路伯寶座、聖殿、
神聖的名、妥拉書卷、黑石、一件聖物，以及一個禱告的方向，全都能夠中介臨在，
而不與那位被敬奉的存有直截了當地等同。各個宗教群體把它們可容許的界線畫在
不同的地方。論戰之所以取勝，靠的是假裝對手根本沒有任何界線。

這並不能把每一尊膜拜神像都從古代的批判中救出來。一件物質的器物可以變成它
本應中介之實在的替代品。祭司可以操縱通路。帝國可以擄走一尊雕像，聲稱綁走了
它的神。禮儀的運作可以在它的宇宙論被遺忘之後仍然延續。先知們或許診斷出了
一次真實的、由象徵坍縮為器物的崩解。但那份診斷所指的是**被誤導的關係**，
而不必然是糟糕的算術。

## 一位至高神並未清空異教的天空

基督教與伊斯蘭教所興起於其中的那個世界，早已包含著種種朝向神聖合一的強勁
潮流。哲學家談論著一個第一原理。頌歌能夠把一位神抬高於一切名號之上。銘文
歡呼「獨一之神」或 *Theos Hypsistos*，即至高之神。地方性的膜拜彼此競爭，
把自己的守護神奉為 *megas*，即偉大，有時甚至是最偉大。

現代學者把這稱作「異教的一神論」，儘管這個說法是刻意挑釁的。敬拜者或許把
眾多有名有姓的神看作權能、名號、顯現、僕役，或一個層級體系中的從屬成員，
而那體系的頂峰是一位。頂峰並沒有使山坡消失。史蒂芬·米切爾（Stephen
Mitchell）與彼得·范·努菲倫（Peter Van Nuffelen）的《獨一之神》（*One
God*）顯示，古代的至高神語言，是何等輕易地跨越了現代版圖視作多神論與一神論
之間一堵牆的東西。居伊·斯特勞姆薩（Guy Stroumsa）晚近則論證說，*單一主神論*
（henotheism）往往可能是更準確的詞：崇拜一位，而不否認其他諸位。

這一古典晚期的背景，改變了皈依這齣戲。那個抉擇，並不總是在「相信有一位
神聖存有存在」與「相信有十二位」之間。它可以是一個決定：捨棄承襲下來的
獻祭，退出城邦諸神，拒絕皇帝的崇拜，並把排他的效忠獻給以色列的神。祭祀與
政治先動了起來，早於本體論被完全系統化之前。

## 保羅那擁擠的一神論

保羅關於祭偶像之食物的論證，用六節經文捕捉住了整個問題。他同意哥林多人的
口號，即「偶像在世上算不得甚麼」，以及「除了一位以外，再沒有別的神」。一份
現代的摘要在此打住。保羅卻沒有：

{% library(book="1-corinthians", chapter=8, verse=5, verse_end=6) %}
因為即便真有那些被稱為神的，無論在天上或在地上——正如那有許多的神、許多的主——
然而在我們，卻只有一位神，就是父，萬物都本於他，我們也歸於他；並有一位主，就是耶穌基督，萬物都藉著他而有，我們也藉著他而有。
{% end %}

那兩個做著歷史工作的字，是**「在我們」**。保羅把那許多者，與這個群體的
一位神、一位主相對照。他的公式是宣信式的、以效忠為本的。它在這宣信之內，
還容納了兩個角色：萬物「本於」他的父，以及萬物「藉著」他而有的耶穌基督。
清點位格，已經不如描摹能動者與敬拜來得有用了。

兩章之後，保羅收緊了這條禁令。他說，異教的獻祭是獻給鬼魔的，而人不能既喝
「主的杯」又喝「鬼魔的杯」，也不能同赴兩席。無論*偶像算不得甚麼*是什麼意思，
它都不意味著透過偶像崇拜什麼也遇不到。那件器物是無能的；那份關係是危險的；
那些對手權能真實到足以要求一個抉擇。因此，肖恩·麥克多諾（Sean McDonough）
把這段經文讀作一場關於中介與效忠之種種形式的較量，而不僅僅是一份對天界的
人口普查。

使徒行傳讓保羅在雅典的眾像之間發表了一篇相關的講辭。那位未識之神，並不被
囚於人手所造的殿宇，也不需要人的服事。然而保羅是從一座早已存在的祭壇開始
的，並引用希臘詩人論人類為神之後裔的話。這篇講辭並不是走進一座無神論的
城市、為它添上第一位神。它是把一個過度飽和的宗教場域，圍繞一位造物者兼
審判者重新排列。

## 伊斯蘭教是尋回，而非發明

正是在這裡，{% wiki(slug="muhammad") %}早期伊斯蘭教{% end %}成了不只是又一個
例子。《古蘭經》並不把穆罕默德呈現為發明了一套更好的形上學。它把他呈現為
把人類召回到一份原初的聖約、確認先前的眾信使，並遵循 *millat Ibrāhīm*，即
{% wiki(slug="abraham") %}亞伯拉罕{% end %}的宗教。

那節核心的經文是斬釘截鐵的：

> 亞伯拉罕既不是猶太教徒，也不是基督徒，而是一位正直者（*hanif*）、一位
> 順服者，他也不屬於以物配主的人。（《古蘭經》3:67）

那個時序本身就是論證。亞伯拉罕先於那些為他的遺產而爭執的建制。因此他不屬於
其中任何一方，而雙方都可以拿他來衡量。穆罕默德受命去「遵循亞伯拉罕的宗教，
那位正直者（*hanif*）的宗教」（16:123）。對猶太人與基督徒，回答是：「不然，
是亞伯拉罕的宗教」（2:135）。亞伯拉罕與以實瑪利築起天房的根基，並為他們
後裔中一個順服的共同體祈禱（2:125–129）。

最早的傳記發展出同一齣尋回之戲。伊本·伊斯哈格（Ibn Ishaq）筆下的四位尋道者
斷定，古萊什人已經敗壞了「他們祖先亞伯拉罕的宗教」。他們拒斥那塊他們族人
所繞行的石頭，因為「它既不能聽，不能看，不能害人，也不能助人」，於是動身去
尋找{% wiki(slug="hanafiyya") %}哈尼夫信仰{% end %}。這段插曲以我們今日所見
的形式寫成，已是穆罕默德之後一個多世紀的事，而穆尼姆·西里（Mun'im Sirry）
已經表明，*hanif* 有著一段比後來「原初一神論者」這一乾淨注解所暗示的更為
不穩定的前史。儘管如此，《古蘭經》自身的尋回主張，並不依賴於這部傳記。

那亞伯拉罕—以實瑪利的關聯，也不是在七世紀從無中憑空開始的。五世紀的基督徒
史學家索佐門（Sozomen）把撒拉森人呈現為以實瑪利的後裔，記下他們的割禮與
對豬肉的避忌，並說他們的先祖曾採納鄰近諸族的神。{{ footnote(id="5") }}他的
記述支持一種阿拉伯—以實瑪利譜系在伊斯蘭之前的流傳。它並不支持一條從夏甲、
經由每一個阿拉伯族群、直到{% wiki(slug="nabataeans") %}納巴泰人{% end %}的、
筆直的生物學世系，而本文也並不需要這樣一條。一份被記住的世系，可以組織身份
認同，而無須充當一份現代的血統證明。

弗雷德·唐納（Fred Donner）的「信士共同體」（Believers' movement）為這場
尋回添上一個社會的維度。依他的重建，最早的那個共同體，在一個界線分明的
伊斯蘭信條完全成形之前，聚集起那些信仰獨一之神與末日的義人。猶太人與基督徒
最初可以站在它的道德與末世論視野之內。後來的界線才變得堅硬。無論唐納模型的
每一個部分是否都成立，它都解釋了為何《古蘭經》能夠聽起來既像一次全新的
宣告，又像一次對一個比種種信條之名更為古老之共同體的籲求。

對眼下這一論點而言，那個要緊之處很簡單：**伊斯蘭教自身的語法就是回歸**。
爭執所關乎的，是誰正確地保存了亞伯拉罕的那份關係，而不是誰能設計出最新奇
的神。[雙魚座時代](/timeline/age-of-pisces/)把那場尋回安置在更長的先知時序
之內，而[*最早的清真寺朝向的是佩特拉嗎？*](/articles/the-first-mosques-faced-petra-not-mecca/)
則檢驗那個爭議大得多的問題：它最早的地理位置。

## 《古蘭經》的以物配主者早已認得一位造物者

常見的圖景，把穆罕默德擺在一群相信石頭造了宇宙的人面前。《古蘭經》本身卻
一再提供一個不同的答案。它說，去問是誰創造了天地、又使日月馴服，那麼「他們
必定會說：阿拉」（29:61）。去問是誰降下雨水、使大地復甦，他們又說是阿拉
（29:63）。同樣的承認在 31:25、39:38 與 43:87 重複出現。

這些是論戰式的提問，而非一次宗教普查的謄本。G·R·霍廷（G. R. Hawting）曾
警告，後來伊斯蘭關於一個偶像林立之漢志的記述，可能把《古蘭經》的論戰，轉換
成了一部關於異教的簡單歷史。艾哈邁德·賈拉德（Ahmad Al-Jallad）與海瑟姆·
西德基（Hythem Sidky）的銘文學重建，如今又添上了物質層面的深度：阿拉伯宗教
實踐的一部分，早已朝著一位至高造物者移動，而更古老的女神與儀式，則能夠作為
名號、說情者、護符、獻祭習俗以及承襲下來的聖所慣例而存續。

《古蘭經》藉著它賦予其對手的口吻，道出了那真正的爭端：

> 我們崇拜他們，只為叫他們使我們更接近阿拉。（《古蘭經》39:3）

而在別處：

> 這些是我們在阿拉那裡的說情者。（《古蘭經》10:18）

那並不是不信一位至高造物者。那是一套關於通路的理論。以物配主者說，那些較低
的存有把敬拜者帶得離阿拉更近，並在他的朝堂上代為求情。《古蘭經》所攻擊的，
是把夥伴、女兒、說情者，以及未經授權的權能與造物者相配。它筆下的亞伯拉罕問
那些偶像為何不能言語；它對拉特（al-Lat）、烏札（al-Uzza）與麥那特（Manat）
的判決是：人的命名，賜給了它們一種阿拉從未降下的權柄。

因此，這場衝突正好橫跨那個矩陣。就本體論而言，《古蘭經》容納了天使、精靈
（jinn）、眾撒但與眾信使。就創造而言，它把萬物歸於阿拉。就祭祀而言，它把
崇拜保留給阿拉。就效忠而言，它要求對他與他的使者順服。就表徵而言，它拒絕
膜拜神像。就中介而言，它只允許阿拉所授權的說情。「一」，是這整個秩序的
壓縮，而不僅僅是第一個整數。

後來的穆斯林神學家，把這一秩序打造得在形上學上難以撼動。哈利勒·安達尼
（Khalil Andani）的綜述顯示，*認主獨一*（tawhid）成了一個爭論的場域，圍繞著
神的單純性、屬性、因果、具身、超越與說情。{{ footnote(id="6") }}穆爾太齊賴派
（Mu'tazila）否認神具有物理與時間的性質；艾什爾里派（Ash'ari）肯認永恆的
屬性，卻不使它們成為分立的諸神；哲學家把神辨識為必然的存在；伊斯瑪儀派
（Isma'ili）的思想家把神聖的合一推到尋常述謂之外；蘇非派（Sufi）則透過神聖
諸名的顯現來描繪創造。沒有任何誠實的辦法，能把這一切稱作一種對有限的生物學
造物者的偽裝信仰。

然而，它所由以興起的那份宣告，其當下的對手往往早已說過阿拉創造了世界。就
歷史而言，第一個施壓點，並不在於一位至高的造物者是否存在。它在於：崇拜、
權柄與通路，是否可以被分派給那些較低的存有及其器物。

## Wheel of Heaven 的倒轉

在[水瓶座時代](/timeline/age-of-aquarius/)之內，Wheel of Heaven 的正典一開始
便倒轉了那個後來的定局。它接受古代的複數性，並拒絕那個超自然的範疇。耶洛因
是眾多的、有限的、具身的製造者。耶和華是一位有名有姓的個體、一位建制上的
主席，而非存在本身。加百列或吉卜利勒（Jibril）被解讀為一位涉入先知接觸的
信使，而不是一個在被造與造物者之間盤旋的非物質物種。{{ footnote(id="7") }}

它關於偶像崇拜的那個核心主張，恰恰出現在與亞伯拉罕相關聯的那個時期：

{% library(book="the-book-which-tells-the-truth", chapter=3, verse=4) %}
但人類，在最聰慧者以及如所多瑪與蛾摩拉那樣的進步中心被摧毀之後，重新跌回一種極為原始的狀態，便愚昧地開始崇拜石塊與偶像，忘卻了是誰創造了他們。
{% end %}

最後那個分句提供了框架：**忘卻了是誰創造了他們**。那過錯並不在於相信實在中
包含了太多有力的存有。正典自己的實在，就包含著一整個由這樣的存有所組成的
文明。那過錯在於記憶的移位——從製造者，移到了器物。

同一部源典後來把三項主張捆在一起：經文把「太多超自然的東西放進了真理之中」；
翻譯者用單數的「神」替換了複數的*耶洛因*（Elohim），把「創造者轉化成了一位
單一而不可理解的神」；而基督徒則被弄得去崇拜釘成十字的木頭，儘管「十字架並
不是基督」。[*翻譯者的賭注*](/articles/the-translators-wager/)在種種現代字面
主義的解讀之中，審視了那場由複數到單數的轉化。

{% library(book="the-book-which-tells-the-truth", chapter=5, verse=21) %}
然而，他們的錯誤是重大的，尤其是把太多超自然的東西放進了真理之中，以及拙劣地翻譯了聖經的著作——在通行的各版聖經裡，用單數的「神」一詞，取代了指稱創造者的「耶洛因」（Elohim）一詞，而「耶洛因」在希伯來文中乃是「Eloha」的複數形，如此便把創造者轉化成了一位單一而不可理解的神。其他的錯誤，是使人們去崇拜一塊為紀念耶穌基督而釘在十字架上的木頭。十字架並不是基督。一塊釘成十字的木頭毫無意義。
{% end %}

這不是一個邊緣的類比。它是正典關於宗教史的理論：

1. 真實的、多位的製造者與人類互動。
2. 特定的代表管理著種種接觸、聖約、考驗與訊息。
3. 在災難與文化喪失之後，記憶收縮成名號、儀式、聖所與形象。
4. 那個記號開始領受本屬於它背後那個實在的東西。
5. 一位先知藉由摧毀或相對化那個記號、並把效忠召回到那個被代表的源頭，來
   尋回那份關係。
6. 後來的神學把那位代表普世化為唯一的超自然之神，並把其他的行事者重新歸類
   為天使、鬼魔、假神或隱喻。

依這個模型，一個古代對「一」的承認，可以在操作上為真，而不必在本體論上是
單一的。一道經授權的命令、一項使命、一位主席、一份聖約、一個效忠的方向，
以及一項關於創造的真理，都可以代表一個複數的創造者群。一位外交官為一個國家
發言，卻並非那國家唯一的公民。一位任務指揮官代表一個機構，卻並非唯一存在的
軍官。那個單數是**代表性的**，而非**窮盡性的**。

這正是一位 Eloha 在先知宗教中被提議擔當的角色。耶和華之所以能作為以色列的
造物者發言，是因為他代表並領導著耶洛因的工程。加百列可以帶著其背後的權威
傳遞一則訊息。《古蘭經》可以藉由尋回一條單一的效忠脈絡，正確地打斷那條由
死寂形象與承襲說情者所構成的鏈條——即便在 Wheel of Heaven 的解讀裡，後來的
穆斯林神學把那條脈絡末端的權威，誤認作了一位非物質的絕對者。

最後那句話，標出了本文中最鋒銳的一道界線。伊斯蘭教並不教導這個。《古蘭經》
一再對阿拉使用單數的創造語言，否認有夥伴，並拒絕神有後裔。古典的*認主獨一*
（tawhid），在其根基之處便排除了耶洛因假說。因此 Wheel of Heaven 可以說，
伊斯蘭教的介入在**功能上是矯正性的**，同時卻與它的**神學自我解釋**相異。它
把注意力從無能的膜拜器物與未經授權的中介者，尋回到創造的源頭；而正典隨後對
那個源頭究竟是什麼，給出了一個不同的說法。{{ footnote(id="8") }}

## 這一解讀所能解釋之物

唯有當這一假說能夠解釋那些被標準的二分法弄得尷尬的文本特徵時，它才配得被
認真對待。

**它解釋了為何排他的誡命會與複數的天界並存。**「在我以外不可有別的耶洛因」、
神的兒子們、詩篇 82 篇的那個議會、保羅的眾神與眾主，以及《古蘭經》的天使與
精靈，都不再看起來像是一神論內部令人難堪的殘骸。那些存有可以留存，而祭祀與
效忠則收窄。

**它解釋了為何反偶像的論戰攻擊的是功能，而非製造。**先知們反覆說一尊像不能
聽、不能說、不能助、不能害、不能救。那些都是對能動性的檢驗。問題主要不在於
雕像的數目，而在於它們未能去做那些活的代表與被記住的製造者據說能做的事。

**它解釋了那反覆出現的尋回語法。**亞伯拉罕站在
[金牛座時代](/timeline/age-of-taurus/)的邊緣；摩西在金牛犢之後，於
[白羊座時代](/timeline/age-of-aries/)把以色列帶回聖約；先知們控告百姓遺忘；
耶穌把聽眾召回到律法中那更重之事；而《古蘭經》則在
[雙魚座時代](/timeline/age-of-pisces/)之內，稱亞伯拉罕既非猶太人亦非基督徒，
並命穆罕默德遵循他的宗教。新的訊息把自己呈現為尋回，因為依這一假說，那被
尋回的歷史是同一段。

**它解釋了在議會與單一發言者之間的那種擺盪。**古代文本可以顯示會眾在作決定，
隨後又讓一個名字代表全體發言。建制性的代表，在政治與軍事的記錄裡是尋常之事。
唯有在那個單數的發言者被界定為唯一可能的神聖存有之後，它才變成一個矛盾。

**它解釋了為何「一」如此頻繁地是倫理與政治的。**《示瑪》（Shema）把耶和華的
合一，繫連於全然的愛。保羅把一位神、一位主，繫連於一個不能在對手的席上進食
的共同體。《古蘭經》把神聖的合一，繫連於崇拜、公義、施捨，以及對承襲下來之
權威的拒斥。「合一」使一個民族凝聚成形，早於它使一套形上學變得簡單之前。

## 這一解讀可能失效之處

一個能解釋一切的假說，通常只是一套拒絕認輸的詞彙。這一個需要一些堅硬的
稜角。

**第一項反對意見是歷史發展。**主流學術可以在不訴諸外星記憶的情況下解釋這些
證據。古代以色列從一個西閃族的宗教環境中浮現。耶和華吸收了種種稱號與職能；
王朝與被擄的危機驅動了排他的崇拜；抄書吏保存、改寫，有時又降格了更古老的
神聖議會語言。基督教與伊斯蘭教承襲了那份檔案，並強化了它那些普世的主張。
這個模型握有文本、銘文、考古，以及種種尋常的文化變遷機制。而 Wheel of
Heaven 的模型，則添上了一個隱藏的歷史所指，一個沒有任何已被接受的銘文將其
辨識為技術文明的所指。

那是最強的反對意見，而它站得住腳。本文並不推翻它。本文所主張的是：由此得出
的那些範疇，仍然錯誤地描述了文本的表面，而一個記憶的假說依然是可能的——並
不是說這種可能性擊敗了那段更有佐證的歷史。

**第二項反對意見是：反偶像的論戰終究是論戰。**倘若聖經與《古蘭經》的作者把
對手漫畫化了，那麼他們的修辭便不能被當作一扇清晰的窗，去窺看一場在先的記憶
危機。那個「啞巴雕像」的論證，所告訴我們的，或許更多是關於群體的邊界劃定，
而非關於偶像崇拜取代了什麼。G·R·霍廷的研究對伊斯蘭教發出同樣的警告：論戰中
的「以物配主者」，不能被徑直變成一份對前伊斯蘭宗教的照相式記述。

**第三項反對意見是範疇替換。**把「神」替換成「先進存有」，並不會自動尋回
歷史。它可能只是把古代神話，翻譯成技術當下的那套威望語言。蘇珊·帕默爾
（Susan Palmer）與喬治·克里賽德斯（George Chryssides）在
{% wiki(slug="raelism") %}雷爾運動{% end %}中所描述的，正是這一操作：超自然的
敘事變成了科學的敘事，卻保留著宗教的結構。斯特凡諾·比利亞爾迪（Stefano
Bigliardi）把這一批評推得更遠，論證說這場運動借用科學的權威，要比借用科學的
方法更為爽快。因此，一套負責任的 Wheel of Heaven 詮釋學，必須指明為何某個
細節更宜於作操作性的解讀，而不是假定每一片雲都是一艘飛行器、每一位信使都是
一名太空人。[*會吏長與龍*](/articles/the-archdeacon-and-the-dragon/)藉由把一種
結構性的趨同、與一場失敗的詞源學區分開來，示範了那一區別。

**第四項反對意見來自那些傳統本身。**猶太教的一神論，不能被化約為一個被誤譯
的複數。基督教三位一體的「一」，不是一個單純數目上的一。伊斯蘭教的阿拉，不是
一個被誤聽成一位個體的集合體。十四個世紀的*認主獨一*（tawhid），包含著關於
合一、屬性、因果與中介的種種精微論證，而這些正是這一假說所直接牴觸的。智識
上的誠實，要求在尋找接觸點之前先把這一點說出來。

**第五項反對意見關乎世系。**亞伯拉罕—夏甲—以實瑪利這條線，是一份有力的經文與
文化譜系。它並不是一條已被證明的、從一個青銅時代的家戶、經由納巴泰人、直到
早期穆斯林的遺傳路線。那尋回的論證，所倚靠的是被記住的亞伯拉罕身份與《古蘭
經》的自我呈現，而不是去證明一棵家族樹上的每一個環節都是生物學上的。

## 什麼會改變這一裁決

有幾項發現會強化這一假說。一份早期的文本，若以毫不含糊的、非超自然的語言，
把一個活的造物者群體與其膜拜神像區別開來，那將是實質性的。同樣有分量的，
還會有一些彼此獨立的記錄——它們橫跨各個文化、卻無文學上的相互依賴，辨識出
同一位有名有姓的代表、同一個機構，以及同一項技術性的操作。而在相關的古代
脈絡中，若有物質證據指向一套先進的製造或通訊系統，那將把這一論證從詮釋學
轉變為歷史。

倘若那些複數與議會的文本被證明一律是晚出的、詩體的、且依賴於一個狹窄的
抄書吏傳統，這一解讀就會削弱；它們並非如此，但它們的分佈確有干係。倘若反
偶像的論戰始終如一地只針對「雕像本身創造了宇宙」這一信念，這一解讀也會削弱，
因為那樣一來，記憶與中介的模型便無所增益。倘若「一」的語言能被表明是永遠地、
處處地在否認每一位別的神聖存有，而非在安排祭祀與效忠，這一解讀還會削弱。而
那些源頭文本，早已使那個普世性的主張難以維持。

而最具決定性的是：倘若那個技術性的層次，永遠產生不出任何為尋常宗教發展所
無法提供的證據，那麼它就應當被放棄。一副只是把諸神重新命名為訪客的鏡片，
只是裝飾而已。一個嚴肅的假說，必須預言文本、考古或物質的痕跡究竟應在何處
呈現差異。

## 雕像與空蕩的天空

回到那個博物館的展櫃。雕像沉默不語。先知在這一點上說對了。但從它的沉默裡，
可以引出兩種錯誤。

第一種，是那論戰所攻擊的古代錯誤：把本屬於它所代表之物的信任、敬畏、獻禮與
順從，交給了那件器物。第二種，是現代的錯誤：想像一旦那件器物被揭穿為木頭或
石頭，那麼唯一的選項就是一個由一位看不見的、非物質的心靈所統治的空蕩宇宙。

那些古老的文本留下了更多敞開的可能。它們記得議會、諸子、信使、民族守護神、
鬼魔、對手之主，以及有名有姓的行事者。它們同時又要求一份忠誠、一份聖約、
一個律法之源、一條經授權的道路，以及一套關於創造的說法。把整份第一份清單
叫作「多神論」、把整份第二份清單叫作「一神論」，便遮蔽了那使前者變成後者的
機制。

Wheel of Heaven 尋回的是一個不同的座標軸。不是**一位神抑或眾多神**，而是
**活的製造者抑或死寂的替代品；被記住的源頭抑或承襲來的記號；經授權的代表
抑或層層堆積的中介者**。在那個座標軸上，伊斯蘭教的*認主獨一*（tawhid），可以
被認作歷史上偉大的矯正之舉之一，而不必被弄得去說它並未說過的話。它打破了
器物的權威，並把亞伯拉罕召回為那位立於諸般分裂之前的人物。本典籍認同那份
矯正，而對隨之而來的形上學提出異議。

這就是這一假說最強的、可辯護的形式。先知們並不必然是在清空天堂。他們是在
清理那條視線。
