# Newsroom

Der Newsroom liest **aktuelle Ereignisse** durch den Korpus. Jeder Eintrag — ein *Dispatch* — knüpft an etwas Geschehenes an: eine Entdeckung, eine Ankündigung, einen Jahrestag, einen kulturellen Moment. Der Winkel ist das, was der Korpus daraus macht.

## Dispatches vs. Explainers

Dispatches sind kurz, datiert und vergänglich. Sie wollen nicht endgültig sein — sie wollen *aktuell* sein. Wenn Sie die zeitlose, thesengetragene Behandlung eines Themas suchen, lesen Sie die [Artikel](/de/articles/), wo Explainers die langsame Arbeit leisten.

Ein Dispatch kann einen späteren Explainer säen; ein Explainer wird nie auf einen Dispatch reduziert.

## Was Sie hier nicht finden

- Breaking-News-Aggregation. Wir wollen keine Nachrichten­agentur sein.
- Sensationelle Aufmachung. Dasselbe wissenschaftliche Register wie im Rest der Seite.
- Missionierung. Dispatches laden zur Untersuchung ein; sie ziehen keinen Schluss für den Leser.
