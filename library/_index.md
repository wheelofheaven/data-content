# Library

The Library holds the primary texts the project reads from — not commentary on them, but the texts themselves, presented for direct reading.

At the centre are the Raëlian source books, which the project takes as its foundational canon. Around them sit the biblical, Mesopotamian, Vedic, and other ancient writings the canon is read alongside. Each book renders in an interactive reader with original-language script, transliteration, and translation side by side where the corpus has them.

If you want the interpretive layer — what the project makes of a passage — start in the [timeline](/timeline/) or the [encyclopedia](/wiki/). The Library is where you read the texts on their own terms first.
