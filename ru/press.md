## Название бренда — как его писать

<div class="press-usage">
  <div class="press-usage__col press-usage__col--do">
    <h3>Используйте</h3>
    <ul>
      <li><strong>Колесо Небес</strong> — название проекта (в скобках при первом упоминании — <em>Wheel of Heaven</em>)</li>
      <li><strong>Элохим</strong> — всегда во множественном числе («Элохим <em>были</em>…»)</li>
      <li><strong>Раэль</strong>, <strong>раэлианский</strong> — с буквой «э»</li>
      <li><strong>рабочая гипотеза</strong> — когда описываете центральное утверждение</li>
      <li><strong>WoH</strong> — только после первого упоминания, в основном тексте</li>
    </ul>
  </div>
  <div class="press-usage__col press-usage__col--dont">
    <h3>Не используйте</h3>
    <ul>
      <li><s>КОЛЕСО НЕБЕС</s> — заглавные буквы зарезервированы для бренд-чипа</li>
      <li><s>колесонебес</s> — слитное написание снижает читаемость</li>
      <li><s>Элохим есть…</s> — существительное грамматически во множественном числе</li>
      <li><s>Раелианский</s>, <s>Раелизм</s> — сохраняйте написание через «э»</li>
      <li><s>доктрина</s>, <s>теория</s>, <s>система верований</s> — говорите <em>прочтение</em></li>
      <li><s>древние инопланетяне</s>, <s>тайная история</s> — это иные рамки, которые мы не разделяем</li>
    </ul>
  </div>

(function(){
  function copyHandler(e) {
    const btn = e.target.closest('.press-boilerplate__copy');
    if (!btn) return;
    e.preventDefault();
    const text = btn.dataset.copy || '';
    const done = () => {
      const original = btn.dataset.original || btn.textContent;
      btn.dataset.original = original;
      btn.textContent = 'Скопировано';
      btn.classList.add('is-copied');
      setTimeout(() => {
        btn.textContent = original;
        btn.classList.remove('is-copied');
      }, 1600);
    };
    if (navigator.clipboard && window.isSecureContext) {
      navigator.clipboard.writeText(text).then(done).catch(() => {});
    } else {
      const ta = document.createElement('textarea');
      ta.value = text;
      ta.style.position = 'fixed'; ta.style.opacity = '0';
      document.body.appendChild(ta); ta.select();
      try { document.execCommand('copy'); done(); } catch (_) {}
      document.body.removeChild(ta);
    }
  }
  document.addEventListener('click', copyHandler);
})();
</script>
