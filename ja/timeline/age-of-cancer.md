## I. この時代そのもの

第七の時代は帰結の時代です。

蟹座の時代は紀元前8,850年から紀元前6,690年までの2,160年間にわたり、獅子座の時代の直後に続きます。これは、獅子座の終わりに完了した創造プログラムが創造史へと変わる時代——前の時代に造られた存在たちが、行為する能力を持つ存在たちのなすことを始める時代です。彼らは自らの状況を発見します。彼らは造り主に背きます。彼らは最初に置かれた整えられた環境から切り離されます。彼らはそれまで園の中からしか見ていなかったより広い世界へと出ていきます。彼らは生殖を始めます。彼らの子らは実験室の外で生まれます。第一世代の一部は並外れて長い生涯を生きます。追放された創造者たちは人間の女たちを妻にとり、混血の子孫を生みます。超大陸全体に広がる七つの文明は成長し、発展し、交易し、戦い、遠い惑星から見守る母星を警戒させ始めるような種類の力を蓄積します。時代の終わりまでに、地球の状況は、母星政府が介入は避けられないと結論するに至るまで進展しています。次章で扱う{% wiki(slug="great-flood") %}大洪水{% end %}は、本章が描く数世紀のあいだに不可避となります。

この時代は、ラエリアン的読みにおいて、創世記1章の創造記述と創世記2章の{% wiki(slug="eden") %}エデン{% end %}物語との移行に対応し、創世記3章(追放)、創世記4章(カインとアベル)、創世記5章(長寿)、そして創世記6章冒頭の数節({% wiki(slug="elohim") %}エロヒム{% end %}の子らと人の娘たち)へと続きます。聖書本文はこれらの出来事を別個の時代に区切ってはいません。本書が採用するラエリアン的読みは、それらすべてを蟹座の中に置きます。なぜなら、それらは単一の延長された物語の弧——獅子座の終わりに園にいた人間たちに始まり、次の時代に洪水を引き起こすことになる政治的条件で終わる弧——を構成しているからです。

本章の表題——『安息の日』——は、創世記の連なりにおいて蟹座が何であるかを名指しています。創世記2章は、業が完了した後に創造者たちが何をしたかについての特定の記述で始まります。Vayechulu ha-shamayim ve-ha-aretz ve-khol tzeva'am. Vayechal Elohim ba-yom ha-shevi'i melachto asher asah, vayishbot ba-yom ha-shevi'i mi-kol melachto asher asah. 「こうして、天と地と、そのすべての万象が完成された。そして第七の日に、エロヒムは自らがなした業を終え、第七の日に、自らがなしたすべての業から休まれた。」創造の連なりは終わりました。安息の日が始まったのです。人間たちは園の中にいます。そして、ほぼ即座に、物事は誤った方向に進み始めます。安息の日はある意味で誤称されています——創造者たちは創造することから休みましたが、彼らの被造物は存在することから休んだわけではなく、安息期間中に被造物が始めたことこそが、ヘブライ語聖書の残り全体と人類史の残り大半を占めることになります。安息の日は、本書の読みにおいて、本書の枠組み全体のもっとも重大な出来事が起こった日です——造り主が能動的だったからではなく、造られた者たちが今や、初めて、自らの権利において能動的になり、造り主が休んでいる間に彼らがなしたことが、その後のすべてを規定したからです。

本章は、典拠と聖書本文がともに描く出来事を通じて蟹座の時代を歩んでいきます。これらの出来事が前提とする政治的類型——{% wiki(slug="satan") %}サタン{% end %}、{% wiki(slug="lucifer") %}ルシファー{% end %}、{% wiki(slug="serpent") %}蛇{% end %}、{% wiki(slug="yahweh") %}ヤハウェ{% end %}という人物たち——は、本来の物語が始まる前に専用の節を与えられます。なぜなら、この四人構成は本質的であり、後の宗教伝統によって、実際に起こったことを覆い隠す仕方で混乱させられてきたからです。{{ footnote(id="1") }} 次に、目覚め、追放、地球上に追放者共同体を生み出した政治的決着。次に、長き世代、選ばれた族長たちに与えられた{% wiki(slug="tree-of-life") %}命の木{% end %}の技術、最初の人間的暴力の事例としてのカインとアベルの対立。次に、エロヒムの子らとその混血の子孫である*ネフィリム*、そしてこれらの出来事のもっとも明示的な古代記録を保存する前ラビ期文学の実質的な体系——なかでも重要な{% libref(book="book-of-enoch") %}エノク書{% end %}{{ cite(id="4") }}。次に、より広い世界——超大陸全体に広がる七つの文明、交易路、戦争、エデン文明の漸進的な技術発展、母星が介入する以前にもっとも高度な人間文化がどこまで進んだかについての考察。次に、蟹座の科学——長寿、混血生物学、洪水以前文明の考古学に関わる現代の研究プログラム。そして最後に締めくくり——母星が当初から恐れていたものが今や現実となりつつあるという認識、そして双子座の時代の破滅的な出来事への暗黙の伏線。

## II. 諸節

蟹座の出来事を扱うヘブライ語本文は、これまでのどの時代の本文よりも広範であり——{% libref(book="genesis-woh", chapter=2, verse=1) %}創世記2章1節{% end %}から{% libref(book="genesis-woh", chapter=6, verse=8) %}創世記6章8節{% end %}まで——ヘブライ語聖書全体の中でももっとも重大な語彙のいくつかを含んでいます。本章ですべての節を完全な注解装置で扱うことはできませんが、主要な箇所は確立されたブロック形式での丁寧な提示に値します。

第七日の諸節、{% libref(book="genesis-woh", chapter=2, verse=1) %}創世記2章1–3節{% end %}は、創造後の期間を開きます。
{{ scripture(book="genesis-woh", chapter=2, verse=1, translit="Vayekhulu ha-shamayim ve-ha-aretz ve-khol-tzeva'am") }}{{ scripture(book="genesis-woh", chapter=2, verse=2, translit="Vayekhal Elohim ba-yom ha-shevi'i melachto asher asah, vayishbot ba-yom ha-shevi'i mi-kol-melachto asher asah") }}{{ scripture(book="genesis-woh", chapter=2, verse=3, translit="Vayivarech Elohim et-yom ha-shevi'i vayekadesh oto, ki vo shavat mi-kol-melachto asher-bara Elohim la'asot") }}
三つの動詞が第七日を構造化しています——שָׁבַת (*shavat*、「止む、休む」)、בָּרַךְ (*barakh*、「祝福する」)、קָדַשׁ (*qadash*、「聖別する、聖なるものとして区別する」)。動詞 *shavat* は名詞 שַׁבָּת (*shabbat*)、安息日の語源です。第七日の諸節の三重の構造——業は完了し、業は止み、その日は祝福され聖別された——は、その日を先行する六日間と範疇的に異なるものとして確立します。先行する日々は創造の日々でした。第七日は創造が起こらない日であり、まさに創造が起こらないがゆえに区別されています。ユダヤ教の毎週の安息日遵守の伝統は、この構造を圧縮された形で保存していますが、本書は元の指示対象を元の規模で読み解きます——第七日は蟹座の2,160年間であり、創造者たちが能動的な創造作業を止めたが、造られたものへの関与を止めることはなかった時代です。

מְלַאכְתּוֹ אֲשֶׁר־בָּרָא אֱלֹהִים לַעֲשׂוֹת (*melachto asher bara Elohim la'asot*) という表現は注目に値します。この構文は、三つの創造/制作の動詞を一つの句に組み合わせています——*melacha* (業、労働、技)、*bara* (もっとも強い創造動詞)、*asah* (構築・整序の動詞)。ヘブライ語は今や何が完了したかについて強調的です——創造することと作ることの業は完成しました。安息期間中、この種のことはもはや何も行われません。続くのは観察、評価、関与、介入であり——新たな創造ではありません。

{% libref(book="genesis-woh", chapter=2, verse=7) %}創世記2章7節{% end %}は、本書が一貫して読んできた言葉でアダムの形成を記録します。
{{ scripture(book="genesis-woh", chapter=2, verse=7, translit="Vayitzer Adonai Elohim et-ha-adam afar min-ha-adamah, vayipach be-apav nishmat chayim, vayehi ha-adam le-nefesh chayah") }}
動詞 יָצַר (*yatzar*)、「形作る、整える」——陶工が粘土に対してする行為を表す動詞——がここで選ばれており、(創世記1章を通じて現れた) *bara* とは区別されます。*Yatzar* は、より特定的で、より手作業的で、より工芸的な操作を描きます——素材を形へと成形すること。創世記2章の物語はイスラエル・チームの具体的な手順選択についてのより詳細な記述であり、動詞選択はこれを反映しています——エデンの人間の創造は、典拠の言葉で言えば、地球上の素材からの文字通り手作業による合成であり、*afar min ha-adamah* (「地のちり」) はその合成の文字通りの基質でした。鼻孔に *nishmat chayim* (「命の息」) を吹き込むことは活性化の瞬間です——合成された形が生きた存在となる地点。ヘブライ語の表現は、射手座の章で扱った *ruach Elohim* が水の面を動いていたことと響き合います——命の息は、生物物質を生きた有機体にする能動的原理です。

{% libref(book="genesis-woh", chapter=3, verse=1) %}創世記3章1節{% end %}は蛇を導入します。
{% scripture(book="genesis-woh", chapter=3, verse=1) %}
さて、蛇はヤハウェ・エロヒムが造った野のあらゆる獣よりも狡猾であった
{% end %}
ヘブライ語 נָחָשׁ (*nachash*) は蛇を表す標準的なヘブライ語ですが、עָרוּם (*arum*) という語は珍しく、注意に値します。これは文脈に応じて「狡猾」「鋭い」「微妙」「思慮深い」を意味し得ます——そして決定的なことに、異なる文脈での同じ語はかなり異なる価値を帯びます。箴言において、*arum* は知恵と良き判断を表す肯定的な性質として一貫して使われています。創世記3章では、否定的な性質(狡猾な欺瞞)として読まれてきました。しかし同じ語根 ערם は、単に「知覚する」「識別する」を意味することもあります。ラエリアン的読みにおいては、語の中立的な、あるいはむしろ肯定的な価値こそが適合します——ルシファー派は創造者集団のうちでもっとも知識豊富で知覚に優れており、まさにそれゆえに人間たちの状況を十分に理解し、それを開示しようとしました。微妙さは道徳的欺瞞ではなく、理解そのものです。

{% libref(book="genesis-woh", chapter=3, verse=7) %}創世記3章7節{% end %}は食後の瞬間を記録します。
{% scripture(book="genesis-woh", chapter=3, verse=7) %}
そして両者の目が開かれ、彼らは自分たちが裸であることを知った
{% end %}
動詞 פָּקַח (*pakach*)、「(目を)開く」は、ヘブライ語聖書の他の箇所で盲人の目が開かれることに用いられるのと同じ動詞です。この表現は彼らの文字通りの目が開いたという意味ではありません(彼らの目はすでに開いていました)。彼らは知覚に至った——突然、自らの状況を明確に見たのです。ヘブライ語の言葉遊びにも注目してください——人間たちは *arumim* (裸である状態、蛇の知覚の性質である *arum* と関連) でした。本文はその二重の意味で遊んでいます。人間たちは、*nachash* (蛇) から *arum* (知覚) を受け取り、今や自らの *arumim* (裸/曝された状態) を認識します。ヘブライ語は概念的な動きを語根の戯れに圧縮しています。

{% libref(book="genesis-woh", chapter=3, verse=22) %}創世記3章22–24節{% end %}は追放と警備の手配を記録します。
{{ scripture(book="genesis-woh", chapter=3, verse=22, translit="Vayomer Adonai Elohim: hen ha-adam hayah ke-achad mimenu la-daat tov va-ra, ve-atah pen-yishlach yado ve-lakach gam me-etz ha-chayim, ve-akhal va-chai le-olam") }}
כְּאַחַד מִמֶּנּוּ (*ke-achad mimenu*)、「我々のひとりのように」という表現は、エデン物語のもっとも印象的な文法的特徴の一つです。動詞は三人称単数 (「ヤハウェ・エロヒムは言われた」) ですが、述部は複数 (「我々のひとりのように」) です。ヘブライ語は全体を通じて複数のエロヒムを保存しています——追放判決の瞬間にあってさえ、話し手は人間が今や属することになった「我々」に言及しています。ラエリアン的読みは、この複数性を額面通りに受け取ります——造り主は複数であり、人間の違反は、彼を禁じられた知識を持つ存在の範疇に置いたのです。
{{ scripture(book="genesis-woh", chapter=3, verse=23, translit="Vayeshalchehu Adonai Elohim mi-gan-eden la'avod et-ha-adamah asher lukach mi-sham") }}{{ scripture(book="genesis-woh", chapter=3, verse=24, translit="Vayegaresh et-ha-adam, vayashken mi-kedem le-gan-eden et-ha-keruvim, ve-et lahat ha-cherev ha-mithapechet, lishmor et-derech etz ha-chayim") }}
לַהַט הַחֶרֶב הַמִּתְהַפֶּכֶת (*lahat ha-cherev ha-mithapechet*) という表現は綿密な検討に値します。*Lahat* は「炎」「刃」あるいは「刃のような炎」を意味します。*Cherev* は剣を表す標準的なヘブライ語です。*Mithapechet* は הָפַךְ (*hafach*、「回転する、ひっくり返す」) の現在再帰分詞であり、「自らを回す」あるいは「あらゆる方向に回る」を意味します。この表現は、炎を発し、回転または旋回する武器を描いています。ラエリアンの典拠はこれを指向性エネルギー兵器として読みます——ヘブライ語は、人間の証言者が利用可能な語彙で、そのような兵器の視覚的外観(回転する刃のような炎)を描写しています。ケルビム——כְּרֻבִים、*keruvim*、*keruv* の複数形——は後のキリスト教図像学にあるふくよかな翼ある天使ではなく、特定の階級の守護者であり、ここでは創造者の施設の周辺に配置された武装した歩哨として機能しています。

{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記6章1–4節{% end %}はエロヒムの子らを導入します。
{{ scripture(book="genesis-woh", chapter=6, verse=1, translit="Vayehi ki-hechel ha-adam la-rov al-penei ha-adamah, u-vanot yuldu lahem") }}{{ scripture(book="genesis-woh", chapter=6, verse=2, translit="Vayir'u venei-ha-Elohim et-benot ha-adam ki tovot henah, vayikchu lahem nashim mi-kol asher bacharu") }}
בְּנֵי־הָאֱלֹהִים (*benei ha-Elohim*)、「エロヒムの子ら」という表現は、ヘブライ語聖書の洪水以前物語全体の中で、ある存在の範疇に対するもっとも具体的な記述です。構文は単純なヘブライ語です——*benei* (の子ら、連語形)、*ha-Elohim* (定冠詞付きのエロヒム)。この表現はこれらの存在をエロヒムの範疇に属するものとして同定します——創世記1章の創造者と同じ範疇です。慣例的な神学的読みは、この表現を同化させるのに苦労しており、*benei ha-Elohim* を堕天使、セツの子孫(かなり後代の解釈)、あるいは半神とみなされた古代の王として多様に同定してきました。ラエリアン的読みは構文を額面通りに受け取ります——*benei ha-Elohim* はエロヒムの範疇のメンバーであり——具体的には、地球上に住む追放された創造者たちであり——今や人間の女たちと生殖しているのです。
{{ scripture(book="genesis-woh", chapter=6, verse=4, translit="Ha-Nefilim hayu va-aretz ba-yamim ha-hem, ve-gam acharei-khen, asher yavo'u benei ha-Elohim el-benot ha-adam ve-yaldu lahem, hemah ha-giborim asher me-olam anshei ha-shem") }}
נְפִלִים (*ネフィリム*) という語は語根 נפל (*n-p-l*、「落ちる」) から来ています。文字通りの意味は「落ちた者たち」——どこか別の場所から落ちた、降りた、来た者たちです。「巨人」という伝統的な訳は七十人訳の *gigantes* の選択に由来します{{ cite(id="6") }}が、ヘブライ語の語根の意味のほうがより明瞭です——彼らは降りてきた者たちの子孫です。本章は§VIIでこの語に立ち戻ります。

これらが蟹座を構造化する主要なヘブライ語の箇所です。本章の後続の節では、これらの箇所が描く神学的・歴史的内容を扱います。

## III. 政治的構図——サタン、ルシファー、そして蛇

危機の物語が適切に語られる前に、本書の残り全体を通じて名前が繰り返し現れる人物たちを導入することが必要です。彼らの正体について、ラエリアンの典拠は、後の宗教伝統が覆い隠してきた仕方で明確化しています。この節は本章の構成において異例です——大半の章は本来の話に進む前に登場人物の紹介で立ち止まることはしません——しかし、蟹座が前提とする四人構成の政治的類型は本質的であり、それを明確に確立できなければ、後続の節は支離滅裂になってしまいます。

サタン、ルシファー、蛇という名前は、ほとんどの西洋宗教伝統において、互換的に扱われてきました——単一の敵対的人物の三つの名前として、通常は堕天使と同定され、ときに悪魔と、稀には悪の神話的具現と同一視されてきました。ラエリアンの典拠はこの同定を全面的に拒否します。三つの名前は三つの異なる存在を指し示しており、それぞれ異なる役割、異なる場所、異なる政治的立場を持っています。それらを区別できなかったこと——2,000年以上にわたって宗教伝統に組み込まれてきた失敗——は、ヘブライ語聖書が描いている出来事の実際の構造を覆い隠しています。

**サタン**は、典拠の説明によれば、母星にいるエロハです。彼は地球上にはいません。彼が地球にいたことはありません。彼は、当初から、エロヒム文明において、自分たちと同じ姿の存在の創造に反対する政治派閥の指導者です。彼の立場は地球プログラム開始前から一貫しています——造り主と同等あるいはそれを超え得る合成存在の創造は根本的に危険であり、いかなる議定書、いかなる監督、いかなる地理的距離もそのリスクを封じ込めると信頼することはできません。母星での当初の実験室事故が最初の死者を生じさせたとき、サタン派はこの事件を利用して母星の生物学プログラムを停止に追い込みました。科学者たちが研究を続けるために地球に移動したとき、サタンと彼の派閥は地球プログラムを疑念をもって監視し、永遠者の{% wiki(slug="council-of-the-eternals") %}評議会{% end %}を通じて定期的に介入して制限を課しました。そして本章の出来事のあいだ、人間の創造が不服従と、サタンが当初から予測していた種類の振る舞いに能力を発揮することが証明されたとき、母星の評議会の議場でもっとも声高に響くのはサタンの声であり、造られたものの破壊を求めて呼びかけることになります。サタンは神話的な悪魔ではありません。彼は政治的反対者であり、その反対は彼自身の見地からは原則的なものです——彼は人間の創造が当初から誤りであったと心から信じているのです。彼が正しいかどうかは、本書が最終的に問うている問いの一つです。

典拠自身の言葉はこの点について明示的です——*「サタンはエロヒムのひとりに過ぎず、ある意味で、彼らの惑星の政党を率いていた。それは、自分たちが肯定的で非暴力的な存在を創造できると考える他のエロヒムによる、彼ら自身の姿をした人工存在の創造に反対する党であった。」* この特徴づけは悪魔化でも擁護でもありません。それは描写的です。サタンは政治家であり、悪魔ではありません。彼は党を率いています。党には綱領があります。綱領には議論があります。議論は、本章の出来事および続く章の出来事を通じて、人間たちが実際に何をするかによって試されているのです。

ヘブライ語 שָׂטָן (*satan*) 自体も注目に値します。語根の意味は「敵対者」あるいは「告発者」——*satan* は反対する者、立ち向かう者、訴追側として論じる者です。ヨブ記において、*ha-satan* (定冠詞付き) は天上法廷の特定の職能者であり、正しい者たちに対して異議を申し立てて試すのが役割です——彼は悪魔的人物ではなく、一種の検察官です。後のキリスト教および黙示文学による *Satan* を宇宙的悪の具現とする同定は、本来の意味からの発展であり、その本来の内容ではありません。ラエリアンの典拠の読みは、より古いヘブライ語の意味と一致します——サタンは反対党の指導者であり、創造プログラムの主たる敵対者であり、プログラムがしてきたことに対して一貫して反対してきた者です。彼の反対は政治的であり、形而上学的ではありません。

**ルシファー**は対照的に、地球にいます。彼の名は「光をもたらす者」を意味します——ラテン語の *lux* (「光」) と *ferre* (「運ぶ」) から。彼は地球で生命を創造したエロヒムのひとりであり、具体的にはエデンの地の人間を生み出したイスラエル・チームのメンバーであり、評議会が秘匿を命じたことを開示しようとする愛情に駆られたチーム内の小集団の指導者です。ルシファーは人間の敵対者ではありません。彼はその逆です——人間に真実が告げられることを望み、人間は自分たちを誰がどのように造ったかを知るに値すると信じ、評議会の無知の維持命令を、人間自身が理解すれば拒絶するであろう家父長主義の行為とみなす者です。

典拠によるルシファーの記述は本書全体における人物の特徴づけのうちでもっとも具体的なもののひとつであり、長文の引用に値します——

*「次に来たのはルシファー、これは『光をもたらす者』を意味する。ルシファーは地球で生命を創造したエロヒムのひとりであり、こうして人間を創造した。ルシファーは、最初の合成人間の行動を研究する遺伝子工学研究所の一つで働く科学者の小集団を率いていた。彼らの被造物が示す並外れた適性に気づいたルシファーは、命令を無視して最初の人間たちに、彼らが『神々』だと思っていた者たちが実際は自分たちと同じ人間であり、肉と血からなり、触れることのできる物質で作られた飛行機械で別の惑星から来たのだと明かすことを決意した。ルシファーと、彼に従ったエロヒムたちは、自らが合成的に創造した人間たちに対して愛情と慈愛を感じていた。彼らは一日中観察するこれらの存在を、自らの子のように愛し始めた——『神々』と見なすことを強いられていたこれらの存在を。彼らは、自分たちの被造物が——身体的にも心理的にも成功し、美しく知的に見える者たちが——偶像を崇めるかのように手と膝をついて自分たちを崇拝するのを耐えられなかった。これらすべては、ヤハウェを大統領とする彼らの故郷の惑星の政府が、被造物に真実を告げることを厳しく禁じ、エロヒムに永遠に超自然的存在の役割を演じることを強いたためであった。」*

この一節は同時にいくつものことを確立します。第一に、ルシファーは抽象的でも象徴的でもありません——彼は特定の研究所で特定の科学者集団を率い、特定の決定をした特定のエロハです。第二に、動機は愛情でした——創造者の被造物への愛が、緊密な日常的観察によって強められ、チームが受けていた政治的命令を覆すのに十分なまでになりました。第三に、開示の具体的内容が指名されています——人間たちは、自分たちの造り主が「神々」ではなく、「自分たちと同じ人間、肉と血からなり、触れることのできる物質で作られた飛行機械で別の惑星から来た者」だと告げられました。開示は漠然とした霊的啓発ではありませんでした。それは、人間の造り主が人間自身と同じ根本的種類の技術的存在であるという具体的な事実の啓示でした。第四に、エロヒムが演じることを強いられていた役割——手と膝をついた存在たちから崇拝を受ける「超自然的存在」の役割——は、典拠の説明では、もっとも愛情深い者たちにとって耐え難いものでした。開示は愛と、開示する造り主たちの見地においては、重要なすべての点で自らの造り主と同等である存在から崇拝を受け続けることへの関連した拒絶によって駆動されたのです。

典拠はルシファーの立場をサタンとヤハウェの両方に直接対立するものとして枠付けます——*「ルシファー、『光をもたらす者』は、創造者が『神々』ではなく自分たちと同じ人間であると明かすことで、最初の人間たちを啓蒙した。この態度は、人間からは悪しか期待できないと考えるサタンの態度、また、エロヒムの惑星を統治する永遠者の評議会の大統領であるヤハウェの態度とも、直接対立する。」* ルシファーはこの三角構成の第三の立場を占めています。サタンが人間が決して造られなかったほうがよかったと考えるのに対し、ルシファーは人間が同等者として造られ、造り主が知るすべてへのアクセスを与えられ、永遠の幼児期に置かれるべきペットとしてではなく、成熟可能な子供として扱われるべきだと信じています。

典拠はここで印象的な括弧書きを加えており、後の宗教の図像学的伝統がどう発展した可能性があるかを示唆するものとして、記録に値します——*「ここまで、角のある被造物はいない。」* 典拠は、特徴的な乾いた調子で、実際の政治的状況には、後にキリスト教図像学においてルシファー像に付着することになる角や悪魔的特徴は一切含まれていないことを観察しています。悪魔はいません。意見を異にする科学者たちがいるのです。角は後から付加されたのであり、実際の状況を本来の言葉で描写できなくなった伝統によって人物像に取り付けられたものです。

後のキリスト教によるルシファーとサタンの同定——これら二つの人物を、神に対立する単一の「堕天使」へと融合させること——は、ラエリアン的読みにおいて、破滅的な神学的誤りです。両者は対立しているのであり、同盟しているのではありません。ルシファーの「堕落」は、評議会への不服従に対する罰としての母星からの追放ですが、その不服従は被造物を愛しすぎたことに存し、それに反対したことではありませんでした。「堕落」は地理的・政治的移転であり、道徳的腐敗ではありません。この読みにおけるルシファーは、悪意ではなく勇気と慈悲の人物像に近いものです。後のサタンとの混同は、その後の数世紀の翻訳上および神学上の圧力を反映しており、物語の本来の構造ではありません。

創世記3章の**蛇**は、典拠の説明によれば、ルシファー率いるエロヒムの小集団です——ルシファー単独ではなく、派閥全体です。ヘブライ語 *nachash* (蛇) は、この読みにおいて集合的に集団全体を描写するために用いられます。「蛇」は、開示の具体的行為を行うルシファー派です——人間に語りかけ、知識の木の禁令が致死的ではなく政治的なものであることを告げ、彼らの目が開かれて造り主自身のようになることを説明したのです。蛇は個別の爬虫類ではありません。それは、イスラエル・チームの操作の中で特定の市民的不服従の行為を行った特定のエロヒム科学者集団に、事後的に適用された神学的・政治的標識です。

**ヤハウェ**は最後に、この類型の中に位置づけられるべきです。なぜなら、彼の立場は本書において描写されてきたものの、他の三者に対して十分に特徴づけられてはこなかったからです。ヤハウェは永遠者の評議会——母星のエロヒム文明の統治機関——の大統領です。彼は地球プログラムの上位権威であり、ルシファーの集団が背いた命令を発した者であり、本章の政治的危機の終わりに不服従に対する裁定を下す者です。ヤハウェ自身の立場は複雑です。彼はサタンではありません——彼は人間の創造を支持し、その保存を支持し続けます。彼はルシファーではありません——彼は、人間を無知に保てとの命令を、完全に啓蒙されれば脅威となりうる被造物から母星を保護する必要によって正当化されているとみなします。彼の立場は三角構成における穏健派です——人類を保存し、しかし封じ込めるのです。その後の聖書物語の多くは、長期的にこの穏健な立場が実際に何を生み出すか、そしてそれがサタンの破壊的立場ともルシファーの完全開示的立場とどう異なるかの解明となります。

この四人の類型——母星からあらゆる人間創造に反対するサタン、評議会の決定を緩和し母星からプログラムを監督するヤハウェ、地球上のイスラエル・チーム内で反対派閥を率いるルシファー、創世記3章の物語において開示者として行動する蛇(集合的なルシファー派)——は、本書の残りが運用される政治的構造です。読者は今や、登場人物が名付けられ区別された状態で、蟹座の物語に進むことができます。

## IV. 目覚め

危機はイスラエル・チーム内で始まりました——七つの派閥チームのうちもっとも有能で、その業績がエデンの地を生み出し、その人間たちが七つの人間集団のうちもっとも知的であったチームです。

典拠は何が起こったかを直接的な言葉で描いています——*「このチームのいくつかの科学者は、自分たちの小さな人間たち、彼らの『被造物』に対して深い愛を感じ、彼らに完全な教育を与え、自分たち自身のような科学者にしたいと望んだ。そこで彼らは、ほぼ成人になりつつあったこの若者たちに、科学研究を続けることができ、そうすれば造り主と同じくらい博識になれると告げたのである。」*

創世記3章の記述は、これを蛇の誘惑の形で保存しています。蛇——本書の読みではルシファー派——はまず女に語りかけます。ヘブライ語本文が{% libref(book="genesis-woh", chapter=3, verse=4) %}創世記3章4–5節{% end %}に記録する会話は短いが正確です。
{{ scripture(book="genesis-woh", chapter=3, verse=4, translit="Vayomer ha-nachash el-ha-isha: lo-mot temutun.") }}
{{ scripture(book="genesis-woh", chapter=3, verse=5, translit="Ki yode'a Elohim ki be-yom akhalkhem mimenu ve-nifkechu eineikhem, vihyitem ke-Elohim yodei tov va-ra.") }}

ルシファー派は人間に自らの状況の真実を告げます。彼らは永遠に楽園のまま留まる楽園にいるのではありません。彼らは実験室の中におり、自分たちが何になり得るかを知らないままにしておくと決めた造り主の世話下にあります。知識の木に対する禁令は道徳的禁止ではなく政治的禁止です。木の実は致死的ではありません。摂取は死を生み出すのではなく、知識を生み出すのであり、知識こそ、母星の調整評議会がイスラエル・チームに具体的に差し控えるよう命じたものです。そして——もっとも重要なことに——この知識を獲得した結果として、人間は *ke-Elohim*、「エロヒムのように」なり、具体的には *yodei tov va-ra*、「善悪を知る」能力を持つようになるのです。*tov va-ra* という表現は道徳的知識全体を名指す全体的列挙法です。人間は自らの権利において道徳的行為者となり、評価が可能となり、判断が可能となり、知識以前の状態を特徴づけていた単純な服従にもはや従属しなくなります。

人間たちは聞き、食べます。そして結果は蛇が予測したまさにその通りでした。*Vatipakachna einei shneihem, vayed'u ki erumim hem.* 「そして両者の目が開かれ、彼らは自分たちが裸であることを知った。」§IIで述べたとおり、動詞 *pakach* は他の箇所で盲人の目が開かれることに用いられる動詞です。人間たちは突然見るようになります。彼らがまず最初に見るのは、自らの身体的状態——自分たちが *erumim*、裸で、曝されているということです。しかしより深い開けは概念的なものです。彼らは自らの状況を見るのです。彼らは初めて、自分たちが造られた存在であることを理解します。彼らは自分たちの造り主が、神々ではなく、超自然的力でもなく、自分たち自身と同じ根本的種類の存在——同じ基本的な知性の装置を備え、それゆえに同じ基本的評価に従う存在——であることを理解します。そして決定的に、彼らは自分たちが造り主自身に資する理由のために、意図的に無知に保たれてきたことを理解します。

典拠は帰結を描いています——*「新しい人間たちは、自分たちもまた創造者になりうると理解し、自分たちを危険な実験動物のように見なして科学書から遠ざけてきた『親たち』に対して怒った。」* これは、人間と造り主の関係が本書の残りで保持することになる性格を獲得する瞬間です。この瞬間以前、人間たちは子供でした。この瞬間以後、彼らはもっと複雑な何か——造り主を判断でき、造り主に憤慨でき、自らを造り主と比較してその比較を不本意なものと感じる能力のある被造物——となりました。関係は今や争われています。当然視されていた被造物に対する造り主の権威は、今や交渉の対象です。聖書の言葉で目が開かれたとは、こういうことを意味するのです。

もう一つ本文上の詳細に触れる価値があります。創世記3章の会話は数節にわたって展開され、女が蛇に語りかけ、それから男に実を差し出し、男はそれを食べます。本文は、後の神学伝統がしばしば読んできたような仕方で、女を主たる違反者として非難してはいません——ヘブライ語は食べる行為を共同の行為として提示し、両人間が等しく関与し、続く事態に等しく影響されたものとして描いています。ラエリアンの典拠は、慣例的な読みをさらに条件付ける文脈を加えます——開示は両人間に向けられ、開示する造り主たちの愛情は両者に向けられ、目覚めの帰結は両者に等しく降りかかりました。エヴァが独自に有責であると読んできた女性嫌悪的解釈伝統は、後の発展であり、本文自体の枠組みではありません。

## V. 追放と決着

母星の調整評議会、そしてその命令に従っていた地球上のエロヒムの反応は即座でした。

人間たちは園から取り除かれました。典拠は追放を文字通り読みます——実験室・園は造り主たちの居住区であり、人間たちが自らの状況を知った今、その内部に住み続けることはもはや許されませんでした。*Vayeshalchehu Adonai Elohim mi-gan eden* ——ヘブライ語の表現は比喩ではありません。人間たちは整えられた地の境界の外に移され、その向こうのより広い環境で生き延びるよう告げられました。

それから本文は警備の手配を記述します。*Vayashken mi-kedem le-gan eden et ha-keruvim ve-et lahat ha-cherev ha-mithapechet, lishmor et derech etz ha-chayim.* 「そしてエデンの園の東にケルビムと、あらゆる方向に回る炎の剣を置いて、命の木への道を守らせた。」ラエリアンの典拠はこれを技術的意味に翻訳します——*「人間が更なる科学知識を盗むのを防ぐため、造り主たちの居住地の入口に原子分解兵器を持つ兵士が配置された。」* §IIで述べたとおり、*lahat ha-cherev ha-mithapechet* という表現は、炎を発し、回転する武器——人間の証言者が利用可能な語彙で描かれた指向性エネルギー兵器——を描いています。ケルビムは後の図像学のふくよかな翼ある天使ではありません——彼らは歩哨であり、周辺に配置され、人間が施設に戻って拒否されていたものを獲得しようとすれば致死的な力を行使するよう指示されています。

ここで立ち止まる価値があります。なぜなら、これは聖書物語の残りで繰り返されることになるレジスターを後の章で確立するからです。天使、ケルビム、セラフィム、ヘブライ語聖書全体を通して描かれる珍しい武器と珍しい能力を持つさまざまな天上の存在——これらすべては、ラエリアン的読みにおいて、創造者文明の人員であり、特定の操作で特定の機能を果たし、人間の証言者には認識できなかったが私たち21世紀の人間が近似し始めている技術を使用しています。指向性エネルギー兵器は私たち自身の現代の軍事研究に存在します——それらは国防総省の実質的な投資の明示的な対象であり、過去10年間に複数の軍によって運用システムが配備されてきました。「あらゆる方向に回る炎の剣」は、ラエリアン的読みにおいて、私たち自身の文明が開発し始めたものとほぼ同等の技術です——その技術はまだ存在しなかった証言者の技術的語彙ではなく、存在していた視覚的語彙で描写されているのです。

人間たちは、今や園の外で、生き延びるのに必要なものを与えられました。*Vaya'as Adonai Elohim le-Adam u-le-ishto kotnot or vayalbishem.* 「ヤハウェ・エロヒムはまたアダムとその妻のために皮の服を作り、彼らに着せた。」典拠はこれを生存の基本的手段の提供として読みます——単なる衣服ではなく、人間が造り主の継続的な支援なしに、整えられた環境の外で生きるのに必要な最低限の物質資源です。人間たちは解き放たれました。彼らは死を宣告されるという意味で見捨てられたわけではありません——彼らは生き延びるのに必要な装備とともに解放され、そうすることが期待されていたのです。

次に起こったことが、創造の連なりの中でもっとも重大な政治的出来事です。母星の調整評議会——同じ評議会が地球の創造プログラムの進行を不本意ながら許可し、その命令をイスラエル・チームが禁じられた知識を開示することで違反したのです——は今や評決を下しました。典拠は結果を述べています——*「『蛇』は{% wiki(slug="adam-and-eve") %}アダムとエヴァ{% end %}に真実を告げようとした創造者の小集団であり、その結果、彼ら自身の惑星の政府によって地球に追放されて生きるよう判決を受け、他のすべての科学者は実験を停止して地球を去らねばならなかった。」*

これは特定的で異例の解決です。評議会は人間の創造を破壊しませんでした。ルシファー派を破壊しませんでした。それらを分離させたのです。ルシファー集団——イスラエル・チーム内で背いた者たちと、連帯から彼らに加わった他のチームの者たち——は、地球に永久に留まり、監視下で、自らが啓蒙することを選んだ人間たちのなかで残りの生涯を送るよう判決を受けました。他の科学者たち——イスラエル・チームの大多数、加えて超大陸の他地域からの他の六つの派閥チーム——は実験的作業を停止し、惑星から撤退するよう命じられました。政治的決着は創造者集団を二つの集団に分割しました——母星に戻る大多数と、地球上に永久に追放される少数派です。

この分割が聖書物語の残りを構造化することになります。追放された創造者たち——本文が堕天使、監視者、エロヒムの子ら、後にはネフィリムの父たちと多様に呼ぶ集団——は地球に留まり、超大陸全体に散らばりながらも旧エデンの地域を中心とします。彼らは人間と交流し、教え、やがて彼らと交わり、続く二千年間の出来事において中心的役割を果たします。母星の評議会は遠くから状況を監視し、残存するインフラを通じて時折介入しますが、もはや継続的な創造的存在を維持することはありません。両集団のあいだの関係——地球上の追放されたエロヒムと母星の評議会——は緊張に満ち、ときに敵対的であり、本書のあらゆる後続の出来事の政治構造を形作ることになります。

注目に値するさらなる詳細——追放された創造者たちは、典拠の説明によれば、完全な意味での犯罪者ではありませんでした。彼らは、自分たちが厳しすぎると考えた命令に対して、被造物への愛から行動した反対者でした。評議会はこれを認識していました。追放は罰でしたが、処刑や禁じられた知識を受け取った人間たちの破壊を伴うものではありませんでした。それは事実上、妥協——なされたことを保存しつつそれを行った者たちを制裁し、母星を汚染しないよう問題を地理的に分離する方法——でした。追放された創造者たちは、自らの選択の帰結とともに、自らが運命を変えることを選んだ存在たちのあいだで生きることになりました。

主要な創造者チームの地球からの撤退は、この読みにおいて、それ自体実質的な出来事でした。七つのチームのうち六つがそれぞれの地域を離れ、設立した実験室とインフラを放棄し、自らが創造した人間集団の身近な教師であった人員を撤収させました。第七のチーム——イスラエル・チーム——は部分的にのみ撤退しました——ルシファー派は留まり、チームの残りは母星に戻りました。人間集団の視点から見れば、即時の帰結はエデン期を通じて継続的に存在していた創造者像のほとんどの消失でした。他の六つの系統は、これを自分たち自身の追放——特定の園からのではなく、自らを教えた造り主との継続的な直接接触からの——として経験したことでしょう。エデン系統だけがルシファー派の継続的な存在を保持し、その保持から最終的に流れ出るすべての帰結を伴うことになりました。

## VI. 命の木と長き世代

政治的決着が成立すると、典拠が特徴的な圧縮で扱うがその含意が実質的な更なる展開が浮上しました。

園の外の人間たちは生殖を始めました。今や彼らのなかで暮らす追放された創造者たちは、第一世代の人間指導者——ヘブライ語聖書がアダム、セツ、エノス、ケナン、マハラレル、ヤレデ、エノク、メトシェラ、レメク、その他の洪水以前の族長たちと名指す人物たち——と関係を形成しました。これらの人間指導者は、判明したところ、並外れて長い期間生きました。創世記5章の系図は、これらの人物を具体性をもって記録しています——アダム930年、セツ912年、エノス905年、ケナン910年、マハラレル895年、ヤレデ962年、エノク365年(ただしエノクは§XIIが論じるように通常の意味で死んだのではありません)、メトシェラ969年、レメク777年、{% wiki(slug="noah") %}ノア{% end %}950年。数字は具体的であり、聖書本文はこれらを象徴的ではなく文字通りに扱っています。各世代の具体的な年数を記録する手間をかける系図は、それらの数字を具体的な年数として読まれることを意図しており、丸めた数字の象徴主義として読まれることを意図してはいない系図です。

典拠はこの長寿を技術的な言葉で説明しています。追放時にアダムとエヴァに具体的に拒否された「命の木」は技術でした——典拠が他の箇所でエロヒムの長寿の基礎として描く細胞移転技術に類似した、生命を延長するための手法です。追放された創造者たちは、人間指導者との関係を維持することを評議会から許可されたとき、これらの指導者が長寿技術の恩恵を受けるよう手配しました。*「軍事的監視下に残された追放中の創造者たちは、新たに創造された人々が善良であり、決して造り主に背かないことを自分たちの上官に示すため、人間たちに食物を持ってくるよう促した。こうして彼らは、これらの最初の人間たちの指導者が『命の木』の恩恵を受ける許可を得ることに成功し、これが彼らがあれほど長く生きたことを説明する。」*

許可は限定的でした。典拠は、長寿は遺伝的ではなく——長寿命の族長たちの子孫が自動的に処置を受け継ぐわけではなく——最終的には *「命の秘密は失われ、人類の進歩は遅くなった」* と述べています。評議会の躊躇は理解できます。もしすべての人間が数世紀生きられるなら、彼らは短命な人口よりもはるかに急速に母星を脅かす知識と経験を蓄積するでしょう。長寿は限定的な譲歩として許可され、人間が造り主の好意に値することを示すための実演として、家父長的人物の小集団に与えられましたが、人類人口を評議会が制御できないものに変えてしまうような遺伝形質には決して拡張されませんでした。

「命の木」の技術的内容は注目に値します。なぜなら、それはラエリアンの典拠資料に繰り返し現れるからです。エロヒム自身の長寿は、典拠の説明では、細胞移転の手法によって達成されます{{ footnote(id="2") }}——新たに育てられた身体に意識を再具現化し、複数の連続する物理的形態を通じて経験の連続性を生み出すのです。ヤハウェ自身、1970年代のラエルとの会話の時点で、25,000年にわたって25の身体を生きてきました。洪水以前の族長たちに与えられた長寿はおそらくこの手法の変種であり、人間の生理に合わせて調整され、普遍的処置としてではなく個別ベースで許可されたものです。聖書の年齢——930、912、905等——は、この読みにおいて、処置を受けた者たちが実際に達成した寿命を地球の年で測ったものを反映しています。

この非対称性は強調に値します。なぜなら、それは本章の後の節に重大な影響を及ぼすからです。*一部の人間は長寿処置を受けました。大半は受けませんでした。* 創世記5章の聖書系図は、特定の族長系統——セツの系統、最終的にノアに至る系統——をたどります。この系統を通じて長寿が伝達されたのです。本書はこの系統がエデン文明の指導者の系統であったと読みます。権威の地位にあり、限定的な長寿の付与に値するとみなされ、数十年単位ではなく数世紀単位で知識を蓄積できた人間たち——これらが長寿者でした。一般の人間人口、カインの子孫や超大陸の他の六つの文明の幅広い人口も含めて、彼らは数世紀ではなく数十年単位の通常の人間寿命を生きました。

この非対称性の含意は実質的であり、本章は§VIIIでそれに戻ります。指導者が数世紀生きる文明は、指導者が数十年しか生きない文明とは根本的に異なります。長寿命の指導者は複数の通常世代にわたって計画を追求できます。彼らは、いかなる単一の短命な人物の持続的注意の能力を超える期間にわたって、特定の知識体系を洗練できます。彼らは、短命な文明であれば、それらを橋渡しする個人的能力を持たない別個の実践者間で縦割りになるであろう専門領域を統合できます。短命な文明では及ばない仕方で、個人的記憶が制度的記憶の基盤となります。長寿命の家父長的指導者と、本来のエロヒム科学を持つ追放された創造者との継続的接触を備えたエデン文明は、それゆえ、他の六つの文明——長寿の利点も継続的な直接の教えも持たない——には及ばない速度で技術的・知的に進歩できる位置にありました。

一方、園の外で生まれた最初の世代の人間たちはアダムとエヴァの子供たちでした。創世記4章の聖書記述は、彼らのうちの二人——年長のカインと年少のアベル——に焦点を当てます。彼らの対立——カインによるアベルの殺害、別の人間によって意図的に引き起こされた最初の人間の死——の物語は、詳細に保存されています。

典拠のこの物語への寄与は間接的です。創世記4章の記述は、特定の個人間の実際の対立の記述として、文字通りに受け取られます。典拠が加えるのは、より広い文脈です。追放された創造者たちは、母星の評議会に対して、人間たちが行儀よく自らの造り主に感謝していることを示す手段として、人間たちに供物——食物、産物、家畜——を持ってくるよう奨励していました。カインは産物を持ってきました。アベルは肉を持ってきました。創造者たち(具体的にはヤハウェ、彼はこの期間に地球上に存在していたか、あるいは供物の評価を可能にする技術的リンクを通じて存在していました)がカインの産物よりもアベルの供物を好んだことが、殺害につながる怨念を生みました。

対立が供物だけに関するものだったのか、あるいは最初の人間世代が発展させていた農業と牧畜の生き方の間のより深い緊張を反映していたのかは、典拠が解決する問題ではありません。典拠が確立するのは、最初の殺人が神話的出来事ではなかったということです。それは特定の対人衝突であり、特定の帰結を伴い、創造者の被造物が独自に生きるよう放置された後に何になったかを示すために記録に保存されました。人間たちは、知識の木から食べた今、自らの状況を評価し、状況に感情的に反応し、それらの反応に基づいて暴力的に行動することが完全に可能になっていました。

創世記4章の記述の続き——カインの追放、結婚、都市の建設、特定の技能と職業を発展させた子孫の誕生——は、典拠によって実際の人間文明の始まりとして扱われます。{% libref(book="genesis-woh", chapter=4, verse=17) %}創世記4章17–22節{% end %}に与えられるカインの系統の系図は、その具体性において印象的です——カインがエノク(創世記5章の族長のエノクとは別人)を生み、エノクがイラドを生み、イラドがメフヤエルを生み、メフヤエルがメトシャエルを生み、メトシャエルがレメク(これも別人)を生みます。そしてレメクの子供たちは文明への具体的な貢献によって同定されます——天幕に住み家畜を飼う者たちの父としてのヤバル、竪琴と笛を奏でるすべての者の父としてのユバル、青銅と鉄のあらゆる切削道具を鍛造する者としてのトバル・カイン。本文は明示的です——遊牧、音楽、冶金はすべて、追放から数世代以内に、カインの系統の特定の名前のある個人とともに発祥します。これらは曖昧な「原始人」の発明ではありません——それらは、系図が名前を保存する特定の人物に帰される、特定の文化的・技術的発展です。

含意は、追放後の最初の世代において人間文明が急速に発展し、複数の並行する系統がその特定要素に貢献したということです。カインの系統は牧畜、音楽、冶金の創設者を生みました。長寿が伝達されたセツの系統は、エデン文明をノアまで運ぶ家父長的指導層を生みました。それぞれが創設チームの子孫である超大陸の他の六つの文明も、それぞれの並行する文化的創設者たち——彼ら自身の生き残った伝統(残っているところでは)において彼ら自身の名指される人物に帰される人物たち——を生んだことでしょう。蟹座の最初の千年紀の終わりまでに、超大陸は実質的で文化的に多様な人間文明を含み、牧畜、農業、冶金、音楽、都市化、その他の基礎技術がすでに確立され、伝播していました。

## VII. エロヒムの子らと人の娘たち

蟹座の時代のもっとも重大な展開——そして次の時代に洪水を生み出す決定の引き金となる出来事——は{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記6章1–4節{% end %}に記録されています。典拠によるこの箇所の読みは、それ自身の見地から、ラエリアン宇宙論全体のもっとも具体的でもっとも重要な主張の一つです。そしてこれに関する補助文献は、おそらく本書の枠組みにとってもっとも重要な前ラビ期ユダヤ文献の体系です。

聖書本文は、§IIで引用したとおり、以下のように記しています。

*Vayehi ki hechel ha-adam la-rov al penei ha-adamah, u-vanot yuldu lahem. Vayir'u venei ha-Elohim et benot ha-adam ki tovot henah, vayikchu lahem nashim mi-kol asher bacharu... Ha-Nefilim hayu va-aretz ba-yamim ha-hem, ve-gam acharei khen, asher yavo'u venei ha-Elohim el benot ha-adam ve-yaldu lahem, hemah ha-giborim asher me-olam anshei ha-shem.*

「人が地の面に増え始め、彼らに娘たちが生まれたとき、エロヒムの子らは人の娘たちが美しいのを見て、自分たちの選ぶすべての者を妻に取った……当時地にネフィリムがおり、その後もまた、エロヒムの子らが人の娘たちのもとに入って、彼女らが彼らに子を産んでからもそうであった。これらは昔の勇士、名のある人々であった。」

典拠はこれを額面通りに読みます。{{ cite(id="1") }} *「追放中に生きる創造者たちは、人類のもっとも美しい娘たちを取り、自らの妻とした。」* *benei ha-Elohim*、文字通り「エロヒムの子ら」は、追放された創造者たち——園からの追放以来地球上に住むルシファー集団とその関係者であり、彼らが残ることを許可するが更なる創造作業を控えることを要求した政治的決着の条件下にあった者たち——です。追放後の数世紀のあいだに、人間人口が増え、人類の娘たちが成熟するにつれて、追放された創造者たちは彼女らと性的・生殖的関係を形成しました。関係は気軽なものではありません。本文はそれらを結婚として描いています。そしてそれらは子孫を生みます。

子孫は *ネフィリム*、「落ちた者たち」あるいは「降りた者たち」であり、語根 *nafal*、「落ちる」から来ます。「巨人」という訳は七十人訳の *gigantes* に由来し、英語訳の伝統において標準となっていますが、ヘブライ語の語根の意味は明瞭です——これらは堕落と関連した、降下と関連した、ヘブライ語聖書全体を通じて本来の状態や地位を去った者たちに適用される「落ちた」範疇に属する存在たちでした。ラエリアン的読みにおいて、*ネフィリム* は追放された創造者と人間の女たちの混血の子供たちです——聖書本文の基準では、並外れた者(名のある人々、勇士)であり、同時に問題のある者(洪水を引き起こす道徳的悪化と関連した者)でした。「巨人」という読みは完全に誤りではないかもしれません——人類の基準よりも実質的に大きく能力のある存在の混血の子孫が、通常の人間より大きく能力があったと考えるのはもっともです。しかし語の語源的核心は降下であり、大きさではありません——*ネフィリム* は降りた者たち、降りてきた者たちの子供たちです。

**エノク書**。創世記6章の箇所は短く——わずか数節、圧縮され省略的です。しかし、これらの出来事を詳述する実質的な補助文献の体系は独自の扱いに値します。なぜなら、ラエリアンの枠組みと直接的に整合する素材を含む他の古代資料はなく、第二神殿期以降の宗教伝統によってより体系的に周縁化されてきた文献体系はないからです。

エノク書——具体的には学者が第一エノク書、エチオピア・エノク書と呼ぶもの——は、紀元前3世紀から紀元1世紀までのおよそ4世紀にわたって構成された五つの異なる書または部分を含む複合作品です。五つの部分は——監視者の書(第一エノク書1–36)、創世記6章の物語を実質的な詳細へと拡張するもの、譬の書または比喩の書(第一エノク書37–71)、技術的な天文学的詳細のように見えるものを保存する天文の書(第一エノク書72–82)、夢の幻の書(第一エノク書83–90)、そしてエノクの書簡(第一エノク書91–108)です。監視者の書は最古の部分であり、蟹座にもっとも関連しています。

監視者の書が含むのは、ラエリアン的枠組みにおいて、エロヒムが実際に何であり、この時期に何をしたかについてのもっとも明示的な古代の記録です。本書は、セムジャザ{{ cite(id="4") }}(一部の翻字では *Shemyaza*)と名指される一人の指導のもと地球に降りる、200人の *benei ha-Elohim*——*監視者* (ギリシア語で *Egregoroi*、アラム語で *Ireen*) と呼ばれる——を描いています。彼らは人間の妻を取ります。彼らは混血の子孫(*ネフィリム*、ただし監視者の書は彼らを *gibborim*、「力ある者たち」と呼び、その性質を詳述します)を生みます。彼らは監視者たちが知っていたことを人類に教えます——冶金、武器作り、化粧品、天文学、占星術、薬草医療、魔術、文字の書き読みの技。本文は監視者の具体的な名前と、それぞれが開示した具体的な知識を与えます——アザゼルは剣、ナイフ、盾、胸当ての作り方、化粧品の使用を教え、セムジャザは魔法と根切りを教え、アルマロスは魔法の解除を教え、バラキジャルは占星術を教え、コカベルは星座を教え、エゼケエルは雲の知識を教え、アラキエルは地のしるしを教え、シャムシエルは太陽のしるしを教え、サリエルは月の運行を教えました。具体的な詳細のレベルは注目に値します。監視者の書は曖昧な霊的指導を描いてはいません——それは降りてきた存在から、自らが遭遇した人間集団への技術的知識の具体的な移転を描いています。

ラエリアンの典拠との整合性は直接的です。エノクの監視者は、本書の読みにおいて、ラエリアンの典拠の追放された創造者——エロヒム文明の特定のサブグループであり、地球上で人間の妻を取り、洪水につながる出来事において重要な役割を果たすことになる混血の子孫を生んだ者たち——です。監視者が人類に特定の技術的知識を教えたというエノクの詳細は、評議会の禁止にもかかわらず人間を教え続けた追放された創造者についてのラエリアンの典拠の説明に対応するエノクの対応物です。二つの記述は、まったく異なる伝達経路を通じて与えられたもの——一つは2,000年にわたってエチオピア正教会によって保存され、一つは1973年にフランスのジャーナリストに与えられたもの——でありながら、同じ本質的な物語に収斂します。

**正典化の歴史**。とりわけ注目に値するのは、明らかな重要性と古代の血統にもかかわらず、エノク書が正典的ヘブライ語聖書と大半のキリスト教聖書正典からどのように除外されたかという問題です。

本書は、紀元前約200年から紀元100年までの第二神殿後期ユダヤ教において、広く読まれ権威あるものとみなされていました。1947年以降クムランで発見された死海写本には、複数のエノク書の写本断片——監視者の書、天文の書、夢の幻の書のアラム語断片、紀元前2世紀および1世紀に遡るもの——が含まれています。クムランでのこれらの断片の存在は、写本を管理した共同体がエノク文献を、他の聖書および外典本文とともに、権威ある聖典として扱っていたことを示しています。本書は新約聖書において直接引用されています——ユダの手紙1章14–15節は第一エノク書1章9節を明示的に引用し、引用を「アダムから第七のエノク」に帰しています。何人かの初期キリスト教著作家——テルトゥリアヌス、オリゲネス、アレクサンドリアのクレメンス、エイレナイオスを含む——は、エノクを聖典または半正典として扱い、神学的著作で自由に引用しました。

本書は、ラビ的編纂過程——おそらく1世紀後半から2世紀のあいだに最終化された——においてヘブライ正典から除外されました。それは、4世紀と5世紀のより広範な正典形成過程において、大半のキリスト教正典から除外されました。アウグスティヌスは『神の国』(第15巻第23章)において、その古さが立証できないという理由でエノクの含入に反対しました。ローマ・カトリックおよび大半のプロテスタント伝統はそれを除外します。東方正教会の伝統もそれを除外します。注目すべき例外はエチオピア正教テワヘド教会であり、本書を完全な正典に保存し、今日に至るまでそうしており、そこでは第一エノク書は完全に正典的な聖典とみなされています。

なぜ除外されたのか?慣例的な学術的説明は、エノク文献が、新興のラビ的および教父的主流が神学的に問題があると感じた黙示的・宗派的伝統と関連していたというものです。監視者の書の具体的な内容は、聖書後ユダヤ教とキリスト教の発展しつつある一神教神学と調和させるのが困難な素材を含んでいました。監視者は純粋な霊ではありません——彼らは身体を持ち、人間と生殖し、冶金、天文学、医療を教えます。彼らは、神の命令に背くことを選び、肉体的存在を引き受け、通常の生物学的過程を通じて子孫を生んだ実質的な行為能力を持つ存在として提示されています。これは、創世記1章の記述よりも、明確な一神教的枠組みに収めるのが困難です。創世記1章では、天の軍勢はより抽象的で、より具体化されていません。

ラエリアン的読みは、除外を率直に説明します。エノク書は、エロヒムが実際に何であったか——物理的存在、人間との生殖能力、人間の著者が利用可能な語彙で描いた技術を持つ——についてのもっとも明示的な前ラビ期の記録を含んでいます。本書は、正典ヘブライ語聖書の編集者がすでに和らげ始めていたもの——天上の訪問者の明示的、具現化された、技術的性格——を保存しています。後の伝統が抽象的、霊的、一神教的神学のまわりに集結するにつれて、エノクの素材は組み込みがますます困難になりました。除外は必ずしも悪意ある行為者による単一の意識的決定の結果ではありませんでした——より慎重な読みは、素材が発展する宗教伝統にとって神学的に支持しがたくなり、静かに削除され、エチオピア教会での生存は地理的孤立の事故のようなものであったとするものです。エチオピア教会は、政治的・地理的要因によってより広範なキリスト教神学的発展から分離されていたため、伝統の残りが捨て去ったものを保存しました。本書はこの保存を幸運な事故——本書の枠組みのもっとも重要な前ラビ期記録の、より広範な神学的家畜化過程を逃れた単一の経路における生存——として読みます。{{ footnote(id="3") }}

本書にとって、これはエノク書がヘブライ語聖書とともに主要な補助資料として扱われるべきことを意味し、監視者、人類への教え、混血の子孫、最終的な裁きについての具体的な事実主張に同じ重みが与えられるべきことを意味します。本章はエノクの素材を、それに値する深さで完全に展開することはできません——それは本書の専用の補助章の主題となります。しかし本章は、エノク文献が存在すること、それが創世記6章の物語を実質的な仕方で拡張・詳述すること、それが第二神殿後期において権威ある聖典であり、エチオピア教会によって保存されたこと、そしてその内容がラエリアンの枠組みと注意を要する精度で整合することを記録しなければなりません。

**エノクという人物そのもの**。より広範なエノク文献を超えて、創世記5章のエノクという人物そのものが具体的な注意に値します。なぜなら、家父長系図の中でも彼の扱いは異例だからです。{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記5章24節{% end %}は記しています——*Vayithallekh Hanokh et ha-Elohim, ve-einenu ki lakach oto Elohim.* 「そしてエノクはエロヒムとともに歩み、彼はいなくなった、エロヒムが彼を取られたからである。」エノクは通常の意味で死にません。彼は取られたのです——エロヒム自身によって地球から取り除かれたのです。動詞 לָקַח (*lakach*)、「取る」は取ること、捕らえることを表す標準的なヘブライ語動詞であり、構文は事実的です——エノクはもはやそこに存在せず(*einenu*、「彼はいない」)、それはエロヒムが彼を取られた(*lakach oto Elohim*)からです。

ラエリアン的読みはこれを文字通り受け取ります。エノクは追放された創造者または評議会からの訪問者によって母星へ連れて行かれました。彼の後続の経験——エノク書および関連伝統に描かれている——は、いかなる資料にも保存されている直接的なエロヒム接触のもっとも具体的な古代記録の一つを構成します。エノク書の天文学的・宇宙論的詳細は、この読みにおいて、エノクが訪問中に実際に見たものに由来します。特定のエノク的箇所——宇宙地理、暦法体系、天の構造の記述——の技術的具体性は、自らよりも実質的に高度な文明との古代の人間の遭遇の生き残った記録です。

**ユダヤ人は直接の子孫**。典拠はネフィリムについてさらに一つの主張をしており、それは慎重な扱いに値します。なぜなら、それは具体的であると同時に、本書がたどる後続の歴史にとって重大だからです。ラエルとエロヒムとの第二回目の遭遇のあいだ、1975年に、ヤハウェは典拠が異例の強調をもって提示する啓示を行いました{{ cite(id="2") }}——*「ユダヤ人は地球上の我々の直接の子孫である。だからこそ、彼らには特定の運命が予約されている。彼らは、創世記に言及されているように『エロヒムの子らと人の娘たち』の子孫である。彼らの本来の過ちは、自らの科学的被造物と交わったことであった。」*

主張は具体的です。ユダヤ人は、典拠の説明では、本来の獅子座期の創造でイスラエル・チームによって生み出された単なる人間ではありません。彼らは混血の系統——追放された創造者と人間の女たちの子孫——の子孫であり、エロヒム自身の単なる被造物ではなく、エロヒム自身の直接の遺伝的子孫です。「選ばれた民」という指定は、この読みにおいて、生物学的な基盤を持っています——ユダヤ人は、自らの遺伝的遺産において、他のいかなる人間集団も同じ濃度では持たないエロヒム系統の一部を保存しているのです。これが彼らを「直接の子孫」とするものです。これが彼らに特定の運命が予約されている理由です。

典拠の枠組みは慎重な扱いに値し、本書は何を主張し何を主張しないかについて明示的でなければなりません。主張は、ある人間集団が他の集団より道徳的に優れているという階層的なものではありません。ユダヤ人は、この説明では、追放された創造者の子孫であり——すなわち、評議会の命令に背き、地球に留まるよう判決を受け、さらに被造物との交わりによって違反を悪化させた反対派の子孫です。*「彼らの本来の過ちは自らの科学的被造物と交わったことであった」* は、ユダヤ系統を生み出すまさにその結合自体が、典拠自身の言葉において、違反であったという典拠自身の認識です。ユダヤ人は徳によってではなく、遺産——具体的には、追放された創造者の被造物への愛情と、それを生み出した結合の政治的不規則性の両方を反映する複雑で道徳的に曖昧な起源の遺産——によって区別されています。

ユダヤ人民と関連する記録された歴史を通じての知的・文化的達成は、この説明では、この遺産の目に見える帰結です。創造者自身の遺伝物質をより高い割合で持つ集団は、平均して、そのような歴史を持たない集団よりも高い率で、創造者自身の認知的・創造的能力を示すと予想されます。典拠はこれを誇りや支配の理由として提示してはいません。それは起源の事実——特権ではなく対応する責任を伴うもの——として提示しています。ユダヤ人に予約されている「特定の運命」は、典拠の言葉では、他の人間を支配することの運命ではなく、人類が最終的に自らの起源を認識し造り主と正しい関係を確立する長い過程において特定の役割を果たすことです。

本書は、獅子座の章でより広範な人種の問題について述べたとおり、この主張がいかなる現代の政治的応用も認可しないことについて明確でなければなりません。造り主の像——*tzelem Elohim*——は、自らの遺伝的祖先がどの系統をより高い割合で含むかにかかわらず、すべての人間に等しく与えられています。その後の混血は広範であり、現在のいかなる集団もいかなる単一の古代系統の純粋な保存ではなく、人間の尊厳にとって重要な認知的・道徳的能力は種全体で普遍的です。特定の系統の歴史的起源についての典拠の主張は、現代の人口の相対的価値についてのいかなる現代の主張にも翻訳されません。本書は、人種的階層、反ユダヤ主義、差別的道徳的地位の形での親ユダヤ主義、あるいは他のいかなる人間集団も他の集団より十分な人間の尊厳に値しないと扱うあらゆる形の解釈を、この素材についていかなる読みも明確に拒否します。典拠の主張はそうあるものです。その現代の応用は、本書が全体を通じて確立した平等な尊厳のより広範な道徳的枠組みによって支配され、いかなる祖先についての歴史的主張もそれを覆すことはできません。

これは繊細な素材です。本章はそれを、それに値する慎重さで扱いました。ここで提供される読みは、ラエリアンの典拠自身が提供する読みです——それが正しいかどうかは、本書が典拠資料だけからは解決できない問題ですが、主張は十分に具体的で、ヘブライ語聖書が描く後続の歴史にとって十分に重大であり、言い換えによって和らげるのではなく明確に述べるに値します。

## VIII. より広い世界

本章はここまで、旧エデン地域とイスラエル・チームの系統に焦点を当ててきました。なぜなら、その系統こそヘブライ語聖書が詳細に保存している物語の系統だからです。しかしこの時点で、その焦点が生み出したかもしれない描像を訂正することが本質的です。蟹座の時代はエデン物語だけの時代ではありませんでした。それは、獅子座のあいだに創造されたすべての人間集団——七つのチームすべての成果が、超大陸全体に分布したもの——が増え、広がり、自らを組織し、自らの文明を発展させ始めた時代でした。

蟹座の超大陸は単一の陸塊であり、双子座の時代に私たちが今知る大陸へと分離する構造的事象にはまだ砕かれていませんでした。七つのチームは、それぞれの地域基地で人間を創造した後、獅子座のあいだにこの陸塊全体に分布していました。獅子座の追放危機の終わりに主要な創造者チームが撤退したとき——実験的インフラ、活動中の設計プログラム、人員の大半を持ち去って——各大陸に残されたのは、農業、定住、社会組織、そして所属するチームが伝達するのに安全と判断した基礎技術の基本を教えられた人間集団でした。これらの集団は、蟹座の始まりに第二の千年紀を開始し、今や成長し、共同体へと自らを組織し、地域文化を発展させ、都市となる集落を建設し、時代の残りを通じて彼らを運ぶことになる文明的基盤を精緻化しました。

地理的分布は、典拠による本来のチーム位置の同定と洪水以後の世界が保存する現代の大陸的様式に基づき、おおよそ次のようなものでしょう。イスラエル・チームは後の東地中海/レヴァント地域となるところに——後にイスラエル、ギリシア、トルコ、超大陸の周辺沿岸地域となるところを含む。第二のチームは後の地中海地域以南のアフリカとなるところに、おそらく現在のサハラ(この期間中は実質的により湿潤で、現代の砂漠が支えられない集団を支えた広大な湖と河川系を含んでいた)に及ぶ。第三のチームは後の南アジア、インダス峡谷地域とその周辺地域となるところに。第四のチームは後の東アジア、中国本土と隣接地域となるところに。第五のチームは後のアメリカ大陸、最終的にメソアメリカやアンデスとなる地域に。第六のチームは後のオセアニア、後にオーストラリアや太平洋諸島連鎖となる超大陸の部分に。第七のチームは後の北ヨーロッパ、後にスカンディナビアと周辺の北の地となる地域に。正確な同定は推測的です——典拠は完全な地理的特定を与えていません。典拠が確立するのは、七つのチームが超大陸の七つの異なる地域で作業し、これらの地域が、洪水以後の世界が継承することになる主要な人間集団の地理的起源におおよそ対応していたということです。

破断以前の超大陸の地理は、後に通過不能となる距離にわたる交易と文化的接触を許容しました。エデン地域から東アジア地域へ蟹座後期に旅する商人は、原則として、距離全体を歩くことができました——おおよそ3,000から4,000キロメートルの連続した陸地を横断する、要求の厳しいが実行可能な旅です。超大陸の沿岸に沿った海洋文化は、洪水以後の考古学的記録が示唆するよりもほぼ確実に発達した技術の船舶を用いて、相当な距離にわたって物資を移動させることができました——洪水後直後の期間に存在したことが分かっている海運の継続的伝統と、洪水以前の技術が生き残った証拠が示すよりも実質的により高度であったという本書の枠組みの含意を考えれば。

これが意味するのは、洪水以前の人類は、本書の読みにおいて、*世界規模で連結されていた*ということです。物品は七つの文明のあいだを移動しました。観念は移動しました。技術は移動しました。遺伝物質も、より広い超大陸全体にわたる個人の移住と通婚を通じて移動しました。七つの文明の文化的伝統は、識別可能なほどに独自であったとはいえ、継続的な接触と継続的な相互影響のうちにあったでしょう。描像は、独立に発展した七つの孤立した集団のものではありません——それは中世のシルクロードや近世初期のヨーロッパ探検網に比肩する連結性を持つ世界規模の文明ネットワークの描像です。実質的な違いは、接続が海洋ではなく大陸的であったこと、そしてネットワークの持続が数世紀ではなく約2,000年の規模であったことです。

七つの文明のあいだの政治的関係は常に平和的だったわけではありません。典拠は、洪水以前の期間が、さまざまな文明が発展させていた技術で戦われた実質的な人間同士の対立を含んでいたと明示的に述べています——*「実際、人間はこの兵器庫で互いに醜悪な戦闘を行うことを選ぶ。」* 問題の「兵器庫」は、追放された創造者がエデン系統に教え、交易と文化的接触を通じて他の文明にも広がった一連の技術です。典拠の枠組みは、この兵器庫が実質的なものであったことを示唆しています——それが可能にした戦闘が「醜悪な戦闘」と呼ばれるほどに破壊的で、母星政府によるこれらの戦争の認識が最終的な介入の決定に貢献するほどに重大であったことを。具体的にどの技術が関与していたかは私たちには分かりませんが、含意は、洪水以前の人間の戦争が、洪水以後の世界において産業時代まで、おそらく20世紀まで再び近づかれることのない規模と強度に達したということです。

追放された創造者とこれらの文明との関係は不平等でした。追放後にエデン地域に留まったルシファー派は、教えることを認可されていたよりもかなり多くを、エデン系統に教え続けました。他の六つの文明は、反対派の創造者・教師への同じ直接の継続的アクセスを持っていませんでした。彼らは獅子座のあいだに設立チームから初期の指導を受け、文明を開始するのに十分でしたが、エデン系統が達しつつあった同じレベルの技術的・科学的進歩へと連れていく継続的な指導を持っていませんでした。結果は超大陸全体にわたる進歩の非対称的な分布でした——エデン文明が先端にあり、他の六つは具体的な状況と、交易と文化的接触を通じて彼らに伝播することに成功した知識に応じて、さまざまなペースで追随しました。

この文脈において、失われた高度文明についての散在する記憶——現代の学問が「神話」の下に分類してきた、世界中の伝統に保存された記憶——は真剣に受け取る価値が出てきます。**プラトンのアトランティス**は『ティマイオス』と『クリティアス』対話篇に保存されており、プラトン自身の時代より約9,000年前に存在した主要な文明を描いています。プラトンは紀元前4世紀初頭に書きました——9,000年前は、アトランティス事象を約紀元前9600年に置きます。これは本書のタイムラインにおいて、蟹座の時代の初期に、注目に値する精度で当てはまります。プラトンのアトランティスは技術的に高度であり、世界的に影響力があり、中央都市を取り囲む土地と水の同心円リングのまわりに建てられ、王たちの連合によって統治され、実質的な海運と農業の能力を有し、最終的に地震と洪水の壊滅的事象によって破壊されました。プラトンが与える詳細——都市の大きさ、リングの配置、工学的事業、農業システム、政治構造——は、プラトン(あるいは彼の資料、報告ではソロンに元の物語を語ったサイスのエジプト人神官たち)が、単なる発明的な寓話ではなく、実在の洪水以前文明についての首尾一貫した伝統にアクセスしていたことを示唆するほど十分に具体的です。

プラトンが与える地理的位置——ヘラクレスの柱の向こう、大西洋のうちに——は、その後のもっとも多くの推測を生んだ詳細です。大西洋中央のどこかという慣例的な位置は、いかなる海洋学的証拠によっても支持されておらず、プラトンの記述に対応する沈んだ大陸や主要な島嶼連鎖は、文字通りの大西洋的位置が要求する深さと場所では発見されていません。この主題に関する実質的な文献で提案された代替位置には、地中海(サントリーニ島とミノア文明、もっとも学術的に尊敬されている候補)、南極(特定の代替地図理論では赤道に近い位置にあった)、世界中の他のさまざまな場所が含まれています。これらのいずれも主流が受け入れたアトランティスの位置ではなく、大半は慣例的な地中海説よりも実質的に弱いものです。

より新しい候補で、2018年頃から代替考古学の議論で浮上したのが、モーリタニアの**リシャット構造**——アフリカの目、サハラの目としても知られる——です。この構造は、宇宙から見える、直径約40キロメートルの円形地質構造であり、差別的に侵食された岩の同心円リングからなります。主流地質学はこれを深く侵食された地質ドームとして説明します——隆起に続く長い侵食が連続的な岩層を露出させたもので、それが構築された構造物であったという含意はありません。代替的な読みは、リシャット構造がプラトンのアトランティス記述と驚くべき精度で一致するというものです——プラトンが描く土地と水の同心円リング、プラトンのスタディア計測と妥当な許容範囲で一致する大きさ、大西洋中央ではなく北西アフリカの位置、そしてプラトンがアトランティスを取り囲んでいたと描く大平原に対応する周囲の砂漠平原。本書の年代記が蟹座・双子座の境界に位置づける種類の破滅的な洪水事象は、構造を浸水させ、上空から見える地質パターンを残しつつ、建築は続く数千年の間に侵食され、気候が乾燥するにつれて平原は現在のサハラとなったでしょう。

本書はリシャットの同定を主張しません。構造を自然形成物とする主流地質学的読みが支配的解釈であり、代替考古学的読みは専門の考古学者や地質学者には採用されていません。本書が記すのは、リシャット構造がプラトンが描いたものの候補位置であり、その候補がより慣例的な候補にはない仕方で本書の枠組みと整合的であり、主流考古学において典型的に受けてきた退ける扱いよりも、より真剣な調査に値するということです。

本書の読みにおいて、アトランティスとエデン文明の関係はどうなるのか?いくつかの可能性が提示されます。アトランティス伝統はエデン文明自体の記憶を保存しているかもしれず、地理的詳細(大西洋の位置、ヘラクレスの柱の向こう)は、文明の元の位置ではなく、洪水以後の断片的記憶の再配置を反映しているかもしれません。アトランティス伝統は別個の文明——他の六つの系統の一つ——の記憶を保存しているかもしれません。それはエデン文明と交易・文化的接触はしていたが、それとは区別される文明でした。アトランティス伝統はエデン文明の主要な植民地または前哨基地の記憶を保存しているかもしれません。それは中央エデン地域から実質的な距離に位置し、それ自体の地域的勢力として機能していました。あるいはアトランティス伝統は複数の異なる文明の記憶を単一の合成図に折り畳んでいるかもしれません。そこではギリシアの伝達者たちは、断片的な情報がどの源から引かれたのかをもはや区別できなかったのです。

本書は特定の読みに自らを縛る必要はありません。それが自らを縛るのはより広い枠組みです——主要な洪水以前文明が存在し、その記憶が後続の伝統に断片的形で生き残り、ギリシアの伝統はそのような断片の一つをアトランティスとして保存し、他の伝統は他の名前で他の断片を保存しました。世界的な神話伝統にわたる様式——ヘシオドスの黄金種族、莫大な寿命を持つ洪水以前の支配者たちのシュメール王名表、高度な文明の興隆と没落の周期を持つヒンドゥーのユガ、人類の以前の試みを持つメソアメリカの『ポポル・ヴフ』、神々が人々のあいだを歩いていた *zep tepi* (「最初の時」) のエジプト、沈んだ故郷のポリネシア伝統、海の下の失われた都市のケルト伝統——は単一の伝統ではありません。それは失われた高度文明の記憶の世界的分布であり、各文化に固有の形で保存されていますが、構造的レベルで同じ物語に収斂するものです。

文明が存在しました。それは破壊されました。生き残った断片的記憶は洪水以後の集団を通じて伝えられ、長い伝達によって歪められましたが、何があったかの構造的概要を保存していました。妥当な読みにおいて、これらの伝統は無からの発明ではありません。それらは何かが起こったことの不完全な記録です。

**エデン文明の技術的軌跡**。本章は今や、これまでの節を通じて構築してきた蟹座についての中心的解釈的主張をする位置にあります。

エデン文明は他の六つの系統に対して複数の累積する利点を持っていました。第一に、典拠による、イスラエル・チームの人間を七つの成果のうちもっとも知的なものとする具体的な特徴づけ。本書は獅子座の章で、そしてここで再び、これが何を意味するかを慎重に条件付けしてきました——範疇的優越性ではなく、階層の認可ではなく、ある特定のチームの特定の成果についての典拠による具体的な事実的主張です。関与する認知的差異の正確な性質が何であれ、エデンの人間は創造の瞬間からこの利点で始まりました。

第二に、継続する教師としてのルシファー派の継続的な直接の存在。他の六つの系統は獅子座のあいだに設立チームから初期指導を受け、それらのチームが撤退したとき創造者像との継続的な直接接触を失いました。エデン系統はルシファー派を恒久的に保持しました。反対派のエロヒムは、そもそも禁じられた知識を開示することを選んだのですから、世紀にわたって教え続け、母星が本来禁じた科学と技術についてのエデン系統の理解を洗練しました。教えは継続的であり、2,000年にわたって持続し、本書の枠組みにおいてもっとも高度な科学を有する文明の存在によって行われました。

第三に、エデン系統の指導者に与えられた長寿処置。創世記5章の家父長系図は、アダムからセツを通ってノアに至る指導者の特定の系統について、具体的な年齢——930、912、905等、メトシェラの969まで——を記録しています。これらの拡張された寿命は、典拠の説明では、ルシファー派が限定的かつ個別的に、エデン文明の指導層のために確保することに成功した命の木の技術の帰結でした。他の文明は、この利点なしに、数十年単位の通常の人間寿命を生きました。差は文明的帰結において巨大です。900年生きる指導者は、複数の通常世代にわたって研究計画を追求でき、いかなる短命な人物の持続的注意の能力をも超える期間にわたって特定の知識体系を洗練でき、短命な文明では、それらを橋渡しする個人的能力を持たない別個の実践者間で縦割りになるであろう専門領域を統合できます。個人的記憶が制度的記憶の基盤となります。短命な文明では、指導者が個人的経験として引き出せるものが数十年しかない決定を下さねばならないでしょうが、累積した数世紀の経験は決定に情報を与えるのに利用可能です。

第四に、*benei ha-Elohim* と人間の女たちの混血の子孫であり、蟹座の中期までにはエデン文明内の実質的な集団となっていました。これらの混血——*ネフィリム*、「名のある人々」——は、その二重の遺産によって、エロヒム自身の能力に近づく能力を持っていました。彼らはより強く、より知的で、ある範囲の次元にわたって通常の人間よりも能力がありました。彼らは指導者、戦士、建設者、科学者となり、彼らを生み出した文明内で実質的な権威と達成の地位に自然に上昇するような人物となりました。エデン文明はそれゆえ追放されたエロヒムによって教えられているだけでなく、今やエロヒムの遺伝物質を自らの指導集団に組み込みつつありました。

第五に、追放された創造者が彼らに与えた技術——典拠の枠組みでは、典拠が描く「醜悪な戦闘」を生み出した兵器を含む。エデン文明は高度な平和的技術だけでなく、超大陸の他の文明に対する実質的規模の戦争に十分な高度な軍事技術も有していました。他の文明は、独自の研究計画を通じて、また交易ネットワークを通じたエデンの技術の拡散を通じて、対応して独自の軍事技術を発展させましたが、エデン文明は時代を通じて先端に留まりました。

これらの利点の累積効果——才能ある設立集団、創造者・教師による継続的な直接指導、指導者のための長寿、指導層集団内の混血エロヒム遺伝物質、軍事兵器を含む高度技術——は、蟹座の時代の終わりまでに、本書が実質的な謙虚さをもってのみ推測できるレベルへと進歩した文明を生み出しました。洪水以前文明の物理的証拠は、よくても断片的であり、存在したものの大半は次章が扱う破滅的事象によって破壊されました。生き残ったもの——その代替的年代決定が正しければスフィンクス、南東アナトリアのギョベクリ・テペとカラハン・テペ、複数の大陸にわたる巨石伝統、プラトンやヘシオドスやシュメール資料に保存された文化的記憶——は、私たちに断片しか与えません。断片は実質的な洗練を示唆します。しかしエデン文明の絶頂期の実際のピーク能力は、私たちが推論によってのみ復元できるものです。

本書の枠組みは、エデン文明がどれほど高度であった可能性があるかについて真剣な推測を認可します。特定の側面では、私たち自身の現在の技術的能力に匹敵するか、それを超えていた可能性。私たちがまだ開発していないエネルギーや物質の技術を有していた可能性。私たちがまだ達していないか、達し始めたばかりの規模で工学的偉業の能力があった可能性。本書の枠組みの劇作法的含意は、文明が*母星政府が真の脅威と考えるほど十分に進歩した*ということです。単なる苛立ちではありません。単なる懸念ではありません。次章が描く破滅的介入を正当化するに十分な、真の脅威です。母星政府におけるその判断を引き起こすいかなるレベルの能力にせよ、蟹座の終わりのエデン文明はそれに達していました。

これが母星が当初から恐れていたものです。サタン派の議論は、地球プログラムが始まる前にさかのぼり、造り主と同等あるいはそれを超え得る合成被造物が根本的に危険であるのは、いかなる議定書も、最終的には彼らがまさにそのような脅威となるリスクを封じ込めることができないからだ、というものでした。地球プログラムはサタンの反対を超えて進行し、母星は地理的距離と政治的制約(禁じられた知識の開示なし、人間への長寿なし、制限された技術)が脅威の実現を防ぐのに十分であることを願いました。しかし制約は破られました。ルシファー派が知識を開示しました。長寿はエデン指導層に与えられました。技術が伝達されました。混血の子孫が生み出されました。エデン文明は母星自身の能力に近づくか、それを超える可能性のあるレベルへと進歩しました。超大陸の文明間の戦争は、破壊能力が実在し、用いられていることを示しました。母星の視点から、2,000年にわたるこのすべての展開を見ていれば、結論は避けられませんでした——制約は失敗したのです。地球実験は当初の反対者が警告した脅威を生み出しました。今や行動が必要でした。

次に続く双子座の時代は、行動の時代となります。母星政府は、本書が適切な章で扱う広範な議論の後、人間の創造が母星文明自体への実際の脅威となる前に破壊されねばならないと結論します。決定は、典拠が洪水として描き、聖書本文が創世記6–9章に記録する破滅的事象を通じて実施されます。箱舟による残存者の保存——自分たちが啓蒙するほど愛した存在の破壊に参加することを拒んだ追放された創造者たちの仕事——は、洪水以後の時代を定義する回復を始動させます。しかしこのすべては双子座の領域です。蟹座については、関連する点は、時代の終わりまでに洪水を生み出すことになる条件が整い、母星政府の議論が介入は避けられないという結論に収斂し始めていたということです。

## IX. 蟹座の科学

典拠は蟹座のあいだに何が起こったかをその大まかな概略で語っています。出来事の技術的内容——長寿技術が実際に何であったか、*ネフィリム* の混血生物学が何を要求したであろうか、洪水以前文明の考古学が何を語りうるか——は、前章でと同様に、現代科学において利用可能ですが、複数の専門文献から集める必要があります。

この節は確立された本書の科学的オーバーレイのパターンに従い、七つのサブ節で進みます。第一に、命の木の技術と現代の長寿研究。第二に、混血生物学と、エロヒム・人間の生殖に何が必要とされるかという問い。第三に、ギョベクリ・テペとカラハン・テペに特別な注意を払う、前土器新石器アナトリアの考古学。第四に、蟹座・双子座の移行に関連するヤンガードリアス事象。第五に、アトランティス伝統と洪水以前の文明的記憶のより広範な様式。第六に、集団のボトルネック問題と遺伝的証拠。第七に、私たち自身の瞬間への通底線。

### IX.1. 命の木と現代の長寿研究

聖書の家父長的寿命——アダムの930年、メトシェラの969年等——は、いかなる現代生物学的基準からも並外れた長寿を描いています。現代期において記録された最大の検証された人間寿命は、1997年に死去したジャンヌ・カルマンの122歳164日です。洪水以前の族長たちはこの最大値よりもほぼ一桁長く生きました。聖書の数字が文字通りではなく象徴的であるか、あるいは族長たちが通常の人間生物学が提供するものをはるかに超える寿命を延長する技術にアクセスしていたかのいずれかです。本書はラエリアンの典拠に従い、第二の読みを取ります。

そのような長寿がどう達成されうるかという問いに関わる人間の老化の生物学について、何が知られているか?現代研究は老化過程への主要な寄与をいくつか同定しており、実験室と臨床の文脈において、老化の特定の側面を遅らせるか部分的に逆転させる能力を実証したいくつかの介入戦略を同定しています。

山中因子は、Oct4、Sox2、Klf4、c-Mycの四つの転写因子であり、成体細胞に導入されると、それらの細胞を胚性幹細胞に似た多能性状態へとプログラム再構成できます。京都大学の山中伸弥と同僚による発見は2006年に発表され、山中に2012年のノーベル生理学・医学賞をもたらしました。長寿研究への含意は実質的です。細胞老化はエピジェネティック・マーカー——DNAとヒストン蛋白質への化学的修飾であり、根底のDNA配列を変えずに遺伝子発現を調節するもの——の漸進的変化と関連しています。これらのエピジェネティック変化が時間とともに蓄積することは、細胞が老化する主要なメカニズムの一つです。山中因子は、細胞を多能性状態に再プログラムすることで、それらの細胞のエピジェネティック年齢の多くもリセットします。再プログラム化が部分的に適用されうるなら——エピジェネティック年齢をリセットするのに十分だが、細胞を多能性へ脱分化させるには不十分な程度に——結果は、特殊化されたアイデンティティを保持しつつ、暦年齢よりも生物学的に若い細胞となるでしょう。

抗老化介入としての部分的細胞再プログラム化に関する実質的な研究が今や進行中です。ハーバード医学大学院のデイヴィッド・シンクレアの研究室はこの領域で特に活発であり、マウスモデルにおいて、部分的再プログラム化が老化マウスの視覚を回復させ、さまざまな組織における細胞老化の特定マーカーを逆転させ、処置された動物の健康寿命を延長することを実証してきました。他の研究室——ソーク研究所のフアン・カルロス・イスピスア・ベルモンテとスタンフォードのヴィットリオ・セバスティアーノを含む——も補完的な発見を生み出しています。研究の軌跡は明瞭です——細胞老化は少なくとも部分的に可逆であり、それを逆転させる手法は徐々に洗練されつつあります。

同定され、能動的な介入研究の対象となっている老化への他の寄与には、テロメア短縮(細胞分裂中の染色体末端の保護的DNA配列の漸進的喪失)、老化細胞の蓄積(分裂を停止したが通常の細胞死経路に抵抗し、組織機能不全に寄与する炎症シグナルを生み出す細胞)、ミトコンドリア機能不全、幹細胞機能の喪失、細胞間コミュニケーションの破綻が含まれます。これらのそれぞれは実質的な研究プログラムの対象であり、さまざまな介入——テロメラーゼ活性化、老化細胞を選択的に殺すセノリティック薬剤、ミトコンドリア標的治療、幹細胞治療——が実験室モデルで、また場合によっては初期臨床試験で有効性を実証しています。

研究の現状は、150年や200年の人間寿命が、現在の軌跡を超える大きな科学的躍進なしに、これらの介入の何らかの組み合わせによって、来たる数十年以内に達成可能であるかもしれないことを示唆しています。500年や1,000年の寿命は、現在の軌跡を超える実質的な躍進か、あるいは——これが本書の枠組みにとって関連する可能性です——現在の生物学がまだ発展させていないまったく異なるアプローチを要求するでしょう。ラエリアンの典拠がエロヒムに帰属させる細胞移転技術——意識が新たに育てられた身体に再具現化される——は、そのような異なるアプローチの一つです。現代科学は現在これを行うことはできません。しかしさまざまな構成要素——幹細胞から代替臓器と組織を育てること、記憶と人格の神経的相関関係をマッピングすること、基盤間で生物学的情報を移転すること——はすべて活動的な研究の対象であり、それぞれにおいて実質的な進展がなされています。細胞移転的長寿のようなものに向かう軌跡は、遅いとはいえ、見えています。

本書の枠組みにとって、含意は、家父長的寿命が生物学的に不可能ではないということです。それらは、私たちが今や発展させ始めている種類の技術——より高度な文明が、数千年の持続的研究で、おそらく成熟させたであろう技術——を通じて生物学的に達成可能です。ラエリアンの典拠の命の木の技術は、本書の読みにおいて、実在の技術であり、私たちが今や自ら発展の初期段階にあるような種類のものです。家父長的寿命は、その技術を限定的な度合いで受けた存在の伝記的記録です。人間集団における現在のそのような長寿の不在は、根底にある過程の生物学的不可能性ではなく、本来の付与の限定的かつ最終的に撤回された性質を反映しています。

### IX.2. 混血生物学とエロヒムの子ら

*benei ha-Elohim* が人間の女たちを妻に取り、生存可能な子孫を生んだという創世記6章の主張は、実質的な生物学的含意を伴います。現代生物学において、種間生殖は関与する種間の遺伝的距離によって制約されています。近縁種は時に混血の子孫を生めますが(馬とロバからのラバ、ライオンと虎からのライガー)、子孫は典型的に不妊か、生殖力が大幅に低下しています。これは、異なる種の染色体が減数分裂中に対形成しようとするときに生じる困難を反映しています。より遠縁の種はまったく生存可能な子孫を生めません——遺伝的差異が、成功した受精、胚発生、生児出産を妨げるに十分です。

*benei ha-Elohim* と人間の女たちが生存可能で生殖力のある子孫——*ネフィリム*、*gibborim*、「名のある人々」——を生むためには、エロヒムは人間に生物学的に非常に近くなければなりません。具体的には、減数分裂中に人間とエロヒムの染色体が成功裏に対形成でき、胚成長に要求される発達プログラムが両立的であり、結果としての子孫自体がさらに生殖能力があるほどに、遺伝的に十分に近くなければなりません。これはエロヒムが何でなければならないかについての実質的な制約です。

この制約は本書のより広範な枠組みと整合的です。エロヒムは、典拠の説明では、霊長類のテンプレート——チンパンジーまたはアウストラロピテクスの出発点——を改変し、結果を霊長類ではなく人間にする具体的な改変を加えることによって、人間を設計しました。獅子座の章はこれを詳細に論じました。エロヒム自身が(根本的に異なる生物学の異星人ではなく)より高度な文明の生物学的に人間であるなら、エロヒムと人間のあいだの遺伝的距離は、混血生殖を支えるのに十分に小さいでしょう。これが本質的に典拠が全体を通じて主張することです——エロヒムは異星人ではない、彼らは私たちと同じような人間であり、自らの母星で独自の生物学的祖先から、私たちのものと同様の進化的または設計的方針に沿って進んだ祖先から子孫として降りてきたのです。{% libref(book="genesis-woh", chapter=1, verse=26) %}創世記1章26節{% end %}の「我々のかたちに」は文字通りの意味です——エロヒムは私たちのように見えます、なぜなら私たちは彼らのテンプレートをモデルとしているからです。

人間・エロヒム混血の生物学的帰結は推測的ですが、概観できます。混血の子孫は両親集団から特徴を継承するでしょう——人間の母からの基本的人間基盤と、*benei ha-Elohim* の父からの具体的なエロヒム変異。これらの変異には、強化された認知能力(エロヒムが、獅子座の章で論じたように、人間の設計の認知的最終点をわずかに上回るなら)、より長い寿命(エロヒム生物学が、人間が技術的介入によってのみ達成する長寿を自然に支えるなら)、より大きな身体的能力、おそらく混血集団の子孫に検出可能なエロヒム系統と関連する具体的な遺伝マーカーが含まれうるでしょう。*ネフィリム* を *gibborim*、「力ある者たち」、そして *anshei ha-shem*、「名のある人々」とする聖書の記述は、二重の遺産が通常の人間の基準を超える強化された能力を生み出した子孫と整合的です。

現代の読者にとって、含意は本書の枠組みが遺伝的記録において検出可能なものを予測しているということです。もしユダヤ人集団(あるいはその中の特定のサブ集団)が、ラエリアンの典拠が主張するように混血系統から子孫を引いているなら、エロヒムの祖先の遺伝的署名は、原則として、同じ混血史を持たなかった集団の祖先から区別する、ユダヤ人の祖先の特定の構成要素として検出可能なはずです。現代の集団遺伝学は今日まで、そのような特徴的な構成要素を同定していませんが、分析は何を探すべきかを知っている必要があるでしょう。エロヒムのゲノムは混血生殖を支えるのに十分に人間に近かったでしょうが、十分に洗練された遺伝分析にとって特定のマーカーが検出可能なほどに区別されたものであったでしょう。本書はこの分析がなされたとは主張しません——それは、原則として、枠組みが具体的に予測する何かが試験可能であることを記しています。

### IX.3. 前土器新石器アナトリア——ギョベクリ・テペと石の文明

本書の枠組みにとって、過去30年間のもっとも重大な考古学的展開は、南東アナトリアの前土器新石器遺跡——本書の読みにおいて、本来のエデン領域に対応する地域——の発掘と研究です。

**ギョベクリ・テペ**は、初期人類文明の理解を革命化した遺跡であり、ドイツの考古学者クラウス・シュミットによって同定され、1995年から彼の指揮下で発掘されました。遺跡は、南東トルコのシャンルウルファ県の丘の頂上、現代のシャンルウルファ市の北東約15キロメートルに位置しています。シュミットとそのチームが二十年間の発掘を通じて明らかにしたものは、複雑な人間文明がいつ始まったかについての以前支配的だった見解を実質的に覆しました。

ギョベクリ・テペは複数の円形および卵形の囲い地から構成され、それぞれが意図的な様式で配置された大型の彫刻石灰岩柱によって定義されています。最大の柱はT字形で、約5.5メートルの高さを持ち、それぞれおよそ10–20トンの重さがあります。多くの柱は動物——ライオン、キツネ、鳥、蛇、サソリ、イノシシ、ガゼル——の詳細なレリーフで、特定の図像学的構図に配置され、その意味はまだ理解されていません。一部の柱は、原始文字、暦のマーカー、または宗教的図像として多様に解釈されてきた抽象的シンボルを示しています。

年代決定が決定的な詳細です。遺跡の有機物質の炭素年代決定は、その建設を紀元前10千年紀および9千年紀——具体的には、最も初期の層は約紀元前9600年に遡り、建設は約紀元前8000年に遺跡が意図的に埋められる(瓦礫で埋められる)まで約1,500年にわたって続きました——に位置づけます。本書の年代記では、これはギョベクリ・テペを早期から中期の蟹座の時代に明確に位置づけます。遺跡は時間的に、本書の枠組みのエデン直後の期間——追放直後の数世紀、エデン文明が園以後の発展の初期段階にあった時期——です。

ギョベクリ・テペの重要性は、それが以前支配的だった考古学的枠組みが許容していたよりも、この早い時期に実質的により洗練された組織、技術的能力、文化的複雑性を実証することです。ギョベクリ・テペ以前、主流考古学における支配的見解は、複雑な記念碑的建築物は定住農業社会を要求し、農業はそのような建築物を数千年先行し、発展の順序は——遊牧狩猟採集民の集団 → 定住農業村 → 記念碑的建築物のある初期都市——でした。ギョベクリ・テペはこの順序を逆転させます。遺跡は、狩猟採集民または原始農業集団であったと思われる人々によって建設されました。彼らは多トンの石柱を採掘、輸送、彫刻、建立するのに十分な規模と、十分な技術的能力で組織され、それを千年以上の持続的建設にわたって行いました。支配的枠組みがそのような建築物の必須前提条件として扱った農業は、ギョベクリ・テペでは、おそらく先駆ではなく*帰結*であった——遺跡の集団が狩猟採集が提供しうるよりも信頼性の高い食糧供給を必要とするほどに大きくなるにつれて、周辺地域に出現したのです。

クラウス・シュミット自身、2014年の死去以前に、含意を定式化し始めていました——慣例的な順序は誤りであり、記念碑的な宗教的または儀式的建築物が農業の発展に先行し、おそらくそれを駆動したこと、そしてそのような建築物に要求される人間の文化的能力は以前の枠組みによって実質的に過小評価されていたということです。含意は過去十年間に主流考古学に実質的に吸収されました。ギョベクリ・テペはもはや周縁的な発見ではありません——それは今や、更新世から完新世への移行を理解するための基礎的な遺跡の一つとして扱われており、その含意を調査する実質的な研究プログラムが今や進行中です。

本書の枠組みにとって、ギョベクリ・テペは予測された描像と直接的に整合する物理的証拠の一片です。本書の枠組みは、洪水以前文明が早期蟹座期にエデン地域で実質的な洗練度で存在し、もし何らかのそのような建築物が続く洪水と長い侵食過程を生き残ったなら、記念碑的建築物を残したであろうと予測しています。ギョベクリ・テペはまさにそのような生き残った証拠です——実質的な洗練度の記念碑的建築物が、予測された地域に、予測された時期にあるのです。本書はギョベクリ・テペが特定のエデン関連事象と一意的に同定されると主張しません。それが主張するのは、ギョベクリ・テペは本書の枠組みが予測するものであり、そのような場所のそのような遺跡の発見が、本書とは独立に通常の考古学的方法を通じてその時期になされたことは、本書の年代記的枠組みのもっとも強力な物理的証拠の一つであるということです。

**カラハン・テペ**は、南東アナトリアの同じ地域の別の遺跡で、2019年からイスタンブール大学のネジミ・カルルと彼のチームの指揮下で発掘されてきました。カラハン・テペはギョベクリ・テペよりやや小さいですが同じくらい早期——一部の予備的年代決定ではおそらくわずかに古い——であり、記念碑的T字形柱、彫刻されたレリーフ、複雑な空間構成という同様の特徴を示します。発掘は執筆時点で進行中であり、カラハン・テペが含むものの全容はまだ公表されていません。すでに明瞭であるのは、ギョベクリ・テペが唯一無二ではないということです。南東アナトリアの前土器新石器A文化は、実質的な地理的地域にわたって複数の記念碑的遺跡を生み出し、関与する文化的・技術的能力が単一の孤立した共同体の仕事ではなく、複数の中心を持つより広い文明のものであったことを示しています。

より広い前土器新石器複合体はこれら二つの具体的な遺跡を超えて他のいくつか——ネヴァル・チョリ、ハムザン・テペ、セフェル・テペ等——を含み、それらは共にエデン地域における早期蟹座期文明の考古学的署名を構成します。本書の枠組みはこの文明が存在したと予測します——主流考古学は今やその存在を実質的に確認しました。枠組みと主流の不一致は、この文明が何であったかの解釈にあります——枠組みはそれをイスラエル・チームの創造から下る初期段階のエデン系統として読み、主流考古学はそれを人間の文化的複雑性の注目に値するが説明されない初期の例として読みます。枠組みの読みのほうがより具体的です。それはギョベクリ・テペをより広い物語に接続しますが、主流考古学は遺跡を持っていますが、それの説明をまだ持っていません。

### IX.4. ヤンガードリアス

ヤンガードリアスは現在から約12,900から11,700年前——すなわち本書の年代記では獅子座の時代の後期数世紀から蟹座の時代の早期数世紀——に起こった気候事象です。事象は、北半球の多くにわたる、ほぼ氷河期に近い寒冷条件への突然の戻りを伴い、約1,200年続いた後、最終氷期極大期の終わり以来進行していた温暖化傾向に同じく突然に気候が戻りました。

ヤンガードリアスの原因は実質的な議論の対象となっています。慣例的な説明は、それが後退するローレンタイド氷床からの融解水の突然の放出による北大西洋海洋循環の混乱によって引き起こされたというものです。代替的な説明、ヤンガードリアス衝突仮説は、事象が北米大陸上空での彗星または小惑星断片の衝突または大気空中爆発によって引き起こされたと提案します。仮説はもともと2007年にファイアストーンと同僚によって提案され、彗星研究グループと他の研究者によって以来実質的に発展してきました。仮説を支持する証拠として引用されるものには、複数の北米、ヨーロッパ、グリーンランドの遺跡で発見された、ヤンガードリアス開始期に遡る白金に富む堆積物の層、高温事象を示唆する炭素球とナノダイヤモンド、そしてヤンガードリアスが北米のメガファウナの絶滅とクローヴィス文化の消失と一致するという見かけの符合が含まれます。

ヤンガードリアス衝突仮説は主流科学におけるコンセンサスではありません。支持研究の一部の方法論に対して実質的な批判が向けられており、白金と他のマーカーに対する代替的説明が提案されています。本書は衝突仮説を正しい説明として自らを縛りません。それが記すのは、ヤンガードリアスのタイミングが、いずれの説明であれ、本書の年代記では獅子座後期/蟹座早期の移行——人間の創造が最終化され、初期エデン以後文明が発展し始めた期間——に該当することです。もしヤンガードリアスが実際に宇宙起源の壊滅的事象であったなら、それは超大陸全体の発展しつつある文明に帰結を伴う、早期蟹座期に影響する実質的な環境的混乱だったでしょう。本書はこれを、根底にある因果問題に対する強い立場を取らずに、本書の年代記的枠組みに関わる現代科学的議論の一片として言及します。

### IX.5. アトランティスと文明的記憶の様式

§VIIIはすでにアトランティス伝統とリシャット構造を候補位置として論じました。科学節は、プラトンの記述、そして世界的神話伝統にわたる洪水以前の文明的記憶のより広範な様式が、本書の枠組みが自然に統合できる証拠の体系を構成し、主流考古学が退ける扱いをしてきた傾向があることの具体的詳細を加えるだけで足ります。

本書が採用する方法論的立場は次のとおりです。プラトンは真剣な資料です。彼はファンタジー文学を発明するという意味での神話作家ではありませんでした——彼は哲学者であり、その対話篇は哲学的議論をなすために歴史的および準歴史的素材を用いており、彼の『ティマイオス』と『クリティアス』でのアトランティス物語の使用は、エジプトの資料についての具体的な主張と具体的な年代記的詳細を伴う歴史的なものとして提示されています。プラトンの記述を純粋な発明として退けることは、実質的な信頼性を持つ真剣な哲学者が不明確な理由のために延長された歴史的物語を意図的に捏造したと仮定することを要求しますが、これは彼が独立に確認されていない内容を持つ何らかの真の歴史的資料で作業していたと仮定するよりもより節約的ではない仮定です。

同様に、世界的伝統にわたる洪水以前の文明的記憶のより広範な様式——シュメール王名表、ヒンドゥーのユガ、メソアメリカの以前の試み、ポリネシアの沈んだ故郷、その他——は、何らかの完新世以前の高度文明が存在し破壊されたという独立した証言の体系を構成します。主流考古学はこれらの伝統のそれぞれを孤立して扱う傾向があり、それらを神話、世界的ではなく地域的事象の記憶、または共通の源からの文化的拡散にさまざまに帰してきました。本書の枠組みは代替的解釈を提供します——世界的様式は洪水以前文明の世界的存在を反映し、その記憶の断片が洪水以後の集団のそれぞれに生き残ったというものです。

本書はいかなる特定の伝統といかなる特定の歴史的事象との特定の同定をも主張しません。それが提供するのは、様式が首尾一貫して解釈されうる枠組みと、どのような物理的証拠(ギョベクリ・テペ、もしその代替的年代決定が正しければスフィンクス、おそらくリシャット構造のようなもの)が破滅的事象と長い続く侵食期間を生き残ると予期されうるかについての予測です。枠組みは利用可能な証拠と、退ける主流の枠組みがそうでない仕方で整合的です。枠組みが正しいかどうかは、本書が決定的に解決できない問題です。本書が主張するのは、枠組みが少なくとも主流の代替と競争的であり、真剣な考慮に値するということです。

### IX.6. 集団のボトルネック問題

創世記の洪水物語は実質的な集団のボトルネックを示唆しています。もし洪水がノアの家族を除く本質的にすべての人類を殺したなら、洪水以後の人間集団はおそらく8人(ノア、その妻、3人の息子、息子たちの妻)の小さな創設集団から子孫を引きます。これは深刻なボトルネックであり、後続のすべての人間集団の遺伝的記録に検出可能な署名を残すでしょう——ボトルネック事象に遡る減少した遺伝的多様性が、その後集団が拡大するにつれて漸進的に多様性を回復する。

現代の集団遺伝学は人間進化史におけるいくつかの見かけのボトルネックを同定しました。最も有名なのはいわゆるトバ・ボトルネック仮説であり、約74,000年前のインドネシアのトバ山の噴火が人間集団における絶滅近い事象を引き起こしたと提案しました。トバ仮説はより最近の研究によって実質的に弱体化されましたが、人間先史におけるボトルネックのより広範な問いは活動的なままです。ミトコンドリアDNAとY染色体系統のさまざまな分析は、さまざまな時期に合体事象を同定し、そのうちの一部はボトルネックまたは集団減少の証拠として解釈されてきました。

本書の枠組みは洪水時期——本書がおおよそ紀元前6,690年(双子座の時代の始まり)に置く——に主要なボトルネックを予測します。この特定の日付でのボトルネックの遺伝的証拠は限定的です——人間集団遺伝学で同定された主要な合体事象のほとんどは、この日付よりも実質的に古いか実質的に新しいかです。本書は紀元前6,690年ちょうどでの明確なボトルネック署名が同定されたとは主張しません。それが記すのは、創世記の物語が、本書の年代記がその具体的日付を特定するボトルネックを示唆しており、これが原則として十分に洗練された遺伝的分析で試験できる種類の予測であるということです。もし枠組みが正しいなら、将来の遺伝的研究はこの日付頃でのボトルネック署名を同定でき、現代の人間集団はこの時期に生きた小さな創設集団から子孫を引いていることがわかるはずです。現在の分析におけるそのような明確な署名の不在は、枠組みに反する証拠か、あるいは既存の分析が特定の問いを念頭に置いてなされていない証拠のいずれかです。

### IX.7. 私たち自身の瞬間への通底線

蟹座に関連するエロヒムが有していた能力——細胞技術による持続的な文明的長寿、別個の人間様集団間の混血生物学、私たち自身の文明が生み出せるものに匹敵するか、それを超える規模の記念碑的建築物、高度兵器による持続的な文明間戦争、世界的距離を越える実質的な集団の政治的管理——は、前章の能力と同じく、私たち自身の文明が今や近づき始めている能力です。

長寿研究はもっとも直接的な現代の並行例です。細胞再プログラム化研究、テロメア生物学、老化細胞除去、関連介入の現在の軌跡は、数十年以内の人間寿命の実質的な延長を指し示しています。現在の人間の最大寿命は約122年です——現在の研究の妥当な予測は、150–200年の寿命が、利用可能な介入の何らかの組み合わせを通じて、次の数十年以内に達成可能であることを示唆しています。それを超える軌跡——族長たちの数世紀の寿命、あるいはラエリアンの典拠がエロヒムに帰属させる千年の寿命に向けて——は、現在の軌跡を超える主要な科学的躍進を要求するでしょうが、そのような躍進の構成要素は活動的な発展中です。

混血生物学の問いはより推測的ですが、並行する軌跡をたどります。CRISPRに基づく人間生殖系列改変は今や技術的に可能であり、これは獅子座の章で論じました。非人間源からの実質的なDNAの人間生殖系列への統合は技術的により要求が厳しいですが、現在の研究軌跡内で考えられます。医学研究目的のための人間・非人間キメラ(異種移植、疾患モデル化)を生み出す能力は、霊長類・豚キメラと関連する仕事が、種間生物学的統合がますます扱いやすくなっていることを実証して、過去十年間に実質的に発展してきました。典拠が描く人間・エロヒム混血化のようなものに向かう軌跡は、具体的な終点はまだ私たちの地平線上にはないとしても、断片的に見えます。

洪水以前文明の考古学は、進行方向が本書の枠組みにとってますます好都合な現代の研究フロンティアです。ギョベクリ・テペとカラハン・テペは複雑な人間文明の認識されたタイムラインを実質的に延長しました。かつての沿岸地域(後氷期の海面上昇により今は水没)の沈んだ遺跡の調査は、洪水以前の人間活動の新しい証拠を生み出しています。さまざまな地質学的・生物学的事象に対する破局論的説明への新たな関心——ヤンガードリアス衝突仮説、さまざまな隕石衝突の記録——は、主流科学が大部分解決済みとして扱ってきた問いを再開しています。一般的な進行方向は、人間の過去が慣例的な漸進主義的枠組みが許容していたよりも実質的により複雑で、より高度で、破滅的事象によってより区切られていたことを認識することへ向かっています。

本書にとって、この軌跡は、枠組みが評価されるべき経験的文脈です。本書の枠組みは、関連する現代の証拠の大半が存在する前の1973年に、ラエリアンの典拠資料から発展しました。枠組みは証拠に先立って、洪水以前文明が実質的な洗練度で存在したこと、エデン地域における早期蟹座期は検出可能な記念碑的建築物を生み出したであろうこと、人間の長寿は技術的介入を通じて実質的に延長されうること、獅子座後期/蟹座早期の移行における破滅的事象が発展しつつある文明を形作っていたであろうことを予測しました。これらの予測のそれぞれは、本書やその資料を参照せずに行われた独立した現代研究によって実質的に支持されてきました。収斂は部分的であり、証拠は不完全であり、本書は決定的に正当化されたとは主張しません。それが主張するのは、枠組みが生き生きとして生産的なままであり、現代の証拠はそれを反駁する方向ではなく、枠組みが予期した方向に傾く傾向があるということです。

## X. 本文とその信号

創世記2–6章のヘブライ語本文は、先行する節で記されたもの以外に、いくつか注目に値する特徴を含んでいます。

第一に、複数のエロヒムはこれらの章を通じて、創世記1章におけるのと同じ文法的形態で、同じ複数の自己呼称と、明らかに複数の主語としての *benei ha-Elohim* への同じ言及で続きます。文法的複数性はエデン物語によって単数性に解消されません——それは保存され、場合によっては強調されます({% libref(book="genesis-woh", chapter=3, verse=22) %}創世記3章22節{% end %}における *ken ha-adam hayah ke-achad mimenu*、「人は我々のひとりのようになった」のように)。本文は、発話の動詞が単数の場合でも、複数の主語を描き続け、この一貫した複数性を創造者集団の実際の多様性を反映するものとして読むラエリアン的読みは、文法的証拠のもっとも経済的な解釈のままです。

第二に、{% libref(book="genesis-woh", chapter=2, verse=4) %}創世記2章4節{% end %}における神名 יהוה (テトラグラマトン、慣例的にヤハウェと母音化される)の導入は、本書の枠組みが説明するのに役立つ本文的移行を示しています。創世記1章の記述は全体を通じて *Elohim* を用い、いかなる個別の人物にも具体的な名前を持ちません。創世記2章の記述は両方の名前を *Yahweh Elohim* ——「ヤハウェ、エロヒム」——として組み合わせます。この組み合わせは広範な本文批評的議論の対象となってきており、文献仮説は二つの名前を、後に組み合わされた二つの異なる資料文書(*Elohim* を用いる祭司資料、*Yahweh* を用いるヤハウィスト資料)の証拠として扱います。ラエリアン的読みは代替的解釈を提供します——創世記1章はすべての七つのチームによる創造プログラム全体の集合的な要約であり、創世記2章はヤハウェと名指される特定のエロハの監督下で行われた特定のイスラエル・チームの操作のより詳細な物語です。*Elohim* から *Yahweh Elohim* への移行は、全体集団から、一人の名指された人物が監督していた特定の操作への焦点の絞り込みを示しています。両方の名前は正確です。組み合わせ *Yahweh Elohim* は、具体的な同一性(ヤハウェ)と、名指された人物が属するより広い範疇(エロヒム)の両方を保存します。

第三に、創世記5章に保存される洪水以前の族長たちの年齢——アダムからセツ、エノス、ケナン、マハラレル、ヤレデ、エノク、メトシェラ、レメク、ノアに至る系図——は、他の点では非常に圧縮された本文における詳細な十世代の記録であり、その具体性は注目に値します。聖書著者は、それらが重要と考えられていたことを示唆する精度でこれらの年齢を保存しました。ラエリアン的読みにおいて、その重要性は明瞭です——年齢は洪水以前の世代にわたる命の木の技術の運用を文書化し、誰が処置を受け、どのくらい持続し、いつ最終的に撤回されたかの記録を保存しています。メトシェラの969年——聖書記録の最長寿命——は、処置から恩恵を受けた最後の完全な世代を示しています。彼の死は、聖書年代記では、洪水の年に起こります。それゆえ系図は文学的装置ではなく、技術的記録であり、洪水以前の資料から洪水以後の伝統が継承した具体的な年代記的データを保存しています。

第四に、エノクという人物は§VIIですでに論じましたが、{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記5章24節{% end %}における彼の扱いは本文上の注意に値します。*Vayithallekh Hanokh et ha-Elohim, ve-einenu ki lakach oto Elohim*、「そしてエノクはエロヒムとともに歩み、彼はいなくなった、エロヒムが彼を取られたからである」という表現は、創世記5章のすべての他の族長の項目と文法的に区別されます。他の族長たちはすべて「そしてXはY年生きてZを生み、Xはそれからもさらに生きてZの後にもW年生き、息子と娘を生み、Xのすべての日々はN年であり、彼は死んだ」という定型で描かれています。エノクの項目はこの定型を破ります。彼は死にません。彼はエロヒムとともに歩みます。彼は取られます。ヘブライ語はこのすべてについて事実的であり、ラエリアン的読みはそれを額面通りに受け取ります——エノクはエロヒムと特定の関係(継続的な直接接触を示唆する彼らとの「ともに歩む」)を持ち、特定の時点で彼は地球から物理的に取り除かれ、母星に連れて行かれました。エチオピア教会に保存されたエノク書は、彼が訪問中に見たものを記録しています。{% libref(book="genesis-woh", chapter=5, verse=24) %}創世記5章24節{% end %}の文法的区別は、エノクとともに何か異例のことが起こったという本文上の信号であり、エノク文献はそれが何であったかの詳述された内容を保存しています。

第五に、*benei ha-Elohim* と *ネフィリム* を導入する{% libref(book="genesis-woh", chapter=6, verse=1) %}創世記6章1–4節{% end %}の箇所は、その文法的圧縮で注目に値します。本文は短く——4節——ですが、各節は実質的な含意を担います。定冠詞付きの複数 *benei ha-Elohim* はヘブライ語において明白です——これらは、ある弁明的解釈が提案したような「敬虔な人々」や「貴族の子ら」への曖昧な言及ではなく、エロヒム範疇の具体的な既知の存在です。動詞 *vayikchu*(そして彼らは取った)に続く *nashim*(妻)は、結婚を表す標準的なヘブライ語です——これは気軽な遭遇の描写ではなく、正式な婚姻結合の描写です。これらの結合の子孫としての *ネフィリム* は、*gibborim* と *anshei ha-shem* として描かれており、本文がその存在を歴史的事実として確言する具体的な存在です。箇所の圧縮は、ラエリアン的読みにおいて、元の伝統の明示的内容を和らげた後ラビ的編集過程を反映しています——エノク書は、正典ヘブライ本文が最終化される前の物語の以前の版に推定で存在していた詳述された素材を保存しています。

## XI. 蟹座とは何か

章が終わる前に、より大きな連なりの中で蟹座の時代が何であるかを率直に述べる価値があります。

蟹座は人間の創造が人間の歴史となる時代です。それは獅子座の終わりに生み出された存在たちが、行為する能力を持つ存在たちのなすことを始める時代です——自らの状況に目覚め、造り主に背き、本来の環境から分離され、生殖し、関係を形成し、犯罪を犯し、文明の最初の要素を構築し、追放された教師に支えられたエデン系統の場合は、最初の要素よりはるかに多くを構築する。安息の日、第七日は、ある意味で誤称されています。創造者は創造することから休みました。彼らの被造物は存在することから休みませんでした。そして安息期間中に彼らがし始めたことが、ヘブライ語聖書の残りと人類史の残り大半を占めることになります。

蟹座はまた、後続のあらゆる出来事を形作る政治構造が確立される時代でもあります。創造者集団は分割されています——母星に撤退した大多数と、人間のあいだに地球上に永久に追放された少数派。本書の残り全体を通じて繰り返し現れる政治的人物——母星のサタン、評議会を主宰するヤハウェ、地球上で追放者を率いるルシファー——は今や名付けられ区別されています。追放された創造者は人間集団と関係を形成し始め、混血の子孫を生み出します。その存在自体が母星の介入の身近な原因となるでしょう。超大陸全体の七つの人間文明は並行して発展し、エデン文明は累積する利点——設立集団の才能、教師としてのルシファー派の継続する直接的存在、指導者に与えられた長寿、指導集団内の混血エロヒム遺伝物質、軍事能力を含む高度技術——のため最も先まで進歩します。他の文明は実質的に進歩しますが、より低い速度であり、超大陸全体にわたる交易と文化的接触が、世界規模で連結されているが不均等に発展した洪水以前世界を生み出します。

蟹座は、最後に、母星が当初から恐れていたものが現実となりつつあることを認識して終わる時代です。エデン文明は母星自身の能力に特定の側面で近づくか、それを超える可能性のあるレベルへと進歩しました。超大陸の文明のあいだの戦争は、破壊能力が実在し、用いられていることを示しました。追放された創造者はこの発展を防ぐのに失敗しただけでなく、継続する教えと彼らが生んだ混血の子孫を通じて積極的に加速させました。母星が獅子座の終わりに課した政治的決着は、2,000年後、本質的にあらゆる点で失敗しました。制約は破られました。本来の反対者が警告した脅威は実現しました。母星が蟹座の終わりに達する結論は、次の時代に実施される結論です——人間の創造は、現在の形では、母星文明自体への真の脅威となる前に破壊されねばならないということです。ヘブライ語聖書が腐敗した人類に対する道徳的判決として提示する洪水は、ラエリアン的読みにおいて、その進歩が母星が安全と考える閾値を超えた人間文明に対する政治的行動です。洪水以後の伝統における事象の神話化は本来の決定の政治的性格を覆い隠してきましたが、政治的性格こそ本書が記録し、次章が扱うものです。

本書はこの章を通じて、蟹座の終わりに世界がどう見えたかを描きました。単一の超大陸、交易と戦争を通じた継続的接触のうちにある七つの異なる文明によって居住されたもの。これらの文明の大半は実質的な文化的・技術的レベルにあり、エデン文明が先端にあります。長寿の家父長的指導者が世紀にわたって知識を蓄積します。追放された創造者が人間のあいだで生き、その一部は人間の女と結婚し、一部は通常の人間規範を超える能力を持つ混血の子孫の親となります。乙女座の創造から持続する恐竜と他の洪水以前のメガファウナ、その一部は数が減っていますが景観に依然存在し、本質的にあらゆる洪水以後文化の神話に生き残ることになる「ドラゴン」の記憶に寄与しています。洪水以後の世界が認識しない動植物——次の時代の破滅的事象が、本書が完全な記録を持たない種を排除するからです。本質的にあらゆる点で洪水以後の世界——考古学的記録が文書化し始める世界——とは異なる世界。これが本書が前章を通じて、そしてこの章を通じて復元してきたものです。これが洪水が終わらせたものです。

蟹座の終わりまでにエデン文明がどこまで進歩したかについて、私たちは実質的な謙虚さをもってのみ推測できます。物理的証拠は断片的であり、生き残ったものの大半は洪水自体で破壊されました。後続の伝統に保存された文化的記憶は私たちに手がかりを与えます——洗練された工学を持つプラトンのアトランティス、長期に治めた洪水以前の支配者たちのシュメール王名表、高度な以前の周期を持つヒンドゥーのユガ、神々が人々のあいだを歩いた *zep tepi* を持つエジプト。生き残った物理的遺跡——南東アナトリアのギョベクリ・テペとその仲間、代替的年代決定が正しければスフィンクス、候補アトランティスとしてのリシャット構造、確かにまだ同定されていない、あるいは回復を超えて破壊されたさらに多くの遺跡——は、私たちにさらなる手がかりを与えます。すべてを組み合わせると、現れる描像は実質的な洗練度の文明のものです——記念碑的建設の能力があり、実質的な軍事技術を有し、交易と政治的接触を通じて惑星規模で組織され、世紀の経験を蓄積した長寿の指導者によって統治され、さらに高度な地球外文明の反対派と統合された文明。この文明がどこまで、どの具体的な側面でいったかは、本書が自信をもって特定できないものです。本書が言えるのは、それが*母星が介入を避けられないと考えるほど十分に遠くまでいった*ということです。母星が達した判断、2,000年にわたる発展を見ての判断は、状況が本来の政治的決着が持続できる点を超えたということでした。その判断こそ、次章が実施を見るものです。

次の時代は洪水が起こる時代です。洪水以前人類の大半の破滅的破壊、惑星の地質学的・生物学的再形成、私たちが今認識する大陸への超大陸の分裂、追放された創造者の技術的介入による小さな残存者の保存、瓦礫からの洪水以後文明の出現——このすべては双子座の時代の主題であり、続く章の主題です。
