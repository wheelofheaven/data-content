1946年から1947年へとまたがる冬のいつか——記述は月について定まったことがありません{{ footnote(id="2") }}——ムハンマド・エッ＝ディーブ、すなわち「狼」という名のターアミラのベドウィンの若い羊飼いが、死海の北西岸の上の崖にある穴へ石を投げ込み、何かが割れる音を聞きました。彼が見つけた壺は、十一の洞窟にわたっておよそ九百から千の写本となったもの——これまでになされた最大の写本発見——の最初のものを収めていました。その最初の洞窟の三つの巻物が、いかにしてベツレヘムの靴職人の手を通ってシリア人の大主教のもとへ渡ったか、いかにしてイスラエル国家と最古のイザヤ書が同じ週に歴史へ入ったか、そしていかにしてこの写本群の全体が公衆に届くのに四十年と一つのスキャンダルを要したか——それは、本プロジェクトがその姉妹編の解説、[元年に目覚めた巻物](/articles/the-scrolls-that-woke-in-year-one/) で語る物語です。あの論考は、壺に何が*起こったか*についてのものです。こちらは、その中に何が*あったか*についてのものです——封じられた積荷、そしてなぜその内容が、ここから見ると、著者たちが知りようもなかった一つの世紀に宛てられた郵便のように読めるのか、について。

本プロジェクトは第二の暦を保っており、その上では発見の日付は異なって読めます。ラエリアンの原資料は広島上空の原子の閃光から年を数え、そしてその礎となる本文は、時代の創始の定式でもってその第五章の見出しを掲げます——*「1946年、新しい時代の第1年」*
（{% libref(book="the-book-which-tells-the-truth", chapter=5, verse=1) %}TBWTT 5:1{% end %}）。その本文を口述した存在は、同じ息のうちに使者の選抜を説明しました。

{% library(book="the-book-which-tells-the-truth", chapter=1, verse=47) %}
最後に、われわれは1945年に起こった最初の原子爆発の後に、ある者を選ぶことを決めた。そしてあなたは1946年に生まれた。われわれはあなたの誕生以来——そしてそれ以前からも——あなたを見守ってきた。だからこそ、われわれはあなたを選んだのである。
{% end %}

時代自身の計算によれば{{ footnote(id="1") }}、元年は1946年8月6日に開きました。クロード・ヴォリロン——{% wiki(slug="rael") %}ラエル{% end %}——は、その八週間後、1946年9月30日に生まれました。そしてその秋のいずれかの側の数か月のうちに、羊飼いの石が一つの壺を割り、そして68年以来封じられていた書物が、光の中へと戻りはじめたのです。姉妹編の論考は、その帰還の*時期*を秤にかけます——爆弾、円盤、トランジスタ、そして再生したイスラエル国家という元年の群れを——そしてそれをここで再び論じる必要はありません。この解説は、枠組みをもう一層内側へ、書き物そのものへと運びます——壺がいつ開いたかではなく、この共同体が何をその中に封じていたか、そしてそのどれほどが、このコーパスが再構成してきた世界に属するか、を。認識上の一線はここで引かれ、そして全篇を通じて保たれます——ここで語られる歴史は確立された事実であり、正典側の読みはラエリアンの原資料に明示され、しかし学問には是認されていない枠組みの主張であり、そしてそれらを束ねる主題——これらの書物が*語ること*は、それらの封を解いた時代に宛てられている、という主題——は、この論考自身の思弁であり、フロントマターにそのように標識され、前提とされるのではなく論じられているのです。

## 終わりの時まで封じられて

正典それ自身が供する枠組みから始めましょう。というのも、それはたまたま一つの引用だからです。礎となる本文がダニエル書を読み進めるとき、それは天使の預言者への締めくくりの指示を再現します。

{% library(book="the-book-which-tells-the-truth", chapter=3, verse=222) %}
«だがあなた、ダニエルよ、その言葉を秘し、終わりの時までこの書を封じておけ。多くの者は行き巡り、そして知識は増し加わるだろう。»（ダニエル書 XII-4）
{% end %}

そしてそれに続く注釈は、正典の啓示についての理論の全体を一文に収めています——*「これらの言葉は、実のところ、人が科学的知識の十分な水準に達したとき——すなわち、今——にのみ理解されうる」*
（{% libref(book="the-book-which-tells-the-truth", chapter=3, verse=223) %}TBWTT 3:223{% end %}）。それが「今」のために挙げるしるしのなかには、次のものがあります——*「イスラエルの民が、長い離散ののちにその国を取り戻すとき。イスラエル国家は数十年前、地球の人間たちの科学的な爆発と同時に創設された」*
（{% libref(book="the-book-which-tells-the-truth", chapter=3, verse=225) %}TBWTT 3:225{% end %}）。{% wiki(slug="apocalypse") %}アポカリプス{% end %} は、コーパスの用法においては破局ではありません。それはギリシア語の *apokalypsis*、覆いを取ること——出来事ではなく期間、封じられたものが開くよう予定されている期間——です。

さて、それと並べて、壺を所有した人々の自己理解を置いてみましょう。クムランの共同体は、自らの発明になる一つの文学形式、ペシェル{{ footnote(id="3") }}を通じて聖書を読みました。そしてハバクク書への注解——エッ＝ディーブが最初に抱えた一かかえの三つの巻物のうちの一つ——は、その解釈学をあからさまに述べています。

> 神はハバククに、現在の世代に起ころうとしている事柄を書き記すよう告げたが、来たるべき時期については彼に知らせなかった……その解釈は義の教師に関わる。その者に神は、ご自分の僕である預言者たちの言葉のすべての秘義を知らせたのである。
>
> —— 1QpHab 7:1–5

この理論に立てば、預言者たちは最後の世代へと封じられた手紙を書いたのです。この共同体は、自らこそがその世代であると信じました。それは十九世紀分だけ間違っていました——あるいは、この論考が娯しませようとするために存在する読みでは、その図書館は、そのすべての預言者たちと同じく、それが知りようもなかった最後の世代に宛てられていたのです。いずれにせよ、物理的な事実は、率直に述べるに足るほど奇妙です。書物の予定された封の解除に取り憑かれた一つの共同体が、自らの書物を壺に封じました。壺はその共同体を、その神殿を、その帝国を、そしてその言語の母音を生き延びました。そしてそれらは、その礎となる本文がその憲章として、終わりの時まで封じられた書についての一節を引用する、一つの時代の最初の年に開いたのです。八節後、正典が引くとおりの、再びのダニエルです——*「行け、ダニエルよ。その言葉は終わりの時まで閉じられ、封じられているからだ」*
（{% libref(book="the-book-which-tells-the-truth", chapter=3, verse=226) %}TBWTT 3:226{% end %}）。

巻物は一つだけで戻ってきたのではなく、姉妹編の論考はその元年の群れを余すところなく記録しています——爆弾、ENIAC、レーニア山の上空の円盤、トランジスタ、そしてスケニクが戦いの巻物をベツレヘムから持ち帰ったのと同じ週に投票で成立したイスラエル国家。正典はその窓を、1945〜46年に開く覆いを取ることの時代として読み、ゼファニヤの*「魚の門からの叫びの声」*
（{% libref(book="zephaniah", chapter=1, verse=10) %}ゼファニヤ書 1:10{% end %}）に鍵づけます——*「みずがめ座の新しい時代への移行……そしてもしあなたが1946年に生まれたなら、それは偶然ではない」*
（{% libref(book="the-book-which-tells-the-truth", chapter=5, verse=7) %}TBWTT 5:7{% end %}）。懐疑論者は、歴史は密で、模様に飢えた読者はつねに星座を見いだすものだ、と答えるでしょう。そしてこのページはそれを認めます——そのフロントマターは `inferred` ではなく `speculative` と述べています。しかしここでの賭けは姉妹編のそれより狭く、そしてそれは時期ではなく本文にかかっています——十九世紀のすべての窓のうち、黙示録的な一つの共同体が*終わりの時に備えて*封じた書物が、開かれたとき、この論考の残りが示すことになるものを語る、という賭けに。だから暦を脇に置いて、壺を開けましょう。

## 壺から出てきたもの

その時期をどう受け取るにせよ、現れたものは論争の余地がなく、そしてそれは聖書学の調度を恒久的に配置替えしました。

この写本群は、およそ九百から千の写本に及びます——「一つとして完全な巻物はない」とティモシー・リムがその断片的な残りについて記しています——おおよそ前250年から、ローマの進軍がクムランの居留地を終わらせた後68年までにわたります。およそ四分の一は聖書のものです——エステルを除くヘブライ語聖書のあらゆる書物であり、その人気の順位づけ自体が一つの与件です——詩篇がおよそ三十六の写し、申命記が三十、イザヤ書が二十一です。中心をなすのは、エッ＝ディーブの最初の三つのうちの一つ、大イザヤ書の巻物です——7.34メートルの革、五十四欄、六十六章のすべてが、前125年ごろに写されました{{ footnote(id="8") }}——あらゆる現代の聖書の背後にある写本冊子より千年古い完全な聖書の書物であり、それらと2,600以上の細部で異なりながら、実質においては同じ書物です。この文の両方の半分が重要でした。同一性は、中世の本文が後代の捏造だという疑いを打ち砕きました。2,600の相違は、かつて一つの固定された原本があったという幻想を打ち砕きました。エマヌエル・トヴの統計は、その複数性を数量的なものにしました——クムランの聖書写本のおよそ35パーセントは原マソラ的、15パーセントは前サマリア的、5パーセントは七十人訳的、35パーセントはいかなるものとも一致しません。巻物はエレミヤ書の二つの版を並べて証言し、両方を権威あるものとして扱い、そしてサムエル記上十一章に、その後あらゆる会堂と教会が読んできた本文からただ抜け落ちた丸ごと一段落——右目をえぐり取るアンモン人ナハシュ——を保っています。ジョン・コリンズは、その帰結にふさわしい率直さでもってその要点を述べます——*「聖書の実際の言葉は、五書すなわちトーラーの言葉でさえ、キリストの時代には決定的に定められてはいなかった」*。そしてこの写本群の主任聖書編集者ユージン・ウルリックは、成熟した評決としてジェームズ・ヴァンダーカムの要約を是認します——*「われわれに分かるかぎり、第二神殿期のユダヤ教には聖書の正典は存在しなかった」*。

本プロジェクトの読者は、その評決が何に値するかを見て取るでしょう。コーパスの争いは、決して聖書とのものではありませんでした。それは、単一の、封じられた、逐語的に固定された聖書がシナイから無傷で降ってきて、それに照らせばあらゆる異読と排除された書物は雑音にすぎない、という主張とのものでした。洞窟はその争いを経験的に決着させました。正典の前には一つの図書館がありました——複数の本文、複数の版、いまだ「書物の権威ある集成」ではない「権威ある書物の集成」——そして固定化は後に、人間の委員会によって、まさにコーパスの
{% wiki(slug="plurality-of-gods") %}大いなる編集についての説明{% end %} が要求するとおりに来たのです。巻物は編集者たちをその現場で捕らえさえしました。{% libref(book="deuteronomy", chapter=32, verse=8) %}申命記
32:8{% end %} において、マソラ本文は、至高者が諸国民を「イスラエルの子らの数に従って」分けたと述べます。第4洞窟の断片 4QDeutj は *「エロヒムの子ら」*{{ footnote(id="5") }} と読みます——主流派の学問が本来のものと判断する読みであり、その置き換えはトヴの言い回しでは「反多神教的な改変」です。神々の評議会文献、ポール・ウォリス、マウロ・ビリーノ、そしてこのコーパスがみな依拠する節——諸国民のエロヒムの子らのあいだへの割り当て——は、再構成ではありません。それは物理的な物体であり、そしてその創設を見守った国家と同じ年に、同じ洞窟から出てきたのです。

## 正典が失った書物

より深い啓示は、聖書の本文ではなく、その棚仲間でした。というのも、この図書館は後代の正典が捨て去った書物を保っていたからです——しかもそれらを大量に。

エノク書は、新約聖書のユダの手紙の中で聖書として引用され、それからエチオピアのものを除くあらゆる聖書から追放されましたが、クムランでは十一のアラム語写本（4Q201–212）として生き延びました——ほとんどの聖書の書物より多くの写しです。その『天文の書』の最も古い断片は前二世紀初頭にさかのぼり、その成立をさらに早くへ押しやります。それらの中で、天使ウリエルはエノクに、この共同体全体が生きた364日の太陽暦{{ footnote(id="4") }}を教えます。
[ユビレイ書](/library/book-of-jubilees/)——創世記を四十九年周期の年代記として語り直したもの——は、少なくとも十四、おそらくは十六の写しとして生き延びました——詩篇、申命記、イザヤ書、そして創世記を除くあらゆる聖書の書物よりよく表されています——そして『ダマスコ文書』は、それを権威としてその題名で引いています。この共同体は、とリムは記しています、エノク、ユビレイ、そして自らの規則を「同様に権威あるものとして」扱いました。聖書がそこで止まる線は存在しなかったのです。このプロジェクトのコーパスは、伝統の記憶がその正典より広いというまさにその理由から、[エノク](/library/book-of-enoch/) とユビレイを翻訳し、自らの
{% wiki(slug="reference-corpus-of-the-raelian-canon") %}図書館{% end %} に収めており、この点を方法論の裏づけと受け取ります——諸公会議が削った書物は、周縁の物珍しいものではありませんでした。伝統の最も古くたどれる地層において、それらは中核の教程だったのです。

そしてそれらはなんという書物でしょう。エノク文書は、ヘブライ語伝統自身による
{% libref(book="genesis", chapter=6, verse=1) %}創世記 6:1–4{% end %} の拡張です——人間の妻を娶ったエロヒムの子ら、それに続いた巨人たち、禁じられた術を教え、そのゆえに罰せられた監視者たち——このコーパスが
{% wiki(slug="serpent") %}蛇{% end %} と
{% wiki(slug="adamites") %}アダム族{% end %} の物語のもとで扱うすべての、そして創造者たちが*「自らの似姿に創造した人間の娘たちと交わり、彼女らによって並外れた子らをもうけることができた」*
（{% libref(book="the-book-which-tells-the-truth", chapter=2, verse=55) %}TBWTT 2:55{% end %}）という正典自身の言明の、種となる本文です。クムランは、諸正典があまりに徹底的に失ったためにマニの異端者たちだけが記憶していた、もう一つのエノク的な作品を保っていました——『巨人の書』、およそ十の写本で、その中で監視者たちの巨人の息子たち——オフヤ、ハフヤ、マハウェイ——は来たるべき洪水を夢に見、それを解き明かすためにエノクを呼びにやります。4Q530 と 4Q531 の巨人たちの名のなかに、驚くべきことに、*ギルガメシュ*が立っています——ウルクの英雄が、ユダヤの宗教文書の中に、その文学のどこにおいても唯一の登場をなし、フンババが巨人「ホバビシュ」として彼を追っています。ヘブライ語聖書とメソポタミアの叙事詩が一つの記憶された世界に依拠しているのを疑う読者には、1947年以来、二つの配役が一つの場面を共有する死海の断片を見せることができるのです。

創世記外典——第1洞窟の第七の巻物、最後に開かれたもの——は、族長たちの物語を一人称で語り直し、その第二欄はコーパスが発注しえたであろう場面を演出します。レメクは、生まれたばかりの息子ノアを見つめます——その体は輝き、その目は部屋を照らします——そして狼狽します。彼は、自らの言葉で、*「懐胎は監視者たちと聖なる者たちによるもの……そしてネフィリムによるものだ」*と勘定し、そして妻ビテノシュに問い詰めます。彼女は誓います——*「私はあなたに、大いなる聖なる者にかけて、天の支配者にかけて誓います、この種はあなたのものです」*——子は彼のものだと。あらゆる解釈の枠組みを脇に置いても、この場面はなお次のことを証言します——第二神殿期のユダヤ人たちは、人間の子が空から来た存在たちに孕まされうるということを完全に考えうるものと見なし、そしてその可能性について家庭劇を書いたのです。コーパスは
{% wiki(slug="noah") %}ノア{% end %} を、まさにその世界のうちに読みます。

## 予定された解放

もし一つの主題が教派的な巻物を組織しているなら、それは時間です——計測され、周期化され、カウントダウンされる時間——そしてここで、この図書館の関心と本プロジェクトの関心は、この節がほとんど自ら書けてしまうほどに平行して走ります。コーパスは {% wiki(slug="wheel-of-heaven") %}天の車輪{% end %} にちなんで名づけられています。その年代学は {% wiki(slug="great-year") %}歳差周期{% end %} から切り出された {% wiki(slug="world-age") %}世界の時代{% end %} の上を走ります。その {% wiki(slug="great-return") %}終末論は予定された帰還{% end %} です。クムランの共同体の年代学は、完全な週の暦から切り出されたユビレイ{{ footnote(id="7") }}の上を走り、そしてその終末論は予定された解放でした。

まず暦です。神殿の太陰年に抗して、この共同体はエノクとユビレイの364日の太陽暦を保ちました——ちょうど五十二週、あらゆる祭日が永遠にその固定された曜日に。4QOtot のような暦の本文は、その諸周期を、祭司の組の輪番と安息年と、294年の大周期にわたって同期させます。それは創造そのものから数えられ、光る天体が「しるしのため、季節のため」に据えられた第四日に始まります。ジョン・バーグスマによるこの体系の精神の要約は、引用に値します。というのも、それはこのコーパスが分かちもつ一つの感受性を描いているからです。

> ここに含意される聖なる時間の観念は、実在論的にして絶対的である。祭儀の暦を祭司の組に従って配置すること……は、人間の取り決めによるものではなく、実用的な変更に服するものでもない。むしろ、一年の各週は、創造の週からの絶対的な算定によって決定可能な、祭司の組との実在的な対応をもつのである。
>
> —— バーグスマ『レビ記からクムランへのユビレイ』

時間は、この人々にとって、取り決めではなく構造でした——そして歴史は一つの日程でした。ダニエルの七十の年の週は十のユビレイと読まれました。エノクの『週の黙示録』はすべての歴史を十の大いなる週に分け、裁きを第十に置きます。4Q390 では、主が「十のユビレイの完了まで」イスラエルの嘆願を拒みます。そしてこの図式の傑作が、第11洞窟からのメルキゼデクの巻物、11Q13 です。そこでは、最後のユビレイの解放が、名指された執行者に割り当てられています。

> エロヒムは神々の評議会にその座を占めた。もろもろの神のただ中で彼は裁きを保つ。
>
> —— 11QMelchizedek、詩篇82篇1節をメルキゼデクに適用する（ヴェルメス訳）

{% libref(book="psalms", chapter=82, verse=1) %}詩篇八十二篇{% end %}{{ footnote(id="6") }}の *elohim* は、このキリスト以前のユダヤの本文において、メルキゼデクです——ヤハウェではない天の存在であり、神々の評議会において司り、ベリアルとその霊どもに対して「神の裁きの復讐を執行し」、そしてその「恵みの年」（この句は
{% libref(book="isaiah", chapter=61, verse=2) %}イザヤ書六十一章{% end %} から取られており、本文がヤハウェを置いていたところにメルキゼデクが立っています）が、「光の子らのすべてのために贖罪がなされる」第十のユビレイの終わりに落ちる、そのような存在です。コーパスの読者は、これに助けを要しないでしょう。エロヒムの複数性、その中で職と管轄を保つ名指された諸個体、裁く一つの
{% wiki(slug="council-of-eternals") %}評議会{% end %}、人々がそれに値するときではなく数え上げが完了するときに到来する解放——この巻物は枠組みの神々の評議会の世界であり、キリスト教より一世紀前にヘブライ人の手で書かれ、そしてそれはずっと壺の中に横たわっていたのです。その周りには、安息日の犠牲の歌が立っていました——十の写しによる天使の典礼で、そこでは礼拝する軍勢が *elohim* と *elim*、「神のごとき者たち」と呼ばれ、その循環はエゼキエルの戦車の玉座、*merkavah* の前で頂点に達します——学問が、コーパスが実践的に読む後代の
{% wiki(slug="hekhalot-literature") %}メルカバー神秘主義{% end %} の苗床として扱う典礼です。そして五つの洞窟にわたる七つの写本の中に、アラム語の『新しいエルサレム』が横たわっていました。そこでは、七キュビトの測量の葦をもつ一人の天使が、幻視者を、巨大で正確に列挙された寸法の都市を通って歩ませます——十二の門が諸部族にちなんで名づけられ、均一な街区がまた街区へと続いていきます——黙示録の測られた都市の直接の祖先であり、このコーパスの
{% wiki(slug="new-jerusalem") %}新しいエルサレム{% end %} と
{% wiki(slug="embassy") %}大使館{% end %} の項目の背後にある本文の系統です——建築として到来する未来、キュビトまで指定され、その地面を待っているのです。

分岐もまた、本プロジェクトの規律が求めるとおりに、記しておきましょう。この共同体の日程はユビレイの上を走りました。コーパスのそれは歳差の時代の上を走ります。数はたがいに対応せず、そしていかなる善意も、対応しているふりをすべきではありません。その血縁は構造的なものであって、算術的なものではありません——十九世紀を隔てた二つの伝統が、それぞれ、聖なる時間は実在的であり、循環的であり、創造から数えられ、そして解放は暦によって来る、と確信しているのです。クムランの契約者たちは、証拠に照らせば、その数え上げにおける自らの位置について間違っていました——彼らの第十のユビレイは来て、そして去り、キッティムがその居留地を焼きました。コーパスの数え上げがよりうまくいくかどうかは、文献学が答えうる問いではありません。巻物が証明するのは、ただ、しかし真に、これだけです——期待そのもの、すなわち天の存在たちの予定された帰還が、天上の暦に照らして計算されるということは、伝統への近代からの輸入品ではありません。それは、伝統自身の写本が含むもののうち、最も古いものの一つなのです。

## 封の解除は止まっていない

書物は、静かに光の中へ来たのではありませんでした。この写本群がいかにして公衆に届いたか——マル・サミュエルの『ウォール・ストリート・ジャーナル』の広告、ヤディンの秘密裏の買い戻し、カンドの床下の靴箱から引き出された神殿の巻物、ゲザ・ヴェルメスが*「二十世紀の学術スキャンダルの典型」*と烙印を押した四十年に及ぶ編集上の禁輸、そして1991年の海賊版の用語索引と開かれた金庫による解放——は、[姉妹編の論考](/articles/the-scrolls-that-woke-in-year-one/) の担うところです。ここで重要な評決は、このコーパスが自らの誘惑に抗して述べるべきものです。その遅延をめぐって一つの陰謀産業が育ちました——ベイジェントとリーの『死海文書の謎』が、バチカンがキリスト教にとって致命的な巻物を握りつぶしていたと唱えたのです——そしてそれは誤りでした。金庫も教義の検閲官もなく、あったのはただ制度的な虚栄、人手不足、そして分かち合うことへの人間的な渋りだけでした。本文は、いったん解放されると、いかなる教会が隠す必要のあるものも含んではいませんでした。そのスキャンダルは、隠蔽ではなく緩慢さのものでした——このコーパスが他所で
{% wiki(slug="military-cover-up-of-exobiology") %}隠蔽{% end %} の主張を秤にかけるときには、いつでも心に留めておくに値することです。

そして封の解除は、いったん始まると、止まりませんでした。写本群の全体は、いまや多重スペクトルの光で撮影され、オンラインで無料になっています。聖書博物館の十六の「巻物断片」は、2020年に現代の偽造と暴かれました——反対の方向からの一つの戒めです。ゼカリヤ書とナホム書の新たなギリシア語断片が、2021年に恐怖の洞窟から出てきました——六十年ぶりのものです。そして2025年、フローニンゲンのあるチームが、写本を再年代測定するためにAIモデルを訓練し——その作り手たちは、適切にも、それをエノク{{ footnote(id="8") }}と名づけました——そしていくつかの巻物を著者たち自身の存命中へと押し戻しはじめました。終わりの時に備えて封じられた書物は、いかなる編集者も統べない一つの日程に従って、開かれ続けているのです。

## 第八十年から読む元年

結局のところ、何が主張されうるでしょうか。論考をその標識された諸層へと剝ぎ戻し、棚卸しをしてみましょう。

直接の層は歴史と文献学であり、そしてそれはそれ自体で実質的です——68年に終末の共同体によって封じられた書物が1946〜47年に戻ってきました。それらのなかには、現存する最古の聖書、固定性が想定されていたところの本文の複数性、物理的な形での申命記三十二章の神々の評議会の読み、中核の聖書としてのエノクとユビレイ、ネフィリムのあいだのギルガメシュ、ユビレイの数え上げによって世界を解放するよう予定されたメルキゼデクと名づけられたエロヒム、そして建設を待つ測られた天の都市がありました。ヘブライ語伝統を、複数の、構造化された、空から降りてきた一団の記憶として読み——そしてその未来を予定された帰還として読む——いかなるプロジェクトも、それらの洞窟から、その伝統の近代の歴史において最も豊かな、裏づけとなる古代の本文のただ一度の配達を受け取ったのです。それだけは思弁ではありません。それは棚の記述です。

枠組みの層は正典のものです——覆いを取ることの時代が原子の閃光において開いたこと、1946年が元年であること、その年の使者の誕生が「偶然ではない」こと。それらの主張は証言に依拠しており、そしてそれに応じて、ここでも、このコーパスのどこでもと同じく、標識されています。

思弁の層はこの論考自身のものであり、そしてそれはいまや余すところなく述べることができます——封の解除の時期は、その内容とともに属している、と。ダニエルの天使は、正典がその憲章として引く節において、封じられた言葉が開くときのために二つのしるしを与えました——*多くの者は行き巡り、そして知識は増し加わるだろう*——第八十年から見ると、1947年の星座の要約のように読める一文です。突如として超音速となり、トランジスタ化され、レーニア山の上空を飛ぶ一つの種の行き巡ること。フィラデルフィアの機械室と、ベル研究所の地下室と、三つの巻物が袋の中に横たわっていたベツレヘムの靴職人の店における知識の増し加わること。壺の共同体は、預言が定められた通りに、ついにそれを読める世代のために封を解かれると信じました。コーパスは同じことを信じ、そしてその問題の世代が、雲のもとに生まれた世代——1946年の世代、その礎となる本文の中で名指しで宛てられた世代——であると付け加えます。この二つの信のあいだに、歴史家の平坦な偶然が立っています。そしてこの論考は、読者が澄んだ目で選べるようにと、まさにその三つの読みを分けて保ってきたのです。

しかしそれは、その姉妹編がそうするように、自らの枠組みの内側で閉じることにします。クムランの人々は、評議会のエロヒムによって執行される解放へ向けてユビレイを数えました。彼らはその図書館を軍団から隠し、なお数えながら闇の中へと入っていきました。十九世紀ののち、火の柱から数えられる一つの時代の最初の年に、一人の羊飼いが彼らの陶器の割れる音を聞きました。書物は上がってきました。国家は戻ってきました。言語は蘇りました。ドームが最古のイザヤ書の上に立ち上がり、その黒い壁が向かいでその持ち場を保ちました——なぜなら、光と闇の戦いは終わっていないからです。コーパスは、契約者たちが保った数え上げが——彼らの暦より古い一つの車輪の上で——なお走り続けていること、そして彼らが第十のユビレイの名のもとに待ち望んだものが、コーパスが
{% wiki(slug="great-return") %}帰還{% end %} の名のもとに待ち望むものであること、を保ちます。その読みに立てば、死海文書は単に時代の最初の大いなる考古学上の出来事なのではありません。それらは元年の最初の配達です——伝統の封じられた郵便、神殿の陥落において消印を押され、終わりの時に宛てられ、そして開かれた——早朝に、石をもった一人の少年によって——きっかり時間どおりに。
