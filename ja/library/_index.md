# ライブラリ

ライブラリは、本プロジェクトが拠り所とする一次テクストを収めています。それらについての注釈ではなく、テクストそのものを、直接読むための形で提示しています。

その中心にあるのは、本プロジェクトが基礎的なカノンとして扱うラエリアンの原典書籍です。その周囲には、カノンが並べて読まれる聖書、メソポタミア、ヴェーダ、そのほかの古代の書物が配されています。各書物は、コーパスが備えている場合には、原語の文字、翻字、翻訳を並列に表示するインタラクティブなリーダーで描画されます。

解釈の層――ある一節について本プロジェクトがどう読み解くか――を知りたい場合は、[タイムライン](/timeline/)または[百科事典](/wiki/)から始めてください。ライブラリは、まずテクストそのものに即して読むための場所です。
