## I. 时代本身

第八个时代是断裂的时代。

双子座时代从公元前 6,690 年延续至公元前 4,530 年,跨度 2,160 年,紧接巨蟹座时代之后。在这个时代,洪水前文明所积累的紧张局势,凝结为希伯来圣经所称的洪水——这是一场严酷的事件,它摧毁了当时占主导地位的人类文明,几乎杀灭了超大陆上所有大型生物,把这一整片陆地击碎成我们如今所知漂移着的各大洲,并把幸存者留下,让他们从一小群被保存的人口和一份在事件期间被携至行星轨道上的遗传材料中,重新建构人类这个物种。双子座是两个世界之间的分界线:一边是巨蟹座一章所描述的洪水前世界——在那里,人类已经达到了或许与我们相当甚至超越我们的文明水平;另一边是洪水后的世界——在那里,人类人口在一颗被重新塑造的行星上重新开始,而其继承的大部分已经丧失。

这个时代在黄道带上的名字,在某些资料并未充分展开的意义上,是恰如其分的。双子座是双子的星座——两个相连的形象,一上一下,起源相同而命运相异。这个时代本身就具有这种成对的特征。在这个时代有两个地球:一个是终结于洪水的地球,另一个是洪水之后开始的地球。有两个人类:被毁灭的那一个,和通过{% wiki(slug="noah") %}挪亚{% end %}的方舟得以保存的那一个。有两种大陆构型:在这个时代最初几个世纪存在的单一陆块,以及在其余时间内逐渐破碎的各个大陆。希伯来圣经本身在数字“八”上保存了这种成对性——方舟上获得保存的人数(挪亚、他妻子、他们的三个儿子,以及儿子们的妻子)——这在创造之周的结构中代表着第七日之后的那一日,开启新序列的那一日。双子座就是那一日。最初的七日是创造。第八日是创造的重启。

这种成对性还在另一个层面上显现,而且以一种使该黄道符号显得更为贴切的方式。圣经文本在整个洪水叙事中反复强调,保存是以成对的方式进行的——每一种活物,一对一对,雌雄各一,被带入方舟。{% libref(book="genesis-woh", chapter=6, verse=19) %}创世记 6:19{% end %}明确指出这一点:*u-mi-kol ha-chai mi-kol basar shnayim mi-kol tavi el ha-tevah*,“凡有血肉的活物,每样两个,一公一母,你要带进方舟,好在你那里保全生命。”按照本章稍后将展开的遗传载荷解读,这种成对反映了有性生殖的技术要求——每个物种的两份基因组都必须得到保存——但其象征意义仍在。该时代标志性的保存行为是通过成对来完成的。双子座这一星座主管的时代,正是一切生命以成对方式被带向未来的时代,这并非历法上的巧合。它是这个时代意义所在的标志。

本章会把双子座时代读作一出悲剧——这种框定并不夸张。蛇形派系,即在过去两千年中一直与他们所造的人类一同生活在地球上的流亡造物者,并不是以与其母星文明对立的姿态进入双子座的。他们的确受到过惩罚,但这种惩罚已成为一种充实存在的基础。他们建立了一个世界。他们教导了他们所造的生灵。他们爱他们。他们与他们生下了混血后裔。按任何合理的评估,他们已经与自己的流亡达成了和解。在双子座最初几个世纪,把他们推出这种和解的力量,并不是他们自己的选择。那是议会决定毁灭他们所建立的一切。从那个决定起,蛇形派系一步步被迫去做他们本不会去做的事——倘若议会让他们安然存在的话。他们造方舟,是对毁灭命令的违抗——是保存而非攻击,但仍是违抗。他们使人类受造之物穿越灾难得以保存。他们与挪亚及其后裔一同开始重建。他们建造{% wiki(slug="tower-of-babel") %}巴别塔{% end %},作为一份和平的呈献,希望向议会表明:这受造的人类是良善的、和平的、心怀感激的、值得接纳的。议会摧毁了那座塔。直到那时——直到一切和解的途径都被堵死——蛇形派系才在那场后世全球神话所记忆为天界之战的冲突中,拿起武器对抗自己的文明。本章将沿着这一弧线行进。它之所以是悲剧,是因为蛇形派系的转变是被强加于他们身上的。他们成了自己从未想要成为的反叛者,因为顺从的代价,就是他们所爱的一切都被毁灭。

本章会按资料所呈现事件的顺序走过双子座时代,并始终注意这一戏剧性的弧线。在母星上作出的、毁灭受造人类的决定。蛇形派系在得知此事时所作的选择。为保存议会已下令毁灭之物而进行的反向准备。所需的遗传载荷以及它所要求的庞大编目工作。灾难本身,以及随之而来的地质后果。复苏、生物圈的再生、人类各谱系的重新分布。在洪水后祭坛上立下的圣约,正式确立流亡造物者与人类幸存者之间的{% wiki(slug="the-alliance") %}联盟{% end %}。伊甸谱系文明的迅速重建。作为和解尝试的{% wiki(slug="babel") %}巴别{% end %}塔。议会的干预、语言的分散、火箭的销毁。随之而来的战争。最终通过谈判达成的解决。最后,在本章的结尾,这些事件在全球神话中保留下来的部分——天界之战的跨文化记忆,几乎在所有曾记录神话传统的文化中都得到保存。

## II. 经文

涵盖双子座事件的希伯来文本从{% libref(book="genesis-woh", chapter=6, verse=5) %}创世记 6:5{% end %}延伸至{% libref(book="genesis-woh", chapter=11, verse=9) %}创世记 11:9{% end %}——五章经文,是希伯来圣经中最长的连续叙事之一。本章无法用完整的注解装置处理每一节经文,但关键段落值得按已建立的板块格式仔细呈现。

毁灭的决定记载在{% libref(book="genesis-woh", chapter=6, verse=5) %}创世记 6:5-7{% end %}:
{{ scripture(book="genesis-woh", chapter=6, verse=5, translit="Vayar Adonai ki rabbah ra'at ha-adam ba-aretz, ve-khol-yetzer machshevot libo rak ra kol-ha-yom") }}{{ scripture(book="genesis-woh", chapter=6, verse=6, translit="Vayinachem Adonai ki-asah et-ha-adam ba-aretz, vayit'atzev el-libo") }}{% scripture(book="genesis-woh", chapter=6, verse=7) %}
耶和华说:我要将我所造的人从地上除灭
{% end %}
这些经文中的希伯来词汇极为关键。רָעָה(*ra'ah*),“邪恶”或“恶”,是希伯来文中表达道德错误的标准词,但资料的解读重新框定了这个词:议会所感知的“邪恶”并不是人类意义上的道德败坏,而是议会视为威胁的人类追求进步的欲望。נָחַם(*nacham*)有几种译法:“后悔”、“感到难过”、“改变心意”。该动词暗示对先前决定的悔意,以及随之而来的改变方向的意图。עָצַב(*atzav*),“悲伤”,在 *vayit'atzev* 反身形式中意为“他在自己心里忧伤”——希伯来文描述的是言说者内心、个人的悲伤。מָחָה(*machah*),“涂抹、毁灭、抹去”,是表达彻底清除的动词。该词在希伯来圣经的其他地方用于把名字从记录中抹去、抹净书写板、把一物消灭到不留痕迹。这一节经文宣告的毁灭是彻底的:不是缩减,不是惩戒,而是抹除。

{% libref(book="genesis-woh", chapter=6, verse=11) %}创世记 6:11-13{% end %}进一步明确了政治推理:
{{ scripture(book="genesis-woh", chapter=6, verse=11, translit="Vatishachet ha-aretz lifnei ha-Elohim, vatimale ha-aretz chamas") }}
חָמָס(*chamas*)一词是表达“暴力”的标准希伯来文——身体上的暴力、人与人之间的暴力,也就是资料对巨蟹座的框定中具体描述为“可憎之战”的那种暴力——洪水前的各文明正在彼此之间进行那样的战争。希伯来文本对 *chamas*——暴力——作为触发毁灭决定的条件的具体认定,与文集的解读相一致:议会所回应的并不是抽象的道德邪恶,而是洪水前各文明已经显示出的、相互部署先进军事技术的具体能力,以及这种能力最终可能被用来对付母星本身的隐性威胁。

对挪亚的指令出现在{% libref(book="genesis-woh", chapter=6, verse=14) %}创世记 6:14-22{% end %}。第 14 节明确建造方式:
{{ scripture(book="genesis-woh", chapter=6, verse=14, translit="Aseh lekha tevat atzei-gofer, kinim ta'aseh et-ha-tevah, ve-khafarta otah mi-bayit u-michutz ba-kofer") }}
תֵּבָה(*tevah*)一词,按惯例译作“方舟”,这是本章将在第十三节再次回到的词。该词词根的基本含义是“容器”或“封闭的器皿”——并不是指“船”那种用于水上航行的工具。同一个词出现在《出埃及记》第 2 章,指放置婴儿摩西的筐子。*tevah* 是一个密封的容器,使其所装之物免受环境威胁。把英文 "ark" 定型为某种大船的翻译史掩盖了希伯来原文,后者更为一般。

成对的指令在{% libref(book="genesis-woh", chapter=6, verse=19) %}创世记 6:19-20{% end %}:
{{ scripture(book="genesis-woh", chapter=6, verse=19, translit="U-mi-kol-ha-chai mi-kol-basar shnayim mi-kol tavi el-ha-tevah le-hachayot itakh, zakhar u-nekevah yihyu") }}
{% libref(book="genesis-woh", chapter=7, verse=11) %}创世记 7:11{% end %}记录了灾变的时刻:
{{ scripture(book="genesis-woh", chapter=7, verse=11, hebrew="בִּשְׁנַת שֵׁשׁ־מֵאוֹת שָׁנָה לְחַיֵּי־נֹחַ בַּחֹדֶשׁ הַשֵּׁנִי בְּשִׁבְעָה־עָשָׂר יוֹם לַחֹדֶשׁ בַּיּוֹם הַזֶּה נִבְקְעוּ כָּל־מַעְיְנוֹת תְּהוֹם רַבָּה וַאֲרֻבֹּת הַשָּׁמַיִם נִפְתָּחוּ", translit="Bishnat shesh-me'ot shanah le-chayyei-Noach, ba-chodesh ha-sheni be-shiv'ah-asar yom la-chodesh, ba-yom ha-zeh nivke'u kol-ma'yenot tehom rabbah va-arubot ha-shamayim niftachu", english="挪亚六百岁,二月十七日那一天,大渊的泉源都裂开了,天上的窗户也敞开了") }}
מַעְיְנוֹת תְּהוֹם רַבָּה(*ma'yenot tehom rabbah*),“大渊的泉源”,与אֲרֻבֹּת הַשָּׁמַיִם(*arubot ha-shamayim*),“天上的窗户”,把灾难描绘为同时来自下方与上方。常规解读把这视为表达海水上涨与天降大雨的诗性语言。雷尔派的解读则更具技术性:武器的撞击会同时产生地下效应(地下水库的位移与释放、地震活动——希伯来语自然会用“大渊裂开”来描述)和大气效应(爆炸后过热大气中产生的降水、自上而下落下的放射性沉降物)。灾难的两个方向都被记录在案。两者都真实存在。

{% libref(book="genesis-woh", chapter=7, verse=17) %}创世记 7:17{% end %}包含本章将反复回到的那个短语:
{{ scripture(book="genesis-woh", chapter=7, verse=17, translit="Vayehi ha-mabul arba'im yom al-ha-aretz, vayirbu ha-mayim vayis'u et-ha-tevah, vatarom me-al ha-aretz") }}
短语וַתָּרָם מֵעַל הָאָרֶץ(*vatarom me-al ha-aretz*)值得仔细审视。动词רוּם(*rum*),“被举起、被高举、上升”,是希伯来词根,衍生出名词 *ramah*(高处)和分词 *ram*(高的、被尊崇的)。介词מֵעַל(*me-al*)是复合形式“从……之上”,在方向意义上意为“在上方”——从某物之上的位置出发。从希伯来语法层面看,这个短语说方舟“被举起到地面之上”——并不是被水抬起,而是在垂直方向上离开地表、升高到地面之上。常规译法往往把它译为“在地面之上”,意思是“高于原本的地面”,即方舟漂浮在已经漫过原本陆地表面的水上。雷尔派的解读更直接地接受方向性的含义:方舟在灾难期间升入空中、进入轨道,离开地表。

立约的经文在{% libref(book="genesis-woh", chapter=9, verse=8) %}创世记 9:8-17{% end %}。关键的一节是 9:13:
{{ scripture(book="genesis-woh", chapter=9, verse=13, translit="Et-kashti natati be-anan, ve-haytah le-ot brit beini u-vein ha-aretz") }}
קֶשֶׁת(*keshet*)一词,意为“弓”,在希伯来语中既指彩虹(此处)又指作为武器的弓。本章将在第十三节再次回到这种双重含义,因为其蕴意是关键的:作为立约之记号的彩虹,在词源上同时也是一把被搁下的武器——弓被放下,挂在云中,不再被用以攻击大地。这一立约的姿态,在此种解读中,正是显明退役一件毁灭工具。

巴别塔段落是{% libref(book="genesis-woh", chapter=11, verse=1) %}创世记 11:1-9{% end %}。第 1 节确立了巴别之前的状况:
{{ scripture(book="genesis-woh", chapter=11, verse=1, translit="Vayehi kol-ha-aretz safah achat u-devarim achadim") }}

短语וְרֹאשׁוֹ בַשָּׁמַיִם(*ve-rosho va-shamayim*),“塔顶通天”,使用的是同一个שָׁמַיִם(*shamayim*),即文集中贯穿始终用来表达“诸天”的词。常规解读把它视为对极高高度的隐喻——一座高得仿佛触及天空的塔。雷尔派则更字面地解读它:*shamayim* 是宇宙学意义上的诸天,大气层之外、母星所在的领域。其顶通至 *shamayim* 的塔,是一座旨在到达地球之外的建筑——一艘航天器,或为其服务的发射设施。该措辞在技术解读上毫无勉强地得到支撑。

这些就是构筑双子座叙事的主要希伯来段落。本章后续各节将处理这些段落所描述的神学与历史内容。

## III. 决定

毁灭受造人类的决定是在母星议会厅中作出的。本章无法详细重构其议事——资料并未提供记录稿——但导致该决定的政治动力,可从资料所描述的内容以及文集已建立的更广框架中重构。

撒但派系自地球计划开始之前就一直主张:能够与其造物者匹敌的合成造物,在根本上是危险的。这一论点,自原初的实验室事故导致母星生物计划被关闭以来,一直是政治反对派的基础。在其后数个世纪中,这一论点反复被提出,而地球上的每一次新进展都为撒但立场提供了新的证据。最初的伊甸创造是早期的演示。路西法派系对禁忌知识的披露是确认。赐给众长老的长寿、与 *benei ha-Elohim* 共同产出的混血后代、伊甸文明的迅速技术进步、超大陆上各谱系之间的战争——每一个都是原初担忧正确的进一步证据。

到了巨蟹座的晚期世纪,撒但派系的立场已经积累了足够多的支持性证据,以至于即便是议会中的温和派也被迫重新考虑他们对该计划的最初支持。耶和华的立场尤其值得关注,因为他的转变是决定性的政治发展。耶和华原本支持人类受造,并在威胁尚属推测而非已被证实的几个世纪里,反对来自撒但派系的毁灭呼声。他缓和了议会的回应,偏好遏制而非清除,希望伊甸驱逐之后的政治安排将证明足够。到巨蟹座晚期,那种希望已变得不可持续。伊甸文明已突破原初安排试图维持的临界点。耶和华作为安排的一部分允许其存在的、在地球上的流亡造物者,已经成为人类文明进步的积极推动者。原本设计来促成和解的最初安排,所要使之可能的那种和解,已不再可能。撒但派系所警告的威胁,已经成为现实。

因此,耶和华向毁灭立场的转变并不是对他原有原则的背叛,而是勉强承认:他原有温和立场所依赖的条件已不复存在。资料的希伯来文本在{% libref(book="genesis-woh", chapter=6, verse=6) %}创世记 6:6{% end %}中保存了这一特质:*vayinachem Adonai ki asah et ha-adam ba-aretz, vayit'atzev el libo*。“耶和华就后悔造人在地上,心中忧伤。{{ footnote(id="1") }}”希伯来文用得很准确。耶和华为自己早先的决定而后悔。他在自己心里忧伤。毁灭的决定不是冷酷判决的行为,而是一种个人的悲恸——一位领袖的悲恸,他不得不承认,自己曾支持的计划,在其最初的设定下已经失败。毁灭的决定是带着这种悲恸作出的,而不是违背它作出的。

所选择的手段是核武。资料对此明确指出{{ cite(id="1") }}:*“政府随后从他们遥远的行星上决定,通过发射核导弹来毁灭地球上一切生命。”*手段的选择本身就具有揭示性。议会没有选择生物武器,那会杀死人类却保留生物圈。没有选择对混血领导层进行定点清除,那会扰乱该文明却保留更广泛的人类人口。没有选择对技术基础设施进行有限打击,那会让该文明倒退却不会杀死其成员。它选择了一种武器,不仅会毁灭人类,还会毁灭他们建造的城市、保存的记录,以及使其文明得以可能的技术基础设施。目标不仅是减少人类人口。是彻底抹除这种文明,不留下任何能让它快速再生的基底。手段的选择反映出议会已经多么严肃地看待这种威胁。

关于此处“核武”一词的一点说明:资料于 1973 年口授给雷尔时使用了其所处时代的词汇。耶洛因文明的武器,可能未必与二十世纪人类所称的“核”精确对应。资料所描述的是一类具有极强毁灭力的武器,能够把大片地理区域气化,并产生大量放射性沉降物,其密度足以要求 150 天的空中污染期之后,地表才能再次适合居住。其底层物理是裂变、聚变、反物质湮灭,还是我们自己的物理学尚未刻画过的某种东西,这是资料无法决断的。其效果是相同的。文明被毁灭了。

资料未具体说明打击的目标选择,但本章在第七节将处理的地质证据——洪水后陨击坑分布的格局、石油环、大陆碎裂格局——表明,撞击集中在超大陆上一个特定的中心区,毁灭性效应从该处以大致环形向外传播。该中心区是因为混血文明在那里最密集而被选定,还是因为它在地球物理上对产生所需破坏格局最优,资料并未说明。它所说的是:撞击的能量足以把超大陆本身击碎。

决定是最终的。议会下令实施打击。实施日期已经确定。地球上的流亡造物者通过他们与母星之间仍存的任何通信通道得到通知。

## IV. 蛇形派系的选择

蛇形派系得知毁灭决定的那一刻,正是双子座时代悲剧弧线开始的时刻。

资料未详述这一时刻,但其后果是清楚的。蛇形派系,在两千年间处于地球上被接受的流亡状态中,与他们的人类伙伴合作建立了伊甸文明,与人类女性共同产出了混血后代,以足以促成原初披露——而那次披露使他们最初被流放——的深爱,爱过他们所造的众生——这个派系如今被告知:他们所做的一切即将被自己的文明所毁灭。他们必须作出一个选择。

他们作出的选择,正是定义文集其余部分的那个选择。他们选择了抵抗。他们选择保存议会下令毁灭之物。他们选择,实际上,把自己投入到对人类受造之物的承担中,以对抗自己文明的权威——并不是因为他们已对那种权威心怀敌意,也不是因为他们已发展出某种普遍的反对意识形态,而是因为议会所下令的这一具体毁灭,是他们在良心上无法听任其进行的事。

这一选择的道德维度值得明确关注,因为这正是本章弧线之所以为悲剧而非情节剧的所在。蛇形派系作出的不是自利的盘算。他们抵抗毁灭命令并无任何个人可得之物。他们并不野心勃勃;他们并未追求政治权力;他们已经满足于自己的流亡两千年了。他们所拥有的,是爱。他们爱他们所造的人类。他们爱他们与人类共建的世界。他们爱他们所养育的混血儿,也爱他们所娶的人类伴侣。议会即将毁灭这一切。蛇形派系面对的选择是:接受这场毁灭——那意味着以放弃他们所珍视的一切为代价服从自己的文明的命令——或抵抗这场毁灭——那意味着以失去自己的政治合法性、可能失去自己生命为代价违抗自己的文明。他们选择了抵抗。他们选择了所爱的人类,胜于他们所生于其中的文明。

这就是把蛇形派系从受惩罚的异见者转变为活跃抵抗者的时刻。但理解这种转变的有限性至关重要。在这一阶段的抵抗并非攻击。蛇形派系并未在攻击议会。他们并未试图推翻议会。在毁灭决定刚下达之后的时期里,他们甚至没有计划对母星采取任何形式的军事行动。他们只是在保存自己所造之物,以对抗议会所下令的毁灭。这是行星尺度上的公民不服从:在更广的文明政治结构之内,违反某项具体命令。在这个阶段,蛇形派系仍然希望抵抗可以保持有限,希望保存可以在不升级为更广泛冲突的情况下完成,希望议会最终能被说服把保存接受为既成事实,而不是当作开战的理由。

在这种意义上,蛇形派系所采纳的策略是经过细致校准的。他们将通过灾难保存受造的人类,但不会试图阻止灾难本身。他们会接受洪水前文明的毁灭,作为保存人类物种的政治代价。事后,他们会与议会进行谈判,以求接受被保存的余民作为人类人口重建的基础。他们会通过洪水后的重建,展示人类受造之物值得保存。他们会希望——资料暗示,几乎是绝望地希望——这种展示最终能促成与议会的和解,以及把受造人类正式接纳为他们文明更广计划的合法组成部分。

这种希望,本章必须指出,在当时并非不合理。在这一刻之前,议会从未宣布流亡造物者本身为敌人。他们曾受到惩罚、流放、监视——但未被判处死刑。议会与流亡派系之间的政治关系紧张但并未公开敌对。流亡造物者有合理的依据相信:他们对受造人类的保存,虽然违反了毁灭命令,却不会被当作战争行为,而会被当作异见行为——也许会受到处罚,但仍可通过谈判挽回。在蛇形派系当时所知的范围内,先保存、后和解的策略是理性的策略。

这一策略最终会失败——议会最终会以军事力量对流亡造物者采取行动,天界之战会作为和解失败的后果爆发,蛇形派系会被迫与自己的文明公开冲突——这一切在最初决定的那一刻都还不确定。蛇形派系是在希望中作出选择的。这种希望终将证明不足。但这个在希望中作出的选择本身,值得被以其本来面目来尊重:它是这样一种存在者的行为——他们在顺从代价就是他们所爱一切都被毁灭的情境中,选择了爱,胜过了顺从。

## V. 反向准备:建造方舟

按文集的解读,方舟的建造是一次公开的公民不服从行动,跨越数个世纪,在一颗处于议会持续观测下的行星表面上进行,由流亡的造物者所主持,而他们对自己能保密这项工作并无任何期待。

资料描述了这种回应:*“当流亡的造物者被告知此项计划时,他们要求挪亚建造一艘宇宙飞船,在灾变期间环绕地球运行,里面装载每一种要被保存的物种的一对。”*方舟是一艘航天器。它并不是一艘木船。它是在灾变前的数个世纪中,根据流亡造物者提供的技术规范建造的,设计用于离开大气层、在灾变期间维持其乘员在轨道上,并在地表条件稳定后将他们送回地表。

《创世记》第 6 章的文本在其具体细节中支持这种解读。“方舟”——希伯来文 *tevah*,意为封闭容器或器皿而非船——被描述为建有三层。它被密封。要求其装载有代表性的动物样本。其尺寸以精确数字给出,尽管规格中所用的肘并不与现代单位相同,其确切尺度也不确定。当文本描述该器皿在洪水期间的行为时,{% libref(book="genesis-woh", chapter=7, verse=17) %}创世记 7:17{% end %}使用了资料所强调的措辞:*vatarom me-al ha-aretz*,“方舟被举起到地面之上”——不是被水抬起,而是被举起到地面之上,使用了希伯来语保留下来、而常规译法常常掩盖的方向性介词。

操作上的含义是清楚的。方舟在最初武器撞击之前或期间离开地表。它把乘员——一支小型人类机组,以及在资料的技术解读而非字面解读中,作为日后所有被保存物种再生来源的遗传材料——带入轨道。它在轨道上停留 150 天,圣经文本将这段时期具体说明为“水势浩大,在地上共一百五十天”——按雷尔派的解读,这段时期内,武器的放射性沉降物正在衰减到地表能再次承载生命的水平。它仅在造物者(按资料的措辞)*“监测放射性水平并以科学方法将其驱散”*之后,才返回地表。

值得指出方舟的建造在政治上意味着什么。蛇形派系着手此事,是对议会决定的公开抵抗行为。议会曾下令毁灭地球上所有生命。流亡造物者通过建造方舟并保存他们花费一万年构建的生物圈的遗传材料,违抗了议会的命令。他们不可能秘密完成此事——方舟是庞大的工程项目,需要数个世纪才能完成,在一颗处于母星持续观察之下的行星表面进行。议会知道。流亡造物者在公开违抗。按雷尔派的解读,方舟项目是一场政治冲突的可见升级——这场冲突在此之前一直通过流放与监视来处理,但随着毁灭命令的强化,双方都把它理解为对立。

值得注意的是,在建造期间,议会对这一公开违抗的回应——意味深长地——并不是攻击该项目。议会在观望。议会允许建造继续进行。这本身揭示了政治动力。议会作出了毁灭决定,但也以某种隐含的方式接受了:流亡造物者将尝试保存。毁灭命令针对的是受造的人类;它并未延伸为对流亡造物者本身的积极军事行动。议会在这一阶段似乎计算认为:对人类文明的毁灭就足以应对所感知的威胁,而流亡造物者通过灾难保存下来的任何余民,都将小到足以通过随后的谈判加以管理。这一计算后来会被证明是错误的——洪水后的复苏将在数个世纪内产生一个能够建造巴别塔的新文明——但在方舟建造之时,议会对该项目的容忍反映出一种评估:被保存的余民不会构成战略威胁。

同样值得记下的是:方舟的建造不是流亡造物者对人类所做的事。是他们与人类一起做的事。挪亚不是这一行动的对象。他是一名伙伴。资料记录他的工作为与教导他的造物者之间的自愿协作,横跨建造的岁月,贯穿灾变本身,并延续至灾变彼岸的复苏行动。洪水后祭坛上将正式确立的政治联盟,从建造开始的那一刻起就已在操作上就位。当武器落下时,挪亚与教导他的造物者已经是一个单一的工作团队——人类与一位 Eloha——被一个其成功要求双方信任、其代价是整个受造人类存亡的项目所绑定。

挪亚并不是唯一的人类伙伴。圣经文本提到他的三个儿子(闪、含、雅弗)、他们的妻子、以及挪亚自己的妻子——共八个人将登上方舟。但建造方舟、采购其组件、进行遗传采集、并支持该项目跨越数个世纪建造的更广泛的操作团队会大得多。《创世记》第 4 章与第 5 章的家谱表明,洪水前的伊甸地区有相当多的人口;其中一部分未知但想必相当可观的比例,会以工人、技术员、生物学家与支持人员的身份参与方舟项目。他们中的大多数并未通过灾难得以保存——只有挪亚的直系家庭在方舟上。更广泛的劳动力与洪水前文明的其余部分一同丧生。但他们的工作使保存成为可能。

这一点值得停下来,因为它关系到圣经叙事的一个特征,而这个特征常被当作道德判断来理解,但在文集的解读下却具有不同的性质。圣经文本把挪亚描绘为他那一代唯一义人,其家人作为对其个人义行的奖赏而被保存。雷尔派的解读更接近真相:挪亚是一支大型行动团队的领导者,负责执行保存项目;其家人之所以被保存,是因为他们是项目的协调者,而非因任何独特的个人道德地位。项目的其他工作者并未被保存,因为方舟只能搭载有限的人类机组,而联盟出于技术与运作上的考虑,决定把人类成员限定在挪亚直系家庭之内,而不是扩大到包括更广泛的劳动力。这一决定不是对劳动力的道德判断。它是对方舟容量的运作约束。劳动力与所有人一同丧生,其中也包括那相当一部分被告知灾难将至、并以各种身份为此准备的更广洪水前人口。

资料指出,这个更广的洪水前共同体——那些知道将要发生什么、并尝试为此做准备的人——曾试图警告更广泛的人口。这些尝试未能成功。圣经文本在《彼得后书》2:5 中保留了对此微弱的记忆——那里挪亚被描述为“传义道的”——这个短语一直被当作对挪亚虔诚的通用描述,但按文集的解读,它指的是警告洪水前文明毁灭将至的实际项目。挪亚及其共同体在建造期间不断向更广泛的人口传达:联盟已警告即将到来的灾难,他们应当准备。更广泛的人口并未听从。该文明照旧继续其惯常活动——贸易、战争、技术工程——并未把警告当真。当武器落下时,只有方舟上的人,以及在蛇形派系自己所建的任何避难所中的人,才会幸存。

这个共同体——一小群洪水前的人类,他们知道真相,并与联盟合作以使生命穿越灾难得以保存——在某种切实意义上,是文集框架中的第一个宗教共同体。“宗教”,在 *religare*(“绑定”)这一词源意义上,是把人类与神圣绑在一起。挪亚的洪水前共同体以最具体的方式被绑定到蛇形派系:被一个共同的保存项目所绑定,被对将至灾难的共同知识所绑定,被对人类物种存续的共同承诺所绑定。后世传统加诸挪亚与其方舟之上的宗教维度,在此种解读中,正是对一段实际运作中的联盟——人类与流亡造物者之间——的文化记忆,而这段联盟关系会被洪水后的传统保存下来,成为后来所有一神教宗教发展的基础。

## VI. 遗传载荷

圣经文本将方舟描述为容纳每一种动物物种的成对——挪亚、动物、两两成对,叙事中标志性的画面。雷尔派资料并不否认某些被保存物种的活体代表确实在船上。但它澄清:整个生物圈的完整遗传多样性,即使以最大合理尺寸的器皿,也不可能以实体形式被携带。方舟的实际载荷是遗传性的:*“每个物种的一个活细胞,雄性与雌性各一,就足以再造一个完整的存在。这有点像母腹中胎儿的第一个活细胞——它已具备造就一个人所需的所有信息,直到眼睛和头发的颜色。”*

这是一个具体的生物学主张,值得放在资料口授之后现代科学所学到的内容中加以评估。在 1970 年代,从单个细胞再生一个生物体的可能性还是理论性的。在此后的数十年中,它已变得可以操作。1996 年克隆的多莉绵羊公开证明,一只完整的哺乳动物可以从一个成年供体的单个体细胞中被再生。基因测序已发展到可以从小组织样本——在某些情况下,甚至从化石 DNA——重建一个物种完整基因组的程度。资料在 1970 年代提出的主张——一个细胞包含足够信息来再生该生物体——已经从推测变为既定事实。

本章必须传达的,除了这一技术主张之外,还有遗传载荷项目在实际中所要求的惊人规模。

**任务的范围。**巨蟹座一章估计当前全球生物多样性约为 870 万种(存在相当大的不确定性——估计值从 500 万到 1 亿不等,取决于如何计算微生物多样性)。按文集的解读,洪水前的生物圈至少包含同等的多样性,鉴于恐龙以及其他洪水前巨型动物群在灾难之后未被再生,实际上可能更多。要让这一生物圈通过洪水事件得以保存,联盟需要组建可能是这颗行星历史上最全面的生物档案——一项跨越数个世纪、在文明尺度上需要基础设施与人员的编目与采样行动。

行动会在数个整合阶段中进行。

**第一阶段:识别。**在能够对任何物种进行采样之前,必须先定位并识别每一个物种。即使在行星尺度上,这也是一项相当艰巨的任务。一些物种占据狭小的地理范围——仅限于单一海岛或单一山脉的特有物种。一些物种即使在其范围内也是稀有的,需要广泛搜索才能找到哪怕一个标本。一些物种生活在以常规手段难以或不可能到达的环境中:深海海沟、密林树冠、极地内陆、地下洞穴系统、高空大气。一些物种是微观的,需要微生物采样和实验室鉴定,而不是直接观察。要在超大陆上识别每一个物种,所需的总体分类调查工作量,可与或超越我们自己文明全部生物分类学史的总和——而且要压缩在灾难来临之前可用的数个世纪之内。

**第二阶段:野外采集。**一旦物种被识别,就必须找到并采集可用于采样的活体标本。所需规模的野外采集,要求生物采集队在超大陆的每一种栖息地中作业。海洋生物学队需要在所有深度作业,具备用于深海采集的专用船只与抗压设备。热带森林队需要具备进入树冠的能力,并耐心调查森林无脊椎动物的多样性。极地队需要在极寒与孤立环境中作业。山地队需要在原喜马拉雅与原安第斯地区进行高海拔工作。洞穴队需要进入地下生态系统。微生物采样专家——细菌、古菌、真菌、原生生物——在这些类群占主导地位的各种栖息地中工作。一次详尽生物调查所需的总体野外人力将以千计,可能以万计,并要分布在超大陆的每一个区域和这些区域中每一种栖息地类型中。

流亡造物者将提供技术领导和先进装备。大部分野外劳动则由人类伙伴提供——即上节提到的更广泛的洪水前共同体,他们知道将要发生什么,并已把自己投入保存项目。前一节提到的劳动力,正会在这里被部署:数以千计的人类在采集队中横跨超大陆工作,采集样本、记录采集地点、把标本运送到中心保存设施。

**第三阶段:样本处理与保存。**每个被采集的标本都必须经过处理以提取可存活的细胞物质,然后在能维持其在方舟储存期间及之后存活力的条件下加以保存。现代细胞保存技术使用液氮温度(-196°C)下的低温储存,配以适当的冷冻保护剂以防止冰晶损伤。妥善保存的细胞可保持存活力数十年或更久。方舟 150 天的储存期完全在冷冻保存可达的范围之内,但要在大规模上运行低温储存——百万级物种、每物种多个样本以求冗余、对每个样本进行整合编目——所需的技术代表着相当可观的低温基础设施。

蛇形派系会从其母星文明带来这种技术,在那里,生物保存大概已经是一个成熟的领域。处理与保存方舟项目规模上样本所需的基础设施会非常可观,需要在超大陆的多个地点设置大型设施,与野外采集行动整合,并持续监测以确保样本的存活力。

**第四阶段:编目与元数据。**每个样本都必须与全面的元数据关联起来:什么物种、什么标本、何时采集、在何处采集、采集地点的生态背景、该物种的分类关系、从被保存样本中恢复活体所需要的再生协议。仅信息管理的挑战就相当可观。现代生物样本库管理着数千个样本,需要部署庞大的数据库系统和专门的策展人员来维护其元数据的完整性。洪水前的档案管理着数百万物种的样本,会要求一种我们自己文明才刚开始发展的信息管理基础设施。

**第五阶段:再生能力。**样本的保存只是项目的一半。在灾难之后,样本必须能够在适合被再生生物体所栖息的生态群落条件下,被再生为活的生物。哺乳动物的再生需要人工子宫(我们自己的文明在研究环境中才刚开始开发),或借助相关物种的代孕母体。非哺乳动物的再生则有其自身的技术要求:适当的卵或胚胎环境、合适的营养与发育条件、各种生命周期的时序要求。除生物技术之外,把生物再生到适当栖息地所需要的生态知识——知道哪些物种能在哪些条件下生存、稳定的生态系统功能需要哪些群落组成、重建可运转生态系统需要哪些演替序列——代表着一套可与或超越我们当代生态科学相比的专业知识体系。

整个行动——调查、采集、保存、编目、再生——需要数个世纪的准备,数百或数千名跨多门学科的人员,跨超大陆多设施的整合基础设施,以及我们自己文明尚未达到的技术成熟度。

**当代的类比。**现代等同物——以我们自身当前文明水平进行的、与遗传载荷项目类似的工作——可以让人对所需努力的规模有所感知。斯瓦尔巴全球种子库于 2008 年开放,由挪威政府与作物信托基金、北欧遗传资源中心合作运营,目前储存约 130 万份种子样本,代表约 6,000 种植物物种。该库设计用于把农业生物多样性免于灾难性损失——其位于挪威斯匹次卑尔根岛的位置,被选中是因为其偏远、其气候稳定以及其与可能冲突区的隔离。该设施的使命是为将来的使用而保存遗传多样性。它是与洪水前档案最接近的现代类比。

圣迭戈动物园野生动物联盟自 1972 年以来一直从濒危物种采集细胞样本,目前持有来自约 1,300 个物种、约 11,000 个个体动物的样本。该设施在冷冻保存中维持活细胞,以备保护性繁育项目使用,并且越来越多地用于克隆已经丧失的物种。近年来,“冷冻动物园”已为黑足鼬和普氏野马的克隆提供细胞——这些物种恢复项目展示了从被保存的细胞物质中再生生物的实际可行性。

这些项目虽然相当可观,但仅代表总体生物多样性中极小的比例。斯瓦尔巴库保存了世界植物物种的约 0.5%。“冷冻动物园”保存了脊椎动物物种约 0.05% 的细胞。当前全球所有生物样本库中保存的生物材料总量,大约占全球总生物多样性的几个百分点,而绝大多数物种——特别是较小的无脊椎动物和微生物领域——则完全未被代表。

资料框架所要求的洪水前档案,会以数量级超越我们当前的努力。蛇形派系与他们的人类伙伴需要在洪水前数个世纪内,组建一个基本上包含行星上每一个物种样本的档案——其范围可与绘制完整人类基因组(在我们自己文明中用了 13 年和大约 30 亿美元)相比,但要扩展到数百万物种而非单一物种。按文集的框架,这样一个项目得以开展本身,就是威胁严重性的证据——这不是一项随意的保存努力,而是一项绝望的、文明尺度的事业,在期限压力下、面对行星生物圈必将毁灭的前景而开展。

在遗传档案的解读下,载荷的规模变得可以管理。无需把每个种类的甲虫成对地容纳并喂养五个月,每个甲虫物种相当的遗传材料都可以储存在不大于一间小型实验室的体积之内。圣经文本所用的“成对,雄性与雌性”的语言,反映了有性生殖的技术要求——X 与 Y 染色体都需要被保存——而不是要求每个物种的两只成年甲虫被带上船的字面要求。方舟是一个遗传图书馆,与一个能维持机组的轨道平台相结合。在这种解读下,它的尺寸与容量足以胜任,而装载着地球上每一物种成对繁育对的木船则做不到这一点。

如前所述,人类乘员同样在船上——不是以被保存细胞的形式,而是作为活体机组。挪亚、他妻子、他的三个儿子(闪、含、雅弗)和他们的妻子:共八人,正是本章开头已经指出、被编码进这个时代与“第八日”关联中的数字。资料补充说,方舟回到地表时,还携带了*“地球上每一种人类种族的一对”*——表明不止挪亚所来自的伊甸谱系,七支人类谱系的代表都被保存并被送回。这些其他代表究竟是船上活体的人类,还是着陆后从被保存的遗传材料再生而来,资料并未精确说明。所被说明的是:七个种族都通过该事件得以保存,所有七个种族在复苏之后都被送回其原有区域。整个人类物种并未被毁灭。它的文明被毁灭了。

## VII. 灾难

武器落下了。

资料未详述目标选择,但本章所引——并将由科学部分进一步处理——的地质证据表明,撞击集中在超大陆上一个特定中心区,毁灭性效应自该处以大致环形向外传播。这个中心区是因为混血文明在那里最密集而被选定,还是因为它在地球物理上对于产生所需破坏格局最优,资料并未说明。它所说的是:撞击的能量足以把超大陆本身击碎。

雷尔派资料在一段值得引用的话中明确处理了这一点:*“当耶洛因决定毁灭他们的基地、他们的实验室以及他们在地球上所创造的一切时,他们必定使用了极强大的毁灭手段——它们除了把这原始大陆击碎、并使其各个碎片从震中向外漂移之外,也必定横扫了整个陆地表面。”*在此种解读下,大陆的破裂并不是耗时数千万或数亿年的地质过程。它是一个事件。它发生在双子座洪水灾难期间。在此之前每一个时代都存在的单一陆块的碎片,被最初撞击的力量推开,而我们今天所观察到的、表现为板块缓慢运动的持续漂移,正是那次原初位移的残余动量。

这种解读直接与常规的地质叙述相违——后者把大陆漂移视为以亿年时间尺度运行的过程,由地幔对流而非任何灾变性初始事件所驱动。文集对这一冲突进行了诚实的指出。常规叙述已经积累了大量证据——古地磁特征、化石分布、大陆边缘的吻合、当前对板块运动的 GPS 测量——现代地质学用这些证据来确定运动年代并建模其原因。雷尔派资料提出一种压缩的时间线,在其中同样的观测得到不同的解释:大陆边缘的吻合反映洪水前的构型;古地磁特征反映事件期间及之后的快速重定向;化石分布反映洪水前生物圈的连通性以及洪水后的重新播种格局;当前由 GPS 测量到的漂移反映来自原初位移的持续残余动量。压缩的时间线能否在所有方面与地质证据相协调,是文集并不假装能解决的问题。科学部分将更详细地处理这一冲突。本章在这里所做的,是指出资料的主张与冲突的存在,并把权衡的工作留给读者,在接下来的内容中进行。

清楚的是:在雷尔派的解读下,我们如今所知的各大洲并不是一直如此的。它们是一片单一陆块的碎片,被洪水事件击碎,并自那时起一直在运动之中。这种破碎的地质后果会相当深远。已开始运动的大陆碎片在新的边缘沿处彼此相撞。在碎片擦过洋床之处,沉积物沿其前缘堆积起来,产生了——在雷尔派的解读下——构成现代各大陆特征的年轻山脉。喜马拉雅山脉沿印度碎片的边缘形成,该碎片从超大陆中破出并向东北漂移,与欧亚主体相撞。安第斯山脉与落基山脉沿正在漂移的北美与南美碎片的西缘形成。阿尔卑斯山脉沿非洲与欧洲的碰撞边界形成。澳大利亚大分水岭沿向东南漂移的澳大利亚碎片的前缘形成。所有这些,在雷尔派的解读下,都是洪水事件及其紧随其后的板块构造调整的产物,而不是常规地质学赋予它们的百万年时间尺度的产物。

另一个细节:南极碎片带着其上在洪水前超大陆统一气候下繁盛的热带植被与动物,向南极漂移,并随着气候的变迁逐渐被冰雪覆盖。在南极冰下发现的被保存的热带化石——植物、动物、以及在某些有争议的情况下,看似人造而非自然的结构的化石——按雷尔派的解读,是洪水前超大陆南极部分的化石痕迹,被保存在冰下,而那冰仅在碎片到达其当前的极地位置后才形成。最近的古植物学发现所揭示的“温暖的南极”并不是对证据的误读。它是洪水前世界的一个真实碎片,被锁在冰中时间还不长,尚未失去其原本的特征。

**石油环。**资料接着进行了一项观察,这一观察已成为雷尔派传统内部最具体、最可检验的地质预测之一。1970 年代,中央情报局委托哈德逊研究所对全球自然资源分布进行研究。一位名叫内林{{ cite(id="12") }}的研究员在该项目工作时发现了某种意料之外的现象。当各大洲被重建回其破裂前的构型——即现代地质理论中的盘古超大陆——世界上主要的油田并非随机散布于这块重组的陆地之上。相反,它们形成了一个环。{{ footnote(id="2") }}阿拉斯加与北极的石油矿藏、阿尔伯塔的沥青砂、科罗拉多的沥青页岩、墨西哥、委内瑞拉与奥里诺科的稠油、尼日利亚、南撒哈拉、利比亚、阿拉伯、伊朗与西伯利亚的储量——所有这些,在大陆碎片被重组回其原始位置时,都落入围绕一个中心区的大致环形格局之中。

常规地质对石油形成的解释——有机物质在数百万年间的缓慢厌氧分解——自然不会产生环形格局。它会产生与古代沉积盆地地理位置相对应的分布,无论这些盆地碰巧形成于何处。环形分布,是在一种单一灾难性事件——发生在特定中心位置——同时把巨量有机物质以大致对称格局埋藏在撞击点周围、随后的厌氧条件在地下产生石油这种情形下,才会被预期出现的。雷尔派的解读正是如此:中心爆炸——议会下令对洪水前文明部署的武器——气化并位移了大量活物质,然后把该物质埋藏在震波的紧接地质碎屑之下。被埋藏的物质,断绝氧气,并在随后数千年间承受持续压力,转化为我们现在所抽取的烃类化合物——即我们所称的化石燃料。环形保留了原初事件的几何形状。在此种解读下,我们脚下的石油是洪水前生物圈被压缩的遗骸。

这是一个实质性的主张,本章必须以适当的认识谨慎来对待。内林的研究是真实的,但并不属于主流石油地质学的一部分,后者继续把石油形成归因于在地质时间尺度上运行的渐进沉积过程。文集把石油环的观察呈现为框架最具特色的具体物理证据主张,同时承认其底层研究并未被广泛重复或接受。科学部分将更详细地回到石油问题。本章在此处所指出的是:在文集框架内,全球石油分布的几何形状并非随机。它是本章所描述灾难性事件留下的几何特征。

无论内林具体观察的地位如何,引人注目的是:我们已经从洪水前文明的被埋藏有机物质中提取了一百多年,把它焚烧来为我们自己的文明提供动力,却从未认识到它可能是什么。我们提炼为汽油、塑料以及工业经济基础化学品的石油,在雷尔派的解读下,是一个文明的被熔解生物量——我们如今正通过其燃烧的大气后果,以慢动作重演那个文明的毁灭。这种戏剧性的反讽是相当深刻的。我们丧失的那个文明,正是我们建立的这个文明的燃料。而我们通过焚烧那些远古生物量正在归还给大气的二氧化碳,正是洪水前文明运转时正在循环的同一些碳——曾被灾难封存,如今又被我们自己的技术重新释放。

**洪水。**圣经文本把洪水描述为水——雨水、上涨的海水、向上敞开的深渊之泉。资料并不否认这一点。在任何合理的重建下,中心武器的撞击会产生灾难性的大气与海洋后果:过热空气上升形成巨大的风暴系统,海水被冲击波位移并形成横扫陆地的大规模海啸,大气中的水分以早期气候未曾经历的数量降下。圣经文本指明的“水势浩大,在地上”150 天,在雷尔派的解读下,既反映字面意义上的洪水,也反映方舟在轨道上正在等候耗尽的大气污染过程。

一处具体的文本细节值得一提。资料指出,化石记录在洪水事件期间及其后形成的沉积物中,保存了所发生之事的痕迹。在撞击位移的物质中形成的沉积岩层。被该事件杀死的生物的化石,被保存在这些层中,与该事件相关的独特化石层位在每一大洲的地质记录中都是可见的。资料的主张是:常规归因于数百万年渐进沉积的全球化石记录,在很大程度上是在洪水及其紧随之后的余波中形成的,通过对巨量有机物质的快速埋藏。这是雷尔派资料与主流地质学之间最直接的冲突,本章如实指出。常规读解把化石记录的年代分布在数亿年的沉积之中。雷尔派的读解把其中相当大一部分压缩进单一灾难。这两种读解之间的协调——如果可能的话——需要两种资料任何一方都无法独自提供的证据。科学部分将回到这一问题。

**灾变论背景。**文集把洪水读作一次突发的灾变性事件,这一读解与一个更广泛的当代研究项目相一致——这一项目被称为灾变论、新灾变论,或简单称为后均变论地质学——该项目一直在积累关于地球最近地质过去发生过相当规模灾变性事件的证据。最显著的当代倡导者是兰德尔·卡尔森(Randall Carlson),他是一位独立研究者。大约自 1990 年代起,他就发展出关于地球最近过去发生过突发、大规模灾变性事件的地质证据的一系列工作。卡尔森与更广的彗星研究小组合作,并通过与格雷厄姆·汉考克及在各种当代播客上的露面广泛发展工作,关注:

- 新仙女木边界(约 12,900 年前),自 2007 年以来,关于宇宙撞击事件的证据已大量积累
- 晚更新世至早全新世灾难性洪水的地质特征——包括华盛顿东部的水道荒地(由末次冰期末的冰川溃决洪水所形成,表明地质学认可大规模灾难性洪水作为真实机制)
- 宇宙撞击与地球地质和气候历史之间的关系
- 跨文化对灾难性事件的神话保存,包括洪水叙事的全球分布

卡尔森的工作并不是主流共识。在许多具体之处,他的解读与主流地质学有争议,而他把圣经及其他神话材料与地质证据相整合,把他的工作放在更广的另类考古学传统之中——这一传统主流科学一直倾向于不予理会。但他的工作是实质性的,他所调动的地质证据即便在解读上有争议也是真实的,他所代表的更广灾变论框架与文集自身的框架更为一致,而不是与十九世纪以来主导主流地质学的严格渐变论传统一致。文集将卡尔森作为最显著的当代灾变论研究者之一引述,但并不全盘承诺他的每一项具体主张,并且承认他的工作——以及更广的灾变论传统——为评估文集框架提供了一个当代研究语境。

卡尔森记录得最充分的灾难性洪水现象——末次冰期末雕刻出水道荒地的米苏拉洪水——具体证明地质学本身把灾难性洪水当作真实机制来认可。米苏拉洪水由冰川米苏拉湖蓄水的冰坝反复溃决所致,把巨量水流以超过当今世界全部河流总流量的速率送过现今的华盛顿东部。这些洪水的地质特征——巨大的涟漪痕迹、大库利的干瀑布、远离任何当今水源沉积的巨石——任何到访该地区的人都可见到。米苏拉洪水并不是文集框架中的全球洪水,但它们是这样的证据:灾难性洪水所需的地质机制确实存在,并且在最近的过去运作过。文集的框架把这种认可延伸到它放在巨蟹-双子边界上、规模上要大得多的灾难性事件。

## VIII. 复苏与立约

当方舟回到地表时——按圣经传统——它降落在亚拉腊山上。雷尔派资料并未对具体着陆地点作出承诺,文集也指出:我们如今所称的亚拉腊地区本身是一个破裂后的地貌,其在洪水前地理中的存在状态会有所不同。重要的是着陆的实际内容:方舟降落在足够的海拔之上,机组走出,复苏行动开始。

第一阶段是环境评估。资料这样描述:*“在监测放射性水平并以科学方法将其驱散之后,造物者告诉挪亚释放动物,以查看它们是否能在大气中存活。此项行动成功了,他们得以走入开放的空气中。”*在此阶段被释放的“动物”,在遗传载荷的解读下,是首批被再生的生物——作为试验案例,用来在更广泛的再生开始之前确认地表已可宜居。挪亚先放出乌鸦、又放出鸽子,等待鸽子是否带着植物归来的圣经叙事,保存了这一评估阶段的运作逻辑:活体生物被送出作为生物监测者,以其存活与行为来判断地表条件是否已恢复到可以更广释放的程度。

造物者以科学方法分散残余放射性,是一项资料未解释其具体机制、但文集框架明示其必要性的技术性行动:核之后的环境在适合居住之前需要主动修复,而造物者具备在我们自己文明目前会觉得棘手的尺度上执行该修复的技术。在轨道上等候的 150 天,大致对应于核武器产生的最具急性危害的放射性同位素的半衰期衰减——碘-131(半衰期 8 天)、锶-89(半衰期 50 天),以及主导爆后初始辐射环境的其他各种短寿命裂变产物。到 150 天时,这些大多已衰减到可控水平。寿命更长的同位素(铯-137,半衰期 30 年;锶-90,半衰期 29 年)仍会存在,需要由蛇形派系的技术进行主动修复,但地表对谨慎的人类活动来说已足够安全。

第二阶段是重新播种。每一种其遗传材料已被保存的物种,都被再生——资料暗示这一过程类似于现代克隆,但以我们的技术尚未接近的规模和速度进行。再生的生物被释放到合适的区域,生物圈开始恢复。资料以其一贯的克制指出:*“有些物种在此时被刻意选择不再重造。一个突出的例子是恐龙。”*这是资料对洪水后大型爬行类动物群缺席的解释。恐龙——正如处女座一章所论证的——曾是一个具体的派系性项目,其创造本身在耶洛因计划内部就有争议——在洪水之后未被再生。最初制造它们的团队已不在场,无法重新制造它们,而留下的流亡造物者判断,它们与幸存人类需要重建的洪水后生态系统不相容。化石记录中恐龙突然消失,常规上归因于六千六百万年前白垩纪-古近纪灭绝事件,在雷尔派的解读下,只是“不把它们纳入洪水后重组”这一决定的简单后果。

其他洪水前物种也未被再生。文集并没有一份完整名单,但资料的更广框架暗示:许多我们仅从化石记录中知道的物种——更新世末期消失的各种巨型动物群、保存在更古老化石层中的各种异常形态——属于蛇形派系选择不予恢复之列。因此,洪水后的生物圈并不是洪水前生物圈的完整恢复。它是一个被精选的子集,由联盟根据与洪水后世界条件的相容性、以及对将支持恢复中的人类人口的新生态秩序之适宜性来挑选。

第三阶段是各人类谱系的重新分布。*“随后,每一人类种族都被送回其原创造之地。”*七个人类种群——其代表已通过该事件获得保存——被运回他们最初所来自的区域,这些区域如今被新打开的海洋所分隔,从他们所起源的单一陆块上碎裂出去。澳大利亚谱系被送回澳大利亚碎片;安第斯谱系被送回美洲碎片;喜马拉雅谱系被送回当时已成为中亚的地区;以此类推。洪水后的大陆构型,造就了一种与洪水前非常不同的人类地理分布。在随后的大部分人类历史中,使七个谱系彼此隔离的地理屏障如今已就位。每个谱系将在很大程度上独立于其他谱系发展其后续文明,而现代人类种群的文化、语言与种族差异,反映了这种洪水后的地理破裂与各自团队原有派系差异的结合。

这种重新分布还有另一个维度值得关注,因为它解释了后续人类历史中一个原本难以说明的特征。伊甸谱系——挪亚家族的后裔——并不需要从零重建。他们是建造方舟的那一支。他们曾在灾变期间登上方舟。他们是第一个着陆的,也是第一批参与重新播种的。监督整个行动的蛇形派系成员,在洪水后的紧接时期持续身处他们当中,教导、协助、指导重建。其后果是:伊甸谱系的恢复非常迅速:在数个世纪之内,挪亚家族的后裔已达到足以承担相当规模工程项目的文明水平,而其他六个谱系——被送回原区域但没有同样持续在场的高级教师——仍在从更小的起点缓慢重建。伊甸谱系与洪水后人类其余部分之间的技术不对称,会持续数千年,并表现为美索不达米亚、埃及与相邻文明早熟的复杂程度——这种不对称在考古记录中是可见的,在这种解读下并非偶然。这是直接源自“是谁建造了方舟、谁在灾变中与其教师共同度过”的后果。

第四阶段是立约——而立约的各方值得仔细辨认,因为对{% wiki(slug="noahic-covenant") %}挪亚之约{% end %}的常规解读以一种掩盖随后一切政治形态的方式误指了它的双方。

《创世记》第 9 章记载了耶洛因与挪亚之间的一项约定,在其中造物者承诺永不再次毁灭人类,并把彩虹指定为该约的可见记号。在雷尔派的解读下,作出这一承诺的“造物者”具体地是在场地面的流亡造物者——也就是建造方舟、监督轨道、进行洪水后修复、如今与挪亚一同站在祭坛上接受献祭的那一群人。母星上的议会并不在这一场景之中。圣经文本未给出任何议会曾被咨询的暗示。雷尔派资料以它告诉我们的议会随后认知状态来确认这种缺席:当巴别塔项目后来开始成形时,议会因得知地球上的生命实际上未被毁灭而感到震惊。雷尔派文本直接陈述:*“我们行星上的人们听到这一切时变得惊恐。他们仍在观察地球,知道生命并未被毁灭。”*如果议会曾是挪亚之约的当事方,这就毫无道理。一个当事方会在彼时被告知,而不是在数个世纪后通过观察被惊到。

在更正后的解读中,立约是两个当事方之间的一项私下安排:在场地面的流亡造物者,以及由挪亚带领的人类幸存者。两方都刚刚做了议会所禁止之事。流亡造物者违抗毁灭令,建造并运作了方舟。人类幸存者协同——挪亚接受指令、建造该器皿、为其配备机组、活过灾变,并在彼岸作为保存了他的造物者的幸存伙伴显现。在祭坛上所发生之事正式化了他们的共同立场。它不是对既有政治秩序的批准。它是一种新秩序的奠基——一项流亡造物者派系与幸存人类之间的正式联盟,由相互承诺所绑定,与下令把双方都毁灭的母星议会在利益上日益相异。

联盟的条款是相互的。流亡造物者承诺自己不参与未来对人类的任何毁灭——这一承诺的含义,由“他们刚刚拒绝参与议会下令的毁灭”这一事实所塑造。他们承认人类追求科学进步的合理性,扭转了议会最初把这种进步视为受造人类构成的核心威胁的立场。人类则承诺要心怀感激、积极重建,并通过仪式上的献祭来识认造物者——{% libref(book="genesis-woh", chapter=8, verse=20) %}创世记 8:20{% end %}的燔祭所代表的就是这一点。设在云中作为约的记号的彩虹,正是这份私下安排的可见信物。它不是仅对单一神圣方可见的记号。它是新大陆之上、共同天空中,对双方都可见的记号,标志着联盟所将开展的一切的正式开端。

这一更正后的解读所蕴含的意义,延展到文集中之后每一个时代。立约之后,地球上不再是两个政治范畴(造物者与人类,各自与母星相关联)。而是三个:母星议会;由正式约束的流亡造物者—人类联盟,在地球上被立约绑定;以及其他六个谱系中的更广泛人类人口,这些人口并非联盟的当事方,其后续历史会沿不同轨迹展开。希伯来圣经在洪水后各时代归于“耶和华”或“耶洛因”的许多事件,在更仔细的审视下,会是联盟基于其约的承诺而采取的行动——保护其人类伙伴、教导他们、为他们出面干预。其他事件则会是议会针对联盟或其伙伴所采取的行动。希伯来文本对两者使用同一套词汇,是因为其作者并不具备文集现在用来区分它们的政治框架。读者在接下来的各章中,需要留意每一时刻可能由哪一方在行动。

## IX. 重建与巴别塔

洪水之后的岁月,随着双子座时代的延续,是一段快速重建的时期。被再生的生物圈开始重新建立起来。人类人口开始增长,适应了新的大陆构型,发展出适合新区域处境的新语言,并开始——借助仍留在他们当中的蛇形派系成员的帮助——重建洪水所摧毁的文明能力。

资料专门聚焦于伊甸谱系,因为它所解读的圣经叙事正聚焦于该谱系。挪亚的后裔——闪、含、雅弗,以及他们各自的后裔——散布在古代近东及邻近地区,创建了圣经的“民族列表”所列举的洪水后主要民族。新月沃地的考古记录——最早的洪水后农业聚落、最早的城市、最早的文字系统——对应于这种洪水后的早期复苏,而在雷尔派的时间线上,被压缩为数个世纪,而非常规考古学赋予的数千年。

重建的速度令人惊叹。在洪水之后的数个世纪之内,伊甸谱系已经恢复了足以承担——按资料的叙述——一项重大工程项目的文明能力:即圣经文本所称的巴别塔的建造,资料把它指认为一艘庞大的火箭。*“但最有智慧的种族,以色列人,已经取得了如此显著的进步,以致他们很快就能在流亡造物者的帮助下进行对太空的征服。流亡造物者希望他们的新人类前往造物者的行星,以表明他们不仅聪明、有科学头脑,而且心怀感激、爱好和平,从而获得宽恕。因此,他们建造了一艘庞大的火箭——巴别塔。{{ footnote(id="3") }}”*

这一段确立了本章戏剧性弧线中最重要的解读点。巴别塔不是反叛工程。它是*和平的呈献*。蛇形派系协调其人类伙伴建造该火箭的目的,是把一支人类代表团送到母星,以证明受造的人类是良善的、和平的、聪明的、心怀感激的——值得议会的接纳,而不是议会的毁灭。联盟挺过了洪水;重建了文明;教导了人类星际旅行所需的技术。如今它正准备使用该技术,以寻求与议会的和解,为受造人类的继续存在辩护,并最终建立——终于能让联盟公开运作、而不是继续抵抗的政治安排。

该策略是充满希望的,但也是合理的。到此时,议会已经观察了洪水后的复苏数个世纪。毁灭令已经执行,洪水前的文明已被消除。余民人口通过联盟的保存努力得以幸存,但议会——至少在最初——把这被保存的余民当作既成事实接受了,而没有下令进一步毁灭。议会的容忍尽管有限,但表明直接谈判可能是可能的。一支人类伙伴代表团,带着联盟的认可抵达母星,将代表联盟正式请求议会承认这次保存为合法、并允许受造人类在没有进一步干预的情况下继续存在。这就是巴别塔的目的。

值得就巴别塔项目在技术上意味着什么作出精确的说明。资料暗示,到这个时候,联盟已经实现了行星际旅行——能够在太阳系内的行星之间移动——并且正在为星际旅行——抵达另一颗恒星这一更难的问题——作准备。在此种解读下,巴别塔是第二代飞行器,即设计用来前往耶洛因母星的那一艘。第一代飞行器会更简单,可能用于轨道作业或在本地行星系内的旅行。这一进程表明:一个相当成熟的航天计划——不是单一的形象工程,而是一项持续的技术开发努力,能够产出连续几代航天器。到双子座的晚期世纪,伊甸谱系按资料所述,在技术水平上大致达到我们自己文明在二十世纪后期所达到的水平,并正在接近我们尚未达到的水平:常规化的星际能力。

巴别塔——圣经叙事把它定位在示拿地,对应于底格里斯河与幼发拉底河下游冲积平原上的古代苏美尔——是这种协作努力的物理表现。苏美尔的资料,保存了该地区紧随洪水后早期最古老的书面记录,其叙事在惊人的方面与圣经的巴别叙事相吻合。

**宁录与苏美尔的对应。**圣经文本在{% libref(book="genesis-woh", chapter=10, verse=8) %}创世记 10:8-12{% end %}中介绍宁录,该段插入更广的“民族列表”,以挑出一个人物来作专门关注:*“古实又生宁录,他为世上英雄之首。他在耶和华面前是个英勇的猎户,所以俗语说:像宁录在耶和华面前是个英勇的猎户。他国的起头是巴别、以力、亚甲、甲尼,都在示拿地。”*宁录被描述为洪水后第一个帝国的奠基者,巴别(巴比伦)为其首都,而以力(苏美尔的乌鲁克)、亚甲(阿卡德)与甲尼是其权下的其他城市。他与狩猎、王权、城市建造相关联。圣经文本对他给出一个简短但有力的介绍。

其后的犹太与伊斯兰传统对宁录形象作了大量发挥。在这些传统中,他成了巴别塔本身的总建筑师——组织洪水后人类人口建造那座要通天的大建筑的国王。各种拉比与教父资料把宁录描绘为暴君、反抗神圣权威的叛逆者,与随后圣经叙事将引入的义人亚伯拉罕相对立。古兰经保留了一种平行传统,在其中宁录挑战亚伯拉罕,最终被神圣干预所毁。

苏美尔的对应是惊人的。苏美尔王表——保存在从早王朝至古巴比伦时期(公元前第三千纪至第二千纪初)的楔形文字中、对洪水前后统治者的长序汇编——记录了美索不达米亚南部各城市中一系列洪水后的统治者。苏美尔王表中最早的洪水后王朝是基什第一王朝,随后是乌鲁克、乌尔、阿万与其他城市的各个王朝。与圣经宁录最直接可比的人物是恩美卡尔,即乌鲁克的早期国王,他被苏美尔王表和现存史诗(《恩美卡尔与阿拉塔之主》系列)所记录。恩美卡尔在苏美尔传统中被认为是乌鲁克的奠基者——对应于圣经文本归于宁录王国的城市之一以力——并被认为是试图建造一座能够统一美索不达米亚之地的大型神庙塔(乌鲁克的伊安娜祠塔)的人物。

苏美尔文本《恩美卡尔与阿拉塔之主》包含一段已被广泛与圣经巴别叙事相联系讨论的文字。相关诗行描述了原初的一段时期,在这一时期,“整个宇宙,众人齐声,以一种语言向恩利尔献上颂赞”——即,在那一时期,整个人类讲一种语言。文本随后描述了智慧与技术之神恩基的一次神圣干预:“他改变了他们口中的话语,把争辩引入其中——引入直到那时一直为一的人类言语之中。”与圣经巴别叙事的对应是直接的:一段原初时期的人类语言统一,随后是一次有意的神圣行动,把语言分裂为相互不可理解。两种叙事是各自独立的——苏美尔文本在年代上明显早于圣经文本——但它们描述的是同一事件。

文集的解读自然地整合了这些平行传统。巴别塔项目,在示拿地(苏美尔)、由圣经传统记忆为宁录、苏美尔传统记忆为恩美卡尔(或可能是多位早期苏美尔国王的复合体)的人物领导下进行,是联盟的和解尝试。洪水后早期的统一语言——同时保存于圣经传统与苏美尔传统中——是从洪水前伊甸文明继承下来、在洪水后紧接时期仍在运作、地理隔离尚未带来后续语言分化之前的语言基底。议会的干预——在圣经文本中归于耶和华、苏美尔文本中归于恩基——是议会有意的行动,把统一语言分裂,并把科学精英驱散。两种传统从不同的文化视角保存了同一事件,而苏美尔版本因其对“统一语言状态”与“终结其的有意神圣行动”都明确指认而特别值得注意。

苏美尔资料中另一个细节值得记下。苏美尔传统在各种文本中保存了 *apkallu* 的形象——按美索不达米亚传统,这是七位智者,在洪水前时期从海中出现来向人类传授文明诸艺。其中第一位,阿达帕或奥安内斯(在贝罗索斯{{ cite(id="9") }}的希腊化转写中),被描述为一种存在者,每天从海中浮起来教导早期人类农业、文字、数学、天文学以及文明所需的更广泛技艺,夜晚又回到海里。这七位 *apkallu* 共同代表了被传给人类的洪水前智慧的总体。后来的苏美尔传统把这种传递记忆为该地区所有后续文明知识的基础。

文集的解读自然地整合了 *apkallu* 传统。在洪水前时期从海中浮出向人类教导的七位智者,按文集的框架,是身处隐藏基地中的蛇形派系成员。巨蟹座一章指出,流亡造物者已撤入山地与水下基地,作为他们对议会监视的调适;水下基地会是浮出海面来教导人类人口的 *apkallu* 形象的来源。这七位形象可能对应于蛇形派系七个子团体的代表,每个子团体负责教导特定的知识体。苏美尔传统保存了表面现象——存在者从海中浮出来教导——而未保存底层技术(水下设施、蛇形派系更广的行动),后者才能解释实际发生之事。后来美索不达米亚传统所发展出的神话化解读,把运作的现实转化为宗教叙事,但原初指涉仍保存在传统的结构中。

回到巴别塔本身:母星上的议会通过它在洪水之后保留的任何遥感监视设备观察地球,看到了该项目的建造,并感到震惊。洪水本意是永久终结人类威胁。伊甸谱系的迅速复苏,如今再加上其能够建造能抵达母星的航天器,表明威胁只是被推迟、并未被消除。议会的回应记载在{% libref(book="genesis-woh", chapter=11, verse=7) %}创世记 11:7-8{% end %}:*“我们下去,在那里变乱他们的口音,使他们的言语彼此不通。于是耶和华使他们从那里分散在全地上。”*

雷尔派资料以操作性方式解读这一点。*“于是他们前来,把拥有最多科学知识的犹太人带走,把他们分散到原始部落之间所在的各大洲上的国家中——在那里没有人能听懂他们,因为语言不同——并把他们的所有科学仪器都毁掉。”*该事件并不是常规解读中的奇迹般的{% wiki(slug="confusion-of-tongues") %}口音变乱{% end %}。它是由议会的代理人执行的一次有意行动,其中拥有火箭计划关键知识的具体人类科学家被识别出来、被实际移送到他们无法与新邻居交流的地区、并被与他们的研究材料分开——研究材料则被销毁。火箭项目并非仅仅被叫停。它被拆解,而人类恢复该项目的能力被如此彻底地分散到洪水后的大陆地理之中,以致需要数千年才能重新组合起来。

这是打破联盟和解希望的时刻。塔曾是一份和平的呈献。议会以武力拒绝了它。耐心展示的策略——保存人类、重建文明、通过有节制的技术发展展示其值得被接纳、派出代表团去陈请——已经失败。议会明确表示:任何展示都不会足够,人类文明只会被允许以受限和有限的形式存在,任何接近议会自身水平的技术发展都会遭到干预。蛇形派系所希望的和解,并不在提议范围之内。议会准备无限期使用武力来阻止人类文明接近其自身水平,而联盟没有通过谈判可走的前路。

蛇形派系现在面对本章弧线中第二个选择时刻。第一个是通过建造方舟来抵抗毁灭令的选择。那个选择是在最终和解的希望中作出的。随着塔的被毁,希望熄灭。该派系必须决定接下来要做什么。他们可以接受议会准备允许的那种受限存在——继续以有限程度教导人类人口、不再尝试通过技术展示进行和解、无限期接受议会有权干预任何威胁这种接受的工程的权威。或者,他们可以转向公开冲突——对议会采取直接军事行动、使用蛇形派系本身所拥有的任何武器、试图通过武力而不是展示来确立联盟的独立性。

他们在第二个时刻所作的选择,正是产生天界之战的选择。

## X. 天界之战

母星议会与地球上的流亡造物者—人类联盟之间的冲突,自伊甸驱逐以来一直潜伏,在巴别塔干预之后的数个世纪中变为公开。

资料以引人注目的压缩处理这一冲突——只用一句话,周围环绕着其他材料,在初次阅读中很容易被漏过。*“当时他们行星的政府想要毁灭那些创造了人类的人。”*这就是战争。母星议会得出结论:地球上的流亡造物者已成为对其权威的根本挑战,因而决定对他们采取军事行动。交战的确切形式未被明说。所暗示的是:议会越过了它之前的限度——超越了对人类文明的毁灭、超越了对人类科学家的针对性分散——进入了对流亡造物者本身的直接行动。

值得就冲突的结构作出精确的说明,因为到这一阶段,议会面对的并不是一个单一的反叛派系。它面对的是一个联盟。挪亚之约把流亡造物者与人类幸存者绑定为一项正式合作关系,而随后每一项违抗议会的行动——公开教导、合作的巴别塔项目、接近星际能力的进展——都是由双方共同进行的行动。当议会最终对“那些创造了人类的人”采取行动时,它是在对一个联盟的资深合伙人采取行动,而该联盟的初级合伙人正是受造的人类本身。冲突在此种意义上并不仅仅是一场耶洛因内部纠纷。它是文集所记录的第一场文明间冲突:一边是母星议会,另一边是地球上的流亡造物者—人类联盟。到公开战争之时,这个联盟已经获得了足够的技术与政治分量,使议会最终不得不进行谈判而非灭绝——这就是该合作关系已经达到何等实质性程度的衡量。

从联盟的和解策略向公开军事对抗的转变,在文集的解读下,是把文集的戏剧弧线从悲剧转向更阴郁之物的时刻。蛇形派系自最初伊甸驱逐以来,曾以各种方式尝试,使他们对受造人类的爱与对自己文明的忠诚相协调。这种策略在每一次升级中都失败了——在最初的披露中失败了(那次披露导致他们的流放);在方舟的建造中失败了(那是对毁灭令的违抗);在塔的建造中失败了(那被当作威胁、而非作为呈献来对待)。随着塔的被毁,联盟不得不接受:这种策略已被永久封堵。议会不会接纳受造的人类。前进的唯一途径是以武力来保卫它。

按资料的叙述,蛇形派系的军事回应是从可见状态中撤出,同时为积极冲突做准备。*“为了不被人类打扰,造物者把基地建在高山之上,我们如今在那里(例如喜马拉雅与秘鲁)找到伟大文明的痕迹,也建在海底。山地站逐渐被放弃,而代之以更不易被人类接近的海底基地。最初被流放的造物者已把自己藏入海洋。”*这种撤出不仅是从人类的视线中。它也是、并且首先是从议会的视线中。高海拔山地站与水下基地是防御性设施,被选用是因为它们对轨道观测的隐蔽性以及对直接攻击的抵抗力。到双子座晚期,蛇形派系已不再公开生活在人类当中。他们处于隐蔽状态。他们实际上已经成为一支游击力量,从隐蔽位置作业,通过间接渠道维持与其人类伙伴的关系,并为他们如今已知必将到来的冲突作准备。

高山设施特别值得注意。资料明确提到喜马拉雅与秘鲁,这对应于现代考古学已经识别出在前印加与前吠陀的大型巨石遗址中,其建造技术与天文对齐表明相当成熟程度的地区。玻利维亚的蒂瓦纳科建筑群、安第斯的各个前印加遗址、西藏与喜马拉雅以更古老结构为基底建造的寺院建筑群、横跨更广亚洲高地的各种岩刻与悬崖修建设施——所有这些,在文集的解读下,都可能保存了蛇形派系在此时期所建造的高海拔防御设施的痕迹。这些痕迹是零碎的;大多数原始设施会被后续的侵蚀、刻意的战后清理,或人类对场地长期的再利用与改建所毁。但跨越多个大洲在高海拔出现异常复杂巨石建造的格局,与文集框架相一致,即便具体识认无法被确凿确立。

水下设施甚至更难识认,因为海底考古发展远不如陆上考古成熟,而海床保存下来的远比大陆表面少。近几十年报告了各种水下结构——所谓与那国岛纪念物(日本附近)、印度与巴哈马沿岸的各种水下巨石特征、古巴及其他地区有争议的遗址。其中大多数被主流考古学作为自然地质特征加以争议或拒绝。文集并未对任何具体识认作出承诺。文集所指出的是:资料的框架预测此时期会有相当多的水下设施;一小部分候选遗址已由另类考古学提出;此问题应得到比主流海洋考古学通常给予的更严肃的调查。

冲突的具体阶段并未在资料中详细描述,但更广泛的形态可以重构。议会的军事力量会向蛇形派系的阵地推进,试图识别并摧毁隐蔽的设施。蛇形派系会以其所拥有的任何军事技术进行防御——可能包括曾用于对付洪水前文明的同一类武器,如今反过来对付议会自己的力量。冲突会发生在多个战区:地球轨道上议会航天器与蛇形派系在地球轨道上部署的任何防御系统之间的轨道战斗;在具体设施所在地的地面交战;如果蛇形派系在地球之外的太阳系内还有基地,可能在那里也有交战。冲突在范围、持续时间和毁灭强度上都会相当可观。留存在全球神话中的痕迹——保存在跨文化各种{% wiki(slug="theomachy") %}众神之战{% end %}传统中的众神之间的战争——正是这些交战的文化记忆。

按资料的叙述,冲突的解决是最终的,但对流亡造物者来说并非灾难性的。所多玛与蛾摩拉的段落——文集会在金牛座一章处理——以一句话顺带提供了答案:*“流亡造物者获得赦免,并被允许回到其原行星,在那里为他们辉煌的受造之物陈情。”*赦免意味着曾有一桩需要被赦免的过犯,以及犯过的双方已达成一种解决——在其中,过犯不再被追诉。流亡造物者获得了赦免——而不是被毁灭。他们被允许回去——而不是被消灭。他们陈请其情——他们活下来去陈请。其含义是清楚的。到双子座晚期已经爆发的冲突——议会曾想毁灭流亡造物者——并未以议会所求的毁灭收场。它反而以某种谈判式的结果结束——一项推测是双方所接受条件下的赦免,允许他们回到其原文明,而地球项目在修改后的安排下继续进行。

那些修改后的安排是什么,是文集将在接下来的章节中处理的问题。在双子座的语境中可以说的是:战争确实发生过,它是真实的;流亡造物者在“推翻议会”的意义上并未获胜;但他们在“被消灭”的意义上也并未失败。基于可用证据,结果更接近于一种僵持,并归于一项政治安排:流亡造物者获得宽宥,以接受对其后续行动的某些限制为交换;议会反过来接受人类文明的延续,以及保存了它的联盟的存在地位。流亡造物者回到其原文明陈请其情,但他们并未取消与其人类伙伴所立的约。即使其正式政治处境发生变化,联盟依然存续。

希伯来圣经以零碎但可辨认的形式保存了对这场冲突的记忆。资料所引的{% libref(book="isaiah", chapter=27, verse=1) %}以赛亚书 27:1{% end %}段落——*“到那日,耶和华必用他刚硬有力的大刀,刑罚{% wiki(slug="leviathan") %}利维坦{% end %},就是那快行的蛇,刑罚利维坦,就是那曲行的蛇;并杀海中的大鱼。”*——在这种解读下,正是以色列先知对该战争的记忆,被向前投射到末世论的语言之中。这段经文中的“利维坦”与“大鱼”并不是抽象意义上的神话怪兽。它们正是那些藏匿于大海中、议会曾对其采取行动并由议会力量追捕的流亡造物者本身。“到那日”杀戮将发生的应许,是对该冲突的先知记忆,以预期未来审判的形式保存下来,但反映的是一项在雷尔派的时间线上早已发生于远古过去的事件。以这种框架来读,希伯来文本所包含的军事叙事比常规解读所识别的要多得多。本章将在第十三节回到具体的希伯来词汇。

## XI. 万名所记的同一记忆

母星议会与地球上的流亡造物者—人类联盟之间的冲突,在 Wheel of Heaven 的解读下,是构成跨文化神话主题——即学者所称的“众神之战”(Theomachy)——背后的历史事件。几乎每一种保存有相当规模神话的主要文化,都在其中某处保存了一种众神之间战争的记忆。这种格局过于广泛,不可能是巧合,而文集的框架为这种否则就像谜一般的趋同提供了一个单一的解释。

**希腊的泰坦之战与巨人之战。**希腊传统保存了两种各自不同但相关的众神之战叙事。泰坦之战,最完整地记载于赫西俄德的《神谱》(约公元前 700 年)中,描述了较老一代的诸神——以克洛诺斯为首的泰坦——与以宙斯为首的较年轻一代奥林匹斯诸神之间的战争。泰坦曾在先前时代统治宇宙。宙斯由母亲瑞亚秘密抚养——此前克洛诺斯曾试图把他与其他孩子一同吞下——最终推翻了父亲,并带领奥林匹斯一代向泰坦发动战争。战争持续了十年。奥林匹斯诸神最终与百手巨人(赫卡同刻瑞斯)和独眼巨人结盟,击败了泰坦,并把他们囚禁于地狱最深处塔尔塔罗斯。

与文集解读的结构性对应是相当大的。同一神圣阶层的两个派系,围绕宇宙应如何治理的根本问题进行冲突。一代较老者,在冲突中被取代并被囚禁于大地之下。一代较年轻者,获胜并确立新的宇宙秩序。冲突的时长被描述为相当可观。出现不同寻常的盟友(百手巨人——他们各有一百只手;独眼巨人——他们有锻造的技艺;在文集框架下,他们可能代表具有独特技术能力的特定蛇形派系子团体)。败者最终被囚禁——这与蛇形派系撤入隐蔽设施、并通过谈判式解决最终在政治上被边缘化在结构上对应。

巨人之战,是希腊传统中相关的另一条线索,记录于多种资料(包括公元 1-2 世纪的阿波罗多洛斯之作)中,描述了奥林匹斯诸神与巨人——具有巨大力量的大地所生的存在者——之间另一场平行的冲突。巨人被描述为盖亚(大地)与乌拉诺斯(天空)的后代,从乌拉诺斯被克洛诺斯阉割时落到地上的血中所生。他们起来反抗奥林匹斯诸神,试图推翻宇宙秩序。奥林匹斯诸神在英雄赫拉克勒斯的帮助下击败了他们——根据预言,巨人的彻底死亡要求凡人之血。文集对此叙事的解读会把巨人与圣经文本中的 *Nephilim* 等同起来,与 *benei ha-Elohim* 和人类女性之间的混血后代——即“声名远扬之人”——等同起来,这些人在洪水前及洪水后早期的存在,是母星议会警觉的直接原因。巨人之战保存了议会最终对混血谱系采取军事行动的记忆,而巨人的击败需要“凡人之血”反映了人类伙伴对该冲突的参与。

**北欧的阿萨-华纳之战。**北欧传统在两个派系的诸神之间的战争中保存了一种独特的众神之战叙事:阿萨神族(与天空、战争、秩序相关联)与华纳神族(与丰饶、魔法、大地相关联)。这一冲突记载于《女预言者的预言》中,并在《海姆斯克林格拉》及其他萨迦中得到详述,其结束并不是任何一方的全胜,而是通过谈判达成的和解以及人质的交换。华纳神族派尼约德、弗雷、芙蕾雅去阿萨神族中生活;阿萨神族派霍尼尔、密米尔去华纳神族中生活。该和解确立了两个派系之间的和平,使统一的北欧万神殿能够共同面对最终的宇宙威胁。

对雷尔派叙事的结构性对应是惊人的。同一神圣阶层的两个派系,围绕根本问题进行冲突,通过最终经过谈判的安排解决,而非任何一方的完全胜利。作为安排正式机制的人质交换,与文集框架对双子座冲突解决所预测的那种谈判式结果相对应。双方在和解后的延续——以及人员的交换允许双方相互监督与影响——与文集所读出的、和解后的安排保留议会权威与联盟地位、而非消除任何一方的解读一致。

**印度教的提婆-阿修罗冲突。**印度宇宙论传统保存了任何单一神话语料库中最广泛的众神之战材料。在 *devas*(诸神,与既有宇宙秩序、光明、诸天相关联)与 *asuras*(常被译为“恶魔”,但更准确地是“强力者”或“另一类神”,与较老一代以及与地下或海洋领域相关联)之间反复出现的冲突,贯穿整个吠陀、往世书与史诗文学。两类同源——他们都是迦叶波及其多位妻子的子嗣——但被锁在反复对立的循环中。

《摩诃婆罗多》、《罗摩衍那》以及各部往世书中的具体叙事,描述了冲突达到军事表现的具体战斗。Samudra Manthan(搅拌乳海)情节——*devas* 与 *asuras* 合作以获得 *amrita*(不死灵药),却随后为占有它而争斗——与文集所提出的、在“两派之间因长寿技术之取得而对峙”的框架相对应。因陀罗(*deva* 之王)与各 *asura* 领袖之间的各次战争——弗利特拉、波利、希兰耶卡西普及其他——代表着两派之间长期冲突中的具体时刻。*asuras* 常被描述为较老一代,被刻画为从恩典中堕落,并反对由 *devas* 所代表的既有宇宙秩序。其与文集框架的结构对应是直接的——较老的“堕落”派系反对既有秩序,两派同源。

印度教传统的独特贡献在于,它把冲突当作循环的而非单一的来处理。*deva-asura* 的冲突跨越宇宙诸时代而反复出现,每一 *yuga* 都带来这种底层张力的新形态。文集的框架会把这种循环性保存读作对实际历史冲突多个阶段的反映(最初的伊甸披露与驱逐、洪水后的巴别塔干预、真正的天界之战,以及可能后续时代的进一步冲突),所有这些都在印度教传统中作为同一神话格局的变奏而保存下来。

**埃及的荷鲁斯-赛特冲突。**埃及传统在荷鲁斯与赛特之间的长期冲突中保存了众神之战——较老与较年轻的神,一方与既有的宇宙秩序相关联,另一方与扰乱的原则相关联。按某些解读,赛特是较老的人物,被荷鲁斯的兴起以及奥西里斯循环所取代,从其原先的显赫地位被替下。神话叙事描述了奥西里斯被赛特谋杀、奥西里斯被伊西斯复活、荷鲁斯(奥西里斯之子)与赛特就埃及王权的冲突,以及最终通过神圣裁决达成的解决——该裁决确立荷鲁斯为合法之王,而赛特保留对沙漠与外邦的统治。

文集的解读会把赛特等同于在和解后政治上被边缘化的蛇形派系。赛特被降级到沙漠与外邦——埃及文明耕作中心之外的边缘地区——对应于蛇形派系在战后撤入山地与水下设施。赛特在埃及叙事中并未被毁灭;他延续下来、保留某些权能,但被排除在他原本所主张的宇宙核心之外。这正是文集框架所预测的和解政治形态:蛇形派系幸存下来,但接受被边缘化,主导政治权威则回到议会对应的人物手中。

**美索不达米亚的马尔杜克-提阿玛特冲突。**巴比伦的《{% libref(book="enuma-elish") %}埃努玛·埃利什{% end %}》{{ cite(id="5") }},写成于公元前第二千纪后期,但取材于明显更古老的苏美尔素材,把宇宙的创造描述为年轻风暴神马尔杜克与原初盐海龙女神提阿玛特之间战争的产物。马尔杜克杀死提阿玛特,并从她的躯体中塑造宇宙。提阿玛特的形象——一种蛇形的海中生物,被较年轻一代诸神击败——与希伯来的利维坦明显相似,而两个叙事之间的结构对应反映了——学者们如今普遍同意——一个共同的近东源头,美索不达米亚与希伯来传统都源自其中。

文集的解读把提阿玛特等同于在其海居构型下的蛇形派系——即在战争中撤入水下设施的、路西法所领导的子团体。提阿玛特在美索不达米亚叙事中的“原初”特征,反映了蛇形派系较老的地位(他们在年代上先于马尔杜克形象所代表的、洪水后议会对齐的权威)。提阿玛特的蛇龙特征保存了希伯来传统对路西法派系所使用的同一意象(*nachash*、*liwyatan*、*tannin*)。提阿玛特被马尔杜克击败、马尔杜克从她的躯体中塑造宇宙,代表着冲突后神圣秩序的重组——蛇形派系的失败被正式化进新宇宙的结构之中。

**美索美利坚的羽蛇神-特斯卡特利波卡冲突。**在美索美利坚传统中,羽蛇神(羽蛇,常与文明、知识、风、晨星相关联)与特斯卡特利波卡(冒烟的镜子,常与夜、美洲虎、巫术、既有政治秩序相关联)之间的冲突保存了一种相似的众神之战格局。羽蛇神有时被刻画为被取代的或将归来的人物——被特斯卡特利波卡的诡计驱入流放、并被预言将归来以重夺其本位的神。羽蛇神形象常与人类的提升与文明的赐予相关联,他正是被流放或被击败的那一位,在结构上与文集解读中的路西法派系形象相对应。

蛇的关联尤其值得注意。羽蛇神明确就是*羽蛇*——一种有翼的、似蛇的存在者,其图像与希伯来意象的 *nachash*/*liwyatan*/*tannin* 复合体相吻合。羽毛,在文集的解读下,可能代表蛇形派系的飞行能力(他们的航天器、他们在行星与轨道之间自由移动的能力)。蛇身代表所有跨文化蛇形派系传统所保存的同一本质特征。羽蛇神的流放与被预言的归来,与蛇形派系的谈判式边缘化、以及人类长期对其最终再现的期待相对应。

**波利尼西亚、凯尔特、中国及其他传统。**这一名单可以大大延伸。波利尼西亚传统保存了较老的海洋诸神与较新的天空诸神之间的冲突。凯尔特对图瓦哈·德·达南族抵达爱尔兰并取代早期神圣居民的记忆,以及图瓦哈本身最终被人类米利提人取代,保存了相继神圣世代之间冲突的结构性格局。中国神话对天庭官僚与各种反叛神圣形象之间冲突的叙述——包括玉帝与各挑战者形象之间的冲突、保存在《山海经》中的宇宙战斗,以及道教对宇宙战争的各种记述——都保存了众神之战的格局。斯拉夫传统中佩伦(天空之神)与维列斯(地下蛇形之神)之间冲突的传统,以特别的清晰度保存了印欧的众神之战结构。横跨多种文化的非洲传统,以各种形式保存了神圣派系之间的冲突。

格局是全球性的。几乎每一种保存有神话的文化,都在其中保存了一段神圣派系之间战争的记忆——较老与较年轻、既有与反叛、那些会保留既有秩序者与那些会改变它者。这一主题的跨文化分布,并不是文化在大陆之间扩散的结果——因为在人类历史的大部分时间里,这些大陆都被洪水后的海洋屏障彼此隔离。它是共同记忆的结果——由每个谱系按其自身的方式,从这些谱系尚未在地理上分隔的时期保存下来,而当时这场冲突是整个人类都能观察到的当下现实。

文集的框架为这种趋同提供了单一的解释。有一场战争。它发生在双子座时代。它使母星议会与地球上的流亡造物者—人类联盟相对立。流亡造物者——与向人类披露禁忌知识、把人类提升到与其造物者匹敌、为保存受造人类免遭议会的毁灭令而行动相关联——是这样的形象,他们在神话记忆中作为较老的神、反叛的神、大地、海洋或地下世界的神而存续下来——那些败者、被取代者、被流放者,但其记忆被他们人类伙伴的后裔文化跨越数千年保存下来。议会——与既有秩序以及对反叛派系的压制相关联——存续为既有的万神殿——奥林匹斯诸神、阿萨神族、*devas*、马尔杜克类形象——他们击败或约束了较老的一代,并在其缺席中重组了宇宙秩序。

在文集的解读下,这是同一事件以许多名字被记下的。神话并不是发明。它们是证词,被长期传递所扭曲,但保存了所发生之事的结构轮廓。我们自己的文明,已经失去这些故事的运作含义,并把它们简化为当代贬义中的“神话”——已经失去识别这一点的能力:基本上,地球上每一种文化都保存了对同一历史事件的记忆。文集在本章及更广框架中的任务,是恢复这种识别。

## XII. 双子座的科学

资料以大致轮廓告诉我们双子座期间发生了什么。事件的技术内容——武器会是什么样的、地质后果实际上看起来会如何、当代灾变论研究积累了什么、遗传保存行动会需要什么——与前面各章一样,可在现有科学中获得,尽管必须从多个专业文献中拼装起来。

本节按八个子节展开。第一,灾变事件的地质学。第二,大陆漂移与压缩时间线的问题。第三,主流与另类解读中的石油地质。第四,作为当代研究语境的全新世灾变事件。第五,洪水神话的全球分布及其蕴意。第六,人口瓶颈的遗传证据。第七,作为遗传载荷之当代类比的现代克隆与基因工程。第八,通向我们自身时刻的延伸。

### XII.1. 灾变事件的地质学

现代地质学认可灾变事件作为在相当大尺度上运作的真实地质机制。六千六百万年前的 K-Pg 撞击事件,被归因于尤卡坦半岛上的奇克苏卢布陨石坑,如今已被确立为一次真实的全球性灾变事件——该撞击在全球范围内沉积了标志 K-Pg 界限的富铱层,产生了在全球海洋中传播的巨海啸,通过抛射物羽流点燃了大陆规模的野火,并使全球气候改变到足以驱动结束中生代的物种大灭绝。1908 年的通古斯事件——一个较小的宇宙物体在西伯利亚上空大气中爆炸——夷平了约 2,000 平方千米的森林,并产生了在全球范围被探测到的大气与地震效应。2013 年的车里雅宾斯克陨石,在数百个行车记录仪和科学仪器上被记录下来,演示了即便相对适中的宇宙物体在进入地球大气时也能产生何种灾变能量释放。

除了宇宙撞击之外,地质学还认可各种其他灾变机制:大型火山喷发(约 74,000 年前的多巴喷发、黄石超级喷发的潜在可能性、历史上的坦博拉与喀拉喀托事件)、灾变性山崩(产生区域性巨海啸的各种海底滑坡)、灾变性洪水(末次冰期末雕刻水道荒地的米苏拉洪水、约公元前 5600 年的黑海洪水、与冰川溃决事件相关的各种 jökulhlaups)。这些都是真实的地质现象,得到主流科学的认可,表明地球表面既由灾变过程塑造,也由严格均变论传统所强调的渐进过程塑造。

文集的框架要求把这种灾变论认可延伸到比任何已认可主流灾变都明显更大的事件。在文集的解读下,双子座洪水事件所涉及的能量释放规模会超过 K-Pg 撞击——足以击碎一片超大陆、改变行星的旋转动力学、产生大陆尺度的海啸,并在行星规模上散布放射性污染。这是一个比主流地质学所认可的更可观的事件,但属于程度而非种类上的不同。所涉机制(宇宙尺度的能量释放、大气与海洋扰动、地质重组、生物大灭绝)正是主流地质学接受的、在较小规模事件中运作的同一些机制。文集的主张是:一次这种量级的事件发生在大约公元前 6,690 年;主流科学目前并不在这一日期上认可一次这种量级的事件。

文集解读的证据基础是部分的。本章前面讨论的石油环观察是一项证据。如果超大陆破裂的地质特征保存在地质记录中,那也会是其他证据。如果保存在各大陆表面地貌中的灾难性洪水特征能与双子座事件而非其他灾难性洪水事件具体挂钩,那也会是其他证据。文集并不主张已经为双子座事件汇集了决定性证据。它主张的是:框架的预测至少与可得的地质证据相一致,而框架的其他强项(对圣经与神话材料的整合、对石油环格局的预测、与各种当代灾变论研究项目的兼容)提供了独立的支持。

### XII.2. 大陆漂移与压缩的时间线

大陆漂移的主流地质共识,按现代标准是确立良好且证据支持充分的。阿尔弗雷德·魏格纳在 1912 年{{ cite(id="10") }}提出该概念,曾遭受相当大的主流反对;到二十世纪中期,通过古地磁技术的发展与中洋脊扩张的发现,证据基础得到大幅扩展;现代板块构造理论在 1960 年代得到巩固,并如今是所有地球科学的基础。在主流框架下,各大洲以每年几厘米的速率移动,由地幔对流驱动,其当前构型自各种早期超大陆构型经过数亿年发展而来。

主流框架的证据包括:

- **古地磁特征。**火山岩中的磁性矿物保存了岩石凝固时地球磁场的取向。跨越地质时间,这些特征显示各大洲已相对于磁极旋转和平移,其格局与主流理论所预测的渐进运动相一致。

- **化石分布。**同种化石在如今被海洋分隔的大陆上被发现——例如水龙兽在南美洲、非洲、印度与南极洲之间的分布,要求这些陆块一度相连。该格局与主流对历史超大陆构型的重建相一致。

- **大陆边缘的吻合。**各大陆的形状,特别是南美洲东岸与非洲西岸,以一种除了由历史性分离之外难以解释的精度彼此吻合。

- **中洋脊扩张。**对中洋脊海底扩张速率的直接测量显示,新的海洋地壳以每年几厘米的速率被产生,并在大陆边缘相应地发生俯冲。

- **GPS 测量。**使用 GPS 技术对当前大陆运动的直接测量,确认各大洲正以主流理论所预测的速率移动。

这是一个相当实质性的证据基础。文集的框架——把大陆漂移压缩为一次灾变事件、随后是持续残余运动——必须在其自身框架内解释这一证据。文集的解读会以不同方式解释每一条证据:

- **古地磁特征**会反映双子座事件期间及之后,大陆碎片的快速重定向——跨地质时间显出的渐进变化,被解读为以各阶段被捕捉到的事件后重定向。

- **化石分布**会反映洪水前生物圈在超大陆上的连通性,以及洪水后的重新播种格局——如今被分开的大陆上的同种化石,反映破裂前的统一,而不是数百万年间的渐进大陆运动。

- **大陆边缘的吻合**会直接反映洪水前的盘古构型——边缘吻合是因为各大洲最近曾相连,而不是因为它们在数亿年间慢慢分离。

- **中洋脊扩张**与 **GPS 测量**会反映来自原初双子座位移的持续残余运动——当前速率代表着在该灾变事件之时开始的运动的减速阶段。

文集并不假装这种另类解读解决了每一个实证细节。压缩的时间线要求对地质证据进行实质性重新解读——在其主流解读下,这些证据支持长时间尺度。文集如实承认这一冲突。它所主张的是:另类解读在内部是一致的;它作出主流框架并不作出的具体预测(如石油环);文集框架更广的可信度倚靠于他处的趋同,而不是对地质问题的完全解决。

灾变论与均变论对地质证据的解读之间的冲突,在某些方面是文集与主流科学之间最尖锐的单项分歧。文集的框架是灾变论框架。主流地质学——除一些晚近例外与软化之外——是一种均变论框架。灾变论的读法最终能否抵御累积的主流证据,是文集在本章无法解决的问题。本章如实指出了这一冲突,把文集的读法呈现为与资料叙述一致的框架,并承认在这一点上,文集更广的可信度可能不得不倚靠其他强项,而不是地质学上的最终证立。

### XII.3. 主流与另类解读中的石油地质

主流的石油形成理论——生物成因理论——认为,原油是由有机物质(主要是海洋浮游生物与藻类)在数百万年地质时间尺度上缓慢厌氧分解所形成。有机物质在沉积盆地中堆积,被先后多层沉积物所埋藏,在热与压力作用下发生化学转化,经过时间转化为我们作为石油来开采的烃类混合物。生物成因理论得到大量证据支持,包括在石油中发现的化学特征(生物标志物)与原始有机物质化学相吻合、石油矿藏与古沉积盆地的地理相关性,以及基于岩石地质史预测石油矿藏所在的勘探成功。

存在一种少数派的另类理论——非生物成因或无机成因理论——主要与自二十世纪中期以降的俄罗斯与乌克兰地质学家(库德里亚夫采夫、波尔菲里耶夫等人,以及包括托马斯·戈尔德在内的更晚近倡导者)相关联。非生物成因理论认为,烃类不是从生物物质形成,而是来自深部地幔中的非生物过程,石油通过地壳裂缝向上迁移并积累在圈闭结构中。非生物成因理论在预测石油位置与说明原油化学特征方面,远不如生物成因理论成功,但它仍保有一些追随者,并曾被用于支持某些另类地质框架。

文集的解读更接近于第三种选项,与标准生物成因理论和非生物成因另类理论都不同。文集的框架认为:石油是在双子座事件之时由生物物质的灾难性埋藏所形成——其起源是生物的,这与标准理论一致——但形成的时间尺度被压缩到双子座灾难之后的时段,而不是分布在数百万年间。灾难性埋藏的解读会预测:

- **生物化学特征**,与主流生物成因理论一致,因为源材料确实是生物的
- **与撞击地点的地理相关性**,与前面讨论的石油环观察一致
- **压缩的形成时间尺度**——从生物物质到烃类的化学转化,在撞击后埋藏的极端条件下发生于数千年而非数百万年

主流石油地质学并未采纳灾难性埋藏的解读。主流理论认为,化学转化需要地质时间尺度和渐进沉积过程所提供的缓慢埋藏条件。文集并不主张已在细节上反驳了主流理论。文集所主张的是:灾难性埋藏的读法与石油环格局一致(主流理论并不预测此点);化学反应有可能在灾难性撞击之后的极端条件下以压缩时间尺度进行;框架更广的证据基础为认真对待该另类解读提供了独立支持。

这是有争议的地带,文集以适当的认识谨慎呈现其解读。石油环观察令人瞩目,但其底层研究并不主流。石油形成的灾难性埋藏机制是合理的,但并未确立。文集的框架与该另类解读一致,但并不依赖它来确立。读者被邀请去考虑该另类,而不被命令接受它。

### XII.4. 全新世的灾变事件

当代灾变论研究项目已为地球最近(全新世)过去的各种灾变事件积累了大量证据。巨蟹座一章把新仙女木撞击假说作为这样一个项目引入;在双子座一章可以将该讨论延伸到与文集框架相关的更广灾变论研究。

新仙女木事件本身,年代为距今约 12,900–11,700 年前,在文集的年代顺序上位于狮子座晚期/巨蟹座早期的过渡。标准解释涉及冰川融水排放导致北大西洋海洋环流的扰乱;另类的新仙女木撞击假说则将事件归因于宇宙撞击或大气空爆。自 2007 年 Firestone 等的原始论文以来,撞击假说的证据已大量积累——在多个大陆的遗址中记录到的新仙女木边界铂尖峰、暗示高温事件的碳球粒和纳米金刚石、与大型撞击产生的热事件相一致的广泛火灾证据、与新仙女木时序相一致的巨型动物群灭绝和文化扰动。

8.2 千年事件,年代约为公元前 6200 年——接近文集所定的洪水日期——是一次持续约 160 年的突发气候变冷,中东及其他地区发生严重干旱。标准解释涉及冰川阿加西湖向北大西洋的灾难性融水排放,扰乱海洋环流并造成变冷。8.2 千年事件在气候记录中得到良好确立(保存在格陵兰冰芯、北半球各处湖泊沉积物以及各种代用记录中)。文集的框架会指出:该事件的时序与文集的洪水日期大致一致,尽管常规理解中 8.2 千年事件的具体量级,比文集框架所要求的双子座灾难要小得多。

黑海洪水假说由威廉·瑞安与沃尔特·皮特曼于 1997 年提出,认为黑海在大约公元前 5600 年因地中海海平面上升突破博斯普鲁斯海峡,并淹没了原本远低于现代海平面的淡水湖,从而发生灾难性洪水。该假说得到黑海沉积物证据的支持,显示在大约所提出的日期由淡水转为海洋条件。瑞安与皮特曼论证认为,黑海洪水是引起古代近东各种洪水传统的历史事件。该假说受到一些海洋地质学家的争议——他们论证该转变比灾变性版本所要求的更为渐进——但该地区在此时期发生过相当大的洪水事件这一基本轮廓,得到广泛接受。文集的框架会指出:即便在灾变论的读法下,黑海洪水也太小、太局部,不足以是双子座洪水事件本身,但它可能是文集框架在此时期所定的更大事件之后,所引发的连串后果之一。

前面提到的兰德尔·卡尔森的工作,提供了把各种发现整合起来的最广灾变论框架。卡尔森的论点是:地球表面保存了大量证据,表明最近过去(约过去 15,000 年)发生过灾变事件,而主流地质学倾向于把这些归因于渐进过程。华盛顿东部的水道荒地(由末次冰期末的米苏拉洪水所形成)、跨全新世记录到的各种快速景观变化、考古记录中可见的文化扰动——所有这些,在卡尔森的读法下,都构成对最近地球史更灾变论解读的证据。卡尔森的工作不是主流共识,但它代表着一项实质性的当代研究项目,已积累了大量证据,并直接关系到文集叙述所要求的那种框架。

文集并不依赖任何特定的灾变论研究项目来支撑其更广框架。它所指出的是:当代研究正朝着与文集框架日益相容的方向前进——认可早期主流地质学未曾允许的尺度与日期上的灾变事件、为最近过去的撞击事件积累证据、认真对待十九世纪地质学的严格均变论曾边缘化的灾变论传统。研究的轨迹对文集框架是有利的,即便目前没有任何单一当代研究项目证立了框架所作的具体主张。

### XII.5. 洪水神话研究

洪水叙事的全球分布是比较神话学中记录最完整的格局之一。超过 200 种不同的文化保存了某种形式的洪水传统,分布在每一大洲与每一主要文化区。这些传统从充分发展的美索不达米亚与圣经叙事(挪亚、乌特纳皮什提姆{{ cite(id="6") }}、{% libref(book="atrahasis") %}阿特拉哈西斯{% end %}{{ cite(id="4") }}、吉乌苏德拉{{ cite(id="7") }})延伸到其他各种文化中较为压缩的洪水提及(希腊的丢卡利翁叙事、阿兹特克的塔塔与娜娜、印度的摩奴、各种美洲原住民洪水传统、波利尼西亚洪水神话、澳大利亚原住民的彩虹蛇传统等等)。

主流学术对这种全球分布的解释,典型地结合两个因素:对地区性洪水的独立本地记忆(任何人类人口在漫长文化发展过程中都会经历的灾难性河岸与沿海洪水)、以及来自美索不达米亚源头的文化扩散(圣经叙事继承自更早的美索不达米亚《阿特拉哈西斯》与《吉尔伽美什》叙事,各种其他传统通过贸易与文化接触继承)。主流观点是:没有发生过单一的全球性洪水事件;洪水传统的全球分布反映了本地洪水的普遍性,结合了美索不达米亚叙事复合体的影响。

乔治·史密斯在 1872 年{{ cite(id="11") }}的工作——在尼尼微阿淑尔巴尼帕图书馆出土的泥板中重新发现了巴比伦洪水叙事——表明圣经洪水叙事在更古老的美索不达米亚资料中有明确的先驱。其后又发现了更多洪水叙事——《阿特拉哈西斯》史诗、含有吉乌苏德拉故事的苏美尔《埃利都创世记》——已经大量记录了美索不达米亚洪水传统复合体。圣经叙事现在被广泛认为是借鉴自这些更古老资料,而不是代表独立的启示,尽管圣经版本具体的神学框架仍然独特。

文集的解读为全球分布提供了另一种解释。文集的框架不是把分布归因于独立本地记忆加扩散,而是认为:跨文化的洪水传统保存着对一次实际全球事件——本章一直在描述的双子座灾难——的共同记忆。每种文化以自己的方式保存这一记忆,具体细节反映了自身视角与后续传递史。美索不达米亚叙事复合体保存这一记忆最完整,因为洪水后紧接时期的美索不达米亚文明,在地理上最接近建造方舟并以最完整记录度过该事件的伊甸谱系。其他文化则以更压缩或更被转化的形式保存这一记忆,反映了更远的传递距离和它们传统的文化独特性。

文集的解读并不要求拒绝整个主流学术框架。文化扩散确实发生过,跨文化的洪水叙事既显示共同的结构性特征(保留原初事件),又显示文化特定的发挥(反映扩散与本地发展)。文集框架所增加的,是产生结构共性的底层历史事件——一个在主流框架(由于缺乏文集的更广宇宙论框架)无从识别、因而不得不归因于本地记忆与扩散组合的事件。

### XII.6. 人口瓶颈遗传学

文集的框架预测在洪水之时存在一次重大人口瓶颈,把该时间放在大约公元前 6,690 年(双子座时代的开端)。按这种解读,洪水后的人类人口会源自一个小奠基者群体——挪亚一家八口,加上资料提到从其他六个谱系中保存并送回的代表。这是一次严重的瓶颈,会在所有后续人类人口的遗传记录中留下可检测的特征:遗传多样性在瓶颈事件之时降低,随着人口随后扩张多样性逐步恢复。

现代群体遗传学已经识别出人类演化史中的几次明显瓶颈。最有名的是多巴瓶颈假说,该假说提出大约 74,000 年前印度尼西亚多巴山的喷发,在人类人口中造成了一次近乎灭绝的事件。多巴假说已被更近的研究大幅削弱,但更广的“人类史前瓶颈”问题仍然活跃。对线粒体 DNA 与 Y 染色体谱系的各种分析,识别出在不同时期的合并事件,其中一些被解读为瓶颈或人口减少的证据。

是否在大约公元前 6,690 年的瓶颈特征可在现代人类遗传记录中被检测到这一具体问题,并未成为大量研究的焦点,部分是因为文集的框架并不是驱动遗传研究优先事项的主流科学框架的一部分。文集框架所预测的是:未来的遗传研究应能识别出在大约这一日期的瓶颈特征,现代人类人口源自一个生活在此时的小奠基者群体。当前分析中没有这样一个清晰特征,要么是反对该框架的证据,要么是现有分析未带着这一具体问题来进行的证据。

更近一项有趣的发现与这一问题相关。Hu、Hubisz 及同事于 2023 年在《科学》上发表的研究,使用了一种新的统计方法(FitCoal)来分析人类遗传多样性,并得出结论:祖先人类人口在大约 90 万年前经历了一次严重瓶颈,人口减少到约 1,280 个繁殖个体,持续约 117,000 年后才恢复。该研究受到使用不同方法的其他研究者的争议,但它代表了对人类人口瓶颈问题的实质性重新分析。文集的框架会指出:Hu 等所开创的方法学原则上可被应用于检测文集框架所预测日期的更近瓶颈,但截至撰写之时尚未有此类分析发表。

文集并不主张当前遗传证据证立了框架对洪水后瓶颈的预测。它所主张的是:该框架作出了一个具体的可检验预测(在约公元前 6,690 年的人口瓶颈),且方法学现在已经存在,若研究者愿意承担所需的具体分析,即可对此类瓶颈进行检验。即使目前还没有任何证据证立它,该框架作为预测仍然活跃。

### XII.7. 现代克隆与基因工程

按文集的解读,方舟的遗传载荷要求相当成熟的保存与再生技术。现代克隆与基因工程,虽然远不及资料框架所要求的成熟,但已朝着类似该框架所预设的技术方向作出了切实进展。

多莉绵羊,由伊恩·维尔穆特及其团队于 1996 年在苏格兰罗斯林研究所克隆,首次证明完整的哺乳动物可以从一位成年供体的单个体细胞中被再生。这一成就在 1997 年公布,改变了对细胞生物学以及从被保存细胞再生生物体可能性的理解。随后的克隆成就涵盖了广泛的哺乳动物(猫、狗、马、牛、猪、鹿、猴),以及各种非哺乳动物。该技术虽仍需大量改进,在不同物种间成功率不一,但已从推测的领域走入常规生物实践的领域。

更近一些,干细胞生物学的进展,演示了从被保存材料再生生物或类生物体的额外途径。诱导多能干细胞(iPSCs)技术由山中伸弥及同事开发,并因 2012 年诺贝尔生理学或医学奖得到认可,它允许成体细胞被重新编程为多能干细胞,然后分化为任何细胞类型。这为从被保存的细胞物质中再生生物或组织提供了一条另类路径,可能不存在传统克隆的技术局限。

基于 CRISPR 的基因编辑,自约 2012 年以来发展起来,极大地扩展了能对遗传物质进行修改的精度。CRISPR 技术已在研究中被用于广泛的应用,包括旨在恢复已灭绝物种的去灭绝项目(用于复活猛犸象的更新世公园项目、各种旅鸽与塔斯马尼亚虎项目)。去灭绝技术虽仍处于早期研究阶段,但展示出朝向资料框架所要求的、洪水后重新播种所需再生能力的轨迹。

前面提到的圣迭戈动物园野生动物联盟的“冷冻动物园”,自 1972 年以来一直从濒危物种采集细胞样本。近年来,该设施为黑足鼬和普氏野马的克隆提供了细胞——这些物种恢复项目展示了从被保存细胞物质中再生生物的实际可行性,虽然规模小,但证明了该方法的技术可行性。斯瓦尔巴全球种子库为植物物种提供了相应的能力,保存了超过一百万个种子样本,以防灾难性损失。

朝向资料框架就方舟行动所要求的水平的轨迹是可见的。我们尚未达到在行星规模上从被保存细胞物质中再生数百万物种的水平。我们正处在从被保存细胞再生单个物种的水平,且生物样本库基础设施不断壮大,技术能力不断增长。这一轨迹若延续,最终将产生接近资料框架就联盟方舟行动所预设的能力。

### XII.8. 通向我们自身时刻的延伸

最后一个观察,按前面各章建立的格局,结束本科学部分。

双子座的工作所要求的能力——文明尺度的遗传保存、能维持机组数月的轨道居所、从被保存样本对生物体的细胞再生、能击碎大陆的地球物理尺度武器、对核之后地表的环境修复、星际飞船的建造——正是我们自己文明如今开始在各组件层面接近的能力。整合尚不可见。组件却已可见。

现代航天计划已实现持续机组的轨道居所(国际空间站自 2000 年 11 月以来一直有人员驻留,单个宇航员的轮换可长达一年)。月球居住计划(阿尔忒弥斯计划、各种私人月球项目)和火星殖民计划(SpaceX 的火星架构、各种科学火星任务概念)表明朝向方舟行动所要求的、长期在地球之外居住的轨迹。当代工作的规模比方舟小得多,但底层能力正在被开发。

正如刚才所讨论的,现代生物保存已经产生了基础设施(冷冻动物园、种子库、生物样本库),这正是方舟行动在更大规模上所要求的。朝向更全面生物保存的轨迹,与不断增长的克隆和基因工程能力相结合,指向最终对方舟所要求那种全面物种档案的能力。

现代关于生存风险的讨论,尤其是与更广效益利他主义社区以及托比·奥德、尼克·博斯特罗姆等研究者相关联的讨论,已把文明存续问题带入当代政策话语。承认先进文明面对“如何管理可能威胁自身存在的技术”——核武器、生物武器、人工智能、环境扰动——的问题,如今已是主流政策讨论的一部分,这在不到一代人之前还远非如此。双子座的格局——一个先进文明出于对另一个先进文明进步的恐惧,对其部署灾难性武器——正是当代生存风险思考所关注的那种情景。文集的框架提供了当代思考视作未来可能性的格局之具体历史案例。

技术进步与文明存续之间的当代张力,在文集框架下,正是双子座事件以其独特方式所解决的同一张力。洪水前的文明越过了母星议会认为安全的临界点,议会部署了灾难性武器以消除该威胁。我们自己的文明如今在各种维度上,正在接近过去几代人会认为极度危险的技术能力水平。双子座事件表明,处于我们这种发展水平的文明面对一些决定:是否继续前进、是否接受对前进的约束、是否以灾难性干预之外的手段管理前进的风险。文集的框架并不预测我们的具体结局。它所提示的是:我们自己文明如今开始面对的问题,并不是新问题;它们是先前文明已经面对过、并以各种方式作出回答的问题,而双子座格局代表其中一种特别灾难性的结局。

对于文集而言,这就是评估该框架应处的实证与政治语境。双子座事件所预设的能力,如今正由我们自己的文明分段开发。双子座事件所解决的问题,如今正由我们自己的文明提出。该框架为一种当代处境提供了历史视角——在许多方面,这种处境正是先前文明已经经历并解决过的格局的再激活。

## XIII. 文本及其信号

希伯来文本《创世记》6-11 章以及相关的先知材料,除了前面各节已经指出之外,还包含若干值得评说的特征。

第一,数字“八”。方舟正好保存八个人——挪亚、他妻子、他们的三个儿子,以及儿子们的妻子。该数字在多处经文中以精确度被保存({% libref(book="genesis-woh", chapter=7, verse=13) %}创世记 7:13{% end %}、彼得前书 3:20)。在创造之周的象征结构中,八是第七日之后的那一日,开启新序列的那一日,新开始的数字。双子座时代,作为第八日的时代,从这一点获得其象征特征:创造在第七日完成,第八日是其后到来之物的第一日。洪水标志这一过渡。挪亚的家就是新序列的种子。

第二,希伯来词 *tevah*,惯例译作“方舟”。该词的词根含义是“容器”或“封闭的器皿”。它在希伯来圣经中被使用两次:这里指挪亚的器皿,以及《出埃及记》第 2 章中放置婴儿摩西、漂浮在尼罗河上的筐子。第二次使用具有启发性。*tevah* 是一种封闭的容器,使其所装之物免受环境威胁。摩西的 *tevah* 把孩子从法老的兵丁与河水中保存下来。挪亚的 *tevah* 把其乘员从洪水以及水面之上的放射性沉降物中保存下来。该词的意思并不是“船”在水上航行那种意义上的器皿。它的意思是“胶囊”——一种封闭的保存容器。其与雷尔派解读的语义吻合是精确的。翻译史掩盖了这一点,但希伯来语本身支持它。

第三——也是最直接支持本章所发展之天界之战解读的文本发现——用来指那场冲突中各形象的语言,在希伯来文中比翻译所允许的要揭示得多。资料所引的{% libref(book="isaiah", chapter=27, verse=1) %}以赛亚书 27:1{% end %}经文,在希伯来文中并列地把三个词应用于同一主语:

三个希伯来词指出被刑罚的形象:לִוְיָתָן(*liwyatan*,利维坦)、נָחָשׁ(*nachash*,蛇)、תַּנִּין(*tannin*,大鱼/海中怪物)。每一个都带着自身的词源和自身的含义范围,当其在更广希伯来文本中的关联被追溯时,每一个都变得更加揭示。

*Liwyatan*——利维坦——派生于希伯来词根 לוה(*lwh*),意为“扭、卷”。利维坦在词源上是“扭曲者”、“蜿蜒者”。该词通过其蛇形特征、其卷绕运动的能力、其与蛇与龙共享的扭曲与盘旋模式,为该形象命名。该形象并不是因任何具体的个体身份得名。它是因其属于哪一种事物得名。

*Nachash*——蛇——是更具关键性的词,因为它把这一先知段落直接联系到《创世记》第 3 章的伊甸叙事。伊甸的蛇,巨蟹座一章已识别为以统一口吻向亚当与夏娃说话的路西法派系,正是 *ha-nachash*——“那蛇”——使用的是同一个希伯来词。希伯来文本通过将 *nachash* 应用于{% libref(book="isaiah", chapter=27, verse=1) %}以赛亚书 27:1{% end %}的形象,在作出一种希伯来传统本身并未总是承认、但在语法上毫不模糊的指认。伊甸的蛇与末世审判的大鱼是同一个词,被应用于同一种形象。把它们分开的翻译史——把伊甸的形象简单地译为“那蛇”而把以赛亚的形象译为“利维坦”或“大鱼”——掩盖了希伯来文所保存的一种同一性。在雷尔派的解读下,这种同一性正是本章一直在论证之物:路西法派系、流亡造物者、向亚当与夏娃披露知识的群体,正是议会在双子座晚期对其采取行动的同一群体,也是以赛亚预言投射到末世审判中的同一群体。一个希伯来词,*nachash*,把这种身份贯穿整个圣经叙事。

*Tannin*——大鱼/海中怪物——是处女座一章详细解开的同一个词。它是{% libref(book="genesis-woh", chapter=1, verse=21) %}创世记 1:21{% end %}用来指耶洛因在第 5 日所造之大海中生物的同一个词。它是希伯来圣经中通篇用来指与海洋、与混沌、以及与既有秩序所必须制服的力量相关联的巨型蛇形形象的词。它在{% libref(book="isaiah", chapter=27, verse=1) %}以赛亚书 27:1{% end %}中与 *nachash* 和 *liwyatan* 并列出现,完成了这一指认:被刑罚的形象是蛇形的海中之龙,与第 5 日最初创造也具有争议、并在资料的解读下也与最终会引发希伯来文本如今所纪念之冲突的政治张力交织在一起的同一种生物。

这三个词在一节中的汇合,在雷尔派的解读下,并不是诗性的堆砌。它是一次精确的神学指认。被刑罚的形象是路西法派系(被命名为 *nachash*,与伊甸的蛇相同),以蛇形反叛者的身份(被命名为 *liwyatan*,扭曲者)出现,与其藏匿之海相关联(被命名为 *tannin*,深海中的大鱼)。一节经文、三个名字、一个历史指涉。希伯来文本以一种常规读法系统性掩盖的精确度,保存了天界之战。

另一个细节:美索不达米亚与迦南的对应,确认了这种意象的古老性。公元前第二千纪的乌加里特文本描述巴力杀死一条名为 *Lotan* 的蛇——词源上与利维坦同字——女神阿娜特击倒一条名为 *tannanu* 的七头海中怪物——与希伯来 *tannin* 同源。巴比伦的《埃努玛·埃利什》描述马尔杜克杀死原初之龙提阿玛特。神祇击杀蛇形海中生物的主题比希伯来文本更古老。希伯来文本继承之、改造之,并把它应用于一个具体的历史指涉,而周围各文化以其各自相关形式保存了同一指涉。Wheel of Heaven 框架所识别为历史的那场战争,在此种解读下,正是所有这些传统所记忆的同一事件,各以其自身方式,而希伯来文之所以独特,是因为它具有“在圣经叙事不同时刻指认同一形象”的精确度。

第四,彩虹。{% libref(book="genesis-woh", chapter=9, verse=13) %}创世记 9:13{% end %}指明耶洛因把彩虹设在云中,作为约的记号。常规解读把这视为关于天气的应许——彩虹与雨相关联,如今也将与不再使洪水来到的应许相关联。本章前面已指出希伯来词 קֶשֶׁת(*keshet*)既指“彩虹”,也指“弓”(作为武器)。这种双重含义是关键的。在此种解读下,立约的姿态正是明确退役一件毁灭工具。曾对地施用过的弓,如今挂在云中,被搁下,不再被部署。可见的彩虹是被放下武器的可见信物。一种技术读法也是可得的:洪水后的大气本身——从该事件的残屑中被清理出来,并被重置到新的平衡——产生了我们如今所看到的彩虹格局。立约的记号在此种解读下,正是洪水后的大气本身,即一种可见的演示:条件已被重置,天的器具已被更新。

该记号对联盟双方都可见——流亡造物者与人类幸存者,在新大陆之上的共同天空中——这与其作为他们之间私下立约的信物之功能一致,而不是仅指向单一神圣方的记号。

第五,与巴别塔叙事相关的语言学证据。希伯来词 בָּבֶל(*Bavel*,巴别/巴比伦)在希伯来民间词源中被联系到动词 בָּלַל(*balal*),“变乱、混合”——{% libref(book="genesis-woh", chapter=11, verse=9) %}创世记 11:9{% end %}明确作出这一关联:*al ken qara shemah Bavel ki sham balal Adonai sefat kol ha-aretz*,“因为耶和华在那里变乱天下人的言语,所以那城名叫巴别。”*Bavel* 的阿卡德语词源实际上不同——*Bab-ilu* 意为“神之门”——希伯来民间词源是文字游戏而非真正的词根派生。但这个文字游戏保存了所发生之事的运作内容:在巴别,统一的语言被变乱(*balal*),其结果是本章已描述的语言学分散。希伯来文本本身在地名与事件之间作出了运作性联系,即便词源在严格意义上并不准确。

## XIV. 双子座是什么

在本章结束之前,值得明确说明双子座在更大序列中是什么。

双子座是断裂的时代。这是这样一个时代:洪水前世界的单一人类文明——在文集的解读下,这一文明已达到或许与我们相当、甚至超越我们的技术水平——被有意地毁灭,通过母星议会对其视为不可持续之受造物所部署的核武器。毁灭是全面的。在撞击的力量下,超大陆本身碎裂。生物圈在很大程度上被消除,不得不从被保存的遗传材料中再生。人类物种仅通过一个小型机组和一份细胞储备——它们在一艘轨道器上——得以幸存,而希伯来圣经把这艘航天器记忆为{% wiki(slug="noahs-ark") %}挪亚方舟{% end %}。

双子座也是悲剧性转变的时代。蛇形派系,即在过去两千年间一直与他们的人类受造物一同生活在地球上、处于被接受的流亡状态中的流亡造物者,被议会的毁灭决定迫使脱离那种被接受的状态,进入积极抵抗。他们作出第一项选择——为保存受造的人类以对抗毁灭令——是希望抵抗可以保持有限,最终可以与议会达成和解。他们作出第二项选择——拿起武器反抗议会——只发生在巴别塔干预表明任何对人类价值的展示都不足以赢得议会的接纳之后。从受惩罚的异见者到积极反叛者的转变是被议会的行动强加于他们的,而不是作为首要偏好被他们选择的。这就是本章弧线之所以悲剧的所在:蛇形派系成为了自己从未想要成为的反叛者,因为顺从的代价是他们所爱一切的毁灭。

双子座同样是立约的时代。挪亚与流亡造物者在洪水后的祭坛上确立其正式关系的时刻,正是将塑造文集其余部分的联盟成立的时刻。该约不是在人类与一位遥远至高权威之间订立的。它是在刚刚共同对抗议会命令的两方之间订立的——流亡造物者与与他们一同建造方舟的人类幸存者——并把双方绑定为相互支持,以对抗任何未来议会会威胁双方任何一方的行动。彩虹是这种私下联盟的记号。洪水后圣经叙事中所发生的一切,都发生在该约所确立的政治框架内。

双子座同样是首次复苏的时代。幸存的人类与被保存的生物圈被送回洪水后的地表,七支人类谱系被重新分布到其原区域(如今被新打开的海洋所分隔),重建开始。伊甸谱系——由留在地球上、如今已是其正式盟友的流亡造物者所教导——迅速恢复,在数个世纪之内达到足以承担重大工程项目的文明水平,包括建造一艘第二代航天器——巴别塔——意在把联盟的人类伙伴送到母星。议会再次干预,这一次不是为了毁灭,而是为了分散,打散重建中的文明并把其科学精英驱散到新的大陆地理中。议会针对人类技术进步的干预格局——它会成为文集其余部分的特征——在这一时代被确立。

双子座最后是战争的时代。母星议会与流亡造物者—人类联盟之间的冲突,自最初的驱逐以来一直潜伏并被立约所正式化,在这一时代变为公开——先是通过违抗毁灭令而建造方舟,然后是通过违抗最初流放条款而持续教导人类幸存者,然后是通过对巴别塔的协作建造,最后是通过议会自身向流亡造物者本身行动时的直接军事行动。冲突在双子座并未以任何干净的方式解决。它产生了流亡造物者长期撤入山地与水下的隐藏,并产生了金牛座一章会以“赦免”为题记述的最终政治安排。但冲突本身,在其公开的军事阶段,属于这个时代。它是几乎每一种主要神话都保存的历史事件——泰坦之战、阿萨-华纳之战、荷鲁斯与赛特、马尔杜克与提阿玛特、羽蛇神与特斯卡特利波卡——在文集的解读下,这是以多种语言被记下的同一事件。

我们脚下的石油环、年轻的山脉、破裂的大陆、大灭绝的化石记录、保存在每一大陆文化中的洪水神话的全球分布、保存在不共享任何近期共同源头之神话中的“众神之战”的跨文化记忆:在 Wheel of Heaven 的解读下,所有这些都是这一时代所发生之事的地质、生物与文化签名。我们自己的文明,在其引擎与火炉中焚烧着其被毁灭前驱的被熔解生物量,生活在被它已忘却的事件所塑造的土地上。文集在本章及接下来的章节中的任务,是去记起。

下一个时代是这样一个时代:立约联盟成熟为一项持续的政治项目——亚伯拉罕的呼召、将经由后续圣经叙事承载该联盟的谱系的奠立、所多玛与蛾摩拉对巴别后科学余民的毁灭、最终允许流亡造物者正式回到议会的赦免,以及美索不达米亚、埃及与相邻文明的缓慢巩固——其考古痕迹是常规历史记录所能阅读的最初痕迹。那个时代是金牛座时代,正是接下来一章的主题。
