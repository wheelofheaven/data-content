# 年表

本文集透过岁差周期阅读人类历史——地球地轴二万六千年的摆动，划分为十二个各约二千一百六十年的时代，每个时代以春分日出背后的星座命名。在这一框架内，十五个篇章追溯耶洛因——那个设计了地球生命的小型高级人类文明——做了什么、何时做的、以及为何而做。

本系列开篇于第一个时代之前的《起初》一章，讲述母星上将流亡者送往地球的政治决裂。它贯穿干预与再干预的远古时期诸篇章——摩羯座、射手座、天蝎座、天秤座、处女座、狮子座、巨蟹座、双子座——进入金牛座、白羊座和双鱼座的历史记录，并以我们当前正身处其中的水瓶座过渡作结，加上关于轮盘本身的综合篇章。

每一篇章透过本典籍的视角阅读各自的时代——比较宗教、科学背景与批判学术均纳入其中——但典籍主张直接陈述，不加遮掩。
