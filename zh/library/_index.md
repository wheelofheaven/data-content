# 图书馆

图书馆收录本项目所参照的原典——不是关于这些文本的评注，而是文本本身，以供直接阅读。

居于中心的是雷尔派源典著作，本项目将其视为根本性的正典。围绕它们的是与正典并读的圣经、美索不达米亚、吠陀及其他古代典籍。每部典籍均以交互式阅读器呈现，凡文集已具备者，原文文字、转写与译文并排显示。

如需阅读诠释层面的内容——即本项目对某一段落的解读——请从[年表](/timeline/)或[百科全书](/wiki/)入手。图书馆是首先按文本自身脉络阅读它们的地方。
