question = "用通俗语言来说，岁差周期是什么？"
answer = """
地球轴倾斜约 23.5°，并且并不指向空间中某个固定方向。像一个旋转的陀螺，地轴在天空中缓慢画出一个圆，每约 25,920 年完成一次完整的循环。这一运动被称为*二分点的岁差*。

实际后果是：春分时升起在太阳背后的星座，沿黄道带缓慢向西移动。两千年前那是白羊座；今天是双鱼座；不久将是水瓶座。该循环每约 26,000 年重复一次。

按黄道带的十二星座划分，该循环产生每个约 2,160 年的*岁差时代*。本文集将这些时代用作其年代学框架。详情请参见 [岁差](/wiki/precession/) 词条。
"""

[[extra.faq_groups.items]]
slug = "what-does-elohim-mean"
question = '"Elohim"是什么意思，项目为何将其读作复数？'
answer = """
*Elohim*（אֱלֹהִים）是传统圣经译作"上帝"的希伯来词。从语法上看，它是复数名词——*-im* 后缀是标准的希伯来语复数标记，与使其他阳性名词复数化的后缀相同（*cherub* / *cherubim*，*Nephil* / *Nephilim*）。

传统的基督教和犹太教神学将该复数解读为"庄严复数"（一位神祇以复数表述以示宏伟，类似于王室的"我们"）或作为三位一体的占位语。本文集按字面意义阅读它：复数 *Elohim* 指多位存在。

希伯来文本的若干特征支持这一阅读：

- 创世记 1:26 使用第一人称复数动词：*na'aseh adam be-tsalmenu kidmutenu*——"让*我们*按*我们的*形象、依*我们的*样式造人。"
- 创世记 3:22 使用类似的复数：*hen ha-adam hayah ke-achad mimennu*——"看哪，那人已经像*我们*中的一个了。"
- 创世记 6:1–4 引入了 *benei ha-Elohim*——"耶洛因的众子"——作为不同于人类的特定类别存在。
- 文本在不同段落中将 *YHWH Elohim*（复数类别中一个具名的具体形象）与单独的 *Elohim*（类别本身）相区分。

详情请参见 [耶洛因](/wiki/elohim/) 词条。
"""

[[extra.faq_groups.items]]
slug = "handling-uncertainty"
question = "项目如何处理不确定性与分歧？"
answer = """
直接处理。项目的编辑风格指南明确要求不确定性被命名出来，而不是被隐藏：

- "该日期存在争议。"
- "这一年表是传统而非学术性的。"
- "该认定是诠释性的，并未在来源中明示。"
- "该平行性具有暗示意味，但并非可证明的谱系关联。"

当学术共识与本文集的框架相分歧时，本文集直接讨论这一分歧，命名替代解读，并陈述本文集得出不同结论的具体推理。在证据确实开放之处，本文集明言之。在本文集进行推测之处，亦明言之。
"""

[[extra.faq_groups.items]]
slug = "claim-badges"
question = '页面徽章上的"直接"、"推断"和"推测"是什么意思？'
answer = """
本文集的每一页都带有一个小徽章，标示其主要主张的认识论状态：

**直接主张**——来源文本自身所主张的内容。*"创世记 1:26 写道'让我们按我们的形象造人。'"* 这些是最可靠的主张；核实只是对照来源的事。

**推断主张**——学术合理地从来源得出的结论。*"复数动词形式表明多个施事者。"* 这些主张涉及诠释工作，但仍贴近证据。

**推测主张**——项目作为诠释综合所提出的内容。*"创世记中复数的耶洛因可以与雷尔派来源所描述的小型文明相认定。"* 这些主张是项目的特色贡献，最容易被修订。

徽章让读者在单个主张的层面校准信任。推测性的主张不因其推测而错；它被*标记*为推测，是为了让读者知道相应地予以权衡。
"""
